# .php-cs-fixer.php

**Path**: `.php-cs-fixer.php`

## Summary
This is a PHP-CS-Fixer configuration file that defines code style and formatting rules for a PHP project. It configures the tool to automatically format code according to PHP 8.2 migration standards, PhpCsFixer and Symfony coding conventions, with custom rules for string concatenation spacing, import ordering, and PHPDoc handling. The configuration applies these rules to the `src`, `spec`, and `tests` directories while enabling parallel processing for faster execution.
