# OrderSummaryHeader.vue

**Path**: `apps\front-ecommerce-b2b\components\OrderSummaryHeader.vue`

## Summary
This Vue component displays a progress indicator for free shipping eligibility in a B2B e-commerce cart. It calculates the minimum order amount required for free shipping from cart validity rules, tracks how much more the customer needs to spend, and visualizes this as a progress bar with percentage completion. The component helps incentivize customers to increase their order value to qualify for free shipping benefits.
