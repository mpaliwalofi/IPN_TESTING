# error.vue

**Path**: `apps\front-ecommerce-b2b\error.vue`

## Summary
This is a Vue error page component for a B2B e-commerce front-end that handles application-wide error states. It displays localized error messages based on error status codes (or falls back to generic error text), loads cross-sell product recommendations, and tracks page views for analytics. The component also manages page context, clears the error state before navigation, and automatically scrolls to the top when mounted.
