# fr-FR.json

**Path**: `apps\front-ecommerce-b2b\i18n\locales\fr-FR.json`

## Summary
This is a French localization file (fr-FR) for a B2B e-commerce application that provides contact and business hours information for "Ultra Premium Direct" company. It contains utility strings including phone numbers in different formats, a contact email address, and operating hours (Monday-Friday, 9:00-17:30) to be displayed throughout the French version of the application.
