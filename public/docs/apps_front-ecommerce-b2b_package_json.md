## Summary
This is the package configuration file for a B2B e-commerce front-end application built with Nuxt.js. It defines the project dependencies, build scripts, and Nx monorepo task orchestration, with the application depending on a shared `@ultrapremiumdirect/front-ecommerce` workspace package. The configuration includes development tools for testing (Vitest), linting (ESLint), internationalization (@nuxtjs/i18n), payment integration (Lyra embedded forms), and search functionality (Algolia).
