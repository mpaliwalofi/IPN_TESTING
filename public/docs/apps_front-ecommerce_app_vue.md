# app.vue

**Path**: `apps\front-ecommerce\app.vue`

## Summary
This is the root application component for a front-end e-commerce application built with Nuxt.js. It initializes reCAPTCHA functionality for bot protection and sets up the core layout structure with a loading indicator and page routing system. The component serves as the main entry point that wraps all pages with a consistent layout and provides visual feedback during navigation.
