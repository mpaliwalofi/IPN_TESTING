# AnimalHeader.vue

**Path**: `apps\front-ecommerce\components\animal\AnimalHeader.vue`

## Summary
This Vue component manages the header section of an animal profile page in an e-commerce application, providing functionality to view animal details (age, birthday, photo) and handle animal deletion. The component allows users to select a deletion reason from a list of available reasons and submit a request to remove an animal from their account, with navigation back to the animals list upon successful deletion.
