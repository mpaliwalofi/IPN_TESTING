# AnimalForm.stories.ts

**Path**: `apps\front-ecommerce\components\animal\stories\AnimalForm.stories.ts`

## Summary
This is a Storybook configuration file that defines UI documentation and testing scenarios for the `AnimalForm` Vue component in an e-commerce application. It sets up a default story that renders the AnimalForm component wrapped in Vue's Suspense component to handle asynchronous loading states, allowing developers to view and test the animal form in isolation with a loading fallback during data fetching.
