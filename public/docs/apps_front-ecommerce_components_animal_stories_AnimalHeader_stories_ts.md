# AnimalHeader.stories.ts

**Path**: `apps\front-ecommerce\components\animal\stories\AnimalHeader.stories.ts`

## Summary
This is a Storybook configuration file that defines stories for the AnimalHeader Vue component in an e-commerce application. It sets up visual testing scenarios across multiple viewport sizes (mobile, tablet, desktop) using Chromatic, and provides mock data representing a dog profile (including breed, gender, condition, activity level, and other pet-related attributes) to render the component in different states for UI documentation and testing purposes.
