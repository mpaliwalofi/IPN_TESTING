# CartButton.vue

**Path**: `apps\front-ecommerce\components\cart\CartButton.vue`

## Summary
This Vue component implements a shopping cart button for an e-commerce application that displays the total number of items in the cart and provides hover-to-preview functionality. On desktop devices, hovering over the cart icon triggers a mini-cart popup that shows cart contents without navigating away from the current page. The component integrates with a Pinia cart store to manage cart state, loads cart data on mount, and provides a clickable link to the full checkout cart page.
