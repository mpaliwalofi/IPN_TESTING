# CartPromoTag.vue

**Path**: `apps\front-ecommerce\components\cart\CartPromoTag.vue`

## Summary
This Vue component renders a promotional code tag for an e-commerce cart that allows users to copy the promo code to their clipboard. When clicked, it copies the promotion code using the VueUse clipboard composable and tracks the copy action via a tracking event that records the promotion code and its display location for analytics purposes.
