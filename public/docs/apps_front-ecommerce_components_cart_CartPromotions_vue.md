# CartPromotions.vue

**Path**: `apps\front-ecommerce\components\cart\CartPromotions.vue`

## Summary
This component manages promotional coupon code functionality for an e-commerce shopping cart, allowing users to add and remove discount coupons to their orders. It handles coupon validation through the cart store, displays error messages when invalid coupons are entered, and tracks coupon application events for analytics purposes.
