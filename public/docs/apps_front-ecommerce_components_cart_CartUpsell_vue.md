# CartUpsell.vue

**Path**: `apps\front-ecommerce\components\cart\CartUpsell.vue`

## Summary
This Vue component handles cart upselling and cross-selling functionality in an e-commerce application. It allows users to add recommended or promotional product variants to their cart from either upsell offers or cross-sell promotions (from Pixie Ad Product Offers), tracking the additions with GTM analytics and managing the appropriate source item list metadata based on the offer type.
