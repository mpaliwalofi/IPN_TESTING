# AvailablePaymentMethods.vue

**Path**: `apps\front-ecommerce\components\checkout\AvailablePaymentMethods.vue`

## Summary
This Vue component manages the display and selection of available payment methods during an e-commerce checkout process. It dynamically loads payment method offers through a provided loader function, allows users to select a payment method, and handles the state of payment form configuration including loading, reloading, and submission states. The component supports displaying payment information modals and tracks whether a selected payment method is ready for submission.
