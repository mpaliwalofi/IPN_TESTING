# CheckoutApplePayPaymentForm.vue

**Path**: `apps\front-ecommerce\components\checkout\Payments\CheckoutApplePayPaymentForm.vue`

## Summary
This Vue component handles Apple Pay payment integration for an e-commerce checkout flow. It dynamically loads the Apple Pay SDK, validates merchant sessions with the backend, and manages the payment submission process by coordinating between the Apple Pay session events and the application's payment backend. The component emits lifecycle events (ready, submitting, cancel) to notify parent components of the payment flow status.
