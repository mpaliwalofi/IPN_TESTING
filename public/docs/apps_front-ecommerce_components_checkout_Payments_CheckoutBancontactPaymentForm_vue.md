# CheckoutBancontactPaymentForm.vue

**Path**: `apps\front-ecommerce\components\checkout\Payments\CheckoutBancontactPaymentForm.vue`

## Summary
This Vue component implements a Bancontact payment form for the e-commerce checkout process. It manages the payment submission workflow by handling a payment offer prop, emitting lifecycle events (ready, submitting, cancel), and displaying a submit button with terms of service link that remains disabled during payment processing to prevent duplicate submissions.
