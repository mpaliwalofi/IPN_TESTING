# CheckoutGooglePayPaymentForm.vue

**Path**: `apps\front-ecommerce\components\checkout\Payments\CheckoutGooglePayPaymentForm.vue`

## Summary
This Vue component implements a Google Pay payment form for the e-commerce checkout process. It configures and manages the Google Pay API integration, handling payment data requests with merchant information, allowed payment methods (cards), and transaction details. The component provides a payment interface that emits lifecycle events (ready, submitting, cancel) and validates Google Pay-specific configuration including public keys and merchant IDs.
