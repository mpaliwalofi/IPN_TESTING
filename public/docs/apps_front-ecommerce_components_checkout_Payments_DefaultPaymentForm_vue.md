# DefaultPaymentForm.vue

**Path**: `apps\front-ecommerce\components\checkout\Payments\DefaultPaymentForm.vue`

## Summary
This Vue component serves as a default payment form for e-commerce checkout, providing a standardized submit button and terms of service agreement text. It manages the payment submission flow by calling the provided payment offer's submit method, handling loading states, and emitting lifecycle events (ready, submitting, cancel) to coordinate with parent components during the payment process.
