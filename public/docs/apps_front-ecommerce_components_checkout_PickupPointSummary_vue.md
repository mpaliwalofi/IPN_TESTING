# PickupPointSummary.vue

**Path**: `apps\front-ecommerce\components\checkout\PickupPointSummary.vue`

## Summary
This Vue component displays a summary of pickup point information for an e-commerce checkout process. It allows users to select between multiple shipping methods for pickup points and displays relevant details like opening hours and method descriptions. The component uses responsive breakpoints to adjust its layout between mobile and desktop views, with transition effects applied on desktop screens.
