# AvailablePaymentMethods.stories.ts

**Path**: `apps\front-ecommerce\components\checkout\stories\AvailablePaymentMethods.stories.ts`

## Summary
This is a Storybook configuration file that defines UI stories for the `AvailablePaymentMethods` Vue component in an e-commerce checkout flow. It provides mock data for testing and showcasing different payment method options (such as Google Pay and PAYZEN) that customers can select during checkout, including their configurations, translations, and pricing information.
