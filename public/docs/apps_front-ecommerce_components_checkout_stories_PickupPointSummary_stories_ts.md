# PickupPointSummary.stories.ts

**Path**: `apps\front-ecommerce\components\checkout\stories\PickupPointSummary.stories.ts`

## Summary
This Storybook configuration file defines visual documentation and testing scenarios for the `PickupPointSummary` component used in the e-commerce checkout flow. It provides sample data for a pickup point delivery option (Mondial Relay in France) including shipping method details, location coordinates, opening hours, and address information. The stories enable developers and designers to preview and test how the pickup point summary displays during the checkout process when customers select a relay point delivery method.
