# AccordionToTab.vue

**Path**: `apps\front-ecommerce\components\content\AccordionToTab.vue`

## Summary
This Vue component provides a responsive UI pattern that displays content as tabs on desktop and as an accordion on mobile. It manages the state for both views, handling tab selection on desktop and multiple accordion panel expansion on mobile, with accordion items being sortable by position and optionally pre-opening the first item on mobile by default.
