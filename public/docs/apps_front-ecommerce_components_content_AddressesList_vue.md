# AddressesList.vue

**Path**: `apps\front-ecommerce\components\content\AddressesList.vue`

## Summary
This Vue component manages the display and operations for a customer's address book in an e-commerce application. It provides functionality to view, set default, edit, and delete customer addresses, with special handling for subscriber users who can add additional addresses. The component integrates with the customer store for persistence operations and includes loading states, error handling, and tracking for address-related actions.
