# CTAListing.vue

**Path**: `apps\front-ecommerce\components\content\CTAListing.vue`

## Summary
This Vue component renders a Call-to-Action (CTA) listing section for an e-commerce storefront, displaying an optional icon, a customizable heading, and a horizontal scrollable row of action buttons. It integrates with Strapi CMS to dynamically configure button styles (outline, ghost, solid), colors (primary, neutral), and link behaviors based on content managed through the CMS. The component is designed to present promotional or navigational CTAs in a centered, responsive layout that adapts between mobile (scrollable) and desktop (auto-overflow) views.
