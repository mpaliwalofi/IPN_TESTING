# CalendlyEntrance.vue

**Path**: `apps\front-ecommerce\components\content\CalendlyEntrance.vue`

## Summary
This Vue component provides a Calendly scheduling widget integration for an e-commerce front-end application. It dynamically loads Calendly's external scripts and stylesheets, configures a popup widget with customizable parameters (including background, image positioning, heading levels, and Calendly URL), and triggers the scheduling interface when initiated. The component serves as a content block for allowing customers to book appointments or consultations directly within the e-commerce site.
