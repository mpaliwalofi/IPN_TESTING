# Calendly.vue

**Path**: `apps\front-ecommerce\components\content\Calendly.vue`

## Summary
This Vue component embeds a Calendly scheduling widget into an e-commerce front-end application. It accepts configuration parameters (site ID, colors, dimensions) to construct a customized Calendly iframe URL and dynamically loads the required Calendly CSS and JavaScript assets. The component serves as a reusable content block for integrating appointment/meeting scheduling functionality within the CMS-driven e-commerce site.
