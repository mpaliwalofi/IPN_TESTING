# CarouselAnimal.vue

**Path**: `apps\front-ecommerce\components\content\CarouselAnimal.vue`

## Summary
This Vue component displays a carousel of customer animals for an e-commerce pet store application. It fetches and displays a list of animals associated with the customer, with optional title/subtitle headers and support for both carousel and grid layouts. The component handles loading states, allows for forced carousel display even without animals, and integrates with a CMS content system for customizable text content.
