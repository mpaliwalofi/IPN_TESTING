# CarouselEdito.vue

**Path**: `apps\front-ecommerce\components\content\CarouselEdito.vue`

## Summary
This Vue component displays an editorial carousel section with accompanying text content, commonly used in e-commerce front-end layouts. It implements a responsive grid layout where an image carousel and editorial content can be positioned side-by-side, with configurable ordering (text left/right, image first on mobile). The component uses Strapi CMS data structures to render formatted images with smart loading and supports various styling options for headings, buttons, and carousel arrows.
