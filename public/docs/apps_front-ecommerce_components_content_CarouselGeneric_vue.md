# CarouselGeneric.vue

**Path**: `apps\front-ecommerce\components\content\CarouselGeneric.vue`

## Summary
This Vue component renders a generic carousel for displaying content items in an e-commerce application, with responsive behavior that adapts layout and navigation controls based on screen size (mobile, tablet, desktop). The carousel supports configurable intervals, displays up to 6 items per slide with horizontal scrolling, and includes an optional title heading and smart image rendering for each item. It integrates with Strapi CMS types and uses VueUse breakpoints to conditionally show arrow controls inside the carousel on tablet devices while maintaining different layouts across device sizes.
