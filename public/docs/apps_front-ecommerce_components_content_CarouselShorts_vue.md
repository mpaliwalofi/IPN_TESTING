# CarouselShorts.vue

**Path**: `apps\front-ecommerce\components\content\CarouselShorts.vue`

## Summary
This Vue component renders a carousel specifically designed for displaying short-form video content (like YouTube Shorts or TikTok-style videos) in an e-commerce application. It displays embedded videos in a horizontal scrollable carousel with 4 items visible at once, featuring an optional H3 heading and vertical aspect ratio (9:16) containers optimized for mobile-first short video formats.
