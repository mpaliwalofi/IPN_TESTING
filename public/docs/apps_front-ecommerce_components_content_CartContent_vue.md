# CartContent.vue

**Path**: `apps\front-ecommerce\components\content\CartContent.vue`

## Summary
This Vue component manages the shopping cart page for an e-commerce application, handling cart display, validation, and checkout initiation. It retrieves cart data from the Pinia store, organizes products into categories (regular items, free products, upsells, cross-sells), and triggers multiple tracking events (view cart, validate cart, begin checkout) before navigating users to the checkout funnel when they proceed with their order.
