# CategoryGrid.vue

**Path**: `apps\front-ecommerce\components\content\CategoryGrid.vue`

## Summary
This Vue component renders a category grid layout for an e-commerce site, displaying up to 4 category cards alongside dynamically changing images. The component implements an interactive hover effect where hovering over a category card switches the displayed image from a default image to the corresponding category's image, creating a visual preview system for browsing product categories.
