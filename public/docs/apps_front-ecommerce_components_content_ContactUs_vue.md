# ContactUs.vue

**Path**: `apps\front-ecommerce\components\content\ContactUs.vue`

## Summary
This Vue component renders a "Contact Us" section for an e-commerce front-end, displaying three contact methods: phone, email, and social media messenger. The component uses internationalization (i18n) for all text content and provides clickable links (via SmartLink) that enable users to directly initiate contact through tel:, mailto:, or external messenger URLs.
