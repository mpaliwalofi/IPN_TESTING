# CrossSellSameCategory.vue

**Path**: `apps\front-ecommerce\components\content\CrossSellSameCategory.vue`

## Summary
This Vue component displays cross-sell product recommendations from the same category as the currently viewed product on a product detail page. It fetches related products using the product's main category ID and SKU, then renders them in a CrossSell component for merchandising purposes. The component includes analytics tracking metadata to monitor user interactions with same-category product recommendations.
