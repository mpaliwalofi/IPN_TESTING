## Summary
# Component Summary

This Vue component implements a **cross-sell product recommendation feature** for an e-commerce platform. It displays related products that can be purchased alongside the currently viewed item, filtering products based on SKUs defined in the CMS, computing their prices, and tracking user interactions (views and selections) through analytics events. The component retrieves product metadata, handles responsive carousel display (4 items per slide), and integrates with Strapi CMS for content management while providing comprehensive e-commerce tracking capabilities.
