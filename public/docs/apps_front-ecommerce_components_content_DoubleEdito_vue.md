# DoubleEdito.vue

**Path**: `apps\front-ecommerce\components\content\DoubleEdito.vue`

## Summary
This Vue component renders a double editorial content section for an e-commerce storefront CMS. It accepts content parameters conforming to the `IDoubleEdito` interface and iterates through multiple editorial items (`editos`), rendering each using the `TheEdito` component with a `double` flag. The component applies a marble-textured background styling and responsive layout (vertical stacking on mobile, increased spacing on larger screens) to display editorial content in pairs.
