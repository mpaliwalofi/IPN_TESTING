# Expertise.vue

**Path**: `apps\front-ecommerce\components\content\Expertise.vue`

## Summary
This Vue component displays an "Expertise" content section for an e-commerce storefront, featuring a responsive layout that showcases company expertise or capabilities with images, titles, and descriptive text. The component implements separate mobile and desktop layouts, with the mobile view showing a full-width image with overlaid title and a secondary card containing multiple description sections and an optional call-to-action button. It integrates with a Strapi CMS backend to dynamically render content including images, text descriptions, and configurable button links.
