# FAQQuestions.vue

**Path**: `apps\front-ecommerce\components\content\FAQQuestions.vue`

## Summary
This Vue component displays and manages a FAQ (Frequently Asked Questions) section for an e-commerce storefront. It handles question selection/toggling via URL hash navigation, maintains the state of opened/closed questions, and generates structured JSON-LD schema markup (FAQPage) for SEO purposes to help search engines understand and display the FAQ content in rich snippets.
