# FiltersProductCategory.vue

**Path**: `apps\front-ecommerce\components\content\FiltersProductCategory.vue`

## Summary
This Vue component provides product filtering and sorting functionality for an e-commerce product category listing page. It manages filter state (active filters and sort options), displays filter groups, handles filter toggling with tracking events, and supports both desktop and mobile filter interfaces through a dialog system. The component bridges user filter selections with the parent component's product catalog, emitting updates when filters or sort options change.
