# FiltersProductsCategoryContent.vue

**Path**: `apps\front-ecommerce\components\content\FiltersProductsCategoryContent.vue`

## Summary
This Vue component manages product filtering and sorting functionality for an e-commerce product category listing page. It processes filter groups, sorts filter options by priority, tracks active filters and sort selections, and emits updates to parent components when users modify their filter choices or sorting preferences. The component serves as a bridge between the CMS-defined filter configurations and the actual product catalog filtering logic, with support for both standard and subscription-based product views.
