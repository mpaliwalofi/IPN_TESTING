# Glossary.vue

**Path**: `apps\front-ecommerce\components\content\Glossary.vue`

## Summary
This Vue component renders a glossary section for an e-commerce storefront, displaying a customizable heading with title and subtitle text, followed by an optional USP (Unique Selling Proposition) content block. It accepts content configuration from a Strapi CMS, including the heading level, title, subtitle, and USP parameters, providing a flexible way to showcase key terminology or product information sections on the site.
