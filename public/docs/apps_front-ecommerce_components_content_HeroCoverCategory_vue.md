# HeroCoverCategory.vue

**Path**: `apps\front-ecommerce\components\content\HeroCoverCategory.vue`

## Summary
This Vue component renders a hero cover section for e-commerce category pages with a customizable title and responsive content display. It displays a prominent headline using the Cedrat font and conditionally shows different markdown content for mobile and desktop viewports, with the mobile version displaying at 5/6 width and desktop at 3/4 width within a container. The component integrates with Strapi CMS to receive structured content data including title, mobile content, and desktop content configurations.
