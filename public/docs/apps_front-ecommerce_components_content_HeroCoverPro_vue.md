# HeroCoverPro.vue

**Path**: `apps\front-ecommerce\components\content\HeroCoverPro.vue`

## Summary
This Vue component renders a hero banner section with a main cover image and horizontal card promotions for an e-commerce storefront. It implements tracking functionality that fires analytics events when promotions are viewed (on mount) or clicked, sending promotion names and locations to a tracking system. The component uses Strapi CMS data structures to display responsive images with configurable URLs and alternative text.
