# HorizontalCard.vue

**Path**: `apps\front-ecommerce\components\content\HorizontalCard.vue`

## Summary
This Vue component renders a horizontal card for an e-commerce storefront that displays promotional content with an image, title, subtitle, and call-to-action button. The card serves as a clickable link wrapper with hover effects and styled borders, designed to showcase featured products or promotional content in a horizontally-oriented layout. It integrates with Strapi CMS for content management and supports configurable button styles and image transformations optimized for 140x140px display.
