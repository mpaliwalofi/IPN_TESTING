# ItemBlogGrid.vue

**Path**: `apps\front-ecommerce\components\content\ItemBlogGrid.vue`

## Summary
This Vue component renders a responsive blog post grid layout for an e-commerce front-end. It displays an optional title and a horizontally scrollable (on mobile) or centered (on desktop) collection of blog items, with each item rendered using the `ItemBlog` child component based on CMS content data structured according to the `IItemBlogGrid` interface.
