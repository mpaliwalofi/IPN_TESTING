# ItemBlog.vue

**Path**: `apps\front-ecommerce\components\content\ItemBlog.vue`

## Summary
This Vue component renders a clickable blog article card that displays an article's image and links to the full blog post. It supports two layout modes (column or row) and constructs URLs dynamically from the article's category and slug, opening the blog article in a new tab when clicked.
