# LastOrder.vue

**Path**: `apps\front-ecommerce\components\content\LastOrder.vue`

## Summary
This Vue component displays a customer's most recent order with product details and tracking information. It fetches the last order from the customer store, enriches it with product catalog data (prices, images, etc.), and presents order status, shipping state, and items in a responsive card layout. The component also integrates analytics tracking for item list views and selections, supporting the e-commerce customer account experience.
