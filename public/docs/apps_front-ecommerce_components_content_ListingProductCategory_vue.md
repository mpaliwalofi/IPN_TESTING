# ListingProductCategory.vue

**Path**: `apps\front-ecommerce\components\content\ListingProductCategory.vue`

## Summary
This Vue component implements a product listing page for an e-commerce category with sorting, filtering, and tracking capabilities. It manages the display of products within a specific category, handling URL-based sort parameters (relevancy, price, etc.), product loading states, and integrating with analytics tracking events (view item list, select item, view/select promotion). The component serves as the main interface for browsing and interacting with products in a category context, coordinating between the catalog system, routing, and marketing/analytics tracking infrastructure.
