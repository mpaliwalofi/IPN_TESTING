# MainProduct.vue

**Path**: `apps\front-ecommerce\components\content\MainProduct.vue`

## Summary
This Vue component displays a main product cross-sell section in an e-commerce storefront. It renders a featured product with an image, heading, and link that tracks user interactions through GTM (Google Tag Manager) analytics, emitting events when the product is viewed or selected. The component integrates with Strapi CMS for content management and supports URL query parameters for tracking the source item list in the analytics flow.
