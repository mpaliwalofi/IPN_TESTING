# MentionMe.vue

**Path**: `apps\front-ecommerce\components\content\MentionMe.vue`

## Summary
This Vue component integrates the MentionMe referral marketing platform into an e-commerce application. It fetches customer personal data from the store and renders either a landing page referral offer or a dashboard view based on configuration parameters, passing the customer's email and name to initialize the MentionMe widget in the designated wrapper div.
