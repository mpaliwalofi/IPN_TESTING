# MutationForm.vue

**Path**: `apps\front-ecommerce\components\content\MutationForm.vue`

## Summary
This Vue component implements a mutation/modification request form for an e-commerce platform, allowing customers to submit requests for changes to their orders or products. It handles form state management using a `MutationRequestForm` class, captures UTM parameters from the URL for tracking, and integrates with the customer store to populate order and product options. The component emits form submission events with the collected data and manages success/error states with optional thank-you message display.
