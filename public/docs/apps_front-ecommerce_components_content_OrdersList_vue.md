# OrdersList.vue

**Path**: `apps\front-ecommerce\components\content\OrdersList.vue`

## Summary
This Vue component displays a paginated list of customer orders in an e-commerce application. It manages order data fetching with pagination (30 items per page after the first 5), handles "load more" functionality to progressively display additional orders, and provides navigation to individual order detail pages. The component serves as the main order history interface for authenticated customers to view and access their past purchases.
