# ReviewsProduct.vue

**Path**: `apps\front-ecommerce\components\content\ReviewsProduct.vue`

## Summary
This Vue component displays product reviews on an e-commerce product detail page. It fetches review data from product metadata, applies rotating color themes to individual reviews, handles anonymous author names through internationalization, and conditionally renders a `ReviewLarge` component with aggregated rating statistics and a link to display review certificates from AvisVerifies (a review verification service).
