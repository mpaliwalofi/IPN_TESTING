# SearchFaq.vue

**Path**: `apps\front-ecommerce\components\content\SearchFaq.vue`

## Summary
This Vue component implements a searchable FAQ interface for an e-commerce front-end, integrating Algolia search to allow users to query frequently asked questions. It handles both direct FAQ subject routes and search queries, dynamically loading FAQ content from a CMS (Strapi) based on URL parameters, and provides filtering/display functionality for FAQ subjects and questions with search autocomplete capabilities.
