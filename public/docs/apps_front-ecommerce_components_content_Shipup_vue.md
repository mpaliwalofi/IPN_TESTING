# Shipup.vue

**Path**: `apps\front-ecommerce\components\content\Shipup.vue`

## Summary
This Vue component integrates Shipup's order tracking functionality into an e-commerce front-end. It dynamically loads the Shipup SDK and renders a tracking widget that displays either a single shipment's tracking number or allows multi-shipment search by order number, configuring the display based on whether an order has multiple shipments.
