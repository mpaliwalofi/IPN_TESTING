# SimilarProducts.vue

**Path**: `apps\front-ecommerce\components\content\SimilarProducts.vue`

## Summary
This Vue component displays a "Similar Products" section on product detail pages in an e-commerce application. It receives product metadata as props and renders a CrossSell component populated with related products based on the current product's similarProducts data, with optional background styling and customizable title. The component also tracks source metadata (slug, category, page type) for analytics or recommendation purposes.
