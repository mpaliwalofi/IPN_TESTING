# StrapiSubscriptionCard.vue

**Path**: `apps\front-ecommerce\components\content\StrapiSubscriptionCard.vue`

## Summary
This Vue component serves as a Strapi CMS wrapper for displaying subscription card content in an e-commerce application. It accepts content configuration from Strapi along with subscription data and loading state, then renders a `SubscriptionCard` component within a responsive grid layout with appropriate spacing and a horizontal divider below it.
