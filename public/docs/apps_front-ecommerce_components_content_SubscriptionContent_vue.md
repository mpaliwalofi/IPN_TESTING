# SubscriptionContent.vue

**Path**: `apps\front-ecommerce\components\content\SubscriptionContent.vue`

## Summary
This Vue component manages subscription content display and editing functionality for an e-commerce platform. It handles the presentation and modification of subscription orders, including adding/removing products, managing upsells and recommended items, and tracking user interactions with subscription items across different states (active/draft). The component serves as the main interface for customers to view and edit their recurring subscription orders, integrating with Sylius e-commerce backend and Strapi CMS for content.
