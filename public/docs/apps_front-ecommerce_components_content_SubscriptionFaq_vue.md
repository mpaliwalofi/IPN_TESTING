# SubscriptionFaq.vue

**Path**: `apps\front-ecommerce\components\content\SubscriptionFaq.vue`

## Summary
This Vue component provides a subscription cancellation FAQ interface for e-commerce customers. It manages a form that allows authenticated users to submit subscription abort requests by selecting a cancellation reason and providing additional details, including reCAPTCHA validation and customer data integration. The component tracks the submission status and handles the complete workflow from form initialization through data submission to the backend system.
