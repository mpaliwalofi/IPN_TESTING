# TheEdito.vue

**Path**: `apps\front-ecommerce\components\content\TheEdito.vue`

## Summary
This Vue component renders an editorial content section (edito) for an e-commerce storefront, featuring configurable image placement, text content, and call-to-action buttons. It provides flexible layout options including left/right text positioning, mobile-first image ordering, customizable themes, and visual overlays for promotional or marketing content blocks. The component integrates with a Strapi CMS backend to dynamically display editorial content with responsive grid-based layouts.
