# Title.vue

**Path**: `apps\front-ecommerce\components\content\Title.vue`

## Summary
This Vue 3 component renders a dynamic title heading for an e-commerce content management system. It accepts content properties and title parameters through TypeScript interfaces, then displays the title text within an H1 element wrapped in a container class. The component serves as a reusable title block for CMS-driven page content in the front-end e-commerce application.
