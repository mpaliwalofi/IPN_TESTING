# USPLarge.vue

**Path**: `apps\front-ecommerce\components\content\USPLarge.vue`

## Summary
This Vue component renders a "USP Large" (Unique Selling Points) content section for an e-commerce storefront, displaying up to 4 USP cards in a responsive grid layout with an optional title, background, and call-to-action button. The component integrates with Strapi CMS to dynamically render marketing content that highlights key product or service benefits, with configurable heading levels, button styles, and layout options. It uses TypeScript interfaces to ensure type-safe content delivery from the CMS and includes responsive design classes for optimal display across mobile and desktop devices.
