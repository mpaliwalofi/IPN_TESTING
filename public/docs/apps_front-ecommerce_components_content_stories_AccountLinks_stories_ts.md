**Path**: `apps\front-ecommerce\components\content\stories\AccountLinks.stories.ts`

## Summary
This is a Storybook configuration file that defines visual test scenarios for the AccountLinks component in an e-commerce application. It sets up a story with mock data and configures Chromatic visual regression testing across three viewport sizes (mobile, tablet, and desktop) to ensure the account navigation links render correctly across different screen sizes.
