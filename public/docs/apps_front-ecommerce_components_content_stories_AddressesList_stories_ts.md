# AddressesList.stories.ts

**Path**: `apps\front-ecommerce\components\content\stories\AddressesList.stories.ts`

## Summary
This is a Storybook configuration file that defines visual test scenarios for an `AddressesList` component in an e-commerce application. It sets up stories to showcase how the component renders customer shipping/billing addresses across different viewport sizes (mobile, tablet, desktop), including test cases for single and multiple address displays with sample French address data.
