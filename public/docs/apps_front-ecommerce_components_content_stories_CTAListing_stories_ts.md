# CTAListing.stories.ts

**Path**: `apps\front-ecommerce\components\content\stories\CTAListing.stories.ts`

## Summary
This is a Storybook configuration file for a CTAListing component in an e-commerce application. It defines visual test stories for a call-to-action listing component that displays platform availability buttons (like Spotify, Deezer) with icons and links, configured for testing across mobile, tablet, and desktop viewports. The story demonstrates a podcast availability widget use case with branded buttons that can be rendered in different styles and include external links.
