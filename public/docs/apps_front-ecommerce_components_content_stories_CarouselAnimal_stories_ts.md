# CarouselAnimal.stories.ts

**Path**: `apps\front-ecommerce\components\content\stories\CarouselAnimal.stories.ts`

## Summary
This Storybook configuration file defines visual test scenarios for a `CarouselAnimal` component in an e-commerce application. It sets up Chromatic visual regression testing across multiple viewport sizes (mobile, tablet, desktop) and includes a default story that initializes session storage with an "animals" value of "2" before rendering the carousel component.
