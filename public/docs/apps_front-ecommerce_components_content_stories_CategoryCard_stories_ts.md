# CategoryCard.stories.ts

**Path**: `apps\front-ecommerce\components\content\stories\CategoryCard.stories.ts`

## Summary
This is a Storybook configuration file that defines visual test stories for the CategoryCard component in an e-commerce application. It sets up a default story with mock category data and configures Chromatic visual regression testing across three viewport sizes (mobile, tablet, desktop) to ensure the category card renders correctly across different screen sizes.
