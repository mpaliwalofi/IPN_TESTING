# CategoryGrid.stories.ts

**Path**: `apps\front-ecommerce\components\content\stories\CategoryGrid.stories.ts`

## Summary
This Storybook configuration file defines visual documentation and testing stories for an e-commerce CategoryGrid component. It sets up a "Default" story with mock data and configures Chromatic visual regression testing across three viewport sizes (mobile, tablet, desktop) to ensure the category grid renders correctly across different screen sizes.
