# ContactForm.stories.ts

**Path**: `apps\front-ecommerce\components\content\stories\ContactForm.stories.ts`

## Summary
This is a Storybook configuration file that defines stories for an e-commerce contact form component. It sets up visual testing scenarios across multiple viewport sizes (mobile, tablet, desktop) using Chromatic, and provides mock data for form fields including a dropdown for user type selection (individual, subscriber, breeder, etc.) and text inputs for contact information like email, first name, and last name.
