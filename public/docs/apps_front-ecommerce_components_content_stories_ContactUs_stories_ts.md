# ContactUs.stories.ts

**Path**: `apps\front-ecommerce\components\content\stories\ContactUs.stories.ts`

## Summary
This is a Storybook configuration file that defines visual test stories for the ContactUs component in an e-commerce application. It sets up the component to be viewed and tested across three different viewport sizes (mobile, tablet, and desktop) using Chromatic for visual regression testing. The file exports a default story that renders the ContactUs component with a margin and padding wrapper for display in the Storybook UI.
