# CrossSell.stories.ts

**Path**: `apps\front-ecommerce\components\content\stories\CrossSell.stories.ts`

## Summary
This Storybook configuration file defines visual test stories for an e-commerce CrossSell component that displays recommended products (like "customers also bought" sections). It sets up two story variants—a default view and a listing display view—with sample product IDs and configures viewport testing across mobile, tablet, and desktop breakpoints using Chromatic for visual regression testing.
