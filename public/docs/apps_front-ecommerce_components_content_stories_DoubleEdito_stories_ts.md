# DoubleEdito.stories.ts

**Path**: `apps\front-ecommerce\components\content\stories\DoubleEdito.stories.ts`

## Summary
This is a Storybook configuration file that defines visual test cases and documentation for the `DoubleEdito` component in an e-commerce front-end application. It creates three story variants (Default, DoubleTextRight, DoubleTextLeft) that demonstrate different text positioning layouts for dual editorial content blocks, with responsive viewport testing configured for mobile, tablet, and desktop views using Chromatic for visual regression testing.
