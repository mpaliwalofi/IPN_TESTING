# EditoImages.stories.ts

**Path**: `apps\front-ecommerce\components\content\stories\EditoImages.stories.ts`

## Summary
This Storybook configuration file defines visual test cases for the `EditoImages` Vue component used in an e-commerce application. It demonstrates different rendering scenarios of the component with varying numbers of images (1-3) and layout options (single vs. double display mode), using sample product/category images from ImageKit CDN. The stories enable developers and designers to preview and test how the EditoImages component behaves with different configurations in isolation.
