# FAQQuestions.stories.ts

**Path**: `apps\front-ecommerce\components\content\stories\FAQQuestions.stories.ts`

## Summary
This Storybook configuration file defines visual testing stories for an FAQ Questions component in an e-commerce application. It sets up a default story with mock FAQ data and configures Chromatic visual regression testing across three responsive viewports (mobile, tablet, and desktop). The file enables developers and designers to preview and test the FAQ Questions component in isolation with different screen sizes.
