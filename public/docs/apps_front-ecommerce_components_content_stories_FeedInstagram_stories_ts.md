# FeedInstagram.stories.ts

**Path**: `apps\front-ecommerce\components\content\stories\FeedInstagram.stories.ts`

## Summary
This is a Storybook configuration file that defines visual test stories for an Instagram feed component in an e-commerce application. It sets up three story variants (Default, QualityZero, Quality100) to demonstrate different image quality settings for the Instagram feed, with responsive viewport testing configured for mobile, tablet, and desktop views using Chromatic for visual regression testing.
