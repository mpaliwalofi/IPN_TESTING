# FiltersProductCategory.stories.ts

**Path**: `apps\front-ecommerce\components\content\stories\FiltersProductCategory.stories.ts`

## Summary
This Storybook configuration file defines visual test stories for the `FiltersProductCategory` component in an e-commerce application. It sets up multiple viewport configurations (mobile, tablet, desktop) for Chromatic visual regression testing and provides mock data for product filter groups (such as age filters like "Chiot", "Adulte", "Senior") that would be displayed in a product catalog filtering interface.
