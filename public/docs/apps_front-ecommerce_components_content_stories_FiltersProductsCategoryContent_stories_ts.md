**Path**: `apps\front-ecommerce\components\content\stories\FiltersProductsCategoryContent.stories.ts`

## Summary
This Storybook configuration file defines visual test stories for the `FiltersProductsCategoryContent` component in an e-commerce application. It sets up multiple viewport modes (mobile, tablet, desktop) for Chromatic visual regression testing and provides mock data for product filters including age-based filters (Chiot/Puppy, Adulte/Adult, Senior) with sort options, enabling UI testing of the product filtering interface across different screen sizes.
