# Glossary.stories.ts

**Path**: `apps\front-ecommerce\components\content\stories\Glossary.stories.ts`

## Summary
This Storybook configuration file defines visual documentation and testing scenarios for a Glossary component in an e-commerce application. It sets up multiple story variants (Default, NoColumnDesktop, NoColumnMobile) that showcase a glossary of product ingredients with different layout configurations across mobile, tablet, and desktop viewports. The component displays ingredient information with a title, subtitle, USP (Unique Selling Points) data, and a call-to-action button, supporting responsive design variations for quality assurance and design system documentation.
