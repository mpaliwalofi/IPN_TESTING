# HeroCoverCategory.stories.ts

**Path**: `apps\front-ecommerce\components\content\stories\HeroCoverCategory.stories.ts`

## Summary
This Storybook configuration file defines visual test stories for the `HeroCoverCategory` e-commerce component, which displays a hero banner for product categories (specifically pet food in the example). It sets up responsive viewport testing across mobile, tablet, and desktop breakpoints using Chromatic, with a default story showcasing a premium pet food category banner including title, hero image, and descriptive content.
