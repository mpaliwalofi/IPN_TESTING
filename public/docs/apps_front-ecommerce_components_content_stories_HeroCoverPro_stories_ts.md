# HeroCoverPro.stories.ts

**Path**: `apps\front-ecommerce\components\content\stories\HeroCoverPro.stories.ts`

## Summary
This is a Storybook configuration file for the `HeroCoverPro` Vue component in an e-commerce application. It defines visual regression testing scenarios using Chromatic across multiple viewport sizes (mobile, tablet, desktop) and includes mock data for a professional hero banner/cover image with various responsive image formats. The story setup enables developers and designers to preview and test the hero cover component in isolation with realistic CMS-sourced image data.
