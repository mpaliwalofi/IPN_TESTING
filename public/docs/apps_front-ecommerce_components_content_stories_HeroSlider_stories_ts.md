# HeroSlider.stories.ts

**Path**: `apps\front-ecommerce\components\content\stories\HeroSlider.stories.ts`

## Summary
This is a Storybook configuration file that defines visual test stories for the HeroSlider component in an e-commerce application. It sets up multiple viewport modes (mobile, tablet, desktop) for Chromatic visual regression testing and exports two story variants: a default hero slider with configurable slide timing and a single-slide version for testing edge cases.
