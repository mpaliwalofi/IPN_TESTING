# HorizontalCard.stories.ts

**Path**: `apps\front-ecommerce\components\content\stories\HorizontalCard.stories.ts`

## Summary
This is a Storybook configuration file for a HorizontalCard Vue component used in an e-commerce application. It defines visual test scenarios and documentation for the component across multiple viewport sizes (mobile, tablet, desktop), with sample data showing a product card displaying dog food with pricing and product imagery. The story enables developers and designers to preview and test how the HorizontalCard component renders product information in different device contexts.
