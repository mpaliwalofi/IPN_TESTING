# ItemBlogGrid.stories.ts

**Path**: `apps\front-ecommerce\components\content\stories\ItemBlogGrid.stories.ts`

## Summary
This file defines Storybook stories for the `ItemBlogGrid` component in an e-commerce application, showcasing how blog items are displayed in a grid layout. It configures two story variants: a default view with all standard properties and an alternative view without a grid title, while also setting up responsive viewport testing across mobile, tablet, and desktop breakpoints for visual regression testing with Chromatic.
