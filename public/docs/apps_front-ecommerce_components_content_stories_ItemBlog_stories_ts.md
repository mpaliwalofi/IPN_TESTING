# ItemBlog.stories.ts

**Path**: `apps\front-ecommerce\components\content\stories\ItemBlog.stories.ts`

## Summary
This is a Storybook configuration file that defines visual documentation and testing scenarios for an `ItemBlog` Vue component in an e-commerce application. It exports two story variants ("Default" and "ColumnLayout") that demonstrate how the blog item component renders with different layout options, and configures Chromatic visual regression testing across mobile, tablet, and desktop viewports.
