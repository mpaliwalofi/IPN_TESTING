# ItemCategory.stories.ts

**Path**: `apps\front-ecommerce\components\content\stories\ItemCategory.stories.ts`

## Summary
This is a Storybook configuration file that defines stories for the `ItemCategory` Vue component in an e-commerce application. It sets up visual testing scenarios across multiple viewport sizes (mobile, tablet, desktop) using Chromatic, and provides mock data for a category page related to cat antiparasitic products, including SEO metadata and product information.
