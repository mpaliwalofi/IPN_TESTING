# LastOrder.stories.ts

**Path**: `apps\front-ecommerce\components\content\stories\LastOrder.stories.ts`

## Summary
This is a Storybook configuration file that defines visual test stories for the `LastOrder` Vue component in an e-commerce application. It creates multiple story variants (Default, DisplayHeader, Inline, Flipped) to showcase different display configurations of the last order component, with each variant testing different parameter combinations like header visibility, layout orientation, and inline rendering. The file also configures Chromatic visual regression testing across mobile, tablet, and desktop viewports to ensure consistent rendering across different screen sizes.
