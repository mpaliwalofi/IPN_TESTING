# ListingProduct.stories.ts

**Path**: `apps\front-ecommerce\components\content\stories\ListingProduct.stories.ts`

## Summary
This is a Storybook configuration file that defines visual test cases and documentation for the `ListingProduct` component in an e-commerce application. It sets up responsive viewport testing across mobile, tablet, and desktop breakpoints using Chromatic, and provides mock product data (specifically dog food croquettes) to demonstrate how product listings are displayed with details like SKU, pricing, ratings, and variants.
