# Newsletter.stories.ts

**Path**: `apps\front-ecommerce\components\content\stories\Newsletter.stories.ts`

## Summary
This Storybook configuration file defines visual test stories for an e-commerce Newsletter component. It sets up the component to be tested across multiple viewport sizes (mobile, tablet, desktop) using Chromatic's visual regression testing, and provides a default story that renders the Newsletter component wrapped in a Suspense boundary for async content handling.
