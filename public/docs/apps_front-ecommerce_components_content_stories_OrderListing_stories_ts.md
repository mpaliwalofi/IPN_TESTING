# OrderListing.stories.ts

**Path**: `apps\front-ecommerce\components\content\stories\OrderListing.stories.ts`

## Summary
This is a Storybook configuration file that defines visual test cases for an OrderListing component in an e-commerce application. It sets up story scenarios with mock order data (including shipping/billing addresses, payment states, and order items) to enable UI testing and documentation of the OrderListing component across different viewport sizes (mobile, tablet, desktop).
