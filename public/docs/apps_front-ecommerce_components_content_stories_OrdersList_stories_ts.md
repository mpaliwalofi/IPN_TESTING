# OrdersList.stories.ts

**Path**: `apps\front-ecommerce\components\content\stories\OrdersList.stories.ts`

## Summary
This is a Storybook configuration file that defines visual test stories for an e-commerce OrdersList component. It creates two story variants ("Default" with 15 total items and "LoadMore" with 100 total items) to showcase the component's behavior with different data volumes, and configures Chromatic visual regression testing across mobile, tablet, and desktop viewports using mock order data.
