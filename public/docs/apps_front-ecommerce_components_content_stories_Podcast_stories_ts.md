# Podcast.stories.ts

**Path**: `apps\front-ecommerce\components\content\stories\Podcast.stories.ts`

## Summary
This Storybook configuration file defines UI component stories for a Podcast component in an e-commerce application. It sets up visual regression testing using Chromatic across three viewport types (mobile, tablet, desktop) and exports a default story that renders the Podcast component with iframe parameters for embedding audio/video content.
