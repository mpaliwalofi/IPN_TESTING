# ReviewLarge.stories.ts

**Path**: `apps\front-ecommerce\components\content\stories\ReviewLarge.stories.ts`

## Summary
This Storybook configuration file defines visual test stories for the `ReviewLarge` component in an e-commerce application. It exports four story variants (with/without images, with/without background, and subscription reviews) to showcase different display states of large customer review cards, with responsive viewport testing configured for mobile, tablet, and desktop breakpoints.
