# SalesForceForm.stories.ts

**Path**: `apps\front-ecommerce\components\content\stories\SalesForceForm.stories.ts`

## Summary
This Storybook configuration file defines stories for a SalesForceForm component used in an e-commerce application. It sets up test data for a lead generation form that collects customer information for a puppy welcome booklet campaign, integrating with Salesforce CRM (campaign ID: 701AZ0000009WtpYAE) and displaying a thank-you message after submission with an accompanying image of a German Shepherd puppy.
