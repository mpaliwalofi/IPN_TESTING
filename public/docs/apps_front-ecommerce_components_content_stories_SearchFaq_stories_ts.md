# SearchFaq.stories.ts

**Path**: `apps\front-ecommerce\components\content\stories\SearchFaq.stories.ts`

## Summary
This is a Storybook configuration file for the SearchFaq component in an e-commerce application. It sets up visual testing scenarios with responsive viewport configurations (mobile, tablet, desktop) for Chromatic, and provides mock data including a hero cover image for the FAQ search interface. The file enables developers and designers to preview and test the FAQ search component in isolation across different screen sizes.
