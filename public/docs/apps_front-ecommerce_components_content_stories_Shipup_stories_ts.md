# Shipup.stories.ts

**Path**: `apps\front-ecommerce\components\content\stories\Shipup.stories.ts`

## Summary
This Storybook configuration file defines stories for the Shipup Vue component used in an e-commerce front-end application. It sets up responsive viewport testing (mobile, tablet, desktop) via Chromatic and provides mock order data including shipping/billing addresses, payment methods, and order totals for visualizing the Shipup component's behavior with realistic e-commerce order information.
