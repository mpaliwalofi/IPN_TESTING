# ShopLocationListing.stories.ts

**Path**: `apps\front-ecommerce\components\content\stories\ShopLocationListing.stories.ts`

## Summary
This is a Storybook configuration file that defines visual component documentation and testing scenarios for the `ShopLocationListing` Vue component in an e-commerce application. It sets up responsive viewport testing across mobile, tablet, and desktop devices using Chromatic, and provides sample data structure for shop locations including details like address, coordinates, contact information, and images. The file serves as both documentation and a testing harness for how shop location data should be displayed in the e-commerce frontend.
