# ShopRelations.stories.ts

**Path**: `apps\front-ecommerce\components\content\stories\ShopRelations.stories.ts`

## Summary
This Storybook configuration file defines visual testing stories for the ShopRelations Vue component in an e-commerce application. It sets up responsive viewport testing (mobile, tablet, desktop) using Chromatic and provides mock data representing shop locations with details like addresses, coordinates, and contact information. The component appears to display a list of related or alternative shop locations to users browsing the e-commerce site.
