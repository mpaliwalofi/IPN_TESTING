# SocialLinks.stories.ts

**Path**: `apps\front-ecommerce\components\content\stories\SocialLinks.stories.ts`

## Summary
This is a Storybook configuration file that defines visual test stories for a SocialLinks component in an e-commerce application. It sets up a default story showcasing social media links (Instagram, Facebook, TikTok, Twitter, Pinterest, YouTube) with grid layout configuration and tests the component across mobile, tablet, and desktop viewports using Chromatic for visual regression testing.
