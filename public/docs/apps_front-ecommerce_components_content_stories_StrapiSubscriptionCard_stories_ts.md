# StrapiSubscriptionCard.stories.ts

**Path**: `apps\front-ecommerce\components\content\stories\StrapiSubscriptionCard.stories.ts`

## Summary
This Storybook configuration file defines visual test stories for the `StrapiSubscriptionCard` component in an e-commerce application. It sets up multiple scenarios (Sent, InProgress, etc.) to showcase different subscription states, including various edge cases like expired cards, stopped subscriptions, and relay-off situations, using mocked subscription and order data. The stories are configured to test responsive behavior across mobile, tablet, and desktop viewports using Chromatic visual testing.
