# SubscriptionFaq.stories.ts

**Path**: `apps\front-ecommerce\components\content\stories\SubscriptionFaq.stories.ts`

## Summary
This Storybook configuration file defines visual testing stories for the `SubscriptionFaq` component in an e-commerce application. It sets up responsive viewport testing across mobile, tablet, and desktop devices, and provides sample data for FAQ questions related to subscription management (adding/modifying products, getting help, etc.). The file enables developers and designers to preview and test how subscription-related FAQ content renders across different screen sizes.
