## Summary
This is a Storybook configuration file that defines visual test stories for the `SubscriptionsListing` component in an e-commerce application. It showcases different subscription states (default, in-progress, expiring credit card, expired, stopped, relay off, and non-eligible for promotion) using mock data to enable UI development and visual regression testing across mobile, tablet, and desktop viewports.
