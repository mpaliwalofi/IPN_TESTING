# Title.stories.ts

**Path**: `apps\front-ecommerce\components\content\stories\Title.stories.ts`

## Summary
This is a Storybook configuration file that defines visual documentation and testing scenarios for an e-commerce Title component. It sets up a single "Default" story that renders the Title component with sample text ("Ceci est un titre") and configures Chromatic visual regression testing across three viewport sizes (mobile, tablet, desktop) to ensure consistent rendering across different screen dimensions.
