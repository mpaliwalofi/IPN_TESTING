# USPLarge.stories.ts

**Path**: `apps\front-ecommerce\components\content\stories\USPLarge.stories.ts`

## Summary
This is a Storybook configuration file that defines visual test stories for the USPLarge component, which displays unique selling propositions (USPs) in an e-commerce context. The file sets up multiple story variants (Default, NoColumn, NoButton) to test different configurations of the component, including with/without columns, buttons, and mock glossary data, across mobile, tablet, and desktop viewports for visual regression testing with Chromatic.
