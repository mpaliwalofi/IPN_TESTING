# USP.stories.ts

**Path**: `apps\front-ecommerce\components\content\stories\USP.stories.ts`

## Summary
This Storybook configuration file defines visual documentation and test scenarios for a USP (Unique Selling Points) component in an e-commerce application. It sets up two story variants: a default version using mock USP data from Strapi, and an "Edito" version that includes additional editorial content with images, with both stories configured to render across mobile, tablet, and desktop viewports for visual regression testing with Chromatic.
