# TheFooter.vue

**Path**: `apps\front-ecommerce\components\footer\TheFooter.vue`

## Summary
This is a Vue 3 footer component for an e-commerce application that displays newsletter subscription, social media links, and footer navigation content. It fetches footer data from a CMS, manages a country selector dropdown with click-outside detection, and includes responsive accordion functionality for mobile devices that auto-closes after 500ms. The component serves as the main site footer with multi-column layout on desktop and collapsible sections on mobile.
