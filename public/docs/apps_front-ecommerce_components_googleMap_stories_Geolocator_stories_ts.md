# Geolocator.stories.ts

**Path**: `apps\front-ecommerce\components\googleMap\stories\Geolocator.stories.ts`

## Summary
This is a Storybook configuration file that defines visual documentation and testing scenarios for a `Geolocator` Vue component used in an e-commerce application. The file sets up two story variants (Default and NoPlaceHolder) for a location search interface that allows users to find pickup relay points by entering their address, with responsive viewport configurations for mobile, tablet, and desktop testing.
