# LangSwitcherHeader.vue

**Path**: `apps\front-ecommerce\components\header\LangSwitcherHeader.vue`

## Summary
This Vue component provides a language switcher dropdown in the e-commerce site header, allowing users to toggle between available locales/languages for the current country. It displays the current language code (e.g., "EN", "FR") and renders a clickable dropdown menu that shows alternative language options, with click-outside functionality to automatically close the menu when users interact elsewhere.
