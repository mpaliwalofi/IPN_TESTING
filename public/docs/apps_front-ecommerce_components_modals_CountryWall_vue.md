# CountryWall.vue

**Path**: `apps\front-ecommerce\components\modals\CountryWall.vue`

## Summary
This component implements a country/language selection wall that detects a user's browser language and prompts them to switch to their region-specific e-commerce site if available. It manages a modal dialog that displays the detected country flag, allows users to either accept the regional redirect (preserving query parameters) or dismiss the prompt, and stores their preference in local storage to avoid repeated prompts. The component handles automatic redirection to country-specific URLs based on browser locale detection.
