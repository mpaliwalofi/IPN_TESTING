# ModalShipup.vue

**Path**: `apps\front-ecommerce\components\modals\ModalShipup.vue`

## Summary
This Vue component implements a modal dialog for displaying shipment tracking information in an e-commerce application. It uses the Shipup service to show order tracking details, with functionality to automatically scroll to the top of the modal when users click tracking links, and provides a close button that emits an event to the parent component.
