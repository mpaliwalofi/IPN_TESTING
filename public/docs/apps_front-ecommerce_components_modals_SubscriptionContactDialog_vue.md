# SubscriptionContactDialog.vue

**Path**: `apps\front-ecommerce\components\modals\SubscriptionContactDialog.vue`

## Summary
This Vue component implements a modal dialog for customers to contact support regarding their subscriptions in an e-commerce application. It provides a form that allows users to request subscription modifications (such as cancellations or updates) by submitting contact requests with their subscription details, customer information, and reCAPTCHA validation to a cloud contact service. The component tracks user interactions for analytics and handles different modes of subscription contact while pre-filling customer data from the store.
