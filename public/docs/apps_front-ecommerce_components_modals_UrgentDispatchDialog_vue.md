# UrgentDispatchDialog.vue

**Path**: `apps\front-ecommerce\components\modals\UrgentDispatchDialog.vue`

## Summary
This Vue component manages an urgent dispatch dialog for subscription orders in an e-commerce system. It allows users to request immediate shipping of their next scheduled subscription delivery, displaying the draft order items and shipping address for confirmation. The component handles the submission flow with success/error states, emits events to notify parent components of successful updates, and includes tracking functionality to monitor user interactions with the urgent dispatch feature.
