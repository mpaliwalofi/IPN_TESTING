# ModalShipup.stories.ts

**Path**: `apps\front-ecommerce\components\modals\stories\ModalShipup.stories.ts`

## Summary
This is a Storybook configuration file that defines UI documentation and testing scenarios for the `ModalShipup` component in an e-commerce application. It provides mock order data including customer addresses, payment methods (Carte Bancaire via PAYZEN), and shipment information to showcase how the modal displays shipping-related information for orders. The file sets up the component's visual testing environment with realistic French e-commerce order data for the Shipup modal integration.
