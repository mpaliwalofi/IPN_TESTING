# OrderAgain.vue

**Path**: `apps\front-ecommerce\components\order\OrderAgain.vue`

## Summary
This Vue component enables customers to quickly reorder items from their previous order in an e-commerce application. It provides functionality to add last order items back to the cart (either as one-time purchases or subscriptions), tracks user interactions for analytics, and manages loading states while handling cart operations. The component is designed to be reusable across different pages (homepage, category pages, static pages) with customizable display parameters and proper tracking context for each placement.
