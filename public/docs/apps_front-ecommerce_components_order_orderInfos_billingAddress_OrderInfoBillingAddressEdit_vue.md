# OrderInfoBillingAddressEdit.vue

**Path**: `apps\front-ecommerce\components\order\orderInfos\billingAddress\OrderInfoBillingAddressEdit.vue`

## Summary
This Vue component enables customers to view and edit the billing address for a subscription order. It manages billing address selection from an address book, allows creating/editing addresses, handles the "billing same as shipping" checkbox option, and updates the subscription's billing address through API calls while maintaining synchronization with the customer's address book.
