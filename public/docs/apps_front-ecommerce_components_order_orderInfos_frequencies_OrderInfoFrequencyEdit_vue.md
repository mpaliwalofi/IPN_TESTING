# OrderInfoFrequencyEdit.vue

**Path**: `apps\front-ecommerce\components\order\orderInfos\frequencies\OrderInfoFrequencyEdit.vue`

## Summary
This Vue component allows customers to edit the delivery frequency of their subscription orders in an e-commerce system. It displays the current subscription interval, provides a dropdown to select a new frequency, and handles the submission to update both the subscription interval and shipping date via the `updateSubscriptionIntervalAndShipping` API call while managing loading and locked states during the update process.
