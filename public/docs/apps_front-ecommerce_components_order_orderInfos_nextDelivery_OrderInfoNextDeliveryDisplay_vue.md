# OrderInfoNextDeliveryDisplay.vue

**Path**: `apps\front-ecommerce\components\order\orderInfos\nextDelivery\OrderInfoNextDeliveryDisplay.vue`

## Summary
This Vue component displays the next scheduled delivery date for a subscription order and provides an option to trigger urgent/expedited dispatch. It shows the delivery information with a call-to-action button (when editing is disabled and urgent dispatch is allowed) that opens a modal dialog for customers to request faster shipping of their next subscription order.
