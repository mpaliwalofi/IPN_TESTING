# OrderInfoNextDeliveryEdit.vue

**Path**: `apps\front-ecommerce\components\order\orderInfos\nextDelivery\OrderInfoNextDeliveryEdit.vue`

## Summary
This Vue 3 component allows customers to edit the next delivery date for their e-commerce subscription. It generates a calendar view of available shipping dates by fetching valid delivery windows, displays them in a monthly format, and enables users to select and update their subscription's next shipping date while tracking changes for analytics purposes.
