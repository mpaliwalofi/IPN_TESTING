# OrderInfoProductsMain.vue

**Path**: `apps\front-ecommerce\components\order\orderInfos\productsList\OrderInfoProductsMain.vue`

## Summary
This Vue component displays and manages the products list section for orders and subscriptions in an e-commerce application. It conditionally renders order items from either a regular order or a subscription's draft order, with support for editing subscription items when enabled. The component integrates tracking functionality to monitor subscription upsell events (add/remove items) and handles reloading the products list after subscription updates.
