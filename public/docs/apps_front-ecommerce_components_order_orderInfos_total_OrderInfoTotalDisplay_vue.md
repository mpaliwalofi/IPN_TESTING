# OrderInfoTotalDisplay.vue

**Path**: `apps\front-ecommerce\components\order\orderInfos\total\OrderInfoTotalDisplay.vue`

## Summary
This Vue component displays a detailed breakdown of order totals for either a regular order or a subscription's draft order in an e-commerce application. It calculates and presents line items including subtotal (sum of items), shipping fees, payment fees, and promotional discounts, with logic to handle different data structures between orders and subscriptions. The component serves as a reusable order summary widget that dynamically computes totals from the provided order or subscription data and formats prices for display.
