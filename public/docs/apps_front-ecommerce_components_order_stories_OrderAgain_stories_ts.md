# OrderAgain.stories.ts

**Path**: `apps\front-ecommerce\components\order\stories\OrderAgain.stories.ts`

## Summary
This is a Storybook configuration file that defines visual test scenarios for the `OrderAgain` Vue component in an e-commerce application. It creates multiple story variations to showcase different states of the "order again" functionality, including default state with an existing order, empty state with no last order, and variations with flipped layout and icon display options. The stories enable developers and designers to visualize and test how the component renders under different conditions using mock order data.
