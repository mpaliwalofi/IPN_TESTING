# OrderInvoice.stories.ts

**Path**: `apps\front-ecommerce\components\order\stories\OrderInvoice.stories.ts`

## Summary
This is a Storybook configuration file that defines stories for the `OrderInvoice` component in an e-commerce application. It sets up a test/documentation story with mock order data including customer information (billing/shipping addresses), payment details (PAYZEN/Carte Bancaire method), and shipment information (DPD delivery) for visualizing and testing the order invoice component in isolation.
