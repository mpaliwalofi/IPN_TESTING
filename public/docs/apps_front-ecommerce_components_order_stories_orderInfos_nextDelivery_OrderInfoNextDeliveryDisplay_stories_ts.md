**Path**: `apps\front-ecommerce\components\order\stories\orderInfos\nextDelivery\OrderInfoNextDeliveryDisplay.stories.ts`

## Summary
This Storybook configuration file defines stories for the `OrderInfoNextDeliveryDisplay` Vue component, which displays next delivery information for e-commerce subscription orders. It sets up mock subscription data including customer details, billing/shipping addresses, and shipping method information to enable visual testing and documentation of how upcoming delivery details are presented to users in the order management interface.
