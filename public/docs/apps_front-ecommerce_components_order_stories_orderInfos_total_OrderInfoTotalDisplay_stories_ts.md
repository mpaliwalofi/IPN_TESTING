# OrderInfoTotalDisplay.stories.ts

**Path**: `apps\front-ecommerce\components\order\stories\orderInfos\total\OrderInfoTotalDisplay.stories.ts`

## Summary
This is a Storybook configuration file for the `OrderInfoTotalDisplay` Vue component in an e-commerce application. It sets up stories to showcase and test the order total display component using sample data from a subscription draft order that includes billing/shipping addresses, payment methods, and shipment information. The file enables visual testing and documentation of how order totals are rendered in the UI with realistic order data.
