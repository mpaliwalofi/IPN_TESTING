# OrderSummaryDetails.vue

**Path**: `apps\front-ecommerce\components\order\summary\OrderSummaryDetails.vue`

## Summary
This Vue 3 component renders an order summary section for an e-commerce checkout flow, displaying order details, adjustments (like payment fees), and checkout actions. It handles both standard cart orders and subscription orders, with responsive behavior that adapts to screen size and conditionally shows payment method USPs (unique selling points) and checkout buttons based on configuration flags. The component also supports displaying payment footer information and processes markdown content for order summary headers while managing loading states during cart updates.
