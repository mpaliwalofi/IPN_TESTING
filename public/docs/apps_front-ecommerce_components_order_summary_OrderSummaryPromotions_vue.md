# OrderSummaryPromotions.vue

**Path**: `apps\front-ecommerce\components\order\summary\OrderSummaryPromotions.vue`

## Summary
This Vue component manages promotional coupon code functionality for e-commerce order summaries (cart or subscription orders). It allows users to add and remove coupon codes, displays active promotions applied to the order, handles coupon validation errors, and integrates with a MentionMe referral system for checkout tracking.
