# LayerProductCard.vue

**Path**: `apps\front-ecommerce\components\overlay\LayerProductCard.vue`

## Summary
This Vue component renders a product card overlay layer for e-commerce purposes, supporting separate desktop and mobile configurations. It fetches product metadata based on SKU from props, displays product information, and implements tracking analytics by firing "select item" events when users interact with the product cards on either mobile or desktop views. The component is designed to be used within an overlay context where products need to be showcased with full tracking capabilities for conversion monitoring.
