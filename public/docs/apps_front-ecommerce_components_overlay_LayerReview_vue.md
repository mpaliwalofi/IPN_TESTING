# LayerReview.vue

**Path**: `apps\front-ecommerce\components\overlay\LayerReview.vue`

## Summary
This Vue component renders a product review overlay layer with responsive design (separate desktop/mobile configurations). It displays a review badge/image alongside text content with configurable font sizes, using client-side rendering to show different layouts based on screen size. The component is specifically designed for an e-commerce catalog to showcase product reviews with customizable styling through the `IOverlayLayerReviews` interface.
