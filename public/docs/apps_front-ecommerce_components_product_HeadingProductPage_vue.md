# HeadingProductPage.vue

**Path**: `apps\front-ecommerce\components\product\HeadingProductPage.vue`

## Summary
This Vue component serves as the main heading section for an e-commerce product page, managing product data fetching from Sylius, displaying product information, and handling add-to-cart functionality. It integrates tracking events (view item, add to cart), manages cart dialog modals, and implements business logic for subscription frequencies and bi-nutrition features based on the product configuration.
