# Ingredients.vue

**Path**: `apps\front-ecommerce\components\product\Ingredients.vue`

## Summary
This Vue component displays a horizontally scrollable list of product ingredients with navigation controls. It uses VueUse's scroll composable to track scroll position and enable/disable left/right arrow buttons based on whether the user has reached the start or end of the ingredient list. The component is designed for an e-commerce product page, rendering ingredient data from a Strapi CMS in a card-based layout with responsive behavior for mobile and desktop views.
