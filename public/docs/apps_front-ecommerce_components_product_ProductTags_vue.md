# ProductTags.vue

**Path**: `apps\front-ecommerce\components\product\ProductTags.vue`

## Summary
This Vue component displays product tags for e-commerce items, combining tags from both Strapi CMS and a Pixie product tags engine. It handles filtering tags based on display mode (PLP - Product Listing Page vs. detail view), limits the display to a maximum of 2 tags, and applies different visual themes (new, promo, info, limited, various) to each tag while showing skeleton loaders during data fetching.
