# RationCalculatorModal.vue

**Path**: `apps\front-ecommerce\components\product\RationCalculatorModal.vue`

## Summary
This Vue component implements a modal dialog for a pet food ration calculator in an e-commerce application. It provides a button that opens a dialog to help users calculate appropriate food portions for their dogs or cats, dynamically adjusting the animal type based on product metadata and tracking user interactions with the calculator modal for analytics purposes.
