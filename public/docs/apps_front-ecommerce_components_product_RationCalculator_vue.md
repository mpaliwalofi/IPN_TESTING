# RationCalculator.vue

**Path**: `apps\front-ecommerce\components\product\RationCalculator.vue`

## Summary
This component implements a pet food ration calculator that determines feeding recommendations based on pet characteristics (weight, age, type). It collects user input through a form, calls a CMS API to retrieve personalized feeding recommendations (daily ration amount, cup measurements, bag size, and bag lifetime), and displays the calculated results to help customers determine the appropriate food portions for their dogs or puppies.
