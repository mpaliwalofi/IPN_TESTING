# ShoppingPanelFooter.vue

**Path**: `apps\front-ecommerce\components\product\ShoppingPanelFooter.vue`

## Summary
This Vue component serves as the footer section of a product shopping panel in an e-commerce application, managing the add-to-cart functionality. It provides quantity selection controls via a Counter component and displays an add-to-cart button that's conditionally shown based on stock availability. The component integrates with the cart store and handles product variants, subscriptions, and error messaging for out-of-stock or invalid selections.
