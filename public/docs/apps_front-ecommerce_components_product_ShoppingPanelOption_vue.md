# ShoppingPanelOption.vue

**Path**: `apps\front-ecommerce\components\product\ShoppingPanelOption.vue`

## Summary
This Vue component renders a product shopping panel option that allows users to select between one-time purchase and subscription variants in an e-commerce context. It manages frequency selection for subscription products, handles stock availability checks across different backend systems (Sylius and Strapi), and tracks subscription-related events while providing price calculations for different purchase options.
