# ShoppingTableSamples.vue

**Path**: `apps\front-ecommerce\components\product\ShoppingTableSamples.vue`

## Summary
This Vue component manages the shopping cart functionality for product samples in an e-commerce application. It allows users to select quantities for multiple sample types (up to 2 total samples), tracks available sample slots, and handles adding selected samples to the cart with optional tracking/analytics data. The component coordinates with a cart store to process the additions and emits events to notify parent components of cart updates.
