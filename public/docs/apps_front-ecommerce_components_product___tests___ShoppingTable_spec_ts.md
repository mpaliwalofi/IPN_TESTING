# ShoppingTable.spec.ts

**Path**: `apps\front-ecommerce\components\product\__tests__\ShoppingTable.spec.ts`

## Summary
This test file verifies the functionality of a `ShoppingTable` Vue component used in an e-commerce application for displaying and managing product variants. It tests the component's ability to mount correctly and handle "add to cart" events for both regular one-time purchases and subscription-based products, ensuring proper event emission with correct product variant IDs, quantities, and frequency options.
