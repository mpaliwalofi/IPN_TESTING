# DisplayImages.stories.ts

**Path**: `apps\front-ecommerce\components\product\stories\DisplayImages.stories.ts`

## Summary
This is a Storybook configuration file that defines visual documentation and testing scenarios for the `DisplayImages` component in an e-commerce application. It provides a default story with sample product image data (main image and multiple additional images) to showcase how the component renders product photo galleries, using cat food product images as example data. The story enables developers and designers to preview and test the DisplayImages component in isolation with realistic product image structures including URLs, alt text, and file hashes.
