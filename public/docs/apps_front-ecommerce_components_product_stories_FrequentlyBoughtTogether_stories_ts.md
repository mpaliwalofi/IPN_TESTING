# FrequentlyBoughtTogether.stories.ts

**Path**: `apps\front-ecommerce\components\product\stories\FrequentlyBoughtTogether.stories.ts`

## Summary
This is a Storybook configuration file that creates a visual documentation story for the `FrequentlyBoughtTogether` Vue component in an e-commerce application. The story demonstrates how the component displays product recommendations (like "You may also like"), using sample data with product IDs, SKUs, and details about pet food products. It provides a default story variant that renders the component within a responsive grid layout with loading state handling via Vue's Suspense component.
