# ShoppingPanel.stories.ts

**Path**: `apps\front-ecommerce\components\product\stories\ShoppingPanel.stories.ts`

## Summary
This Storybook configuration file defines visual test stories for the ShoppingPanel component in an e-commerce application. It showcases different states of the product shopping panel including default view, subscription-only mode, and scenarios with out-of-stock product variants, using mock data from both Strapi (CMS) and Sylius (e-commerce platform) to demonstrate how the component handles product metadata and inventory states.
