# SubscriptionSearchResults.vue

**Path**: `apps\front-ecommerce\components\search\SubscriptionSearchResults.vue`

## Summary
This Vue component implements a product search interface specifically for adding items to subscription draft orders in an e-commerce system. It integrates Algolia's InstantSearch to display searchable product results with pagination, filters results based on the subscription's draft order token, and includes cross-sell recommendations based on items already in the cart. The component tracks search interactions for analytics and allows users to add discovered products to their subscription draft.
