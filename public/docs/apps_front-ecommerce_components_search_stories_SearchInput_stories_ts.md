# SearchInput.stories.ts

**Path**: `apps\front-ecommerce\components\search\stories\SearchInput.stories.ts`

## Summary
This Storybook configuration file defines visual documentation and interactive examples for the SearchInput component in an e-commerce application. It exports three story variants (Default, Filled, NoPlaceholder) that demonstrate different states of the search input component, including empty state, pre-filled search text ("croquettes pour petit chien"), and a version without placeholder text, with configurable props like delay and currentRefinement.
