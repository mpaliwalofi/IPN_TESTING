# AddressCard.vue

**Path**: `apps\front-ecommerce\components\shared\AddressCard.vue`

## Summary
This Vue component renders a formatted address card for an e-commerce application. It displays customer contact information including name, company, street address, postal code, city, and country, while also formatting phone numbers using the libphonenumber-js library to display them in national format based on the country code. The component supports internationalization for country names and provides a slot for custom content injection before the address details.
