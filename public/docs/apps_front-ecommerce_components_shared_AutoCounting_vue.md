# AutoCounting.vue

**Path**: `apps\front-ecommerce\components\shared\AutoCounting.vue`

## Summary
This Vue 3 component provides an animated number counter that smoothly transitions from a start value to an end value over a specified duration. It formats the displayed number with customizable prefixes, suffixes, thousand separators, and decimal places, making it suitable for displaying animated statistics, prices, or metrics in an e-commerce interface. The component supports play/pause controls and emits events when the animation finishes, allowing parent components to trigger or respond to counting animations.
