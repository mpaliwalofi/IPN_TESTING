# Bancontact.vue

**Path**: `apps\front-ecommerce\components\shared\Bancontact.vue`

## Summary
This Vue component displays the Bancontact payment method logo as an icon. It imports the Bancontact SVG graphic and renders it as a 12-unit wide image with accessible screen reader support, allowing the e-commerce application to show this payment option to users during checkout or in payment method listings.
