# BiNutritionProductCard.vue

**Path**: `apps\front-ecommerce\components\shared\BiNutritionProductCard.vue`

## Summary
This Vue component displays a nutrition product card for an e-commerce application, showing product details including an image, title, quantity, and pricing information. It supports both regular and subscription pricing models, and can display a "pack" badge for bundled products. The component integrates data from multiple sources (Strapi CMS for content/images and Sylius for pricing/commerce data) to present a comprehensive product summary card.
