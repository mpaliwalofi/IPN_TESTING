# CategoryQuickLinks.vue

**Path**: `apps\front-ecommerce\components\shared\CategoryQuickLinks.vue`

## Summary
This Vue component implements a horizontally scrollable carousel of category quick links for an e-commerce storefront. It provides navigation buttons with gradient overlays that appear/disappear based on scroll position, and adjusts scroll behavior responsively across mobile, tablet, and desktop viewports using VueUse composables for scroll management and media queries.
