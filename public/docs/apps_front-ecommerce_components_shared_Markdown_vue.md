## Summary
This Vue component provides a flexible Markdown rendering wrapper for the e-commerce front-end application. It accepts markdown content as a prop and renders it as HTML using a markdown parser (`$md`), with configurable options for inline rendering, custom HTML tag wrapping, and heading level offsets. The component enables dynamic display of formatted content (likely product descriptions, help text, or content blocks) throughout the e-commerce site while maintaining consistent markdown-to-HTML transformation.
