# OrderCard.vue

**Path**: `apps\front-ecommerce\components\shared\OrderCard.vue`

## Summary
This Vue component renders an order card for an e-commerce front-end application, displaying order details such as order number, status, date, payment state, shipping information, and the most expensive item with its image. The component supports loading states with skeleton UI, handles click navigation to order detail pages, and integrates a ShipUp modal for shipment tracking. It manages various order states including awaiting payment, payment errors, and cancellations, providing a comprehensive overview of customer orders in account management views.
