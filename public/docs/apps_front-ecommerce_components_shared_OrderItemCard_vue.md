# OrderItemCard.vue

**Path**: `apps\front-ecommerce\components\shared\OrderItemCard.vue`

## Summary
This Vue component renders an order/cart item card that displays product details and provides controls for managing items in an e-commerce shopping cart or subscription order. It handles user interactions like updating item quantities, changing subscription frequencies, and removing items from the cart, while integrating with tracking events to monitor customer behavior. The component supports both one-time purchases and subscription-based products with configurable mutability, quantity limits, and pricing display options.
