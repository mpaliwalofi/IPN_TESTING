# OrderItemUpsell.vue

**Path**: `apps\front-ecommerce\components\shared\OrderItemUpsell.vue`

## Summary
This Vue component displays upsell and cross-sell product recommendations in the shopping cart or checkout flow for an e-commerce application. It conditionally renders promotional product cards with discount tags, product images, and an "add to cart" button, while handling different states like loading, out-of-stock items, and subscription products. The component includes responsive behavior for mobile and desktop viewports and integrates with Strapi CMS for product data.
