# ProductContentDetails.vue

**Path**: `apps\front-ecommerce\components\shared\ProductContentDetails.vue`

## Summary
This Vue component displays detailed product content tabs for an e-commerce product page, including descriptions, reviews, composition information, and ration (feeding) guidelines. It processes product metadata to dynamically generate and filter active tabs based on available data, organizing them by position with computed properties for product reviews and tab configurations. The component uses internationalization (i18n) for multi-language support and includes formatting utilities for nutrition/ration data display.
