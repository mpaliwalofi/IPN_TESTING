# ProductsSlider.vue

**Path**: `apps\front-ecommerce\components\shared\ProductsSlider.vue`

## Summary
This Vue component implements a product slider/carousel for an e-commerce storefront, capable of displaying products from multiple sources (Strapi CMS, Algolia search hits, or SKU strings). It fetches and normalizes product metadata, handles lazy loading of product details, and provides slide navigation while supporting analytics tracking through item list sources and excluding sample products from certain operations.
