# ReviewContent.vue

**Path**: `apps\front-ecommerce\components\shared\ReviewContent.vue`

## Summary
This Vue component displays customer review content for an e-commerce application, showing the reviewer's name, date (in relative format), star rating, comment text, and an optional image. The component supports a "clamped" mode that truncates the review text to 3 lines and makes it clickable to open the full review in a modal dialog, with theme customization options (moutarde, beige, bleu-nuit, vert-eau) for different visual styles.
