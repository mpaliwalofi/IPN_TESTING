# ReviewTitle.vue

**Path**: `apps\front-ecommerce\components\shared\ReviewTitle.vue`

## Summary
This Vue component displays a review section title with associated rating information for e-commerce products. It renders a customizable title, star rating visualization, numerical rating score, and review counts, with support for internationalization and responsive layout that adapts between mobile and desktop views. The component accepts content parameters defining the review data (title, rating, number of reviews) and uses a smart image component to display rating overlays.
