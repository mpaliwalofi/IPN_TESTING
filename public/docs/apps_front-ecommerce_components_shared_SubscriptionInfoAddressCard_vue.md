# SubscriptionInfoAddressCard.vue

**Path**: `apps\front-ecommerce\components\shared\SubscriptionInfoAddressCard.vue`

## Summary
This Vue component displays an informational card about subscription-related addresses in an e-commerce account management interface. It alerts users that their addresses are associated with active subscriptions and provides a call-to-action button that navigates to the account subscriptions page where they can update their subscription addresses. The component uses a tertiary-colored card layout with localized text content and a styled link button.
