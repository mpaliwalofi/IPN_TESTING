# TransitionComponent.vue

**Path**: `apps\front-ecommerce\components\shared\TransitionComponent.vue`

## Summary
This Vue component provides a reusable transition wrapper that animates elements when they enter or leave the DOM. It supports two modes: a simple fade transition (opacity-based) and an optional height-based transition that smoothly expands/collapses elements by animating their height from 0 to full scrollHeight, with configurable top/bottom margins. The component is designed for UI animations in an e-commerce application, allowing developers to easily add enter/exit animations to any content by wrapping it in this component.
