# AddressCard.stories.ts

**Path**: `apps\front-ecommerce\components\shared\stories\AddressCard.stories.ts`

## Summary
This is a Storybook configuration file that defines visual documentation and testing scenarios for an AddressCard Vue component in an e-commerce application. It provides mock data representing various customer address formats (with/without company names, secondary streets, default addresses, etc.) to showcase how the AddressCard component renders different address types used in shipping and billing workflows.
