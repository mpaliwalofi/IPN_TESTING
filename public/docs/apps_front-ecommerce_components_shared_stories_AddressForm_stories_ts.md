# AddressForm.stories.ts

**Path**: `apps\front-ecommerce\components\shared\stories\AddressForm.stories.ts`

## Summary
This is a Storybook configuration file that defines visual documentation and testing scenarios for an e-commerce AddressForm component. It provides a default story showcasing the form with pre-populated French address data, including fields for personal information, location details, and delivery options like Mondial Relay integration. The story enables developers and designers to preview and interact with the address form component in isolation during development.
