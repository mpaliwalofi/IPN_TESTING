# AnimalCard.stories.ts

**Path**: `apps\front-ecommerce\components\shared\stories\AnimalCard.stories.ts`

## Summary
This is a Storybook configuration file that defines visual component stories for an `AnimalCard` Vue component in an e-commerce application. It provides example data and multiple story variants (DefaultCat, DefaultDog) showcasing how the AnimalCard component renders different animal profiles (cats and dogs) with properties like name, type, age, and optional photos, allowing developers and designers to preview and test the component in isolation.
