# Entrance.stories.ts

**Path**: `apps\front-ecommerce\components\shared\stories\Entrance.stories.ts`

## Summary
This is a Storybook configuration file that documents and demonstrates the `Entrance` component used in an e-commerce application. It defines two visual states: a default entrance component displaying a pet food guide prompt with help text and a call-to-action button, and a Calendly variant that integrates a calendar scheduling interface. The stories are specifically configured to showcase an interactive guide entrance UI element that helps customers find appropriate pet food products based on their needs.
