# Modal.stories.ts

**Path**: `apps\front-ecommerce\components\shared\stories\Modal.stories.ts`

## Summary
This file defines Storybook stories for an e-commerce Modal component, providing visual documentation and testing scenarios. It includes two specific story variants: one demonstrating a payment fees information modal with header icon and confirmation button, and another showing an address deletion confirmation modal with yes/no action buttons. The stories serve as a component library reference for developers to understand different modal use cases within the e-commerce application.
