# OrderCard.stories.ts

**Path**: `apps\front-ecommerce\components\shared\stories\OrderCard.stories.ts`

## Summary
This Storybook configuration file defines visual documentation and testing scenarios for an e-commerce OrderCard component. It showcases the OrderCard component in four different order states (Completed, Sent, AwaitingPayment, and Cancelled) using mock order data, allowing developers and designers to preview how the component renders across different order statuses in a desktop layout.
