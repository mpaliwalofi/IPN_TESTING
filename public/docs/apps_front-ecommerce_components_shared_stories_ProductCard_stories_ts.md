# ProductCard.stories.ts

**Path**: `apps\front-ecommerce\components\shared\stories\ProductCard.stories.ts`

## Summary
This Storybook configuration file defines visual test cases and documentation for the `ProductCard` Vue component used in an e-commerce application. It sets up multiple story variants including default product display, loading states, subscription-only mode, out-of-stock scenarios, and relevancy-based display, using mock product data retrieved from a catalog store to demonstrate different component behaviors and states.
