# SubscriptionCard.stories.ts

**Path**: `apps\front-ecommerce\components\shared\stories\SubscriptionCard.stories.ts`

## Summary
This is a Storybook configuration file that defines visual test scenarios for the `SubscriptionCard` Vue component in an e-commerce application. It sets up multiple story variants (Active, OrderSent, PaymentFailed, etc.) to showcase different subscription states including active subscriptions, expired payment methods, stopped subscriptions, out-of-stock products, and various error conditions. The stories are configured with responsive viewport testing across mobile, tablet, and desktop breakpoints using Chromatic visual regression testing.
