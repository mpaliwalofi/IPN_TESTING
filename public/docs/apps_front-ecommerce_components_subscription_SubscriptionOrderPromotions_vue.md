# SubscriptionOrderPromotions.vue

**Path**: `apps\front-ecommerce\components\subscription\SubscriptionOrderPromotions.vue`

## Summary
This Vue component manages promotional coupon codes for subscription orders in an e-commerce system. It provides functionality to add and remove coupon codes from draft subscription orders, handles validation errors from the API, and tracks coupon application events for analytics purposes. The component serves as a wrapper that connects the OrderSummaryPromotions UI component with subscription order management logic and tracking infrastructure.
