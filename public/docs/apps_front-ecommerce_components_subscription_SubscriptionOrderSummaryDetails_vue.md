# SubscriptionOrderSummaryDetails.vue

**Path**: `apps\front-ecommerce\components\subscription\SubscriptionOrderSummaryDetails.vue`

## Summary
This Vue component renders an order summary specifically for subscription-based purchases in an e-commerce application. It wraps the generic `OrderSummaryDetails` component with subscription-specific configuration, accepting subscription page content, order data, and an optional updating state, while providing a customized title header for the checkout cart order details section.
