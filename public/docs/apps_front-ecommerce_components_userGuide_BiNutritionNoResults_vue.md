# BiNutritionNoResults.vue

**Path**: `apps\front-ecommerce\components\userGuide\BiNutritionNoResults.vue`

## Summary
This Vue component displays a "no results" message for the BI Nutrition feature when a pet is not eligible for the service. It presents a centered (on mobile) or left-aligned (on desktop) layout showing a title explaining no results were found, informational text about pet ineligibility, and a customer service phone number for users to contact for assistance.
