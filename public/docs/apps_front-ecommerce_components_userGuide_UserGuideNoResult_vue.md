# UserGuideNoResult.vue

**Path**: `apps\front-ecommerce\components\userGuide\UserGuideNoResult.vue`

## Summary
This Vue component displays a "no results" page for a pet food e-commerce user guide when no matching products are found for a pet's criteria. It shows an apologetic message and provides action buttons to either browse all recipes for the pet type (cat/dog) or add a new pet (if logged in), along with cross-sell product recommendations and contact information for customer assistance.
