# GuideFormButtons.vue

**Path**: `apps\front-ecommerce\components\userGuide\form\GuideFormButtons.vue`

## Summary
This Vue component provides a button-based form input for the user guide, allowing users to select one or multiple options from a set of buttons. It manages the selection state, handles both single and multiple selection modes, and emits updates to the parent form when buttons are toggled, with visual feedback for checked states.
