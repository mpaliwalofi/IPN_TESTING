# GuideFormChoice.vue

**Path**: `apps\front-ecommerce\components\userGuide\form\GuideFormChoice.vue`

## Summary
This Vue component implements a multi-select or single-select choice input field for a user guide form in an e-commerce application. It manages the selection state of choices (rendered as buttons or checkboxes), handling both single and multiple selection modes by tracking selected values through v-model binding. The component provides toggle functionality to add/remove choices from the selection and emits updates to the parent form component.
