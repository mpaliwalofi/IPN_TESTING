# GuideFormRadio.vue

**Path**: `apps\front-ecommerce\components\userGuide\form\GuideFormRadio.vue`

## Summary
This Vue 3 component implements a custom radio button input for user guide forms in an e-commerce application. It provides two-way data binding through v-model, allowing users to toggle a boolean value on/off, with support for disabled states and custom labels. The component wraps a native radio input with custom styling and emits value updates to parent components when toggled.
