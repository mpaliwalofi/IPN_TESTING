# GuideFormSelect.vue

**Path**: `apps\front-ecommerce\components\userGuide\form\GuideFormSelect.vue`

## Summary
This is a reusable Vue 3 select dropdown component for an e-commerce user guide form system. It provides a customizable select field with label floating animation, optional search icon, error display, and flexible option rendering through slots or default name property. The component implements two-way data binding via v-model and supports dynamic option lists with configurable property keys for value extraction.
