# GuideFormSwitch.vue

**Path**: `apps\front-ecommerce\components\userGuide\form\GuideFormSwitch.vue`

## Summary
This is a Vue 3 component that implements a custom toggle switch form control with two-way data binding via v-model. It manages a boolean state that can be toggled on/off through user clicks, emitting value changes to its parent component while also responding to external prop updates. The component includes disabled state handling and visual feedback through CSS classes for hover effects and checked/unchecked positions.
