# GuideFormToggle.vue

**Path**: `apps\front-ecommerce\components\userGuide\form\GuideFormToggle.vue`

## Summary
This Vue component implements a toggle button group for user guide forms in an e-commerce application. It displays a set of buttons (with optional icons) where users can select one option, emitting the selected value back to the parent form via v-model binding, with support for custom styling of selected/unselected states and error message display.
