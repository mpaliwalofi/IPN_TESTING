# useCmsCategoryContext.ts

**Path**: `apps\front-ecommerce\composables\PageContexts\useCmsCategoryContext.ts`

## Summary
This TypeScript class extends `CmsPageContext` to handle category page-specific functionality in an e-commerce CMS system. It builds structured data (schema.org microdata) for category pages including Product and FAQPage schemas, constructs canonical URLs for category pages using their slugs, and aggregates product ratings and offers to provide rich search engine metadata for category listings.
