# useCmsHomepageContext.ts

**Path**: `apps\front-ecommerce\composables\PageContexts\useCmsHomepageContext.ts`

## Summary
This class extends `CmsPageContext` to handle homepage-specific context and metadata for an e-commerce CMS system. Its primary purpose is to build structured data (schema.org) for the organization, including brand information, contact details, business hours, and address, which enhances SEO and search engine visibility for the homepage. It also generates the canonical URL for the homepage route and aggregates this metadata with inherited page context functionality.
