# useCmsPageContext.ts

**Path**: `apps\front-ecommerce\composables\PageContexts\useCmsPageContext.ts`

## Summary
This code defines a `CmsPageContext` class that manages SEO metadata and page context for CMS-driven pages sourced from Strapi. It extends a base `PageContext` class to build page metadata including OpenGraph tags, meta descriptions, canonical URLs, and hreflang attributes from Strapi composite page data, ensuring proper indexing and social media sharing capabilities for dynamically-generated CMS content.
