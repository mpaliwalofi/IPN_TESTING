## Summary
This class manages the page context and metadata for product pages in an e-commerce CMS system. It handles SEO elements (title, description, indexing rules), structured data/schema markup for products, OpenGraph metadata, and tracking parameters, while integrating product information from Strapi CMS. The class extends a base PageContext to provide product-specific functionality including canonical URLs, microdata generation, and conditional indexing based on query parameters.
