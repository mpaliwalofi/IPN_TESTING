# useCmsShopContext.ts

**Path**: `apps\front-ecommerce\composables\PageContexts\useCmsShopContext.ts`

## Summary
This class extends `CmsPageContext` to handle SEO and structured data for e-commerce shop location pages. It generates Schema.org Store/LocalBusiness microdata including address, opening hours, contact details, and geo-coordinates for individual physical store pages, while also building proper canonical URLs using the shop's slug identifier.
