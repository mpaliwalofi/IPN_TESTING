# useCmsShopsContext.ts

**Path**: `apps\front-ecommerce\composables\PageContexts\useCmsShopsContext.ts`

## Summary
This class extends `CmsPageContext` to handle the shops listing page in an e-commerce CMS system. It generates SEO-optimized metadata including canonical URLs and Schema.org structured data (microdata) for physical store locations, transforming Strapi CMS shop data into standardized JSON-LD format with store details like addresses, opening hours, and images for better search engine visibility.
