# useAddress.ts

**Path**: `apps\front-ecommerce\composables\useAddress.ts`

## Summary
This composable provides address management functionality for an e-commerce application, including formatting addresses for display in select dropdowns, retrieving customer address books, and comparing addresses for equality. It serves as a utility layer for handling customer shipping/billing addresses throughout the checkout and account management flows, with features like address linearization (converting structured address objects into readable strings) and address book retrieval from the customer store.
