# useBooxi.ts

**Path**: `apps\front-ecommerce\composables\useBooxi.ts`

## Summary
This composable manages the integration of the Booxi booking widget system for e-commerce appointment scheduling. It handles loading the external Booxi script, opening the booking widget with proper API configuration, and integrating booking events with Google Tag Manager for analytics tracking. The code also supports preloading the widget and auto-opening it based on URL parameters.
