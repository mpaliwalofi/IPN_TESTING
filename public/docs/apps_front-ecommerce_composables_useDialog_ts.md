# useDialog.ts

**Path**: `apps\front-ecommerce\composables\useDialog.ts`

## Summary
This composable provides dialog management functionality for HTML `&lt;dialog&gt;` elements in a Vue 3 e-commerce application. It handles showing/hiding modal dialogs, tracks whether the dialog is mounted in the DOM, and implements click-outside-to-close behavior by detecting when users click on the backdrop area outside the dialog boundaries. The composable abstracts common dialog interaction patterns to be reused across the application's UI components.
