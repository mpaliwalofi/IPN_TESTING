# useFetchCloudContact.ts

**Path**: `apps\front-ecommerce\composables\useFetchCloudContact.ts`

## Summary
This composable provides a centralized HTTP client for communicating with a cloud-based contact service API in an e-commerce application. It creates a configured fetch instance with automatic locale injection and exposes three methods (`fetchContact`, `fetchNewsletter`, `fetchSalesforce`) for submitting contact forms, newsletter subscriptions, and Salesforce integrations to the contact service endpoints.
