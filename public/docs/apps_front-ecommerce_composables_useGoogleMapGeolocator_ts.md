# useGoogleMapGeolocator.ts

**Path**: `apps\front-ecommerce\composables\useGoogleMapGeolocator.ts`

## Summary
This composable provides Google Maps-based geolocation and address autocomplete functionality for an e-commerce application. It manages geocoding (converting addresses to coordinates and vice versa), browser-based geolocation to get user's current position, and integrates with Google Places Autocomplete for address search input. The composable handles the bidirectional conversion between human-readable addresses and geographic coordinates, essential for delivery address selection and validation in the e-commerce checkout flow.
