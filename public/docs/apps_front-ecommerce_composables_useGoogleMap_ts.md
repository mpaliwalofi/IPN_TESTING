# useGoogleMap.ts

**Path**: `apps\front-ecommerce\composables\useGoogleMap.ts`

## Summary
This TypeScript composable provides a centralized integration layer for Google Maps API functionality within an e-commerce front-end application. It manages the initialization and lifecycle of various Google Maps libraries (Maps, Places, Geocoding, Geometry, etc.) and exposes a structured API for implementing location-based features like address autocomplete, interactive maps with markers, and place listings. The composable serves as an abstraction layer that coordinates multiple related components (Geolocator, GoogleMapContainer, GoogleMapListing) to enable store locator or location-based shopping features in the e-commerce application.
