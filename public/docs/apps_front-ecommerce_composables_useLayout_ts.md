# useLayout.ts

**Path**: `apps\front-ecommerce\composables\useLayout.ts`

## Summary
This composable manages layout-related functionality for the e-commerce front-end, specifically handling body CSS class attributes based on user authentication state. It initializes pages with an 'is-anonymous' body class by default (for both SSR and client-side rendering) and provides access to a custom router for handling navigation during layout transitions.
