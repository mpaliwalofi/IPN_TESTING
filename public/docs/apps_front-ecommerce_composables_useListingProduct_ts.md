# useListingProduct.ts

**Path**: `apps\front-ecommerce\composables\useListingProduct.ts`

## Summary
This composable manages product listing functionality for an e-commerce application, including filtering, sorting, and relevancy scoring of products. It filters products based on viewability status and user-selected filters, then calculates relevancy scores by combining product order scores with external recommendations and filter-based recommendations. The composable supports multiple sorting strategies (relevancy, best sellers, first sell date) to display products in category listings.
