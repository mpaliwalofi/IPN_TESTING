# usePickupPoint.ts

**Path**: `apps\front-ecommerce\composables\usePickupPoint.ts`

## Summary
This composable manages pickup point functionality for e-commerce shipping, providing computed properties and utilities for displaying and selecting shipping methods associated with physical pickup locations. It formats pickup point opening hours for display, filters and sorts available shipping methods for the pickup point, and maintains a two-way binding proxy for the selected shipping method that emits updates to parent components.
