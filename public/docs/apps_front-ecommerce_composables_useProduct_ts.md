# useProduct.ts

**Path**: `apps\front-ecommerce\composables\useProduct.ts`

## Summary
This composable provides product data retrieval functionality for an e-commerce front-end built on the Sylius platform. It exposes a `getSyliusProduct` function that fetches detailed product information from the backend API using a product SKU and a configured channel code, returning the data as a typed `ISyliusProduct` object.
