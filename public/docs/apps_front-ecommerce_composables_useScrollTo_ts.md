# useScrollTo.ts

**Path**: `apps\front-ecommerce\composables\useScrollTo.ts`

## Summary
This composable provides a comprehensive set of utility functions for programmatic scrolling within an e-commerce front-end application. It offers multiple scrolling methods including scrolling to elements by ID or class, scrolling to top/bottom of the page, scrolling with custom margins/offsets, and handling hash-based navigation, all with configurable smooth scroll behavior and SSR-safe window checks.
