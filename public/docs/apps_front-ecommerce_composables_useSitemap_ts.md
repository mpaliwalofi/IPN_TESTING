# useSitemap.ts

**Path**: `apps\front-ecommerce\composables\useSitemap.ts`

## Summary
This composable generates sitemap entries for an e-commerce front-end application by fetching content from a Strapi CMS. It retrieves localized data for various page types (samples, categories, products, shops, composite pages) and transforms them into sitemap URL objects with proper locale handling and routing, supporting both core pages (home, contact, FAQ, shop) and dynamic content-driven pages.
