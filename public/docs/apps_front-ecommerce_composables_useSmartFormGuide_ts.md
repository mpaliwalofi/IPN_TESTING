# useSmartFormGuide.ts

**Path**: `apps\front-ecommerce\composables\useSmartFormGuide.ts`

## Summary
This composable manages a smart form for collecting pet/animal information in an e-commerce context, including breed selection, gender, type (cat/dog), and birth date details. It provides reactive computed properties that filter breed options based on selected animal type, and handles the calculation and updating of pet anniversary/birthday data. The code serves as a bridge between the SmartForm component and pet-specific business logic for the e-commerce application.
