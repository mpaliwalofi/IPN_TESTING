# useUserGuide.ts

**Path**: `apps\front-ecommerce\composables\useUserGuide.ts`

## Summary
This composable manages a multi-step user guide/wizard for pet profiling in an e-commerce application. It fetches and caches various animal-related options (types, breeds, genders, activities, conditions, feeding preferences, etc.) from the backend API, then provides functionality to navigate through the guided form steps and submit animal profile data to create or update customer pet records.
