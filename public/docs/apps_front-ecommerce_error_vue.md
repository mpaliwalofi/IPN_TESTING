# error.vue

**Path**: `apps\front-ecommerce\error.vue`

## Summary
This is a Nuxt.js error page component for an e-commerce application that displays user-friendly error messages when page errors occur. It dynamically renders error titles and messages based on the error status code (or falls back to generic error text), loads cross-sell content to keep users engaged, and tracks error page views for analytics. The component also handles clearing the error state before route navigation and sets up page context with appropriate metadata.
