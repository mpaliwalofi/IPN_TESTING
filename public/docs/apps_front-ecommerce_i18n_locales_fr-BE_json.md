# fr-BE.json

**Path**: `apps\front-ecommerce\i18n\locales\fr-BE.json`

## Summary
This JSON file contains French Belgian (fr-BE) localization configuration for an e-commerce front-end application, specifically for Ultra Premium Direct, a pet food company. It defines locale codes for various services (Sylius, PayZen, checkout, tracking), company contact information for the Belgian market (address, phone, email, business hours), and SEO robot directives to control search engine indexing of account pages, cart pages, and pet food guides in both French and Dutch languages.
