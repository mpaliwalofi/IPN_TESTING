# fr-FR.json

**Path**: `apps\front-ecommerce\i18n\locales\fr-FR.json`

## Summary
This is a French (France) localization file for an e-commerce front-end application, specifically for Ultra Premium Direct, a pet food company. It contains French translations and locale-specific configuration including payment methods (bank transfer, check, PayPal), company contact information (address in Agen, France, phone numbers, emails), and order confirmation messages with instructions for delayed payment methods.
