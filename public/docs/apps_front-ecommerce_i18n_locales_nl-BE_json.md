# nl-BE.json

**Path**: `apps\front-ecommerce\i18n\locales\nl-BE.json`

## Summary
This is a Dutch (Belgium) localization file for an e-commerce front-end application, specifically for Ultra Premium Direct pet food products. It contains translated strings and locale-specific configuration for various services (payment gateways, checkout, tracking) along with company contact information, business hours, and payment method instructions (bank transfer, check, PayPal) tailored for the Belgian market.
