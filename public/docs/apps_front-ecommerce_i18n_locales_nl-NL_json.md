# nl-NL.json

**Path**: `apps\front-ecommerce\i18n\locales\nl-NL.json`

## Summary
This JSON file contains Dutch (Netherlands) localization strings for an e-commerce front-end application, specifically for Ultra Premium Direct, a pet food company. It provides translated text for various UI components including checkout processes, payment methods (bank transfer, check, PayPal), contact information, business hours, and order confirmation messages tailored for the Belgian/Dutch market.
