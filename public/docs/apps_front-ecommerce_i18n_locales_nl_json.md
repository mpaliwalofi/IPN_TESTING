# nl.json

**Path**: `apps\front-ecommerce\i18n\locales\nl.json`

## Summary
This JSON file contains Dutch (nl) language route translations for an e-commerce front-end application, specifically for a pet food/supplies business. It maps application route paths to their Dutch URL equivalents, covering key features like shopping cart ("winkelmand"), user account management ("mijn-account"), pet profiles ("mijn-huisdieren"), product catalog ("producten"), subscriptions ("abonnementen"), and a product guide ("brokkengids"). The file enables localized, SEO-friendly URLs for the Dutch version of the e-commerce platform.
