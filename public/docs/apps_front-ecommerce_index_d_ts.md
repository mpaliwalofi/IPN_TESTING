# index.d.ts

**Path**: `apps\front-ecommerce\index.d.ts`

## Summary
This TypeScript declaration file extends Nuxt 3's `#app` module to augment the `PageMeta` interface with a custom `layoutProperties` property. It allows pages in the e-commerce application to pass dynamic configuration data to their layouts through the type-safe `layoutProperties` object. This is a common pattern in Nuxt applications for customizing layout behavior on a per-page basis.
