# user-guide.vue

**Path**: `apps\front-ecommerce\layouts\user-guide.vue`

## Summary
This Vue layout component manages the user guide interface for a pet e-commerce application, specifically handling the pet nutrition questionnaire flow. It orchestrates the display logic for different menu layouts (default, bi-nutrition, edit modes) based on the user's progress through a multi-step form, tracking whether users are completing a standard user guide or a bi-nutrition assessment. The component integrates with customer store data and dynamically adjusts the UI background and stepper visibility depending on which step and nutrition flow variant the user is in.
