# [landing].vue

**Path**: `apps\front-ecommerce\pages\[landing].vue`

## Summary
This is a dynamic landing page component for an e-commerce application that renders customizable B2C landing pages based on URL parameters. It fetches page content from a CMS system using the route parameter `[landing]`, sets up tracking for page views and navigation events, and displays a header with logo and optional CTA button along with the page content. The page supports internationalization, uses a custom layout system, and integrates analytics tracking for user interactions with menu navigation.
