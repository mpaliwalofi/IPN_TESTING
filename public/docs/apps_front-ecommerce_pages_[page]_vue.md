# [page].vue

**Path**: `apps\front-ecommerce\pages\[page].vue`

## Summary
This is a dynamic page component for rendering CMS-driven static pages in an e-commerce application. It fetches page content based on the URL parameter, handles SEO metadata and breadcrumb navigation, and uses a composite renderer to dynamically display page sections from the CMS. The component also tracks page views for analytics purposes when the page mounts.
