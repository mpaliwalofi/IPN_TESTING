# addresses.vue

**Path**: `apps\front-ecommerce\pages\account\addresses.vue`

## Summary
This Vue page component displays and manages a customer's saved addresses in the e-commerce front-end application. It fetches the customer's address list from the store on load, displays either an addresses listing or a no-addresses message based on availability, and includes CMS-driven content headers and page metadata. The page is protected by authentication middleware and includes analytics tracking for the account addresses view.
