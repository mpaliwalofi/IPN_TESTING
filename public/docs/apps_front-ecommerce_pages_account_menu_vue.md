# menu.vue

**Path**: `apps\front-ecommerce\pages\account\menu.vue`

## Summary
This Vue component serves as the main account dashboard page for authenticated e-commerce customers. It displays a personalized welcome message with the customer's account information, fetches CMS-based composite page content for the account section, and provides a logout button that redirects users to the home page upon signing out. The page includes tracking analytics integration and enforces authentication via middleware.
