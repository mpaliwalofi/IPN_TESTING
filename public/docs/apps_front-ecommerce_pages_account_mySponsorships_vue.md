# mySponsorships.vue

**Path**: `apps\front-ecommerce\pages\account\mySponsorships.vue`

## Summary
This Vue page component manages the "My Sponsorships" section of a user's account in an e-commerce application. It fetches and renders CMS-driven content for the sponsorships page while enforcing authentication and a "mention-me" feature flag through middleware guards. The page includes navigation back to the account menu, tracks page views for analytics, and dynamically renders its content using a composite page renderer pattern connected to a Strapi CMS backend.
