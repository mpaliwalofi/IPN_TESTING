**Path**: `apps\front-ecommerce\pages\account\orders\[order].vue`

## Summary
This Vue page displays a detailed view of a specific customer order in an e-commerce application. It fetches order data based on the order ID from the URL route parameter, renders the order information using a composite page system from a CMS, and includes analytics tracking for the account section. The page is protected by authentication middleware and provides navigation back to the orders list page.
