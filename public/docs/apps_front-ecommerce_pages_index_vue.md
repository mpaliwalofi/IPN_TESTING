# index.vue

**Path**: `apps\front-ecommerce\pages\index.vue`

## Summary
This is the homepage route component for an e-commerce storefront that dynamically renders CMS-managed content. It fetches homepage configuration and metadata from a CMS context, applies middleware to identify returning customers, and uses a CompositeRenderer to display modular page components based on the CMS data structure. The component also tracks page views when mounted for analytics purposes.
