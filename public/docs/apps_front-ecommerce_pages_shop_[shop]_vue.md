# [shop].vue

**Path**: `apps\front-ecommerce\pages\shop\[shop].vue`

## Summary
This is a dynamic shop page component for an e-commerce application that renders individual shop pages based on a URL slug parameter. It fetches shop-specific content and metadata from a CMS, displays breadcrumb navigation and the shop title, then uses a CompositeRenderer to dynamically build the page layout from CMS-defined components. The page also initializes analytics tracking and preloads a Booxi booking widget for appointment scheduling functionality.
