# guide-[animal].vue

**Path**: `apps\front-ecommerce\pages\userGuide\guide-[animal].vue`

## Summary
This Vue page component implements a user guide form for selecting or specifying an animal type in an e-commerce application. It handles routing with dynamic animal codes from URL parameters, manages form state through the `useUserGuide` composable (loading states, errors, results navigation), and tracks page view events for analytics purposes. The page uses a specialized 'user-guide' layout with results page functionality and includes middleware protection via 'guide-entrance-feature-guard'.
