# [animal].vue

**Path**: `apps\front-ecommerce\pages\userGuide\guide-nutrition-full\[animal].vue`

## Summary
This is a Nuxt.js page component for a full bi-nutrition user guide for animals in an e-commerce application. It handles the display of an animal selection form with tracking analytics, manages the user guide workflow state (loading, errors, navigation to results), and watches for page view events to trigger analytics tracking when users access the bi-nutrition guide for a specific animal type passed via route parameters.
