# index.vue

**Path**: `apps\front-ecommerce\pages\userGuide\index.vue`

## Summary
This Vue page serves as the landing page for the user guide feature in an e-commerce application, specifically designed for logged-in users who have pets/animals registered in the system. It displays a welcome screen with headings and a call-to-action button that directs users to begin the guided experience for their first animal, while also implementing page tracking, enforcing authentication/animal existence middleware, and loading page context for the user guide flow.
