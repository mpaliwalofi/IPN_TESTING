# demo.vue

**Path**: `apps\front-ecommerce\pages\webView\demo.vue`

## Summary
This Vue component serves as a debug/demonstration page for displaying browser and user authentication information in a web view context. It renders diagnostic data including the browser's navigator properties (user agent, standalone mode, Safari/iOS detection) and customer authentication state (login status, token, username, full name) from the customer store. The page is specifically designed to help developers verify WebView integration and authentication state in the front-ecommerce application.
