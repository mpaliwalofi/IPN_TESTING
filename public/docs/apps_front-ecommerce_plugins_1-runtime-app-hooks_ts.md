# 1-runtime-app-hooks.ts

**Path**: `apps\front-ecommerce\plugins\1-runtime-app-hooks.ts`

## Summary
This Nuxt plugin creates a type-safe, centralized hook system for runtime application events in an e-commerce application. It defines hooks for tracking analytics events, Google identity integration, exit intent behavior, modal displays, and marketing header visibility, enabling different parts of the application to communicate through a pub-sub pattern without tight coupling.

- `RuntimeAppHooks`
