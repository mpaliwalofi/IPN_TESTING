# contactFormSupportedFiles.ts

**Path**: `apps\front-ecommerce\static\contactFormSupportedFiles.ts`

## Summary
This file exports a whitelist array of MIME types that are allowed for file uploads in the e-commerce application's contact form. It supports a comprehensive range of file formats including documents (PDF, Word, Excel), images (JPEG, PNG, GIF), audio/video files (MP3, MP4, MOV), and archives (ZIP, TAR), enabling users to attach various file types when contacting support or submitting inquiries.
