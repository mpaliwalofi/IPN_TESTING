# metadata.custom.json

**Path**: `apps\front-ecommerce\static\metadata.custom.json`

## Summary
This JSON file contains international phone number formatting and validation metadata for multiple European countries (Belgium, Switzerland, Germany, France, Italy, Luxembourg, Monaco, and Namibia). It stores country calling codes, regular expressions for validating phone number patterns, and formatting rules that define how to display phone numbers with proper spacing and prefixes for each country's different number types (landline, mobile, toll-free, premium rate, etc.).
