# userGuideMenuLayout.ts

**Path**: `apps\front-ecommerce\static\userGuideMenuLayout.ts`

## Summary
This TypeScript configuration file defines layout constants for a user guide menu system in an e-commerce application, specifically related to nutrition information workflows. It specifies five different menu layout types (default, edit, and three variations of nutrition-related guides) along with the number of steps associated with each layout (ranging from 4-5 steps), providing a centralized configuration for multi-step user guidance flows.

- `TUserGuideMenuLayout`
