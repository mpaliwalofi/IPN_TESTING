# upd-spa-loading-template.html

**Path**: `apps\front-ecommerce\template\upd-spa-loading-template.html`

## Summary
This HTML template file defines an animated SVG loading spinner for a front-end e-commerce application. The SVG contains complex path animations with leaf-like shapes that morph and transform using keyframe animations, creating a visual loading indicator while the SPA (Single Page Application) content loads. The animation uses SVG `&lt;animate&gt;` elements to smoothly transition between different path coordinates over a 0.5-second duration.
