## Summary
This file exports a mock data object for a B2C e-commerce account navigation menu component. It provides test data containing French-language account dashboard links (orders, subscriptions, addresses, personal information, pets, and referrals) with associated icons and URLs, used for testing the account links interface without requiring real backend data.
