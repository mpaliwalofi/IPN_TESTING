# addressesMock.ts

**Path**: `apps\front-ecommerce\test\mocks\addressesMock.ts`

## Summary
This file exports mock address data for testing purposes in an e-commerce front-end application. It provides two sample French addresses with complete Address object structures including contact information, location details, and flags for default address selection and Mondial Relay (a shipping service) integration, allowing developers to test address-related features without making actual API calls.
