# calendlyMock.ts

**Path**: `apps\front-ecommerce\test\mocks\calendlyMock.ts`

## Summary
This file provides mock data for Calendly widget integration testing in an e-commerce front-end application. It exports two test fixtures: `calendlyData` containing widget configuration (colors, dimensions, site ID) and `calendlyMetaScrip` containing the external Calendly script tag definition for embedding the scheduling widget into pages.
