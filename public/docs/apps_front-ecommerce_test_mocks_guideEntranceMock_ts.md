**Path**: `apps\front-ecommerce\test\mocks\guideEntranceMock.ts`

## Summary
This file exports a mock data object for a "guide entrance" component in a B2C e-commerce application, specifically for helping pet owners select appropriate food for their animals. The mock contains UI configuration data including titles, text, images, buttons, and display settings for both desktop and mobile versions of a guided selection interface that leads users to a pet food recommendation guide.
