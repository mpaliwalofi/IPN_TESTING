# linksSubscriptionMock.ts

**Path**: `apps\front-ecommerce\test\mocks\linksSubscriptionMock.ts`

## Summary
This file provides mock data for testing a subscription links component in a French e-commerce application for pet products. It exports two test scenarios: one with a default title ("Choisissez vos produits") and another without a title, both containing button configurations for dog and cat subscription pages with different styling options (solid vs outline, tertiary color scheme).
