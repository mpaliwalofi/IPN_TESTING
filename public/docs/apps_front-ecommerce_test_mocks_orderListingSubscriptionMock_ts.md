## Summary
This file exports a mock object representing a subscription order listing in an e-commerce system. It contains detailed test data for a subscription with draft orders, including customer addresses (both shipping and billing, with support for Mondial Relay pickup points), order items, payment states, and localization information for the French market. The mock is used for testing subscription-related features in the front-end e-commerce application.
