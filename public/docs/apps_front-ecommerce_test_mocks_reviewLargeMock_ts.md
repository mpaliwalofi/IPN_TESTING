# reviewLargeMock.ts

**Path**: `apps\front-ecommerce\test\mocks\reviewLargeMock.ts`

## Summary
This file provides mock data for testing a large review display component in an e-commerce front-end application. It contains sample customer reviews (in French) with ratings, comments, and dates, along with aggregate statistics like total reviews (60,000), satisfaction percentage (96.3%), and overall rating (4.7), designed to simulate the display of product/service reviews with a call-to-action button linking to a third-party review verification service (Avis Vérifiés).
