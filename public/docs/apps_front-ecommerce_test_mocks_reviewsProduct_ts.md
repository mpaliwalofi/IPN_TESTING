# reviewsProduct.ts

**Path**: `apps\front-ecommerce\test\mocks\reviewsProduct.ts`

## Summary
This TypeScript file exports mock data for product reviews in an e-commerce application, specifically containing French-language reviews for cat food products. The mock includes aggregated review statistics (average rating of 4.67 and 9,275 total reviews) along with an array of individual customer reviews, each containing the reviewer's name, rating, review text, and timestamp, used for testing the product review display functionality.
