# syliusProduct.ts

**Path**: `apps\front-ecommerce\test\mocks\syliusProduct.ts`

## Summary
This file exports a mock object representing a Sylius e-commerce product entity for testing purposes. The mock data includes product metadata (ID, code, context), taxonomic classifications (productTaxons), product variants, reviews, and images, structured according to the Sylius API v2 shop endpoint format. It serves as test fixture data to simulate API responses when testing front-end e-commerce functionality without requiring actual backend calls.
