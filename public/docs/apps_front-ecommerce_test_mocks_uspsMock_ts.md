# uspsMock.ts

**Path**: `apps\front-ecommerce\test\mocks\uspsMock.ts`

## Summary
This file contains a mock data object representing USP (Unique Selling Points) content from a Strapi CMS for an e-commerce pet food website (Ultra Premium Direct). The mock data structures a component with marketing content including a title, button CTAs, disclaimer text, and an array of USP items with icons highlighting product benefits like "high-quality ingredients" and "complete &amp; balanced recipes" - used for testing purposes in the front-end application.
