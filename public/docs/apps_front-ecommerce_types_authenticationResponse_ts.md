# authenticationResponse.ts

**Path**: `apps\front-ecommerce\types\authenticationResponse.ts`

## Summary
This TypeScript file defines the structure of an authentication response for an e-commerce application. It specifies that after successful authentication, the system returns JWT tokens (access and refresh), user personal data, and optional references to the user's cart and customer records. This interface serves as a contract for handling login/authentication API responses throughout the front-end application.

## Interfaces
- `IAuthenticationResponse`
