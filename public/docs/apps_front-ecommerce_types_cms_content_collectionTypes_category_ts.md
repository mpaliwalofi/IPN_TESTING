# category.ts

**Path**: `apps\front-ecommerce\types\cms\content\collectionTypes\category.ts`

## Summary
This TypeScript file defines the `ICategory` interface, which represents the data structure for an e-commerce category page in a Strapi CMS-based system. It extends `IStrapiCompositePage` and includes category-specific properties such as navigation data (slug, breadcrumb), visibility settings, SEO metadata with rating/pricing aggregates, guide entrance configurations for mobile/desktop, product recommendations, and a list of products to display on the category page.
