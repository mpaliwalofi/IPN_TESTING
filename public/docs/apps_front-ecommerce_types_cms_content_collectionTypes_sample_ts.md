# sample.ts

**Path**: `apps\front-ecommerce\types\cms\content\collectionTypes\sample.ts`

## Summary
This TypeScript file defines interfaces for a sample product content type in a Strapi CMS-based e-commerce application. It structures data for sample products including images, categorized lists of sample content, related product pushes, and cross-sell links, extending the base Strapi composite page type to enable presentation of product samples with associated marketing content.

## Interfaces
- `ISampleListContent`
- `ISampleList`
- `IStrapiSample`
