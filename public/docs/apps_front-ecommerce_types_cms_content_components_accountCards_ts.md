# accountCards.ts

**Path**: `apps\front-ecommerce\types\cms\content\components\accountCards.ts`

## Summary
This TypeScript type definition file defines the structure for account-related card components in an e-commerce CMS system. It extends a Strapi CMS component interface to specify which account cards (order card and subscription card) should be displayed, using boolean flags to control their visibility in the front-end application.

## Interfaces
- `IAccountCards`
