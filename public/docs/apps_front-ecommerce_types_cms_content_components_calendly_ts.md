# calendly.ts

**Path**: `apps\front-ecommerce\types\cms\content\components\calendly.ts`

## Summary
This TypeScript file defines a type interface for integrating Calendly (a scheduling/calendar service) into the e-commerce CMS. The `ICalendly` interface specifies the configuration properties needed to embed and customize a Calendly widget, including site identification, styling options (colors), and dimensional constraints (width/height).
