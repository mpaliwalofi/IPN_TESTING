# carouselGeneric.ts

**Path**: `apps\front-ecommerce\types\cms\content\components\carouselGeneric.ts`

## Summary
This TypeScript file defines the type interfaces for a generic carousel component in an e-commerce CMS system. It specifies the structure for carousel items (including responsive titles, text, images, and action buttons) and the carousel container configuration (interval timing, number of items per slide for desktop/mobile views). The types are designed to work with Strapi CMS data structures, enabling consistent rendering of product or content carousels across the front-end application.

## Interfaces
- `ICarouselGenericItem`
- `ICarouselGeneric`
