# carouselShorts.ts

**Path**: `apps\front-ecommerce\types\cms\content\components\carouselShorts.ts`

## Summary
This TypeScript type definition file defines the structure for a carousel component that displays short-form video content (similar to YouTube Shorts or TikTok) in an e-commerce CMS. It extends the base Strapi component interface and specifies that the carousel contains an optional title and a collection of embeds (likely video embeds), enabling the front-end to render a scrollable shorts video carousel feature.

## Interfaces
- `ICarouselShorts`
