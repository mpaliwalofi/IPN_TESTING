# categoryGrid.ts

**Path**: `apps\front-ecommerce\types\cms\content\components\categoryGrid.ts`

## Summary
This TypeScript file defines type interfaces for a CMS-managed category grid component in an e-commerce front-end application. It structures data for displaying a grid of product categories, where each category card contains an image, name, price, and URL, and the overall grid component includes multiple category cards plus a main image, all integrated with Strapi CMS types.

## Interfaces
- `ICategoryCard`
- `ICategoryGrid`
