# crossSellLink.ts

**Path**: `apps\front-ecommerce\types\cms\content\components\crossSellLink.ts`

## Summary
This TypeScript definition file defines the `ICrossSellLink` interface for representing cross-sell link components in an e-commerce CMS (Strapi). It extends the base Strapi component structure to include cross-sell product references along with standard CMS metadata (timestamps, locale, publishing status, and slug) for managing product cross-selling relationships in the front-end e-commerce application.
