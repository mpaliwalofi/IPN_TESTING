# ctaListing.ts

**Path**: `apps\front-ecommerce\types\cms\content\components\ctaListing.ts`

## Summary
This TypeScript type definition file defines the structure for a CTA (Call-to-Action) listing component used in a Strapi-based CMS for an e-commerce front-end application. The `ICTAListing` interface extends a base Strapi component and specifies that a CTA listing must have a title and can optionally include an icon and multiple button elements, enabling the creation of promotional or action-oriented content blocks in the storefront.

## Interfaces
- `ICTAListing`
