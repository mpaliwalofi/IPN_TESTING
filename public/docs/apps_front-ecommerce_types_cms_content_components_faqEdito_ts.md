## Summary
This TypeScript file defines the `IFaqEdito` interface, which represents a FAQ editorial component in a Strapi-based e-commerce CMS. It structures a rich FAQ section that combines visual elements (image with overlay), a list of questions, and layout configuration options (text positioning and mobile image ordering) for flexible content presentation on the storefront.
