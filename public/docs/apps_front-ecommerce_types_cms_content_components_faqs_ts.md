# faqs.ts

**Path**: `apps\front-ecommerce\types\cms\content\components\faqs.ts`

## Summary
This TypeScript file defines type definitions for a FAQ (Frequently Asked Questions) component used in a Strapi-based e-commerce CMS. It structures FAQ data including questions with responses, categorized subjects with optional icons, abort reasons for user interactions, and configurable heading levels for accessibility and SEO purposes.

## Interfaces
- `IQuestion`
- `ISubject`
- `IFaqs`

## Type Aliases
- `TFAQQuestions`
- `TFAQSubjects`
- `TAbortReason`
