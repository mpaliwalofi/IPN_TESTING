# glossary.ts

**Path**: `apps\front-ecommerce\types\cms\content\components\glossary.ts`

## Summary
This TypeScript file defines the type interface for a Glossary content component in a Strapi-based e-commerce CMS. It specifies the structure of glossary entries with required title and subtitle fields, plus optional USP (Unique Selling Proposition) content and configurable heading levels for display hierarchy. This type enables type-safe handling of glossary components throughout the front-end e-commerce application when fetching and rendering content from the CMS.

## Interfaces
