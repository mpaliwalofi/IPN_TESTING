# guideEntrance.ts

**Path**: `apps\front-ecommerce\types\cms\content\components\guideEntrance.ts`

## Summary
This TypeScript file defines type interfaces for guide entrance components in a Strapi-based e-commerce CMS. It specifies the structure for two types of entrance components: a basic `IGuideEntrance` with desktop/mobile images and overlay support, and an extended `ICalendlyEntrance` that adds Calendly scheduling integration via a URL property. These types ensure type safety when rendering guide or scheduling entry points in the front-end e-commerce application.

## Interfaces
- `IGuideEntrance`
- `ICalendlyEntrance`
