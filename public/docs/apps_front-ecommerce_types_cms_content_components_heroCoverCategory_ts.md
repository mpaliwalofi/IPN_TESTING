# heroCoverCategory.ts

**Path**: `apps\front-ecommerce\types\cms\content\components\heroCoverCategory.ts`

## Summary
This TypeScript type definition file defines the structure for a hero cover component specifically designed for category pages in an e-commerce CMS. The `IHeroCoverCategory` interface extends a base Strapi component and specifies that category hero sections must contain a title string and responsive text content, ensuring consistent data structure for prominent category page banners across different screen sizes.
