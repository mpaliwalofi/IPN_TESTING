# heroCoverSubscription.ts

**Path**: `apps\front-ecommerce\types\cms\content\components\heroCoverSubscription.ts`

## Summary
This TypeScript file defines a type interface for a hero cover subscription component in a Strapi-based e-commerce CMS. It structures the data model for a promotional hero section that includes configurable elements like title, SEO text fields, multiple button configurations (top and bottom), heading levels, and a USP (Unique Selling Proposition) list to highlight subscription benefits or features.
