# heroSlider.ts

**Path**: `apps\front-ecommerce\types\cms\content\components\heroSlider.ts`

## Summary
This TypeScript definition file defines the data structure for a hero slider component used in a Strapi-based e-commerce CMS. It specifies two interfaces: `IHeroSlide` (individual slides with content like images, titles, buttons, and mobile variants) and `IHeroSlider` (the slider component with timing configuration and slide collection), enabling type-safe handling of carousel/banner content on the storefront.

## Interfaces
- `IHeroSlide`
- `IHeroSlider`
