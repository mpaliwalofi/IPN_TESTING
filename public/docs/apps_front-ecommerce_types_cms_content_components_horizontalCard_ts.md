## Summary
This TypeScript type definition file defines the structure for a horizontal card component used in a CMS-driven e-commerce front-end. The `IHorizontalCard` interface specifies a reusable card layout containing an image, title, subtitle, and call-to-action button, following Strapi CMS data structures for content management and display purposes.
