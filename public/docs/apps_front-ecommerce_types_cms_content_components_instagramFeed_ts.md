# instagramFeed.ts

**Path**: `apps\front-ecommerce\types\cms\content\components\instagramFeed.ts`

## Summary
This TypeScript file defines type interfaces for an Instagram feed component used in a Strapi-based e-commerce CMS. It structures the data model for displaying Instagram content, where an `IInstagramFeed` component contains an array of `IInstagramItem` objects, each holding an ID and a Strapi-managed image entity. The types ensure type safety when integrating Instagram feed displays into the front-end e-commerce application.

## Interfaces
- `IInstagramItem`
- `IInstagramFeed`
