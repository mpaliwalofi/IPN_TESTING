# itemBlogGrid.ts

**Path**: `apps\front-ecommerce\types\cms\content\components\itemBlogGrid.ts`

## Summary
This TypeScript file defines type interfaces for a blog grid component in a Strapi-based e-commerce CMS. It structures the data model for displaying multiple blog articles in a grid layout, where each blog item includes an article (with image, slug, title, teaser), its associated category, and a teaser text, all organized under a titled grid component that extends the base Strapi component interface.

## Interfaces
- `IBlogArticle`
- `IBlogCategory`
- `IItemBlog`
- `IItemBlogGrid`
