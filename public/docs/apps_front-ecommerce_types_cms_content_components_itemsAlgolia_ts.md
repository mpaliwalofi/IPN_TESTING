# itemsAlgolia.ts

**Path**: `apps\front-ecommerce\types\cms\content\components\itemsAlgolia.ts`

## Summary
This TypeScript file defines type interfaces for Algolia search index items in an e-commerce CMS. It specifies the data structure for two searchable content types: product categories (`IItemCategoryAlgolia`) with SKU, slug, and search metadata, and blog posts (`IItemBlogAlgolia`) with article details like title, teaser, reading time, images, and category information, enabling structured search functionality across the e-commerce platform's content.

## Interfaces
- `IItemCategoryAlgolia`
- `IItemBlogAlgolia`
