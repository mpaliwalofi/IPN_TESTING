# jumbotron.ts

**Path**: `apps\front-ecommerce\types\cms\content\components\jumbotron.ts`

## Summary
This TypeScript type definition file defines the structure for a Jumbotron CMS component used in an e-commerce front-end application. It extends the base Strapi component interface to specify a responsive hero/banner section with optional desktop and mobile images plus an overlay effect. This type ensures type safety when handling jumbotron content retrieved from the Strapi headless CMS system.

## Interfaces
- `IJumbotron`
