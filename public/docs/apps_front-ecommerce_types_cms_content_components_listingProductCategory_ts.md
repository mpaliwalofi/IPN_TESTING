# listingProductCategory.ts

**Path**: `apps\front-ecommerce\types\cms\content\components\listingProductCategory.ts`

## Summary
This TypeScript type definition file defines the structure for a product category listing component in an e-commerce CMS. It specifies two interfaces: `IGroupFilter` (representing a filter group with title, icon, and associated filters) and `IListingProductCategory` (a Strapi component that manages subscription options and collections of filter groups for organizing and filtering products in category listings).

## Interfaces
- `IGroupFilter`
- `IListingProductCategory`
