# mentionMe.ts

**Path**: `apps\front-ecommerce\types\cms\content\components\mentionMe.ts`

## Summary
This TypeScript definition file specifies the data structure for a MentionMe referral program integration component in the e-commerce CMS. It defines a single interface `IMentionMe` that controls which UI model variant to display—either a 'landing' page version or a 'dashboard' version—allowing the CMS to configure different presentations of the referral program in different contexts.
