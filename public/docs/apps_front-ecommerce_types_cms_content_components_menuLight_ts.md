# menuLight.ts

**Path**: `apps\front-ecommerce\types\cms\content\components\menuLight.ts`

## Summary
This TypeScript file defines type interfaces for a lightweight menu system in an e-commerce CMS, specifically handling both desktop and mobile navigation structures. It describes a hierarchical menu with configurable properties including visibility settings, button styling, sub-menus with links, and mobile-specific features like icons and push notifications. The types support various link behaviors (mail_to, nofollow, target_blank) and special menu modes like "all products" views for category navigation.

## Interfaces
- `ISubMenuLink`
- `ISubMenu`
- `IMenuLight`
