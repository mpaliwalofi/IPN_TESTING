# menu.ts

**Path**: `apps\front-ecommerce\types\cms\content\components\menu.ts`

## Summary
This TypeScript file defines type interfaces for a multi-level e-commerce navigation menu system in a CMS. It structures hierarchical menu data including top-level menu items, expandable panels with images, and nested panel configurations, while supporting features like user visibility controls (anonymous, logged-in, subscriber status), mobile-specific assets, and button styling options for the front-end e-commerce application.
