**Path**: `apps\front-ecommerce\types\cms\content\components\recommendations.ts`

## Summary
This TypeScript interface defines the structure for a pet food recommendation system in an e-commerce application. It specifies numerical weight or scoring fields for various pet characteristics (age, size, health conditions, activity level) and dietary preferences for both cats and dogs, which are likely used to calculate personalized product recommendations based on pet profiles and needs.
