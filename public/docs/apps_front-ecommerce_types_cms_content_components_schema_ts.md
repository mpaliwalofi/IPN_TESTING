# schema.ts

**Path**: `apps\front-ecommerce\types\cms\content\components\schema.ts`

## Summary
This TypeScript file defines type interfaces for structured schema data (likely Schema.org markup) for an e-commerce business. It models organization information including business details (contact, address, awards, founders), physical shop locations with opening hours, and product categories, primarily used for SEO and rich snippets in search results.

## Interfaces
- `IAdditionalInfo`
- `ICategory`
- `IFounder`
- `IOpeningHourSpecification`
- `IShop`
- `ISchema`
