# searchFaq.ts

**Path**: `apps\front-ecommerce\types\cms\content\components\searchFaq.ts`

## Summary
This TypeScript type definition file defines the structure for a search-enabled FAQ component in an e-commerce CMS. It specifies the `ISearchFaq` interface which combines FAQ excerpts (question/answer pairs), categorized subjects, and an optional hero cover image, all wrapped in Strapi CMS entity structures to enable FAQ search and display functionality.

## Interfaces
- `ISearchFaq`
