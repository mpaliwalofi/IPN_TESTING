# socialLink.ts

**Path**: `apps\front-ecommerce\types\cms\content\components\socialLink.ts`

## Summary
This TypeScript file defines a type interface for social media links in an e-commerce CMS system. The `ISocialLink` interface structures data for displaying social media connections, including properties for identification, display text, URLs, accessibility text, and icon references. It serves as a type contract for rendering social media links consistently across the front-end e-commerce application's CMS-managed content.

## Interfaces
- `ISocialLink`
