# strapiForm.ts

**Path**: `apps\front-ecommerce\types\cms\content\components\strapiForm.ts`

## Summary
This TypeScript definition file defines the `IStrapiForm` interface, which represents a configurable form component from the Strapi CMS for an e-commerce front-end application. The interface specifies the structure of form data including optional fields for image, form inputs, submit button text, titles, USPs (Unique Selling Points), layout options, thank-you message content, and campaign tracking information.
