# title.ts

**Path**: `apps\front-ecommerce\types\cms\content\components\title.ts`

## Summary
This TypeScript file defines a type interface for a CMS title component in an e-commerce application. It extends the base Strapi CMS component interface to add a `title` string property, allowing the front-end to consistently type and handle title content blocks retrieved from the Strapi headless CMS.

## Interfaces
- `ITitle`
