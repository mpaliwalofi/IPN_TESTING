# usp.ts

**Path**: `apps\front-ecommerce\types\cms\content\components\usp.ts`

## Summary
This TypeScript definition file specifies the data structure for a USP (Unique Selling Points) component in a Strapi-based e-commerce CMS. It defines three interfaces for displaying marketing/promotional cards with icons, images, and text that can be configured for different layouts (mobile/desktop columns, sizes, backgrounds) and includes optional CTAs and editorial content.

## Interfaces
- `IStrapiUSPCard`
- `IStrapiUSPLarge`
- `IStrapiUSP`
