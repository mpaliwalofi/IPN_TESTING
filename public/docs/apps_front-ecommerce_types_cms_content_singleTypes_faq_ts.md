# faq.ts

**Path**: `apps\front-ecommerce\types\cms\content\singleTypes\faq.ts`

## Summary
This TypeScript file defines type definitions for FAQ (Frequently Asked Questions) content pages managed through Strapi CMS in an e-commerce application. It extends the base Strapi composite page interface to include FAQ-specific properties such as index configuration, FAQ enablement flags, and B2C FAQ questions, allowing the front-end to properly type and render FAQ content retrieved from the CMS.

## Interfaces
- `IStrapiCompositePageFAQ`
- `IStrapiCompositePageFAQSubjects`
