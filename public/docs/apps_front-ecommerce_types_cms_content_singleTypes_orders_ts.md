# orders.ts

**Path**: `apps\front-ecommerce\types\cms\content\singleTypes\orders.ts`

## Summary
This TypeScript file defines a type interface for an e-commerce orders page within a Strapi CMS content structure. It extends the base `IStrapiCompositePage` interface with optional order-specific header fields: `orders_header` (for displaying when orders exist) and `no_orders_header` (for displaying when no orders are present). This type is used to provide type safety for CMS content related to the orders page UI in the front-end e-commerce application.

## Interfaces
- `IStrapiCompositePageOrders`
