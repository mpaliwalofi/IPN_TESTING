# shops.ts

**Path**: `apps\front-ecommerce\types\cms\content\singleTypes\shops.ts`

## Summary
This TypeScript file defines a type interface for a Strapi CMS page that displays multiple shops in an e-commerce application. It extends the base `IStrapiCompositePage` interface to include an optional `shops` property that contains an array of shop entities wrapped in Strapi's data structure format, enabling the front-end to render a page listing physical store locations or shop information.

## Interfaces
- `IStrapiCompositePageShops`
