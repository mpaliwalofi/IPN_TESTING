# staticPage.ts

**Path**: `apps\front-ecommerce\types\cms\content\staticPage.ts`

## Summary
This TypeScript file defines the `IStaticPage` interface for CMS-managed static pages in an e-commerce front-end application. It extends the `IStrapiCompositePage` interface from Strapi CMS and adds optional properties for controlling page title display (`title` and `display_title`), enabling the application to render static content pages with configurable title visibility.

## Interfaces
- `IStaticPage`
