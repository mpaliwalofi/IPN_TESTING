# recommendations.ts

**Path**: `apps\front-ecommerce\types\cms\recommendations.ts`

## Summary
This TypeScript file defines type interfaces for a pet food recommendation and ration calculation system in an e-commerce application. It structures data for product recommendations (including classification, scores, and ration calculations), recommendation lists, and detailed ration ratios that handle subscription-based purchases with wet/dry food binutrition support and frequency calculations.

## Interfaces
- `IRecommendationProduct`
- `IRecommendationsList`

## Type Aliases
- `TRecommendationRatio`
