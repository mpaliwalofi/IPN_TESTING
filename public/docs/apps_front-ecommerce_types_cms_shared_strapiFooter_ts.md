# strapiFooter.ts

**Path**: `apps\front-ecommerce\types\cms\shared\strapiFooter.ts`

## Summary
This TypeScript file defines the `IStrapiFooter` interface that structures footer content retrieved from a Strapi CMS for an e-commerce application. It specifies the data schema for footer elements including navigation link groups (information, services, brand), promotional banners for desktop/mobile, social media links, payment options, contact information, USP (Unique Selling Proposition) content, and partner information. This type definition ensures type safety when rendering and managing footer content throughout the front-end e-commerce application.

## Interfaces
- `IStrapiFooter`
