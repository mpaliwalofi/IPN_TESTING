# strapiHeader.ts

**Path**: `apps\front-ecommerce\types\cms\shared\strapiHeader.ts`

## Summary
This TypeScript type definition file defines the structure of a header component for a Strapi CMS-based e-commerce front-end. It specifies the data contract for header elements including promotional banners (desktop/mobile variants), contact information (phone number and availability), navigation menus (legacy and light versions), shop links, and an optional footer banner with a call-to-action button.

## Interfaces
- `IStrapiHeader`
