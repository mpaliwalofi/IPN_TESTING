# cartContent.ts

**Path**: `apps\front-ecommerce\types\components\cartContent.ts`

## Summary
This TypeScript file defines type definitions for a shopping cart content component in an e-commerce application. It specifies the structure of cart-related data including separate content sections for when items are present or absent in the cart, order summary information (header/footer text), and available payment options, while also providing composite types that combine cart content with Strapi CMS component metadata and page structure.
