# googleMap.ts

**Path**: `apps\front-ecommerce\types\components\googleMap.ts`

## Summary
This TypeScript definitions file provides type interfaces for Google Maps integration in an e-commerce application. It defines the structure for map markers (`IMarker`) that represent physical shop locations with coordinates and custom icons, and a geolocation interface (`IGeolocator`) for handling latitude/longitude data. The file exports a constant for a custom pin icon SVG used to display store locations on the map.

## Interfaces
- `IMarker`
- `IGeolocator`
