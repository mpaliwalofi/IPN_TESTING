# draftOrder.ts

**Path**: `apps\front-ecommerce\types\ecommerce\draftOrder.ts`

## Summary
This TypeScript definition file defines the structure of a draft order in an e-commerce system built on Sylius. It extends the base Sylius component interface to include order code, pricing totals (draft and shipping), and a reference to the full order object, allowing the system to track and manage orders before they are finalized.

## Interfaces
- `IDraftOrder`
