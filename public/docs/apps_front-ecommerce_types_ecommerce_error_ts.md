# error.ts

**Path**: `apps\front-ecommerce\types\ecommerce\error.ts`

## Summary
This TypeScript file defines error type interfaces for handling Sylius e-commerce platform API errors in the front-end application. It structures error responses with support for Hydra JSON-LD format (indicated by `'hydra:description'`) and includes a violations array for validation errors, enabling consistent error handling and display of API error messages, validation failures, and property-specific error details throughout the e-commerce application.

## Interfaces
- `IViolations`
- `IErrorSylius`
