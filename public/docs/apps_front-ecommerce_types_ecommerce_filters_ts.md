# filters.ts

**Path**: `apps\front-ecommerce\types\ecommerce\filters.ts`

## Summary
This file defines filtering and sorting configuration for an e-commerce product catalog. It exports a constant array of filter options (relevance, best sellers, new arrivals) with their corresponding labels and values, along with TypeScript types for type-safe filter handling and a Strapi CMS integration interface for managing filter data with localization support.

## Interfaces
- `IStrapiFilter`

## Type Aliases
- `TFilters`
- `TFilterValues`
