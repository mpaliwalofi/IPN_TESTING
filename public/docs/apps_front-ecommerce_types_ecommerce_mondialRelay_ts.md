# mondialRelay.ts

**Path**: `apps\front-ecommerce\types\ecommerce\mondialRelay.ts`

## Summary
This TypeScript file defines a type interface for Mondial Relay shipping information in an e-commerce application. The `IMondialRelayShipping` interface structures data required for managing Mondial Relay pickup point deliveries, including customer contact details, pickup location information (address, GPS coordinates, images), and the pickup point's operational schedule (opening hours and closings). This type ensures type safety when handling Mondial Relay shipping operations throughout the front-end e-commerce application.
