## Summary
This TypeScript file defines type interfaces for an e-commerce order system, specifically modeling Sylius e-commerce platform entities. It provides comprehensive type definitions for order-related data structures including promotions, shipments, order items, adjustments, and their associated properties like pricing, tracking, subscriptions, and product variants. The types support complex order workflows including checkout states, payment states, shipping states, and promotional adjustments for a front-end e-commerce application.
