# product.ts

**Path**: `apps\front-ecommerce\types\ecommerce\product.ts`

## Summary
This TypeScript file defines type definitions and interfaces for an e-commerce product system in a pet food/animal nutrition application. It structures product-related data including product variants with pricing (regular and subscription), composition information with nutritional constituents and metabolizable energy, and special product types like samples and push products. The types support features like stock management, SKU tracking, breadcrumbs, reviews, and promotional overlays, serving as the core data contracts for the product catalog and ordering functionality.
