# variant.ts

**Path**: `apps\front-ecommerce\types\ecommerce\shared\variant.ts`

## Summary
This TypeScript file defines the `IVariant` interface for e-commerce product variants in a Sylius-based system. It specifies the structure for variant data including product information, pricing (regular and discounted), date ranges for availability/promotions, translations, and discount configurations (amount/percentage). The interface serves as a type definition for managing product variations and their associated pricing rules throughout the front-end e-commerce application.
