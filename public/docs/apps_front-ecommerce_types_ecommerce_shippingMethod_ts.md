# shippingMethod.ts

**Path**: `apps\front-ecommerce\types\ecommerce\shippingMethod.ts`

## Summary
This TypeScript file defines type interfaces for e-commerce shipping methods in a Sylius-based system. It structures shipping-related data including transporters (with pricing, configuration, pickup point support, and categorization), shipping method categories, and pickup point filters used throughout the front-end e-commerce application.

## Interfaces
- `IShippingMethodCategory`
- `IShippingMethodTransporter`
- `IShippingMethodsPickupPointFilters`
- `IShippingMethod`
