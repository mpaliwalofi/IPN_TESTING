**Path**: `apps\front-ecommerce\types\ecommerce\shop.ts`

## Summary
This TypeScript definition file provides type definitions for an e-commerce shop management system. It defines interfaces for shop entities including their location data (address, coordinates), operational details (opening hours, special hours, phone), visual content (images), and relationships to other components in a Strapi CMS-based e-commerce application. The file also includes precise time formatting types and status indicators for displaying shop opening hours to users.
