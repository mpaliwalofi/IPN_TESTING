# subscription.ts

**Path**: `apps\front-ecommerce\types\ecommerce\subscription.ts`

## Summary
This TypeScript file defines the type definitions and interfaces for a subscription-based e-commerce system. It models the complete subscription lifecycle including subscription items, billing/shipping addresses, payment methods, shipping methods, and subscription states (active, stopped, etc.), along with related order management for recurring purchases. The types support features like subscription intervals, upsells, renewal dates, and tracking of subscription-related orders and their states.

## Interfaces
- `ISubscriptionShippingMethod`
- `ISubscriptionVariant`
- `ISubscriptionItem`
- `ISubscription`
- `ISubscriptionAlerts`
- `ISubscriptionOrders`
- `ISubscriptionsListing`

## Type Aliases
- `ISubscriptionOrder`
