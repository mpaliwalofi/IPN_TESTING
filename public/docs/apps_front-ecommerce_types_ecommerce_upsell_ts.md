# upsell.ts

**Path**: `apps\front-ecommerce\types\ecommerce\upsell.ts`

## Summary
This TypeScript definition file establishes the data structure for product upsell recommendations in an e-commerce system. It defines the `IUpsell` interface that extends a Sylius component (Sylius being an e-commerce framework) and includes properties for product identification, categorization (typology/classification), and available product variants that can be offered as upsells to customers during their shopping experience.

- `IUpsell`
