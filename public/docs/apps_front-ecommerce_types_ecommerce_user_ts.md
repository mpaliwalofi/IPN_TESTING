# user.ts

**Path**: `apps\front-ecommerce\types\ecommerce\user.ts`

## Summary
This TypeScript file defines the data structure for user personal information in an e-commerce application. It exports an `IUserPersonalData` interface that specifies user profile fields including contact details (email, name), demographics (gender, birthday), account status (verified, new customer), marketing preferences (newsletter subscription), and shopping history (default address, last order date). The file also provides a `defaultUserPersonalData` object with empty/undefined initial values for use when creating or initializing user profiles.

## Interfaces
- `IUserPersonalData`
