# abortSubscriptionRequestForm.ts

**Path**: `apps\front-ecommerce\types\form\abortSubscriptionRequestForm.ts`

## Summary
This class defines a form for handling subscription cancellation requests in an e-commerce application. It creates a two-field form that collects a reason for cancellation (from a predefined list of choices) and a required text message, validating the input against `AbortSubscriptionSchema` before submitting the data via a provided callback function.
