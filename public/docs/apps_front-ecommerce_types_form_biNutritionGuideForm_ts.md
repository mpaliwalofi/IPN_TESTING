# biNutritionGuideForm.ts

**Path**: `apps\front-ecommerce\types\form\biNutritionGuideForm.ts`

## Summary
This file defines form classes for a bi-nutrition guide questionnaire used to collect pet/animal information for nutritional recommendations. The `BiNutritionStep1Form` and `FullBiNutritionStep1Form` classes extend a smart form framework to create multi-step forms that gather data such as the animal's birthday/anniversary, weight, sterilization status, and physical condition, with the full version including additional fields like animal type and breed selection.

- `BiNutritionStep1Form`
- `FullBiNutritionStep1Form`
