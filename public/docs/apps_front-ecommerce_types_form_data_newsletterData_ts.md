# newsletterData.ts

**Path**: `apps\front-ecommerce\types\form\data\newsletterData.ts`

## Summary
This file defines a Zod validation schema and TypeScript type for newsletter subscription requests in an e-commerce application. It enforces that newsletter signups contain a valid email address with a maximum length of 255 characters, providing both runtime validation through Zod and compile-time type safety through the inferred TypeScript type.
