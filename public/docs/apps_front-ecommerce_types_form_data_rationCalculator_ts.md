# rationCalculator.ts

**Path**: `apps\front-ecommerce\types\form\data\rationCalculator.ts`

## Summary
This file defines TypeScript types and a Zod validation schema for a pet food ration calculator feature in an e-commerce application. It specifies the input schema (pet weight and optional age in months) and the expected output structure, which includes daily feeding amounts, frequency codes, cup measurements, and optional extras like weight loss feeding plans.
