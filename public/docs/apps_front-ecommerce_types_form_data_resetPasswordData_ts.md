# resetPasswordData.ts

**Path**: `apps\front-ecommerce\types\form\data\resetPasswordData.ts`

## Summary
This file defines the data structure and validation schema for a password reset form in an e-commerce application. It uses Zod to create a schema that validates a single email field (must be a valid email format with max 255 characters) and exports both the schema and its TypeScript type for use in password reset functionality throughout the application.
