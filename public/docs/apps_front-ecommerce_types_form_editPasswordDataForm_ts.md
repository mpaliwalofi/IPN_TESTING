# editPasswordDataForm.ts

**Path**: `apps\front-ecommerce\types\form\editPasswordDataForm.ts`

## Summary
This class defines a smart form for editing user passwords in an e-commerce application. It extends `SmartFormDefinition` to create a form with two password input fields (`currentPassword` and `newPassword`), both required, and validates submissions against the `EditPasswordDataSchema` before executing the provided callback with the validated password data.
