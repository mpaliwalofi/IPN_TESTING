# emailLookupForm.ts

**Path**: `apps\front-ecommerce\types\form\emailLookupForm.ts`

## Summary
This class defines a form configuration for email lookup functionality in an e-commerce application. It extends `SmartFormDefinition` to create a validated form with a single required email input field, using `EmailLookupSchema` for validation and accepting a callback function to handle submission of the email data.
