# forgotPasswordDataForm.ts

**Path**: `apps\front-ecommerce\types\form\forgotPasswordDataForm.ts`

## Summary
This class defines a smart form for password reset functionality in an e-commerce application. It extends `SmartFormDefinition` to create a structured form with two password input fields (`newPassword` and `confirmNewPassword`) that are validated against a `ForgotPasswordDataSchema` and triggers a callback function when submitted with valid data.
