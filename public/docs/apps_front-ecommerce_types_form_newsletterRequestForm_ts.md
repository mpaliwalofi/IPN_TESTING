# newsletterRequestForm.ts

**Path**: `apps\front-ecommerce\types\form\newsletterRequestForm.ts`

## Summary
This class defines a smart form for newsletter subscription requests in an e-commerce application. It extends `SmartFormDefinition` to create a form with a single email input field, validated against `NewsletterRequestSchema`, and handles form submission through a provided callback. The class uses an `instanceName` parameter to generate unique form IDs, allowing multiple newsletter subscription forms to exist on the same page.
