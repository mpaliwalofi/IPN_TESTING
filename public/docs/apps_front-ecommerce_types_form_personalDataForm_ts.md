# personalDataForm.ts

**Path**: `apps\front-ecommerce\types\form\personalDataForm.ts`

## Summary
This file defines two form classes for managing user personal data in an e-commerce application. `PersonalDataForm` handles user profile information (gender, name, email, birthday, newsletter subscription), while `PersonalPreferencesForm` manages user locale/language preferences. Both classes extend a SmartForm framework to provide structured form definitions with validation schemas and field configurations for user account management.
