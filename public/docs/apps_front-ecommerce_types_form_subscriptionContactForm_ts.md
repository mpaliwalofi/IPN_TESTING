# subscriptionContactForm.ts

**Path**: `apps\front-ecommerce\types\form\subscriptionContactForm.ts`

## Summary
This class defines a form for handling subscription-related contact messages in an e-commerce application. It extends `SmartFormDefinition` to create a simple form with a single required textarea field for the message, validated against a `subscriptionContactSchema`, and executes a callback function when the form is successfully submitted and validated.
