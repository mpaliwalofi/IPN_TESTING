# userGuideForm.ts

**Path**: `apps\front-ecommerce\types\form\userGuideForm.ts`

## Summary
This file defines three form classes (`UserGuideStep1Form`, `UserGuideStep2Form`, `UserGuideStep3Form`) that create multi-step smart forms for collecting pet/animal information in an e-commerce user guide workflow. Each form class extends `SmartFormDefinition` and constructs field definitions (toggles, inputs, choices, etc.) for capturing animal data such as type, gender, breed, name, birthday, and other characteristics, with internationalized labels and validation based on the provided schema and animal type options.
