## Summary
This TypeScript declaration file extends Nuxt 3's `PageMeta` interface to support custom layout properties for the e-commerce front-end application. It adds optional configuration flags like `minimalDisplay` and `enableResultsPage` that allow individual pages to customize their layout behavior and visual presentation. This enables page-level control over layout features without modifying the core layout components.
