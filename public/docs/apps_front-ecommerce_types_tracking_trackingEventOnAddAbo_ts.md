# trackingEventOnAddAbo.ts

**Path**: `apps\front-ecommerce\types\tracking\trackingEventOnAddAbo.ts`

## Summary
This class implements a Google Tag Manager tracking event specifically for adding subscription ("Abo") products to a cart in an e-commerce system. It transforms Sylius product data into GTM-compliant format, handling product details like pricing (including discounts and upsells), quantities, variants, and item list associations for analytics tracking purposes.
