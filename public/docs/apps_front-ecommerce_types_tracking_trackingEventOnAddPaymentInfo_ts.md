# trackingEventOnAddPaymentInfo.ts

**Path**: `apps\front-ecommerce\types\tracking\trackingEventOnAddPaymentInfo.ts`

## Summary
This class defines a tracking event for e-commerce payment information capture, specifically when a user selects or adds a payment method during checkout. It extends a base `TrackingEvent` class to format payment and cart item data into a standardized GTM (Google Tag Manager) ecommerce tracking payload, including product details (name, variant, price, quantity) and the selected payment method code for analytics purposes.
