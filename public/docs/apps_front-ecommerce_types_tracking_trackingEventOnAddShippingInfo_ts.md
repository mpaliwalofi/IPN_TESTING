# trackingEventOnAddShippingInfo.ts

**Path**: `apps\front-ecommerce\types\tracking\trackingEventOnAddShippingInfo.ts`

## Summary
This class represents a tracking event that fires when a customer adds shipping information during the e-commerce checkout process. It captures the selected shipping method and cart items, then transforms this data into a Google Tag Manager (GTM) compatible format with detailed product information (variant codes, prices, quantities, subscription status) and shipping tier details for analytics tracking.
