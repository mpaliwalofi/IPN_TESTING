# trackingEventOnAddToCart.ts

**Path**: `apps\front-ecommerce\types\tracking\trackingEventOnAddToCart.ts`

## Summary
This class implements tracking functionality for "add to cart" events in an e-commerce application. It extends a base `TrackingEvent` class and transforms product data from the Sylius e-commerce system into Google Tag Manager (GTM) format, mapping product details like variant codes, prices, quantities, ratings, and subscription intervals into a standardized tracking data structure for analytics purposes.
