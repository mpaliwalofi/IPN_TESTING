# trackingEventOnAddressCreated.ts

**Path**: `apps\front-ecommerce\types\tracking\trackingEventOnAddressCreated.ts`

## Summary
This class implements a tracking event specifically for monitoring when a user creates a new address in the e-commerce application. It extends the base `TrackingEvent` class to send Google Tag Manager events with contextual information about whether the address was created during checkout or in the user's account settings, along with associated user data for analytics purposes.
