# trackingEventOnDeliverySuccess.ts

**Path**: `apps\front-ecommerce\types\tracking\trackingEventOnDeliverySuccess.ts`

## Summary
This class defines a tracking event specifically for successful delivery completion in an e-commerce checkout flow. It extends a base `TrackingEvent` class and maps the internal 'delivery_success' event to a 'checkout' event name for analytics tracking purposes. The class enables consistent tracking of when customers successfully complete the delivery step during the purchase process.
