# trackingEventOnExitIntent.ts

**Path**: `apps\front-ecommerce\types\tracking\trackingEventOnExitIntent.ts`

## Summary
This class defines a specialized tracking event for monitoring user exit intent behavior in an e-commerce application. It extends the base `TrackingEvent` class to capture when users attempt to leave the site, recording the specific exit target (e.g., close button, navigation away) and formatting it as a "Click" action for analytics purposes.
