# trackingEventOnLoggingInOut.ts

**Path**: `apps\front-ecommerce\types\tracking\trackingEventOnLoggingInOut.ts`

## Summary
This class implements a Google Tag Manager tracking event specifically for user login and logout actions in an e-commerce application. It extends a base TrackingEvent class to capture login attempts, recording whether the authentication was successful (OK/KO based on the customer's logged-in status) along with user data for analytics purposes.
