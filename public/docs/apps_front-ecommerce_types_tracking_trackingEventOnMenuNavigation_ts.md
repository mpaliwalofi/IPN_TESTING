# trackingEventOnMenuNavigation.ts

**Path**: `apps\front-ecommerce\types\tracking\trackingEventOnMenuNavigation.ts`

## Summary
This class defines a Google Tag Manager tracking event specifically for menu navigation interactions in an e-commerce application. It captures click events on menu items in either the header or footer, recording the menu label and location to enable analytics on user navigation patterns through the site's menu structure.
