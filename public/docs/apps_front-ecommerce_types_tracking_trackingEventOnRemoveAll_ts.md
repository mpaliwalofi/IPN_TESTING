# trackingEventOnRemoveAll.ts

**Path**: `apps\front-ecommerce\types\tracking\trackingEventOnRemoveAll.ts`

## Summary
This class implements a tracking event specifically for "remove all" actions in an e-commerce shopping cart, extending a base TrackingEvent class. It transforms Sylius product data (cart items) into Google Tag Manager (GTM) formatted event data, mapping product details like SKU, name, price, ratings, and subscription information into a standardized tracking structure for analytics purposes.
