## Summary
This class tracks user interactions when they select a promotional banner or advertisement in an e-commerce application. It extends a base `TrackingEvent` class to capture promotion details (name, location, and optional Pixie ad tracking data) and formats them into a standardized tracking payload with creative information, slot positioning, and promotion identifiers for analytics purposes.
