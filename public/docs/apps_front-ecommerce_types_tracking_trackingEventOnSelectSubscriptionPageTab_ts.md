# trackingEventOnSelectSubscriptionPageTab.ts

**Path**: `apps\front-ecommerce\types\tracking\trackingEventOnSelectSubscriptionPageTab.ts`

## Summary
This class defines a Google Tag Manager tracking event specifically for monitoring tab selection interactions on subscription pages in an e-commerce application. It extends a base TrackingEvent class to capture custom event data including the tab name and label, formatting them into a standardized structure with "Click" as the action type for analytics purposes.
