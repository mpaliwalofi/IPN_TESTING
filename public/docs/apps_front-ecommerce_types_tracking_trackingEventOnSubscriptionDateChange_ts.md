# trackingEventOnSubscriptionDateChange.ts

**Path**: `apps\front-ecommerce\types\tracking\trackingEventOnSubscriptionDateChange.ts`

## Summary
This class defines a tracking event for monitoring user interactions with subscription date changes in an e-commerce application. It extends a base `TrackingEvent` class to capture analytics data when users view or click through the subscription date modification flow, specifically tracking three stages: 'Edit', 'Timetable', and 'Confirm'. The event data is formatted for Google Tag Manager (GTM) with a custom event name 'abo_date_change' and distinguishes between view and click actions.
