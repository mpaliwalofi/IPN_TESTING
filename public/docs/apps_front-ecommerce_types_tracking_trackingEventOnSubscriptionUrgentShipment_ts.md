# trackingEventOnSubscriptionUrgentShipment.ts

**Path**: `apps\front-ecommerce\types\tracking\trackingEventOnSubscriptionUrgentShipment.ts`

## Summary
This class defines a tracking event specifically for monitoring user interactions with urgent shipment options in a subscription management context. It captures three types of user actions ('Ship_now', 'Confirm', or 'Cancel') and distinguishes between view and click events, sending this data to Google Tag Manager with the event name 'abo_date_change' for analytics purposes related to subscription date modifications.
