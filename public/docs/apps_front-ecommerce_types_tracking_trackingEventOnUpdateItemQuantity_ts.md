# trackingEventOnUpdateItemQuantity.ts

**Path**: `apps\front-ecommerce\types\tracking\trackingEventOnUpdateItemQuantity.ts`

## Summary
This class defines a tracking event specifically for monitoring when users update item quantities in their shopping cart. It extends a base `TrackingEvent` class and enriches the event data with cart-specific metadata (event group and cart method) to categorize this action within the e-commerce analytics system. The event is used to track user interactions with cart quantity controls for business intelligence and user behavior analysis.
