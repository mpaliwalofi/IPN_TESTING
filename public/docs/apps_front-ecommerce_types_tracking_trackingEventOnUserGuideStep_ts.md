# trackingEventOnUserGuideStep.ts

**Path**: `apps\front-ecommerce\types\tracking\trackingEventOnUserGuideStep.ts`

## Summary
This class defines a custom tracking event for monitoring user progression through guide steps in an e-commerce application. It extends a base `TrackingEvent` class to create Google Tag Manager events that capture the guide name and current step number, formatting them as click events with the step as a label for analytics purposes.
