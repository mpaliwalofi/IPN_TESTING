# trackingEventOnViewItemList.ts

**Path**: `apps\front-ecommerce\types\tracking\trackingEventOnViewItemList.ts`

## Summary
This class implements a Google Tag Manager tracking event specifically for tracking when users view a list of products in an e-commerce application. It transforms product data (including SKU, title, price, ratings, and reviews) into the standardized GTM `view_item_list` event format, enriching each item with list source information, brand data, and positioning index for analytics purposes.
