# StoryColorsNeutral.vue

**Path**: `apps\storybook\components\core\colors\StoryColorsNeutral.vue`

## Summary
This is a Storybook component that serves as a color palette showcase for neutral colors in the design system. It displays a visual reference grid of neutral color variations, including both dark and light variants with different lightness levels (ranging from default to 70% lightened), helping designers and developers understand and select appropriate neutral colors for the application's UI.
