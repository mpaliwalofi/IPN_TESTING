# StoryColorsReview.vue

**Path**: `apps\storybook\components\core\colors\StoryColorsReview.vue`

## Summary
This Vue component serves as a visual style guide for displaying the "Review" color palette used in the application's design system. It showcases three primary review colors (orange, yellow, green) and two neutral review colors (dark, light) through a grid layout with color swatches, demonstrating how these colors appear both as backgrounds and text, along with gradient combinations in the headings.
