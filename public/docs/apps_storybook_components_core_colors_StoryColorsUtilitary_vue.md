# StoryColorsUtilitary.vue

**Path**: `apps\storybook\components\core\colors\StoryColorsUtilitary.vue`

## Summary
This Vue component serves as a Storybook documentation page that displays the utilitarian color palette for a design system. It visually demonstrates three semantic color categories (Valid/success, Warn/warning, and Error) with their medium and light variants, helping designers and developers understand and select appropriate colors for UI feedback states.
