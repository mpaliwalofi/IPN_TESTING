# StoryEdito.vue

**Path**: `apps\storybook\components\ecommerce\StoryEdito.vue`

## Summary
This is a Storybook wrapper component for showcasing the e-commerce Edito content component in isolation. It receives editorial content parameters conforming to the `EditoInterface` type and renders them within a containerized, centered layout with a grid system, allowing developers/designers to preview and test the Edito component's appearance and behavior in the Storybook environment.
