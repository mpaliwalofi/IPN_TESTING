# StoryFlags.vue

**Path**: `apps\storybook\components\ecommerce\StoryFlags.vue`

## Summary
This is a Storybook documentation component that showcases SVG flag assets available in the e-commerce application. It displays a visual reference gallery of country flags (Belgium, France, Germany, Italy, Luxembourg) using CSS background classes, and demonstrates their usage in both static displays and within a select dropdown for country/language selection.
