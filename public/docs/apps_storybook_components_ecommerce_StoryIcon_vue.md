# StoryIcon.vue

**Path**: `apps\storybook\components\ecommerce\StoryIcon.vue`

## Summary
This is a Storybook component that provides an interactive icon browser/gallery for displaying and searching through ecommerce-related icon sets. It allows users to filter icons by name using a search input, view icon previews with their class names, and click to copy icon class names to the clipboard for easy integration into other components.
