# StoryStars.vue

**Path**: `apps\storybook\components\ecommerce\StoryStars.vue`

## Summary
This Storybook component demonstrates the visual variations of a star rating component used in the e-commerce application. It renders multiple instances of the `Stars` component with values ranging from 0 to 5 (in 0.5 increments) to showcase all possible rating states, including a custom colored example using the primary theme color.
