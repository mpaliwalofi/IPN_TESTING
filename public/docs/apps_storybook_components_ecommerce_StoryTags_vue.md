# StoryTags.vue

**Path**: `apps\storybook\components\ecommerce\StoryTags.vue`

## Summary
This Vue component serves as a Storybook documentation page showcasing various tag styles and variants for an e-commerce application. It displays a visual catalog of different tag types (primary, tertiary, order, review, neutral, error) in both default and reverse color schemes, along with interactive button tag variants, allowing developers to preview and select appropriate tag styles for use throughout the application.
