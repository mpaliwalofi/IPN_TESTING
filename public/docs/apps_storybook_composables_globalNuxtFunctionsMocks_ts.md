## Summary
This file provides mock implementations of Nuxt global functions and configuration specifically for Storybook component testing and development. It sets up locale configurations for multiple countries (France, Belgium, Netherlands), A/B testing flags, and environment settings that would normally be provided by Nuxt's runtime but need to be mocked in Storybook's isolated environment.
