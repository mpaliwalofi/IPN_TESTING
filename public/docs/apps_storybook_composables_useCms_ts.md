# useCms.ts

**Path**: `apps\storybook\composables\useCms.ts`

## Summary
This is a mock CMS composable for Storybook that provides static header/footer data for component development and testing. It returns hardcoded French e-commerce promotional banner content and navigation menu structure (including pet food categories like dog/puppy products) without making actual API calls. The composable serves as a stub implementation to enable UI development in isolation from the real Strapi CMS backend.
