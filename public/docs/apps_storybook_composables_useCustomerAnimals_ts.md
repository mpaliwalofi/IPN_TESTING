# useCustomerAnimals.ts

**Path**: `apps\storybook\composables\useCustomerAnimals.ts`

## Summary
This Storybook composable provides mock data for testing customer animal-related UI components in an e-commerce pet supplies application. It returns hardcoded animal records (cats and dogs with properties like name, age, type, and photos) based on a sessionStorage flag, and provides mock deletion reasons for removing animals from a customer's account.
