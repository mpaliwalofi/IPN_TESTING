# useFetchCms.ts

**Path**: `apps\storybook\composables\useFetchCms.ts`

## Summary
This is a mock CMS data fetching composable for Storybook that provides a stub implementation of a CMS fetch function. It returns a generic fetch function that immediately resolves with an empty object, allowing Storybook stories to be developed and tested without requiring actual CMS API calls or data dependencies.
