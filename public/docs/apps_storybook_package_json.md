# package.json

**Path**: `apps\storybook\package.json`

## Summary
This is a Storybook configuration package for the UltraPremiumDirect project that provides a component documentation and development environment. It integrates multiple custom UI components (accordion, carousel, smart form/image/link, YouTube player, etc.) from the workspace into a unified Storybook interface with Markdown support and testing capabilities. The package uses Vite as the build tool and has build dependencies on the front-ecommerce-b2b application, enabling isolated component development and visual testing for the e-commerce platform.
