# modes.ts

**Path**: `apps\storybook\static\modes.ts`

## Summary
This configuration file defines viewport breakpoint constants for Storybook's responsive design testing. It establishes four standard device categories (mobile at 375px, tablet at 1024px, desktop at 1260px, and extra-large desktop at 1440px) that are used to preview and test UI components across different screen sizes within the Storybook development environment.
