# customer.ts

**Path**: `apps\storybook\stores\customer.ts`

## Summary
This is a Pinia store for the Storybook app that provides mock customer data and functionality for testing/development purposes. It manages customer authentication state and returns hardcoded Promise-based data for customer personal information, animals (pets), and subscription shipping dates. The store serves as a mock API layer to simulate customer-related operations without requiring actual backend calls during component development and visual testing in Storybook.
