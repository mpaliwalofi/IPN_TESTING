# Colors.stories.ts

**Path**: `apps\storybook\stories\core\Colors.stories.ts`

## Summary
This Storybook configuration file defines visual documentation stories for a design system's color palette in a Vue 3 application. It exports four distinct color category stories (Main, Neural/Neutral, Utilitary, and Review) that showcase different color schemes used throughout the application, with each story rendering a dedicated component to display its respective color collection. The file is configured with Chromatic visual testing support for desktop viewport to ensure consistent color rendering across the design system.
