# Typography.stories.ts

**Path**: `apps\storybook\stories\core\Typography.stories.ts`

## Summary
This is a Storybook configuration file that defines a story for displaying typography styles and text formatting components in the application's design system. It sets up the `StoryTypography` component to be showcased in Storybook's "Core/Base/Typography" section with desktop viewport settings for visual regression testing via Chromatic. The file serves as documentation and a visual reference for developers to see all available typography styles in a centralized location.
