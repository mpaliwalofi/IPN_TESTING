# Icons.stories.ts

**Path**: `apps\storybook\stories\ecommerce\Icons.stories.ts`

## Summary
This Storybook configuration file defines visual documentation stories for displaying three categories of icon fonts used in the e-commerce application: general icons (UpdIcons), FAQ icons (UpdFaqs), and USP (Unique Selling Proposition) icons (UpdUsps). It sets up interactive component stories that showcase all available icons in each font set through the StoryIcon component, configured to render in desktop viewport for visual regression testing with Chromatic.
