# Tags.stories.ts

**Path**: `apps\storybook\stories\ecommerce\Tags.stories.ts`

## Summary
This is a Storybook configuration file that creates a documentation story for an e-commerce tags component. It sets up a single "Default" story that showcases the `StoryTags` Vue component, allowing developers to view and test different states of product/category tags in isolation within the Storybook UI.
