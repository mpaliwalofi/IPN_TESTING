# tailwind.config.js

**Path**: `apps\storybook\tailwind.config.js`

## Summary
This Tailwind CSS configuration file sets up styling for a Storybook instance in an e-commerce application. It extends the Nuxt e-commerce Tailwind config and defines content paths to scan for Tailwind classes across multiple directories, including Vue components, Storybook stories, and front-end e-commerce files from both the main app and shared packages.
