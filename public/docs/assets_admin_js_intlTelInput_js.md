# intlTelInput.js

**Path**: `assets\admin\js\intlTelInput.js`

## Summary
This module imports and globally exposes the intlTelInput library, which provides international telephone number input functionality with country code selection and validation. It serves as a bridge to make the intlTelInput library available throughout the admin interface for handling phone number inputs with proper formatting and country detection. The module also imports the associated CSS styles needed for the telephone input component's visual presentation.
