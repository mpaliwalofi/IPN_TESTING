# behat.yml

**Path**: `behat.yml`

## Summary
This is a Behat configuration file that sets up automated behavioral testing for a Sylius-based e-commerce application. It configures multiple browser testing sessions (Chrome, Panther, Symfony) with support for JavaScript testing, defines the base URL (https://127.0.0.1:8080/), and imports test suites for registration and login features. The configuration includes debugging extensions for screenshots, Chrome driver integration for headless browser testing, and Mink extension for web acceptance testing with various session managers.
