# AnimalDeletedReason.xml

**Path**: `config\api_platform\properties\AnimalDeletedReason.xml`

## Summary
This API Platform configuration file defines metadata for the `AnimalDeletedReason` resource property. It specifies that the `id` field serves as the resource identifier and is read-only (non-writable), establishing how this property should be handled in API operations related to tracking reasons for animal deletions in the system.
