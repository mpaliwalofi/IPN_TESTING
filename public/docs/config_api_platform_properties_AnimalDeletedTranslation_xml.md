# AnimalDeletedTranslation.xml

**Path**: `config\api_platform\properties\AnimalDeletedTranslation.xml`

## Summary
This API Platform configuration file defines property metadata for the `AnimalDeletedTranslation` resource, which handles translations of animal deletion reasons. It specifies that `locale` serves as the identifier (instead of `id`) and is required, enabling the API to manage multi-language support for animal deletion reason descriptions.
