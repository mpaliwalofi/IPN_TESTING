# PickupPointClosingPeriod.xml

**Path**: `config\api_platform\properties\PickupPointClosingPeriod.xml`

## Summary
This API Platform configuration file defines metadata properties for the `PickupPointClosingPeriod` entity, which represents time periods when a pickup point is closed for business. It configures the `id` property as the resource identifier that is read-only (non-writable), establishing how this entity is exposed through the API for managing pickup point availability schedules.
