# UpSellVariant.xml

**Path**: `config\api_platform\properties\UpSellVariant.xml`

## Summary
This API Platform configuration file defines read-only property metadata for an UpSellVariant resource, which represents product variant information used in upselling scenarios. The configuration specifies that all properties (id, variant details, product/variant names, pricing including discounted prices, and reduction type) are non-writable, indicating this is a read-only API resource for displaying upsell product options to customers with their associated pricing and discount information.
