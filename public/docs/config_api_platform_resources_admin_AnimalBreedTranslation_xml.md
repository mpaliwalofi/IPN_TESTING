# AnimalBreedTranslation.xml

**Path**: `config\api_platform\resources\admin\AnimalBreedTranslation.xml`

## Summary
This API Platform configuration file defines a resource endpoint for managing animal breed translations in an admin context. It exposes a non-public GET operation at `/admin/animal-breeds/{id}/translations/{localeCode}` that retrieves translation data for a specific animal breed by linking the breed ID with its corresponding locale-specific translation, enabling multi-language support for animal breed information in the system.
