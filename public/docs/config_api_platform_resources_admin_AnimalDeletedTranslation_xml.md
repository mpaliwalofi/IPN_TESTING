# AnimalDeletedTranslation.xml

**Path**: `config\api_platform\resources\admin\AnimalDeletedTranslation.xml`

## Summary
This API Platform resource configuration defines a non-exposed API endpoint for accessing translations of animal deletion reasons in an admin context. It configures a URI template that retrieves specific translation records by combining an animal deletion reason ID with a locale code, enabling multilingual support for categorizing why animals were removed from the system.
