# AnimalFeatureTranslation.xml

**Path**: `config\api_platform\resources\admin\AnimalFeatureTranslation.xml`

## Summary
This API Platform configuration file defines a resource endpoint for accessing AnimalFeatureTranslation entities in an admin context. It configures a `NotExposed` operation (meaning it's defined but not publicly accessible) that establishes the URI structure `/admin/animal-features/{id}/translations/{localeCode}` for retrieving translations of animal features, linking translations to their parent animal feature through URI variables.
