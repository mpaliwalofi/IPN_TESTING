# AnimalFeedTranslation.xml

**Path**: `config\api_platform\resources\admin\AnimalFeedTranslation.xml`

## Summary
This API Platform configuration file defines a resource endpoint for accessing AnimalFeedTranslation entities in the admin context. It configures a "NotExposed" operation (meaning it's defined but not publicly exposed) at the URI `/admin/animal-feeds/{id}/translations/{localeCode}`, which retrieves translations for a specific animal feed by linking the animal feed ID with a locale code through URI variables.
