# AnimalGenderTranslation.xml

**Path**: `config\api_platform\resources\admin\AnimalGenderTranslation.xml`

## Summary
This API Platform configuration file defines a REST API resource for managing animal gender translations in a multi-language system. It configures a single "NotExposed" operation that maps to the URI template `/admin/animal-genders/{id}/translations/{localeCode}`, establishing the relationship between animal genders and their localized translations through URI variables linking the gender ID and locale code.
