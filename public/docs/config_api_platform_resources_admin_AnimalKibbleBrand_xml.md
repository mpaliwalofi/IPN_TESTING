# AnimalKibbleBrand.xml

**Path**: `config\api_platform\resources\admin\AnimalKibbleBrand.xml`

## Summary
This API Platform configuration file defines REST API endpoints for managing AnimalKibbleBrand resources in an admin context. It configures two operations: a GET collection endpoint with ordering by priority and filtering capabilities, and a POST endpoint for creating new kibble brand records using a custom persist processor. The configuration specifies serialization groups for controlling data exposure and disables pagination for the collection endpoint.
