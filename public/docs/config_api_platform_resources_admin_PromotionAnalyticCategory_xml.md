# PromotionAnalyticCategory.xml

**Path**: `config\api_platform\resources\admin\PromotionAnalyticCategory.xml`

## Summary
This API Platform configuration file defines read-only API endpoints for the PromotionAnalyticCategory resource in the admin area of a Sylius e-commerce application. It exposes two operations: retrieving a collection of promotion analytic categories and fetching a single category, both using the `sylius:admin:promotion:show` serialization group for controlling data output. The configuration enables administrators to access promotional analytics data categorized for reporting and analysis purposes.
