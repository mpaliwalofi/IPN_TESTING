# web_profiler.yaml

**Path**: `config\packages\dev\web_profiler.yaml`

## Summary
This Symfony configuration file enables and configures the Web Profiler bundle for the development environment. It activates the debug toolbar at the bottom of web pages (`toolbar: true`) while allowing automatic redirects to proceed without interruption (`intercept_redirects: false`), providing developers with debugging and profiling information during development without interfering with redirect flows.
