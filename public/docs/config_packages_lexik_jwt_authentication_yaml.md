# lexik_jwt_authentication.yaml

**Path**: `config\packages\lexik_jwt_authentication.yaml`

## Summary
This configuration file sets up JWT (JSON Web Token) authentication for a Symfony application using the LexikJWTAuthenticationBundle. It specifies the paths to the secret/public key pair used for signing and verifying JWT tokens, along with an optional passphrase for encrypted private keys, all loaded from environment variables for security.
