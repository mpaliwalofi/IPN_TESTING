# ressources.yaml

**Path**: `config\packages\ressources.yaml`

## Summary
This YAML configuration file defines Sylius resource mappings for a subscription-based business with animal-related features. It registers multiple entities and their repositories with the Sylius resource system, including subscription management components (frequency groups, monitoring, stop reasons, comments), holiday dates, and animal-related entities (animals, activities, conditions, features, feeds) with their corresponding translation models for multi-language support.
