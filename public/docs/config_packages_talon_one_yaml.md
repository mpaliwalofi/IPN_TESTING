# talon_one.yaml

**Path**: `config\packages\talon_one.yaml`

## Summary
This configuration file sets up integration with Talon.One, a promotional campaign and coupon management platform. It defines environment parameters for API authentication (management and integration API keys), campaign settings, and configures two HTTP clients for making requests to Talon.One's management and integration APIs with their respective authorization headers. The configuration includes feature flags for controlling the Talon.One integration behavior and settings for catalog synchronization and customer service campaigns.
