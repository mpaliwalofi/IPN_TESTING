# sylius_api.yaml

**Path**: `config\routes\sylius_api.yaml`

## Summary
This routing configuration file defines custom API endpoints for a Sylius e-commerce application, extending the default API with specialized functionality. It includes routes for JWT token refresh authentication, customer state verification, and schema retrieval endpoints for configuring promotions, payment methods, shipping methods, and order rules. These routes provide admin and shop-facing APIs for managing business logic configuration, customer reporting, and retrieving dynamic form schemas needed for the platform's administrative interfaces.
