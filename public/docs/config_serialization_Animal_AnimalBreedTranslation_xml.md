# AnimalBreedTranslation.xml

**Path**: `config\serialization\Animal\AnimalBreedTranslation.xml`

## Summary
This Symfony serializer configuration file defines serialization groups for the `AnimalBreedTranslation` entity, which handles translations of animal breed names across different locales. It specifies which attributes (`id`, `name`, `locale`) should be included when serializing the entity for different admin operations: reading breed translations, creating new translations, and updating existing ones.
