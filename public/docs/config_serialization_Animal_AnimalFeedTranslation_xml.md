# AnimalFeedTranslation.xml

**Path**: `config\serialization\Animal\AnimalFeedTranslation.xml`

## Summary
This is a Symfony serializer configuration file that defines serialization groups for the `AnimalFeedTranslation` entity, which handles translations of animal feed names. It specifies which attributes (id, name, locale) should be included when serializing this entity for different admin operations (read, create, update), enabling proper API exposure and access control for multilingual animal feed data.
