# ChangeItemQuantityInCart.xml

**Path**: `config\serialization\Commands\Cart\ChangeItemQuantityInCart.xml`

## Summary
This XML configuration file defines serialization groups for the `ChangeItemQuantityInCart` command in a shopping cart system. It specifies that the `quantity` and `subscriptionInterval` attributes should be serialized/deserialized for both admin and shop contexts when changing item quantities in a cart, enabling controlled API exposure of these fields based on user roles.
