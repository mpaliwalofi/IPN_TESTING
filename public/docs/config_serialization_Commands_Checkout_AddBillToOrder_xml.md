# AddBillToOrder.xml

**Path**: `config\serialization\Commands\Checkout\AddBillToOrder.xml`

## Summary
This XML file configures Symfony serialization for the `AddBillToOrder` command in the checkout domain. It defines which attributes (`number` and `billPath`) should be serialized/deserialized when adding a bill document to an order, specifically for the `admin:order:add_bill` serialization group used in admin API operations.
