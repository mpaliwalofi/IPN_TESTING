# ChoosePaymentMethod.xml

**Path**: `config\serialization\Commands\PaymentInstrument\ChoosePaymentMethod.xml`

## Summary
This XML configuration file defines serialization groups for the `ChoosePaymentMethod` command in a payment instrument context. It specifies that the `paymentMethod` attribute should be included when serializing/deserializing data for both admin and shop user roles updating payment instruments, enabling role-based access control for payment method selection operations.
