# GeneratePromotionCoupon.xml

**Path**: `config\serialization\Commands\Promotion\GeneratePromotionCoupon.xml`

## Summary
This XML configuration file defines serialization groups for the `GeneratePromotionCoupon` command class in a Sylius e-commerce application. It specifies that the command's attributes (prefix, suffix, codeLength, and quantityCouponsGenerated) should be serialized when creating promotion coupons through the admin API, enabling bulk generation of coupon codes with customizable parameters.
