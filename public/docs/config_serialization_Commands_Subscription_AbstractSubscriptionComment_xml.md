# AbstractSubscriptionComment.xml

**Path**: `config\serialization\Commands\Subscription\AbstractSubscriptionComment.xml`

## Summary
This Symfony serializer configuration file defines serialization groups for the `AbstractSubscriptionComment` command class. It specifies that the `message` attribute should be included when serializing subscription comment operations (create, remove, and update) in the admin context. This configuration enables controlled exposure of comment data across different API endpoints for subscription comment management.
