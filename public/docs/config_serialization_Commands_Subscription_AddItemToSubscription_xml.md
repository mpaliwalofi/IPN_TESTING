# AddItemToSubscription.xml

**Path**: `config\serialization\Commands\Subscription\AddItemToSubscription.xml`

## Summary
This is a Symfony serializer configuration file that defines serialization groups for the `AddItemToSubscription` command class. It specifies that the `variant` and `spreadOnDraft` attributes should be included when serializing/deserializing data in the `admin:subscription:add_item` context, which is used for adding items to a subscription through an admin interface.
