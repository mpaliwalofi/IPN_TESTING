# DebitSubscription.xml

**Path**: `config\serialization\Commands\Subscription\DebitSubscription.xml`

## Summary
This Symfony serializer configuration file defines serialization rules for the `DebitSubscription` command class. It specifies that the `subscriptionCode` attribute should be included when serializing objects in the `admin:subscription:debit` group, enabling controlled exposure of subscription debit command data for administrative operations.
