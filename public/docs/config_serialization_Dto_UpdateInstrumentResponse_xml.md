# UpdateInstrumentResponse.xml

**Path**: `config\serialization\Dto\UpdateInstrumentResponse.xml`

## Summary
This Symfony serializer configuration file defines serialization groups for the `UpdateInstrumentResponse` DTO class. It specifies that three attributes (`result`, `instrument`, and `redirection`) should be included when serializing responses for both admin and shop instrument update operations, enabling context-specific API response formatting based on user roles.
