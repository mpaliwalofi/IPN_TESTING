# ProductOptionTranslation.xml

**Path**: `config\serialization\Product\ProductOptionTranslation.xml`

## Summary
This Symfony serializer configuration file defines serialization rules for the `ProductOptionTranslation` model in the Sylius e-commerce framework. It specifies which attributes (id, name, locale) should be included when serializing product option translations, and organizes them into different serialization groups (read, create, update) for controlling API access in the admin interface. This configuration enables proper translation handling for product options across different locales in the system.
