# ChangeItemQuantityInCart.xml

**Path**: `config\validator\validation\Checkout\ChangeItemQuantityInCart.xml`

## Summary
This XML configuration file defines Symfony validation constraints for the `ChangeItemQuantityInCart` command in the shopping cart domain. It applies a custom validator (`ChangeItemQuantityInCartEligibility`) to ensure that cart item quantity changes meet business rules and eligibility requirements, with validation grouped under the "sylius" validation group for the Sylius e-commerce framework.
