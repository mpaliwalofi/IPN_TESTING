# AddItemToSubscriptionOrder.xml

**Path**: `config\validator\validation\SubscriptionOrder\AddItemToSubscriptionOrder.xml`

## Summary
This XML configuration file defines validation rules for adding items to subscription orders in a Sylius-based e-commerce system. It applies a custom constraint (`AddingEligibleProductVariantToSubscriptionOrder`) to the `AddItemToSubscriptionOrder` command class, ensuring that only eligible product variants can be added to subscription orders during the validation process.
