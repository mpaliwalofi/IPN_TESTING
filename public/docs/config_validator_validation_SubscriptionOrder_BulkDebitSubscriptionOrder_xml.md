# BulkDebitSubscriptionOrder.xml

**Path**: `config\validator\validation\SubscriptionOrder\BulkDebitSubscriptionOrder.xml`

## Summary
This XML configuration file defines Symfony validation rules for the `BulkDebitSubscriptionOrder` CQRS command. It applies a custom `DebitSubscriptionOrder` validation constraint to ensure bulk subscription debiting operations meet required business rules before execution, with validation applied across both Default and Sylius validation groups.
