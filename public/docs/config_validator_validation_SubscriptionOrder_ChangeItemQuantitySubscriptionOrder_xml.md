# ChangeItemQuantitySubscriptionOrder.xml

**Path**: `config\validator\validation\SubscriptionOrder\ChangeItemQuantitySubscriptionOrder.xml`

## Summary
This XML configuration file defines validation rules for the `ChangeItemQuantitySubscriptionOrder` command in a subscription management system. It enforces that the quantity field must not be blank and must be a positive value, while also applying a custom `ItemQuantitySubscriptionOrder` validator to the entire command class, all within the Default and Sylius validation groups.
