# AddItemToSubscription.xml

**Path**: `config\validator\validation\Subscription\AddItemToSubscription.xml`

## Summary
This XML configuration file defines Symfony validation constraints for the `AddItemToSubscription` command in a subscription management system. It applies a custom validator (`AddingEligibleProductVariantToSubscription`) to ensure that only eligible product variants can be added to a subscription, with validation applied within the "sylius" validation group (indicating integration with the Sylius e-commerce framework).
