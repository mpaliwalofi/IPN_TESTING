# ChangeItemQuantitySubscription.xml

**Path**: `config\validator\validation\Subscription\ChangeItemQuantitySubscription.xml`

## Summary
This XML configuration file defines Symfony validation constraints for the `ChangeItemQuantitySubscription` command in a subscription management system. It enforces that the `quantity` property must not be blank and must be a positive number, applying these validation rules to both the "Default" and "sylius" validation groups to ensure data integrity when modifying subscription item quantities.
