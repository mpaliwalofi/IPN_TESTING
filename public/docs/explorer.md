# Interactive Explorer

Use the Explorer to search across every generated markdown file with rich
category and repository filters. It renders directly from the MkDocs build
artifacts, so results always stay in sync with the documentation site.

- <a href="../viewer/index.html" target="_blank">Open Explorer</a> — opens the interactive UI in a new tab.
- <a href="../docs_index.json" target="_blank">Download JSON index</a> — inspect the raw data feed.

!!! tip
    The Explorer works best when you regenerate docs (`python generate_docs.py`)
    before starting `mkdocs serve`, ensuring the JSON index and markdown content
    are aligned.
