# registration.feature

**Path**: `features\api\shop\registration.feature`

## Summary
This Gherkin feature file defines API acceptance tests for the shop customer registration functionality in a multi-channel e-commerce system. It verifies that users can successfully register a new account through the API by providing personal information (first name, last name, birthday) and optionally subscribing to a newsletter within a specific store channel (identified by "fr" code). The feature ensures the registration endpoint properly validates and creates new customer accounts with appropriate success notifications.
