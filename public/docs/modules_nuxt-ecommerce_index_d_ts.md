# index.d.ts

**Path**: `modules\nuxt-ecommerce\index.d.ts`

## Summary
This TypeScript declaration file defines the type structure for Nuxt's public runtime configuration in an e-commerce module. It extends the `@nuxt/schema` module to specify configuration options for a pet-focused e-commerce platform, including integration settings for third-party services like Algolia search, payment processing (Payzen), image delivery (ImageKit), analytics tracking (GTM, Pixie), GDPR compliance (Didomi), and feature flags for functionality like subscriptions, samples, animal profiles, and cart behavior.
