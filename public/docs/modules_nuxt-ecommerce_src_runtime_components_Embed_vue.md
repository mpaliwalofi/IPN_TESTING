# Embed.vue

**Path**: `modules\nuxt-ecommerce\src\runtime\components\Embed.vue`

## Summary
This Vue component renders embedded content (specifically YouTube videos) with configurable layout options. It parses embed parameters from props, conditionally applies container styling and optional background effects, and delegates YouTube video rendering to a specialized `Youtube` component while supporting aspect ratio customization through the `ytClass` prop.
