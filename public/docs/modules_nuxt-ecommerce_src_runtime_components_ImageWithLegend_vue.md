# ImageWithLegend.vue

**Path**: `modules\nuxt-ecommerce\src\runtime\components\ImageWithLegend.vue`

## Summary
This Vue component renders a responsive image with optional overlay and legend functionality for an e-commerce platform. It displays different image versions based on screen size (mobile vs desktop), wraps them in a clickable link, and supports overlay content through a slot pattern. The component is designed to create promotional or informational banner sections that adapt to different viewport sizes while maintaining consistent linking behavior.
