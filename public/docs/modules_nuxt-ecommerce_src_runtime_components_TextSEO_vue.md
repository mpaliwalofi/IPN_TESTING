# TextSEO.vue

**Path**: `modules\nuxt-ecommerce\src\runtime\components\TextSEO.vue`

## Summary
This Vue component renders an SEO-optimized text section with expandable/collapsible content functionality. It displays a title, markdown content (both visible and foldable sections), and related links, with the ability to toggle the folded content and smoothly scroll back to the section when collapsed. The component is designed for e-commerce pages to present SEO-rich content that can be expanded for additional details while maintaining a clean initial view.
