# TimeLine.vue

**Path**: `modules\nuxt-ecommerce\src\runtime\components\TimeLine.vue`

## Summary
This Vue component renders a timeline display for e-commerce content, showing chronological blocks with dates, headings, descriptions, and associated media. Each timeline block can contain either an embedded video or an image with an optional link, along with a "more info" call-to-action button, making it suitable for displaying company history, product evolution, or milestone events in an e-commerce context.
