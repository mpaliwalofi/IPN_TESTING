# USPCard.vue

**Path**: `modules\nuxt-ecommerce\src\runtime\components\USPCard.vue`

## Summary
This Vue component renders a USP (Unique Selling Proposition) card for an e-commerce application, displaying promotional or feature highlights with either an icon or image alongside text content. It supports flexible layouts (mobile/desktop column configurations), customizable styling, and integrates with Strapi CMS data structures to render markdown-formatted titles and descriptions. The component is designed to showcase product benefits, shipping information, or other selling points in a consistent, reusable format across the e-commerce site.
