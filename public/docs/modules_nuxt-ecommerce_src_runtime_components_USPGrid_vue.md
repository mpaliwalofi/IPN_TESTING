# USPGrid.vue

**Path**: `modules\nuxt-ecommerce\src\runtime\components\USPGrid.vue`

## Summary
This Vue component renders a grid of USP (Unique Selling Point) cards for an e-commerce product page, displaying features like valuable ingredients or product benefits. On mobile devices, it shows the cards in a carousel/slider format, while on desktop it displays them in a static grid layout with 4 columns. The component supports responsive breakpoints and customizable styling options for text, images, and layout orientation (row/column).
