# Breadcrumb.stories.ts

**Path**: `modules\nuxt-ecommerce\src\runtime\components\stories\Breadcrumb.stories.ts`

## Summary
This is a Storybook configuration file that defines stories (visual test cases) for a Breadcrumb navigation component in an e-commerce module. It exports two story variants: a "Default" story showing the breadcrumb with mock data, and a "WithCurrent" story that additionally displays the current page label, allowing developers to preview and test different states of the breadcrumb component in isolation.
