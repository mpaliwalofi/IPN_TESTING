# badge.css

**Path**: `modules\nuxt-ecommerce\src\runtime\css\ecommerce\shared\badge.css`

## Summary
This CSS file defines badge component styles for an e-commerce application, creating visual notification indicators (dots and count badges) that can be overlaid on elements like cart icons or notifications. The badges support multiple variants including standard dots, mobile-optimized versions, green status indicators, and numbered count badges that display dynamic content via data attributes.
