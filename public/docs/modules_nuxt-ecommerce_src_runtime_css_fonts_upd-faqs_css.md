# upd-faqs.css

**Path**: `modules\nuxt-ecommerce\src\runtime\css\fonts\upd-faqs.css`

## Summary
This CSS file defines a custom icon font named "upd-faqs" for displaying FAQ-related icons in an e-commerce application. It provides 13 icon classes covering various customer service topics (cancellation/returns, delivery, payment, account management, subscriptions, etc.) using Unicode characters, with cross-browser font format support including EOT, WOFF, TTF, and SVG.
