# upd-icons.css

**Path**: `modules\nuxt-ecommerce\src\runtime\css\fonts\upd-icons.css`

## Summary
This CSS file defines a custom icon font named "upd-icons" for the Nuxt e-commerce module. It includes font-face declarations with multiple format fallbacks for cross-browser compatibility and defines CSS classes for individual icons (arrows, avatars, UI controls, etc.) that can be applied using the `upd-icons-*` class prefix, with each icon mapped to a specific Unicode character rendered via the ::before pseudo-element.
