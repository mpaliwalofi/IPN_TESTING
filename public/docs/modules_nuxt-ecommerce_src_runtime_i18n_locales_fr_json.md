## Summary
This is a French language localization file for a Nuxt e-commerce module that provides internationalization (i18n) translations. It contains French translations for common UI elements including header navigation text, generic actions (yes/no), time units (years/months), utility text (more info, next/previous), and SVG icon accessibility labels (arrows, add/subtract buttons).
