## Summary
This is a Dutch (nl) language translation file for the nuxt-ecommerce module's internationalization (i18n) system. It contains localized text strings for common UI elements including navigation controls (skip links, arrows), generic actions (yes/no), time periods (years/months), and utility functions (more info, next/previous buttons), enabling the e-commerce application to display Dutch language content to users.
