# supportedFiles.ts

**Path**: `modules\nuxt-ecommerce\src\runtime\static\supportedFiles.ts`

## Summary
This file exports a static array of MIME types that defines the whitelist of acceptable file formats for file uploads in the Nuxt e-commerce module. It supports a comprehensive range of document types (Office, PDF, text), image formats, audio/video files, and compressed archives, enabling the application to validate and restrict file uploads to these specific supported types for security and compatibility purposes.
