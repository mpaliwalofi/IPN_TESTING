# pixieSupport.ts

**Path**: `modules\nuxt-ecommerce\src\runtime\tracking\pixieSupport.ts`

## Summary
This class provides integration with a marketing campaign tracking system called "Pixie" for an e-commerce platform. It manages the lifecycle of marketing campaigns (popups, banners, A/B tests, product offers) by fetching campaign configurations from a Pixie API endpoint, tracking user interactions and conversions, and coordinating with the site's A/B testing engine to display appropriate promotional content based on user behavior and locale.
