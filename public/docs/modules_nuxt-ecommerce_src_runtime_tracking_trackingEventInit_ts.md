# trackingEventInit.ts

**Path**: `modules\nuxt-ecommerce\src\runtime\tracking\trackingEventInit.ts`

## Summary
This class defines a tracking event for initializing analytics or tracking systems in an e-commerce context. It extends the base `TrackingEvent` class with an 'init' event type and provides structured data including the current page URL, placeholder category fields, and user data that would be captured when the tracking system first loads on a page.
