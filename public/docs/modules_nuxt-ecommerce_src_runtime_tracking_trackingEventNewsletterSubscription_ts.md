# trackingEventNewsletterSubscription.ts

**Path**: `modules\nuxt-ecommerce\src\runtime\tracking\trackingEventNewsletterSubscription.ts`

## Summary
This class implements a tracking event specifically for newsletter subscription actions in an e-commerce application. It extends a base `TrackingEvent` class to capture subscription-related data including event labels, user information, and configurable event names/actions (defaulting to 'newsletter_subscription' and 'Click'), which are formatted for Google Tag Manager custom event tracking.
