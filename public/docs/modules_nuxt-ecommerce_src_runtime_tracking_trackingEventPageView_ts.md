# trackingEventPageView.ts

**Path**: `modules\nuxt-ecommerce\src\runtime\tracking\trackingEventPageView.ts`

## Summary
This class represents a page view tracking event for an e-commerce analytics system. It extends a base `TrackingEvent` class to capture page-specific metadata including platform environment, country/store information, page template type, hierarchical category data (up to 3 levels), and user information, then serializes this data into a standardized format for tracking/analytics purposes.
