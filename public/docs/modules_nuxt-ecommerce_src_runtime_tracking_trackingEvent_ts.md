**Path**: `modules\nuxt-ecommerce\src\runtime\tracking\trackingEvent.ts`

## Summary
This class represents a tracking event entity in the Nuxt e-commerce module, used for analytics and monitoring purposes. It encapsulates event data with an event identifier and optional metadata, providing a structured way to represent user interactions or system events in the e-commerce application. The `toData()` method serializes the event into a flat object format suitable for transmission to analytics services or logging systems.
