# Embed.spec.ts

**Path**: `modules\nuxt-ecommerce\test\components\Embed.spec.ts`

## Summary
This is a test suite for an Embed Vue component in an e-commerce Nuxt module, specifically testing the rendering and behavior of embedded YouTube videos. The tests verify that the component correctly parses oembed data, conditionally renders a YouTube child component based on the media type, and handles various layout configurations (contained vs. full-width, custom aspect ratios). The snapshot tests ensure the component's HTML output remains consistent across different prop configurations.
