## Summary
This file exports a mock data object representing a text-based SEO component for testing purposes in a Nuxt e-commerce application. The mock includes typical SEO text content fields such as a title, markdown content for desktop and mobile views, formatted with Lorem Ipsum placeholder text to simulate real SEO-optimized content that would be displayed on product or category pages.
