# package.json

**Path**: `package.json`

## Summary
This is a package.json configuration file for a Strapi-based content management system (CMS) that integrates with multiple cloud services and third-party platforms. The application uses Strapi v4.15.5 as its core framework and includes dependencies for Google Cloud services (Firestore, Pub/Sub), Algolia search, ImageKit media management, MySQL database connectivity, and various Strapi plugins for internationalization, user permissions, and content management features. It's configured to run on Node.js 18-20 with Yarn package manager and provides standard Strapi development, build, and deployment scripts.
