# package.json

**Path**: `packages\accordionFold\package.json`

## Summary
This package configuration defines an npm package for an accordion-fold UI component within the UltraPremiumDirect design system. It's a Vue 3 component library that provides both JavaScript modules and Tailwind CSS plugin integration, built with Vite and including TypeScript definitions for type safety. The package is designed to be published and consumed by other applications in the monorepo workspace, offering a reusable collapsible/expandable accordion interface component with internationalization support via vue-i18n.
