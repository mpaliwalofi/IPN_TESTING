# AccordionFold.vue

**Path**: `packages\accordionFold\src\components\AccordionFold.vue`

## Summary
This Vue 3 component implements an accordion/collapsible fold UI element that can toggle between open and closed states. It supports both controlled (external) and uncontrolled (internal) state management, renders markdown content using a Nuxt markdown-it plugin, and provides customizable styling options including desktop-specific behavior (stay open, hide icons), reversible layouts, and smooth CSS height transitions for expand/collapse animations. The component is designed for displaying structured content like FAQs or collapsible sections with configurable headers and icons.
