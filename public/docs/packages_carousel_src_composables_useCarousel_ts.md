# useCarousel.ts

**Path**: `packages\carousel\src\composables\useCarousel.ts`

## Summary
This Vue 3 composable manages carousel/slider functionality by handling slide navigation, touch/swipe gestures, and autoplay behavior. It calculates slide positioning and transitions based on configurable properties like items per slide, scroll count, and infinite looping, while integrating VueUse utilities for swipe detection, hover states, and responsive breakpoints. The composable returns methods for programmatic slide control (next/previous/setSlide) and reactive state for rendering the carousel with proper transforms and animations.
