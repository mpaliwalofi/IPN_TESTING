# vite.config.ts

**Path**: `packages\carousel\vite.config.ts`

## Summary
This is a Vite build configuration file for a carousel component package within the UltraPremiumDirect design system. It configures the package to be built as an ES module library with CSS automatically injected via JavaScript, merging custom settings with a shared base Vite configuration while specifying the entry point and output format for the carousel component bundle.
