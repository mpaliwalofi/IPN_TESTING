# package.json

**Path**: `packages\counter\package.json`

## Summary
This is a package configuration file for a Vue 3 counter component that is part of the UltraPremiumDirect design system. The package exports a built counter component module with TypeScript definitions and includes a Tailwind CSS plugin integration, supporting both ES modules and CommonJS formats. It's configured as a workspace package with Vite for building, Vitest for testing, and includes internationalization support through vue-i18n, allowing it to be consumed as a reusable UI component across the organization's applications.
