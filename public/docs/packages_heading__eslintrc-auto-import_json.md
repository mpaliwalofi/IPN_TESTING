# .eslintrc-auto-import.json

**Path**: `packages\heading\.eslintrc-auto-import.json`

## Summary
This is an ESLint configuration file that declares global variables for auto-import functionality in a Vue.js heading component package. It whitelists Vue 3 composition API functions, lifecycle hooks, TypeScript type utilities, and testing framework globals (Vitest/Chai) so they can be used without explicit imports, preventing ESLint from flagging them as undefined variables.
