# Heading.vue

**Path**: `packages\heading\src\components\Heading.vue`

## Summary
This Vue component provides a dynamic heading renderer that accepts a configurable heading level (H1-H6) from Strapi CMS. It uses Vue's dynamic `&lt;Component&gt;` element to render the appropriate HTML heading tag based on the `level` prop, defaulting to H1 if not specified. This allows for flexible, CMS-driven heading hierarchies while maintaining proper semantic HTML structure.
