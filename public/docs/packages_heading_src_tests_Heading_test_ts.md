# Heading.test.ts

**Path**: `packages\heading\src\tests\Heading.test.ts`

## Summary
This is a test suite for a Vue Heading component that verifies the component correctly renders different HTML heading levels (h1, h2, etc.) based on props. The tests ensure the component defaults to h1, properly respects the `level` prop to render different heading tags, and maintains correct snapshot outputs for various configurations including undefined levels.
