# vite.config.ts

**Path**: `packages\heading\vite.config.ts`

## Summary
This is a Vite build configuration file for the `heading` package within the UltraPremiumDirect design system. It configures the package to be built as an ES module library with the entry point at `src/index.ts`, outputting as `upd-ds-heading.es.js`. The configuration extends a base Vite config while defining library-specific settings for bundling the heading component as a distributable package.
