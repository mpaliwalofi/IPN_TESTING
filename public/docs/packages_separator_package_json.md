# package.json

**Path**: `packages\separator\package.json`

## Summary
This package configuration defines a Vue 3 component library for a separator UI element within the `@ultrapremiumdirect` design system. It exports both an ES module build of the separator component and a Tailwind CSS plugin, providing a reusable visual divider component with internationalization support (vue-i18n) for use across the Ultra Premium Direct application ecosystem.
