# Separator.vue

**Path**: `packages\separator\src\components\Separator.vue`

## Summary
This Vue 3 component renders a customizable horizontal separator (hr element) for UI layout purposes. It accepts configuration parameters through the `ISeparator` interface to control the separator's visual style (dashed vs solid line) and spacing (top/bottom margins). The component provides a reusable way to add visual dividers between content sections with consistent styling options across the application.
