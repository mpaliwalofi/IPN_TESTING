# plugin.ts

**Path**: `packages\separator\tailwind\plugin.ts`

## Summary
This Tailwind CSS plugin provides reusable separator/divider components for UI layouts. It defines a base `.separator` class that renders a full-width border with primary color styling, along with modifier classes for dashed borders (`--dashed`) and vertical spacing (`--margin-top`, `--margin-bottom`). The plugin enables consistent visual separation between content sections throughout the application using Tailwind's utility-first approach.
