# package.json

**Path**: `packages\smartForm\package.json`

## Summary
This is a package configuration file for `@ultrapremiumdirect/smartform`, a Vue 3-based form component library that includes phone number validation capabilities (via libphonenumber-js) and internationalization support (vue-i18n). The package is built as an ES module using Vite, exports both the main library and a Tailwind CSS plugin, and is designed to be distributed as part of a monorepo workspace with TypeScript support and comprehensive testing infrastructure.
