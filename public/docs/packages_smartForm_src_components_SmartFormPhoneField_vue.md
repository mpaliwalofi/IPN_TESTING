# SmartFormPhoneField.vue

**Path**: `packages\smartForm\src\components\SmartFormPhoneField.vue`

## Summary
This Vue component implements a phone number input field with international formatting and validation capabilities. It uses the `libphonenumber-js` library to provide real-time phone number formatting as users type, automatically detecting and adjusting for different country codes. The component integrates with a smart form system, supporting i18n translations, markdown rendering for labels/help text, and error handling while maintaining reactive two-way data binding with the parent form.
