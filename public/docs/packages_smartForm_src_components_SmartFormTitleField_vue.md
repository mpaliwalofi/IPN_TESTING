# SmartFormTitleField.vue

**Path**: `packages\smartForm\src\components\SmartFormTitleField.vue`

## Summary
This Vue component renders a title field for a smart form system. It displays either custom slot content or a title from the field's parameters as an `&lt;h2&gt;` heading, serving as a section header within dynamically-generated forms. The component accepts an optional `SmartFieldTitleDefinition` prop that contains the title configuration data.
