# SmartFormToggleField.vue

**Path**: `packages\smartForm\src\components\SmartFormToggleField.vue`

## Summary
This Vue component renders a toggle button field for smart forms, allowing users to select between multiple predefined options by clicking buttons. It manages two-way data binding through `modelValue`, displays internationalized labels for each button option, handles validation errors, and supports optional icons and disabled states for the toggle buttons.
