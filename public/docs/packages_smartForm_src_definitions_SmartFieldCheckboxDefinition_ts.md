# SmartFieldCheckboxDefinition.ts

**Path**: `packages\smartForm\src\definitions\SmartFieldCheckboxDefinition.ts`

## Summary
This class defines a checkbox field configuration for a smart form system, extending the base `SmartFieldDefinition` class to provide checkbox-specific functionality. It encapsulates checkbox field properties (label, required, disabled states) and specifies that instances should render using the 'SmartFormCheckboxField' component, enabling declarative creation of checkbox form fields with validation and accessibility options.
