# SmartFieldEmailDefinition.ts

**Path**: `packages\smartForm\src\definitions\SmartFieldEmailDefinition.ts`

## Summary
This class defines a specialized email input field type for a smart form system. It extends the base `SmartFieldInputDefinition` class to create email-specific form fields by setting the input type to 'email' and configuring the `inputmode` attribute to 'email' for optimized mobile keyboard display. This allows the form system to handle email input fields with appropriate validation and user interface behavior distinct from generic text inputs.
