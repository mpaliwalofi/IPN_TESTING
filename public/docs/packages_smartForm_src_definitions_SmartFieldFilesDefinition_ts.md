# SmartFieldFilesDefinition.ts

**Path**: `packages\smartForm\src\definitions\SmartFieldFilesDefinition.ts`

## Summary
This class defines a smart form field specifically for file input handling within a form builder system. It extends `SmartFieldDefinition` to configure file upload fields with options like label, disabled state, search functionality, required validation, and a reduce property for data transformation. The class maps to a `SmartFormFilesField` Vue component and consolidates field configuration parameters for rendering file input controls in dynamic forms.
