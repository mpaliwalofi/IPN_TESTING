# SmartFieldHiddenDefinition.ts

**Path**: `packages\smartForm\src\definitions\SmartFieldHiddenDefinition.ts`

## Summary
This class defines the configuration for hidden form fields in a smart form system. It extends the base `SmartFieldDefinition` class and specifies that hidden fields should be rendered using the `SmartFormHiddenField` component, allowing forms to store and submit data that isn't visible to users. The `vmodel` property enables two-way data binding for the hidden field's value.
