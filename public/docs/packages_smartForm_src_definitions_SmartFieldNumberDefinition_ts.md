# SmartFieldNumberDefinition.ts

**Path**: `packages\smartForm\src\definitions\SmartFieldNumberDefinition.ts`

## Summary
This class defines a specialized form field for numeric input by extending the base `SmartFieldInputDefinition` class. It automatically configures the field type as 'number' and sets the inputmode to 'numeric', which optimizes mobile keyboard display for number entry. The class serves as a reusable definition for creating number input fields within the smartForm system with consistent behavior and mobile-friendly input handling.
