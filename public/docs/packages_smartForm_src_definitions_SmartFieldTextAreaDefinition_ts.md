# SmartFieldTextAreaDefinition.ts

**Path**: `packages\smartForm\src\definitions\SmartFieldTextAreaDefinition.ts`

## Summary
This class defines a text area field type for a dynamic form system. It extends `SmartFieldDefinition` to specify configuration options (label, disabled state, required validation) and maps to the `SmartFormTextAreaField` Vue component for rendering multi-line text input fields in smart forms.
