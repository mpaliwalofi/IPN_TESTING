# SmartFieldTitleDefinition.ts

**Path**: `packages\smartForm\src\definitions\SmartFieldTitleDefinition.ts`

## Summary
This class defines a title field type for a smart form system, extending the base `SmartFieldDefinition` class to create title/heading elements within forms. It specifies that instances should render using the `SmartFormTitleField` component and allows configuration of an optional title text through its constructor options, which gets passed to the component via the `Params` getter.
