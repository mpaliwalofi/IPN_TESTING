# SmartFieldToggleBoxesDefinition.ts

**Path**: `packages\smartForm\src\definitions\SmartFieldToggleBoxesDefinition.ts`

## Summary
This TypeScript class defines a toggle boxes field type for a smart form system, extending the base `SmartFieldDefinition` to support multiple-choice selection UI with configurable options. It manages toggle box-specific configuration including available choices (with properties like name, ID, priority), disabled states, and default values, ultimately rendering through the `SmartFormToggleBoxes` Vue component. The class serves as a strongly-typed definition layer that bridges form field configuration with the rendering layer in a form builder system.
