# SmartFieldToggleDefinition.ts

**Path**: `packages\smartForm\src\definitions\SmartFieldToggleDefinition.ts`

## Summary
This class defines a toggle button field type for a smart form system, extending the base `SmartFieldDefinition` class. It configures a multi-option toggle component with customizable buttons (each with value, label, and optional icon), supporting standard form field properties like disabled and required states, and maps to the `SmartFormToggleField` Vue component for rendering.
