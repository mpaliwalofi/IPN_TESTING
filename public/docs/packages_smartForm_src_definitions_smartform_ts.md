# smartform.ts

**Path**: `packages\smartForm\src\definitions\smartform.ts`

## Summary
This file serves as a central export barrel/index module for the SmartForm package's field definition types. It re-exports all individual field definition modules (birthday, email, phone, checkbox, etc.) and the core SmartForm definition, allowing consumers to import any field type definition from a single entry point rather than navigating to individual definition files.
