# supportedFiles.ts

**Path**: `packages\smartForm\src\static\supportedFiles.ts`

## Summary
This file exports a static array of supported MIME types for file upload/processing functionality in the smartForm package. It defines an allowlist of acceptable file formats including office documents (Excel, Word, PowerPoint), images, audio/video files, PDFs, and compressed archives, enabling the application to validate and handle a wide range of user-uploaded content types.
