## Summary
This is a Storybook configuration file that defines interactive documentation stories for a `SmartFormButtonsField` Vue component. It demonstrates two primary use cases: a radio button mode (single selection) and a checkbox mode (multiple selection), allowing developers to visualize and test different button field configurations within a form context. The stories showcase how the component handles button arrays with customizable properties like values, labels, and selection behavior.
