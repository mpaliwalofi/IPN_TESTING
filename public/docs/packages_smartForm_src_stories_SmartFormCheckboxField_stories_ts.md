## Summary
This is a Storybook configuration file that defines interactive documentation and visual test cases for a checkbox form field component (`SmartFormCheckboxField`). It exports three stories demonstrating different checkbox states: a default unchecked state, a checked state, and a disabled state, allowing developers to preview and test the checkbox component's behavior in isolation with various configurations including labels, form context, and model values.
