# SmartFormToggleField.stories.ts

**Path**: `packages\smartForm\src\stories\SmartFormToggleField.stories.ts`

## Summary
This is a Storybook configuration file that documents and demonstrates the `SmartFormToggleField` Vue component, which provides a toggle-style form field for selecting between mutually exclusive options (like choosing between dog or cat as animal gender). The file defines multiple story variants including a default empty state, a pre-selected value state, and a disabled state, using example data of animal gender selection with icons to showcase the component's functionality in different configurations.
