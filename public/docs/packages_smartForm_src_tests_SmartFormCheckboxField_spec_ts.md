# SmartFormCheckboxField.spec.ts

**Path**: `packages\smartForm\src\tests\SmartFormCheckboxField.spec.ts`

## Summary
This test file validates the `SmartFormCheckboxField` Vue component, which renders checkbox inputs within a smart form system. It tests that the component properly mounts with checkbox field definitions, handles two-way data binding through `modelValue` props, and correctly emits update events when the checkbox state changes via user interaction. The tests use a `FakeForm` class with Zod schema validation to simulate a real form context with a boolean checkbox field.
