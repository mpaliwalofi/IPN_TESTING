# .eslintrc-auto-import.json

**Path**: `packages\smartImage\.eslintrc-auto-import.json`

## Summary
This is an ESLint configuration file that defines global variables for auto-import functionality in a Vue.js project. It whitelists common Vue 3 Composition API functions, Vue Router composables, i18n utilities, and testing framework globals (like Vitest) so they can be used without explicit imports while avoiding ESLint's "undefined variable" warnings. This configuration enables a cleaner development experience by allowing developers to use these framework APIs directly without import statements in every file.
