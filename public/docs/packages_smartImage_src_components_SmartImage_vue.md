# SmartImage.vue

**Path**: `packages\smartImage\src\components\SmartImage.vue`

## Summary
This Vue 3 component provides a smart image loader with ImageKit integration for optimized image delivery. It handles responsive images with configurable sizes across different screen breakpoints, supports multiple image formats (WebP, AVIF), lazy loading, low-quality image placeholders (LQIP), and hover image switching. The component is designed for e-commerce use cases (note the `product` prop) and generates appropriate srcset/sizes attributes for responsive images with transformations applied through ImageKit's CDN.
