# SmartImage.stories.ts

**Path**: `packages\smartImage\src\stories\SmartImage.stories.ts`

## Summary
This Storybook configuration file defines visual documentation and test scenarios for a SmartImage Vue3 component. It demonstrates three main use cases: a default image display, an image with hover state functionality, and a squared image variant, all utilizing ImageKit for image optimization and transformation with responsive sizing configurations. The stories provide a development environment for testing the SmartImage component's integration with ImageKit's CDN service.
