# types.ts

**Path**: `packages\smartImage\src\types.ts`

## Summary
This TypeScript file defines the configuration types for a smart image management system that integrates with ImageKit CDN. It specifies the structure for ImageKit configuration options including base URLs and path settings for different image categories (general images, animals, and SVGs), along with URL detection patterns and the ImageKit client instance needed for image optimization and delivery operations.
