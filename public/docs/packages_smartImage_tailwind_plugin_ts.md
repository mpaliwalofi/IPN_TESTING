# plugin.ts

**Path**: `packages\smartImage\tailwind\plugin.ts`

## Summary
This file exports a Tailwind CSS plugin that provides utility classes for a smart image component system. The plugin defines styles for progressive image loading with LQIP (Low Quality Image Placeholder), overlay functionality, and hover effects, enabling features like blur-to-sharp transitions, grid-based image wrappers, and interactive overlays that appear on hover or focus.
