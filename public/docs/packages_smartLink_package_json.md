# package.json

**Path**: `packages\smartLink\package.json`

## Summary
This is a package configuration for a Vue 3-based "SmartLink" component library within the Ultra Premium Direct design system. It defines a modular package that exports ES modules for the SmartLink component, TypeScript type definitions, and a Tailwind CSS plugin, with build tooling configured using Vite and testing via Vitest. The package is designed to be consumed as part of a workspace monorepo architecture, providing reusable link components with internationalization support through vue-i18n.
