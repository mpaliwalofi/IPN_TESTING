# SmartLink.vue

**Path**: `packages\smartLink\src\components\SmartLink.vue`

## Summary
This Vue 3 component provides an intelligent link wrapper that automatically selects the appropriate HTML element (`&lt;a&gt;`, `&lt;span&gt;`, or NuxtLink) based on the link destination. It handles internationalized routing with `vue-i18n-routing`, distinguishes between internal and external URLs, manages trailing slashes, and includes special handling for hash-based navigation within the same route. The component serves as a flexible abstraction layer for navigation across different contexts in a Vue/Nuxt application while respecting locale paths and disabled states.
