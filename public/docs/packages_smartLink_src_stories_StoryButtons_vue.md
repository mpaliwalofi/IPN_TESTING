# StoryButtons.vue

**Path**: `packages\smartLink\src\stories\StoryButtons.vue`

## Summary
This is a Storybook story component that demonstrates and tests different variations of button/link components using the SmartLink component. It provides an interactive showcase of button styles, sizes, and HTML element types (links, spans, buttons) with a toggleable background for visual testing, and includes event tracking through Storybook actions when buttons are clicked.
