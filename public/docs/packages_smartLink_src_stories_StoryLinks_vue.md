# StoryLinks.vue

**Path**: `packages\smartLink\src\stories\StoryLinks.vue`

## Summary
This Vue component serves as a visual showcase/demo page for the SmartLink component, displaying various styling variants and use cases. It demonstrates different link types (external, internal, mailto, tel), styling options (primary, secondary, tertiary, neutral, error, disabled), and icon placements to help developers understand the SmartLink component's capabilities and available CSS classes.
