**Path**: `packages\tracking\package.json`

## Summary
This is a package configuration for a private TypeScript tracking module within the ultrapremiumdirect monorepo. It provides analytics/tracking functionality, specifically integrating Google Tag Manager for Vue applications through the `@gtm-support/vue-gtm` dependency. The package is built using unbuild and exports TypeScript definitions and ES modules for use by other packages in the workspace.
