# embed.ts

**Path**: `packages\types\src\cms\shared\embed.ts`

## Summary
This TypeScript file defines a type definition for an embeddable content component in a Strapi CMS system. The `IEmbed` interface extends the base Strapi component structure and specifies two properties: a boolean flag to control background display and an `oembed` string field for storing oEmbed data (typically used for embedding third-party content like videos, tweets, or other rich media). This type enables type-safe handling of embedded content blocks within the CMS's content management workflow.

## Interfaces
- `IEmbed`
