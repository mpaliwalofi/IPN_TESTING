# ingredient.ts

**Path**: `packages\types\src\cms\shared\ingredient.ts`

## Summary
This TypeScript file defines the data structure for a recipe or product ingredient within a CMS (Content Management System) context. It establishes an `IIngredient` interface that models ingredients with a title, description, and an associated image, where the image is wrapped in Strapi's entity relationship structure for content management purposes.

## Interfaces
- `IIngredient`
