# opengraph.ts

**Path**: `packages\types\src\cms\shared\opengraph.ts`

## Summary
This TypeScript file defines type interfaces for Open Graph metadata and SEO configuration in a CMS system. It provides type definitions for managing multilingual page metadata (`IHrefLang`, `ILang`) and Open Graph properties (`IOpengraph`) including SEO attributes like title, description, image, indexing directives, and internationalized URL variants. These types enable type-safe handling of social media preview data and search engine optimization settings within a Strapi-based content management system.

## Interfaces
- `IHrefLang`
- `ILang`
- `IOpengraph`
