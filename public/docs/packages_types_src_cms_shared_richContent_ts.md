# richContent.ts

**Path**: `packages\types\src\cms\shared\richContent.ts`

## Summary
This TypeScript file defines a type interface for rich content components in a Strapi CMS system. It extends the base `IStrapiComponent` interface to include a `markdown` field, enabling the storage and handling of markdown-formatted content within CMS components. This type is used to ensure type safety when working with rich text/markdown content blocks in the CMS data structure.

## Interfaces
- `IRichContent`
