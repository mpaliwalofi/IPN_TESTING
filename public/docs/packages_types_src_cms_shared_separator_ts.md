# separator.ts

**Path**: `packages\types\src\cms\shared\separator.ts`

## Summary
This TypeScript file defines a type interface for a CMS separator component in a Strapi-based content management system. The `ISeparator` interface extends the base Strapi component and provides optional boolean properties to control the separator's visual styling (dashed line style) and spacing behavior (top and bottom margins), allowing content editors to configure visual dividers between content sections.

- `ISeparator`
