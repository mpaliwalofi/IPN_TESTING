# strapiButton.ts

**Path**: `packages\types\src\cms\shared\strapiButton.ts`

## Summary
This TypeScript file defines the type definitions and constants for configuring buttons in a Strapi CMS integration. It specifies the available button color schemes (primary, neutral, order, etc.) and visual styles (solid, ghost, outline), along with an interface (`IStrapiButton`) that includes properties for link behavior (URL, mailto, target blank, nofollow), appearance settings, and optional icon imagery.
