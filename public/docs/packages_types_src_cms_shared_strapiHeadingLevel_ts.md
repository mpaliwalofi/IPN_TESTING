# strapiHeadingLevel.ts

**Path**: `packages\types\src\cms\shared\strapiHeadingLevel.ts`

## Summary
This TypeScript module defines a constant object mapping for HTML heading levels used in Strapi CMS content management. It establishes a strict enumeration of valid heading tags (h1-h6 plus span) that can be used throughout the application, and exports a corresponding type (`TStrapiHeadingLevel`) for type-safe heading level references in components that render CMS content.
