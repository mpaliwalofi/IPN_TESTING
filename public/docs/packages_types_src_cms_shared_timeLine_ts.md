# timeLine.ts

**Path**: `packages\types\src\cms\shared\timeLine.ts`

## Summary
This TypeScript file defines type interfaces for a CMS timeline component used in a Strapi-based content management system. It structures timeline data with individual timeline blocks (`ITimeLineBocks`) that can contain dates, titles, text content, images, videos, URLs, and SEO properties (nofollow), and an overall timeline container (`ITimeLine`) that holds an array of these blocks.

## Interfaces
- `ITimeLineBocks`
- `ITimeLine`
