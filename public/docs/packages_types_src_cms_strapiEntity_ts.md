# strapiEntity.ts

**Path**: `packages\types\src\cms\strapiEntity.ts`

## Summary
This file defines a TypeScript interface for representing entities returned by the Strapi CMS (Content Management System). It provides a generic type structure where Strapi entities have a numeric `id` field and an `attributes` field of generic type `T`, which matches Strapi's standard API response format for content entries.

## Interfaces
- `IStrapiEntity`
