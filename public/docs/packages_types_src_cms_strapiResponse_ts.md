# strapiResponse.ts

**Path**: `packages\types\src\cms\strapiResponse.ts`

## Summary
This TypeScript file defines the type structure for Strapi CMS API responses, including pagination metadata and response wrapper types. It provides interfaces for handling both single entity and array responses from Strapi, with optional pagination information (page number, page count, page size, and total records). These types enable type-safe handling of data fetched from Strapi's REST API in a TypeScript application.

## Interfaces
- `IPagination`
- `IMeta`
- `IStrapiResponse`
