# carousel.ts

**Path**: `packages\types\src\core\carousel.ts`

## Summary
This TypeScript file defines type definitions and interfaces for a carousel/slider component system. It specifies configuration options for carousel behavior (slides, scrolling, navigation arrows, dots, autoplay interval, infinite loop) and visual styling (different carousel styles like 'carousel', 'pdp', 'hero-slider', and responsive height settings across breakpoints). The types support both generic carousel implementations and specific use cases like product detail pages (PDP) and hero sliders in an e-commerce or content presentation context.
