# markdownIt.ts

**Path**: `packages\types\src\core\markdownIt.ts`

## Summary
This TypeScript definition file provides type interfaces for the markdown-it library, a Markdown parser. It defines the `MarkdownIt` interface with methods for parsing and rendering Markdown text (like `parse()`, `render()`, `parseInline()`), plus configuration methods for plugins and options, and the `MarkdownItToken` interface representing parsed Markdown tokens with their attributes, content, and nesting structure.

## Interfaces
- `MarkdownItToken`
- `MarkdownIt`
