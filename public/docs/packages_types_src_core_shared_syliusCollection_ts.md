# syliusCollection.ts

**Path**: `packages\types\src\core\shared\syliusCollection.ts`

## Summary
This TypeScript file defines a generic interface for Sylius API collection responses that follow the Hydra specification. It extends the base Sylius component interface to standardize paginated collection data structures, including an array of items (`hydra:member`) and a total count (`hydra:totalItems`), which are commonly used in Sylius e-commerce platform API responses.

## Interfaces
- `ISyliusCollection`
