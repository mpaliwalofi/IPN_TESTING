# supportedFiles.ts

**Path**: `packages\types\src\static\supportedFiles.ts`

## Summary
This file exports a default array of MIME type strings that defines the file types supported by the application for upload or processing. It includes a comprehensive list of common document formats (Office, PDF, text), images, audio/video files, and compressed archives, with inline comments indicating the corresponding file extensions. This serves as a whitelist configuration for file validation and filtering throughout the application.
