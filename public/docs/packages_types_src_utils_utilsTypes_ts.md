# utilsTypes.ts

**Path**: `packages\types\src\utils\utilsTypes.ts`

## Summary
This TypeScript utility file defines a generic type helper `TObjectValues&lt;T&gt;` that extracts the union type of all values from an object type. It serves as a foundational building block for creating type-safe enum-like patterns in TypeScript, allowing developers to define constant objects with `as const` and then derive types representing all possible values those objects can contain, ensuring compile-time type safety when working with enumerated values.
