# .eslintrc-auto-import.json

**Path**: `packages\verbatimFullWidth\.eslintrc-auto-import.json`

## Summary
This is an ESLint configuration file that defines auto-imported global variables for a Vue.js project with additional tooling. It allows developers to use Vue 3 composition API functions, Vue Router hooks, i18n utilities, and testing framework functions (like Vitest's `afterAll`, `beforeEach`, `assert`) without explicitly importing them, preventing ESLint from flagging them as undefined variables.
