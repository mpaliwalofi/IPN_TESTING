# VerbatimFullWidth.stories.ts

**Path**: `packages\verbatimFullWidth\src\stories\VerbatimFullWidth.stories.ts`

## Summary
This is a Storybook configuration file that defines visual test stories for a VerbatimFullWidth Vue3 component, which displays full-width quoted text or testimonials. It provides two story variants (Default and LongText) with automated play function tests that verify the component renders text correctly, applies appropriate CSS classes (verbatim-full-width, verbatim-full-width__text), and maintains proper structure across different viewport sizes (mobile, tablet, desktop) for visual regression testing with Chromatic.
