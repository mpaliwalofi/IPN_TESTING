# Youtube.stories.ts

**Path**: `packages\youtube\src\stories\Youtube.stories.ts`

## Summary
This Storybook configuration file defines interactive documentation and test scenarios for a YouTube video embedding component. It specifies multiple story variants (Default, ShortUrl, MobileUrl) that demonstrate how the component handles different YouTube URL formats (standard, shortened, mobile) and includes automated play functions that verify the iframe is correctly rendered with proper attributes, ID, and embed URL conversion. The stories also configure viewport settings for testing responsive behavior across mobile, tablet, and desktop breakpoints.
