# Youtube.spec.ts

**Path**: `packages\youtube\src\tests\Youtube.spec.ts`

## Summary
This is a test suite for a Vue.js YouTube embed component that verifies the component correctly transforms YouTube watch URLs into embeddable iframe elements. The tests ensure that the component mounts properly, converts standard YouTube URLs (like `/watch?v=`) into embed format (`/embed/`), assigns proper IDs with a `video-yt-` prefix, and includes required iframe attributes for YouTube video playback (allow permissions, allowfullscreen, etc.). The file exists to validate that the YouTube component correctly handles URL transformation and iframe rendering for embedding YouTube videos in a Vue application.
