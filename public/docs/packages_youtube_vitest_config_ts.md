# vitest.config.ts

**Path**: `packages\youtube\vitest.config.ts`

## Summary
This file configures Vitest (a testing framework) for the YouTube package by extending a base configuration. It merges the workspace-level base Vitest configuration without adding any package-specific test settings or overrides. This allows the YouTube package to inherit standardized testing behavior from the monorepo's shared configuration while maintaining the flexibility to add custom settings in the future.
