# README.md

**Path**: `plugins\PayPalPlugin\README.md`

## Summary
This README file documents the official PayPal Commerce Platform plugin for Sylius e-commerce framework. The plugin integrates PayPal payment processing into Sylius, providing features like multi-currency support (100+ currencies across 200+ markets), various payment methods (credit/debit cards, regional options like iDeal and Sofort), bidirectional payment operations between Sylius and PayPal admin panels, and streamlined checkout from product and cart pages. The documentation covers installation, onboarding, payment processing, and highlights key business benefits like PCI DSS compliance, fraud management, and access to PayPal's global network of 280+ million consumers.
