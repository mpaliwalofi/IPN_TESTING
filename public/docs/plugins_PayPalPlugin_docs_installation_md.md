# installation.md

**Path**: `plugins\PayPalPlugin\docs\installation.md`

## Summary
This installation guide provides step-by-step instructions for integrating the PayPal payment plugin into a Sylius e-commerce application. It covers installing the composer package, configuring routes for shop/admin/webhook endpoints, importing plugin configuration, overriding templates, and running database migrations. The guide emphasizes that the local Sylius installation must be publicly accessible for PayPal servers to communicate with webhooks and callbacks.
