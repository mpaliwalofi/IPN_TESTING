# onboarding.md

**Path**: `plugins\PayPalPlugin\docs\onboarding.md`

## Summary
This documentation provides step-by-step instructions for store owners to onboard their PayPal business account with the Sylius PayPal plugin. It guides users through the payment method setup process, including enabling PayPal in the admin panel, completing OAuth authorization with PayPal, and obtaining API credentials (client ID and secret) necessary for payment processing integration. The guide emphasizes critical requirements like using sandbox mode by default, ensuring HTTPS connectivity, and proper internet accessibility for webhook configuration.
