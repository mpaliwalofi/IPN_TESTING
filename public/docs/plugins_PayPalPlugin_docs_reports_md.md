# reports.md

**Path**: `plugins\PayPalPlugin\docs\reports.md`

## Summary
This documentation explains how to configure PayPal's SFTP-based reporting feature to download payout reports within the PayPalPlugin. It provides step-by-step instructions for enabling the reports feature on a PayPal account, creating an SFTP account through PayPal's profile settings, and configuring the SFTP credentials in the application's environment configuration to enable automated report retrieval.
