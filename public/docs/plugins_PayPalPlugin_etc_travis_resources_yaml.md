# resources.yaml

**Path**: `plugins\PayPalPlugin\etc\travis\resources.yaml`

## Summary
This YAML configuration file defines a Sylius resource mapping for the PayPal plugin, specifically overriding the default Order entity to use a custom `Sylius\PayPalPlugin\Entity\Order` model class. It exists to integrate PayPal-specific functionality into Sylius's order management system by extending or modifying the standard order entity with PayPal-related features and properties.
