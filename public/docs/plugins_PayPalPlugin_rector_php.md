# rector.php

**Path**: `plugins\PayPalPlugin\rector.php`

## Summary
This is a Rector configuration file for the PayPal plugin that automates code refactoring and modernization. It configures Rector to automatically add PHP 7.4 typed properties to class properties while enabling auto-import of namespaces (but excluding short class names). The configuration helps maintain code quality standards by automatically upgrading legacy code to use modern PHP type declarations.
