## Summary
This is a PHPSpec test specification for a PayPal SFTP payouts report downloader in the Sylius e-commerce platform. It tests the functionality of downloading payout report files from PayPal's SFTP server using payment method credentials, specifically retrieving PYT (payout) CSV reports for a given date and returning them as Report objects with the file content and standardized filename.
