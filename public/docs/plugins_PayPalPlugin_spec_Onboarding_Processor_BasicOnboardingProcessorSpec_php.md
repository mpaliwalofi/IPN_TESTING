## Summary
This is a PhpSpec test specification file for the `BasicOnboardingProcessor` class in the PayPal plugin. It tests the onboarding process that configures PayPal payment methods by verifying merchant credentials, exchanging onboarding tokens with PayPal's facilitator API, and registering seller webhooks. The spec ensures the processor correctly handles the OAuth flow for merchant account setup, validates permissions, and updates gateway configurations with merchant IDs and authentication tokens.
