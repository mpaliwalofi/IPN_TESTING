# RefundPaymentApiInterface.php

**Path**: `plugins\PayPalPlugin\src\Api\RefundPaymentApiInterface.php`

## Summary
This interface defines a contract for refunding PayPal payments within the Sylius PayPal plugin. It specifies a single method that processes refunds by accepting authentication credentials (token and assertion), payment identification details (payment ID and invoice number), and refund amount information (amount and currency code), returning the refund operation result as an array.
