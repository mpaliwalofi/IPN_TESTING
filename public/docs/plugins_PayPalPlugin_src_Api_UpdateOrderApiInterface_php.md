# UpdateOrderApiInterface.php

**Path**: `plugins\PayPalPlugin\src\Api\UpdateOrderApiInterface.php`

## Summary
This interface defines a contract for updating PayPal orders within a Sylius e-commerce payment plugin. It specifies a single method that takes PayPal authentication/identification parameters (token, order ID, merchant ID, reference ID) along with a Sylius payment object to synchronize or modify an existing PayPal order, returning the updated order data as an array.
