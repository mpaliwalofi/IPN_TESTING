## Summary
The `UpdateOrderApi` class is responsible for updating PayPal order details during the payment process in a Sylius e-commerce application. It constructs and sends updated order information (including purchase units, items, shipping address, and reference numbers) to PayPal's API via the PayPal client, synchronizing the local order state with PayPal's system during checkout or payment modifications.
