## Summary
This file defines a custom exception class for handling PayPal client authorization failures in the Sylius PayPal plugin. It extends PHP's base Exception class and provides a predefined error message indicating that the PayPal client could not be authenticated or authorized. This exception is thrown when OAuth or API credential validation fails during PayPal integration operations.
