## Summary
This exception class is thrown when a PayPal order refund operation fails within the Sylius PayPal Plugin. It extends PHP's base Exception class and provides a specific, pre-defined error message indicating that a refund transaction could not be completed, allowing the application to handle PayPal refund failures distinctly from other exceptions.
