# PayPalOrderUpdateException.php

**Path**: `plugins\PayPalPlugin\src\Exception\PayPalOrderUpdateException.php`

## Summary
This is a custom exception class for the Sylius PayPal plugin that handles failures when updating PayPal orders. It extends PHP's base Exception class and provides a predefined error message "Could not update PayPal order" to signal when PayPal order update operations fail. This exception allows the plugin to distinguish PayPal order update failures from other types of errors in the payment processing workflow.
