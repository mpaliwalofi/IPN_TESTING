# PayPalPurchaseUnit.php

**Path**: `plugins\PayPalPlugin\src\Model\PayPalPurchaseUnit.php`

## Summary
The `PayPalPurchaseUnit` class is a data model that represents a purchase unit for PayPal transactions within the Sylius e-commerce platform. It encapsulates all the financial and shipping details needed for a PayPal order, including amounts (total, shipping, items, tax, discounts), currency, merchant information, line items, and optional shipping address. This class serves as a structured container to transfer complete purchase information between the Sylius system and PayPal's payment API.
