# CaptureAction.php

**Path**: `plugins\PayPalPlugin\src\Payum\Action\CaptureAction.php`

## Summary
The `CaptureAction` class implements a Payum payment action that handles PayPal payment capture requests within the Sylius e-commerce framework. It authorizes the PayPal client, creates a PayPal order with a unique reference ID, and updates the payment details with the capture status and PayPal order information when the order creation is successful.
