**Path**: `plugins\PayPalPlugin\src\Payum\Action\ResolveNextRouteAction.php`

## Summary
This Payum action determines the next route to redirect users to after a PayPal payment attempt in a Sylius e-commerce system. It routes users to different pages based on payment state: to the PayPal payment form if payment is new, to the thank-you page if completed, or to the order details page for other states.
