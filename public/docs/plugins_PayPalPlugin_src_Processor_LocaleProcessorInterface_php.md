# LocaleProcessorInterface.php

**Path**: `plugins\PayPalPlugin\src\Processor\LocaleProcessorInterface.php`

## Summary
This interface defines a contract for processing locale strings within the PayPal plugin for Sylius e-commerce platform. It exists to standardize the transformation of Sylius locale codes into PayPal-compatible locale formats, ensuring proper internationalization when communicating with PayPal's API.
