# AvailableCountriesProviderInterface.php

**Path**: `plugins\PayPalPlugin\src\Provider\AvailableCountriesProviderInterface.php`

## Summary
This interface defines a contract for providers that retrieve a list of available countries where PayPal payment services can be used. It serves as part of the PayPal plugin's configuration system, allowing Sylius e-commerce applications to determine which countries support PayPal transactions and restrict payment options accordingly based on geographic availability.
