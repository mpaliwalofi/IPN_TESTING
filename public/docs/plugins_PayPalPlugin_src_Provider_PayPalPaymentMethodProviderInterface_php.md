# PayPalPaymentMethodProviderInterface.php

**Path**: `plugins\PayPalPlugin\src\Provider\PayPalPaymentMethodProviderInterface.php`

## Summary
This interface defines a contract for retrieving the PayPal payment method configuration within a Sylius e-commerce application. It provides a single method that returns a `PaymentMethodInterface` instance representing the PayPal payment gateway, throwing a specific exception if the PayPal payment method is not found or configured in the system.
