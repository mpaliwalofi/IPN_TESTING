# flashes.en.yml

**Path**: `plugins\PayPalPlugin\src\Resources\translations\flashes.en.yml`

## Summary
This YAML file defines English translation messages for flash notifications in the PayPal plugin for Sylius e-commerce platform. It contains user-facing error and success messages related to PayPal seller onboarding, order/payment cancellations, payment enablement status, and webhook URL validation, providing standardized feedback for various PayPal integration scenarios.
