# flashes.fr.yml

**Path**: `plugins\PayPalPlugin\src\Resources\translations\flashes.fr.yml`

## Summary
This YAML file contains French translations for flash messages in the PayPal plugin for Sylius e-commerce platform. It defines user-facing error and status messages related to PayPal payment operations, including seller management, order/payment cancellations, payment gateway activation failures, and webhook validation errors.
