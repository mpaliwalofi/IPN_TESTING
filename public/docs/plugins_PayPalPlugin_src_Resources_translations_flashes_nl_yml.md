# flashes.nl.yml

**Path**: `plugins\PayPalPlugin\src\Resources\translations\flashes.nl.yml`

## Summary
This is a Dutch (nl) language translation file for PayPal plugin flash messages in a Sylius e-commerce application. It provides user-facing error and status messages for PayPal integration scenarios including payment cancellations, seller configuration limits, payment enablement failures, and webhook URL validation errors.
