# messages.en.yml

**Path**: `plugins\PayPalPlugin\src\Resources\translations\messages.en.yml`

## Summary
This is an English translation file for the PayPal payment integration plugin in a Sylius e-commerce platform. It provides user-facing text labels and messages for PayPal payment configuration, checkout flow (including credit card fields, billing addresses, and payment method selection), and administrative settings like API credentials (Client ID/Secret) and SFTP configuration for transaction reporting.
