# messages.fr.yml

**Path**: `plugins\PayPalPlugin\src\Resources\translations\messages.fr.yml`

## Summary
This file contains French language translations for the PayPal payment plugin in a Sylius e-commerce application. It provides localized UI labels, error messages, and help text for PayPal payment configuration and checkout flow, including credit card fields, billing addresses, SFTP credentials, and dynamic URL parameters for payment return/cancellation handling.
