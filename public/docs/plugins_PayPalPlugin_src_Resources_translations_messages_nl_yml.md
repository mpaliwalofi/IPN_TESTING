# messages.nl.yml

**Path**: `plugins\PayPalPlugin\src\Resources\translations\messages.nl.yml`

## Summary
This is a Dutch (nl) language translation file for the PayPal payment plugin in a Sylius e-commerce platform. It provides localized text strings for PayPal payment integration features including payment forms, configuration settings (client credentials, SFTP details), error messages, and user-facing payment interface elements like credit card fields and return URLs.
