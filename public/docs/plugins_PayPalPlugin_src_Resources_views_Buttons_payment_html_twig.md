# payment.html.twig

**Path**: `plugins\PayPalPlugin\src\Resources\views\Buttons\payment.html.twig`

## Summary
This Twig template renders PayPal payment buttons on the payment page of a Sylius e-commerce checkout flow. It delegates to a controller action that generates the appropriate PayPal button UI for the current shopping cart/order, enabling customers to complete their purchase using PayPal as a payment method.
