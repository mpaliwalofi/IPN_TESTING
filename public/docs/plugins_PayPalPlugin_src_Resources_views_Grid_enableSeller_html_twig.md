# enableSeller.html.twig

**Path**: `plugins\PayPalPlugin\src\Resources\views\Grid\enableSeller.html.twig`

## Summary
This Twig template renders a PayPal seller account enablement button in a grid view. It conditionally displays a form with an "Enable" button only when the gateway configuration is for PayPal (`sylius.pay_pal`) and the seller account is currently disabled, allowing administrators to activate PayPal payment processing for merchants.
