# _payment.html.twig

**Path**: `plugins\PayPalPlugin\src\Resources\views\bundles\SyliusAdminBundle\Order\Show\_payment.html.twig`

## Summary
This Twig template renders the payment details section for an order in the Sylius admin panel, specifically for the PayPal plugin. It displays payment information including the payment method, amount, and current state, while providing action buttons to complete or refund payments based on state machine permissions, with special handling for PayPal-specific refunded payment states.
