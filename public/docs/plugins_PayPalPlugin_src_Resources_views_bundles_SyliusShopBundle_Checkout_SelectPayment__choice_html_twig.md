# _choice.html.twig

**Path**: `plugins\PayPalPlugin\src\Resources\views\bundles\SyliusShopBundle\Checkout\SelectPayment\_choice.html.twig`

## Summary
This Twig template renders a payment method selection option in the Sylius shop checkout process. It displays a radio button for selecting the payment method along with its label and description, and specifically for PayPal payment methods (identified by the 'sylius.pay_pal' factory name), it additionally renders PayPal-specific payment buttons by invoking the PayPalButtonsController.
