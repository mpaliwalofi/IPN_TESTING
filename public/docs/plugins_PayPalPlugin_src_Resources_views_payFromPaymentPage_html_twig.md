# payFromPaymentPage.html.twig

**Path**: `plugins\PayPalPlugin\src\Resources\views\payFromPaymentPage.html.twig`

## Summary
This Twig template renders a PayPal payment button integration for a Sylius e-commerce payment page. It loads the PayPal SDK and implements the complete payment flow including order creation, approval handling, cancellation, and error management through AJAX calls to backend endpoints. The template configures the PayPal button with customizable styling, locale/currency settings, and handles the full payment lifecycle from order initiation through completion or cancellation.
