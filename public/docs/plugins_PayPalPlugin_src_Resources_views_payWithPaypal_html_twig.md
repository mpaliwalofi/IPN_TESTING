# payWithPaypal.html.twig

**Path**: `plugins\PayPalPlugin\src\Resources\views\payWithPaypal.html.twig`

## Summary
This Twig template file provides a standalone PayPal payment page for the Sylius e-commerce platform's PayPal plugin. It renders a dedicated checkout interface where customers are redirected to complete their PayPal payment, featuring a centered layout with the Sylius logo, payment UI, and a back button for navigation. The template is designed as a self-contained payment flow page with responsive styling optimized for mobile and desktop viewports.
