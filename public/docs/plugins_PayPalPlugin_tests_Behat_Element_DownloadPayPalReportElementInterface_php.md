# DownloadPayPalReportElementInterface.php

**Path**: `plugins\PayPalPlugin\tests\Behat\Element\DownloadPayPalReportElementInterface.php`

## Summary
This interface defines the contract for a Behat test element that handles downloading PayPal payment reports in CSV format. It provides methods to trigger report downloads for specific payment methods and verify that the CSV report file was successfully downloaded, supporting end-to-end testing of PayPal reporting functionality in the Sylius e-commerce platform.
