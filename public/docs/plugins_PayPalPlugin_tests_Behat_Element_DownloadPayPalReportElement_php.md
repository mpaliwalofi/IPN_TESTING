**Path**: `plugins\PayPalPlugin\tests\Behat\Element\DownloadPayPalReportElement.php`

## Summary
This Behat test element class provides functionality for testing PayPal report downloads in the Sylius PayPal plugin. It enables automated tests to trigger report downloads by clicking the "Report" link for a specific payment method and verify that a CSV file is successfully downloaded by checking the HTTP response status and content-type headers.
