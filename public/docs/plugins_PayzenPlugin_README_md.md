# README.md

**Path**: `plugins\PayzenPlugin\README.md`

## Summary
This is a README documentation file for a Sylius e-commerce plugin that integrates PayZen payment gateway functionality via the Payum payment abstraction library. The plugin enables merchants to accept payments through PayZen by installing the bundle via Composer and configuring it as a payment method in the Sylius admin interface. It provides both automated installation instructions and manual setup steps for integrating PayZen payment processing into Sylius-based online stores.
