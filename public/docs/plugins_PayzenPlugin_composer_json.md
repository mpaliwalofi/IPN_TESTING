# composer.json

**Path**: `plugins\PayzenPlugin\composer.json`

## Summary
This is a Composer package configuration file for a Sylius e-commerce plugin that integrates the Payzen payment gateway. The bundle bridges Sylius's payment system with Lyra's REST PHP SDK to enable merchants to process payments through the Payzen/Lyra payment service, supporting Sylius versions 1.12+ and 2.0+ on PHP 8.0+.
