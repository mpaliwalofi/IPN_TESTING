# AntilopSyliusPayzenBundle.php

**Path**: `plugins\PayzenPlugin\src\AntilopSyliusPayzenBundle.php`

## Summary
This is a Symfony bundle class that integrates Payzen payment gateway functionality into a Sylius e-commerce application. It serves as the main entry point for the bundle, registering the dependency injection container extension (`AntilopSyliusPayzenExtension`) that configures the Payzen payment services and resources needed for processing payments through the Payzen platform.
