# AntilopSyliusPayzenExtension.php

**Path**: `plugins\PayzenPlugin\src\DependencyInjection\AntilopSyliusPayzenExtension.php`

## Summary
This is a Symfony Dependency Injection extension class for the Payzen payment gateway plugin in a Sylius e-commerce application. It loads the plugin's service definitions from a YAML configuration file and registers them with the Symfony service container under the 'payzen' alias, enabling the integration of Payzen payment processing functionality into the Sylius shop.
