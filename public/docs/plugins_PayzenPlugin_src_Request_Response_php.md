# Response.php

**Path**: `plugins\PayzenPlugin\src\Request\Response.php`

## Summary
This class serves as a custom Payum request type for handling Payzen payment gateway responses within a Sylius e-commerce application. It extends Payum's Generic request class to provide a typed wrapper for processing payment response data from the Payzen payment provider, enabling proper request/response handling in the Payum payment processing workflow.
