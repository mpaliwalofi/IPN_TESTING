# messages.en_US.yml

**Path**: `plugins\PayzenPlugin\src\Resources\translations\messages.en_US.yml`

## Summary
This YAML file defines English (US) translations for the Payzen payment gateway plugin configuration form in Sylius e-commerce platform. It provides human-readable labels for gateway settings such as site credentials (Site ID, certificate), operational modes (CTX mode, debug), installment payment options (N times payment with count and period), and REST API configuration parameters (endpoint URL, password, public key).
