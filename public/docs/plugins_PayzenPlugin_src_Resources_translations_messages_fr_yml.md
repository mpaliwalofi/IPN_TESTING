# messages.fr.yml

**Path**: `plugins\PayzenPlugin\src\Resources\translations\messages.fr.yml`

## Summary
This is a French language translation file for the Payzen payment gateway plugin in Sylius e-commerce platform. It provides French labels for Payzen gateway configuration fields including endpoint URLs, site credentials, payment installment options (n-times payment with count and period settings), and REST API authentication parameters. The file enables French-speaking merchants to configure Payzen payment processing with localized interface text.
