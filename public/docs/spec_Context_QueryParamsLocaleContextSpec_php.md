**Path**: `spec\Context\QueryParamsLocaleContextSpec.php`

## Summary
This is a PHPSpec test specification for `QueryParamsLocaleContext`, which determines the user's locale from URL query parameters in a Sylius-based application. The spec verifies that the context correctly extracts locale codes (like 'nl_BE', 'fr_FR') from the `locale` query parameter in HTTP requests, and throws a `LocaleNotFoundException` when no locale parameter is present.
