# PlanificationChangeSubscriptionActivity.php

**Path**: `src\Activity\Command\Subscription\PlanificationChangeSubscriptionActivity.php`

## Summary
This class is a Data Transfer Object (DTO) that represents an activity for tracking changes to a subscription's planification (scheduling). It captures both the old and new values when a subscription's interval or shipping date is modified, providing an audit trail for subscription schedule modifications in what appears to be a subscription-based delivery or service system.
