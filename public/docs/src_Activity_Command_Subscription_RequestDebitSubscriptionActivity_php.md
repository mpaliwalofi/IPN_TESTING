# RequestDebitSubscriptionActivity.php

**Path**: `src\Activity\Command\Subscription\RequestDebitSubscriptionActivity.php`

## Summary
This is a command activity Data Transfer Object (DTO) for initiating subscription payment debits in a subscription billing system. It encapsulates the necessary information to request a debit transaction, including the subscription ID, the specific order being charged, and a priority flag to handle urgent payment processing differently from standard recurring charges.
