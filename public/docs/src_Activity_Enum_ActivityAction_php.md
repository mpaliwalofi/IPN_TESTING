## Summary
This file defines an enumeration of activity actions related to subscription management in the application's activity tracking system. It provides a type-safe set of string-backed constants representing different subscription lifecycle events, including creation, plan changes, payment method updates, and debit operations. The enum serves as a standardized vocabulary for logging or auditing subscription-related activities throughout the application.
