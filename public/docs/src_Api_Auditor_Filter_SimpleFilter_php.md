# SimpleFilter.php

**Path**: `src\Api\Auditor\Filter\SimpleFilter.php`

## Summary
The `SimpleFilter` class is a decorator for the DHAuditor library's filter system that generates SQL WHERE clause conditions for querying audit logs in Doctrine. It supports three filtering modes: IN clause for multiple values, LIKE clause with wildcards for custom multi-field searches, and equals comparison for single values, enabling flexible filtering of audit trail data based on field names and values.
