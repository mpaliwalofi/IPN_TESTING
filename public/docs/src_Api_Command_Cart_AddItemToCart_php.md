# AddItemToCart.php

**Path**: `src\Api\Command\Cart\AddItemToCart.php`

## Summary
This is a Data Transfer Object (DTO) command class for adding items to a shopping cart in a Sylius-based e-commerce system. It encapsulates all necessary parameters for the cart addition operation, including the product variant, quantity, optional subscription interval, upsell flag, and source item tracking, while using the `OrderTokenValueAware` attribute to identify which cart to add the item to.
