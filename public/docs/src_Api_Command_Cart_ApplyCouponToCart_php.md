# ApplyCouponToCart.php

**Path**: `src\Api\Command\Cart\ApplyCouponToCart.php`

## Summary
This class represents a command for applying a promotion coupon code to a shopping cart in a Sylius e-commerce system. It extends `OrderPromotionCoupon` and uses the `OrderTokenValueAware` attribute to enable token-based cart identification, allowing customers to add discount coupons to their orders through the API.
