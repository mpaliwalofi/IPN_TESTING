# CreatePromotionForCart.php

**Path**: `src\Api\Command\Cart\CreatePromotionForCart.php`

## Summary
This command class represents a data transfer object (DTO) for creating a promotion and applying it to a shopping cart in a Sylius e-commerce system. It carries the cart's order token identifier and promotion details, and is specifically designed for API operations where administrators need to add promotions to carts, as indicated by the serialization group configuration.
