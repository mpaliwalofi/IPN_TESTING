# ItemQuantityEligibilityInterface.php

**Path**: `src\Api\Command\Cart\ItemQuantityEligibilityInterface.php`

## Summary
This interface defines a contract for checking whether a cart item's quantity is eligible or valid within the cart context. It serves as a marker interface in the e-commerce cart domain to enable polymorphic eligibility checking strategies for item quantities, such as validating stock availability, quantity limits, or business rules before adding or updating items in a shopping cart.
