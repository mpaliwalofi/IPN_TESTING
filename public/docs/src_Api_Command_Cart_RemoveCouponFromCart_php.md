# RemoveCouponFromCart.php

**Path**: `src\Api\Command\Cart\RemoveCouponFromCart.php`

## Summary
This class represents a command for removing a promotion coupon from a shopping cart in a Sylius e-commerce application. It extends the `OrderPromotionCoupon` base class and uses the `OrderTokenValueAware` attribute to ensure proper cart identification via token, serving as a data transfer object (DTO) in the CQRS pattern for the cart coupon removal operation.
