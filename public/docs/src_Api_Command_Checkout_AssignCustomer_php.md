# AssignCustomer.php

**Path**: `src\Api\Command\Checkout\AssignCustomer.php`

## Summary
This is a Sylius e-commerce command object that represents the action of assigning a customer to an existing checkout/order. It serves as a data transfer object (DTO) carrying the order token identifier and customer information, used within the API layer to process customer assignment during the checkout flow. The `OrderTokenValueAware` attribute indicates this command is part of Sylius's order token-based checkout system for managing cart-to-order transitions.
