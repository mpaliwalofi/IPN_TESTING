# PatchOrder.php

**Path**: `src\Api\Command\Checkout\PatchOrder.php`

## Summary
This is a Data Transfer Object (DTO) command class for patching/updating order information during the checkout process in a Sylius e-commerce application. It captures marketing and analytics-related data including Google Click ID (gclid), Google Analytics session information, user agent details, and consent storage preferences for advertising and analytics tracking, allowing this data to be associated with an order identified by its token value.
