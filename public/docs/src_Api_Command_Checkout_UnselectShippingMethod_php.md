# UnselectShippingMethod.php

**Path**: `src\Api\Command\Checkout\UnselectShippingMethod.php`

## Summary
This class represents a command for deselecting a shipping method from a specific shipment during the checkout process in a Sylius-based e-commerce application. It carries the order token and shipment ID required to identify which shipment's shipping method selection should be removed, acting as a data transfer object (DTO) for the unselect shipping method operation.
