## Summary
This command class represents an intent to add a tracking number to a shipment in an e-commerce order system built with Sylius. It captures the order token (for authentication/identification) and the tracking number to be associated with the order's shipment, serving as a data transfer object in a CQRS-style architecture for order fulfillment operations.
