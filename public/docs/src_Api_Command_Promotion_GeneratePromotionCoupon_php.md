# GeneratePromotionCoupon.php

**Path**: `src\Api\Command\Promotion\GeneratePromotionCoupon.php`

## Summary
This class represents a command for generating multiple promotion coupons in bulk for a Sylius e-commerce platform. It extends the base promotion coupon functionality to add bulk generation capabilities, allowing specification of quantity (default 5 coupons), code length (default 6 characters), and optional prefix/suffix for the generated coupon codes, while inheriting standard coupon properties like usage limits and expiration dates.
