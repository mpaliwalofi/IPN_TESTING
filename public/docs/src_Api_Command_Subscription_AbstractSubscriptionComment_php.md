## Summary
This abstract class serves as a base data transfer object (DTO) for subscription-related comment commands in the API layer. It encapsulates the essential data needed when adding or managing comments on subscriptions: a required subscription identifier (`subscriptionCode`) and an optional comment message, providing a common structure for concrete subscription comment command implementations to extend.
