# ChangeItemQuantitySubscription.php

**Path**: `src\Api\Command\Subscription\ChangeItemQuantitySubscription.php`

## Summary
This class is a command DTO (Data Transfer Object) for modifying the quantity of an item within a subscription. It encapsulates the parameters needed to change a subscription item's quantity, including the subscription identifier, item identifier, new quantity value, and an option to spread the change across subscription orders, primarily used in admin contexts for subscription management operations.
