**Path**: `src\Api\Dto\FrequencyDto.php`

## Summary
This Data Transfer Object (DTO) represents frequency data in an admin context for a Sylius-based application, exposing frequency information through both REST and GraphQL APIs. It contains frequency properties including an identifier, label, maximum capacity, and command count, primarily used for read-only collection operations to display frequency statistics or management data in the admin interface.
