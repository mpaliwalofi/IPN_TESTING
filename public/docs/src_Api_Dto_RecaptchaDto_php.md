# RecaptchaDto.php

**Path**: `src\Api\Dto\RecaptchaDto.php`

## Summary
This Data Transfer Object (DTO) represents reCAPTCHA configuration data in the admin API, exposing the reCAPTCHA site key through both REST and GraphQL endpoints. It serves as a read-only resource that allows admin users to retrieve the public site key needed for implementing Google reCAPTCHA on the frontend, with data provided by a custom `ItemProvider` state provider.
