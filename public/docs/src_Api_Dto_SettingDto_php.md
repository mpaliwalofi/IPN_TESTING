# SettingDto.php

**Path**: `src\Api\Dto\SettingDto.php`

## Summary
This DTO (Data Transfer Object) defines an API resource for exposing application settings through both REST and GraphQL endpoints in the admin area. It provides read-only operations to retrieve settings either as a collection (`/admin/settings`) or individually by key (`/admin/settings/{key}`), with each setting containing a key-value pair structure handled by custom state providers.
