# SubscriptionMonitoringDto.php

**Path**: `src\Api\Dto\SubscriptionMonitoringDto.php`

## Summary
This Data Transfer Object (DTO) represents subscription monitoring data for an admin API, exposing metrics about subscription capacity and order volumes over time. It provides both REST and GraphQL endpoints with date-range filtering capabilities to retrieve monitoring statistics including maximum capacity, number of commands/orders, and frequency breakdowns for subscription management and analysis purposes.
