## Summary
This class serves as a Data Transfer Object (DTO) for representing a shipping method option in a shopping cart context within the Sylius e-commerce framework. It encapsulates a shipping method along with its calculated price, providing a simple view model that combines shipping method details with their associated cost for API responses or cart displays.
