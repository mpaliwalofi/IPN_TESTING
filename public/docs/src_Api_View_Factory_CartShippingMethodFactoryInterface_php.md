# CartShippingMethodFactoryInterface.php

**Path**: `src\Api\View\Factory\CartShippingMethodFactoryInterface.php`

## Summary
This interface defines a factory contract for creating cart shipping method view objects in an e-commerce API context. It transforms Sylius shipping method entities along with their associated cost into API-friendly view representations (`CartShippingMethodInterface`) that can be consumed by frontend clients during the checkout process.
