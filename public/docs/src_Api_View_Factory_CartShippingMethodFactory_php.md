# CartShippingMethodFactory.php

**Path**: `src\Api\View\Factory\CartShippingMethodFactory.php`

## Summary
This factory class creates `CartShippingMethod` view objects by combining a Sylius shipping method with its calculated cost. It serves as part of the API layer to transform domain shipping method entities into API-ready view models that include pricing information for cart operations.
