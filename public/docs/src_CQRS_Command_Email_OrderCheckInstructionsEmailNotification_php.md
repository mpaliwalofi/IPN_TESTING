# OrderCheckInstructionsEmailNotification.php

**Path**: `src\CQRS\Command\Email\OrderCheckInstructionsEmailNotification.php`

## Summary
This class represents a CQRS command for sending email notifications with check payment instructions to customers who selected the check/cheque payment method for their order. It extends an abstract order number class and defines a constant identifying the specific payment method ("Cheque") that triggers this notification, serving as part of the order fulfillment workflow for offline payment processing.
