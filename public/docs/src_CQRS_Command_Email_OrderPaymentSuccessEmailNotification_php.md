**Path**: `src\CQRS\Command\Email\OrderPaymentSuccessEmailNotification.php`

## Summary
This class represents a CQRS command for triggering a payment success email notification to customers when their order payment is completed successfully. It extends `AbstractOrderNumber`, indicating it carries order identification data needed to send the appropriate confirmation email, serving as a message object in the application's command bus for handling post-payment email communications.
