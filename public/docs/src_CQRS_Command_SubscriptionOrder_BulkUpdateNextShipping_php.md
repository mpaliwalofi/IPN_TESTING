# BulkUpdateNextShipping.php

**Path**: `src\CQRS\Command\SubscriptionOrder\BulkUpdateNextShipping.php`

## Summary
This is a CQRS command class that represents a bulk operation to update the next shipping date for multiple subscription orders. It serves as a data transfer object (DTO) that encapsulates an array of subscription orders and their new next shipping date, enabling administrators to efficiently reschedule multiple subscription deliveries in a single operation. The command is part of the subscription order management domain and is specifically marked for admin-level access through serialization groups.
