**Path**: `src\CQRS\Query\Subscription\GetNextShippingSubscriptionsQuery.php`

## Summary
This query class is used to retrieve subscriptions that are scheduled for their next shipping cycle. It serves as a data transfer object in a CQRS (Command Query Responsibility Segregation) pattern to fetch subscription records that need to be processed for upcoming shipment fulfillment. The class acts as a request object that will be handled by a corresponding query handler to identify subscriptions due for delivery.
