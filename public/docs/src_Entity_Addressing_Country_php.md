# Country.php

**Path**: `src\Entity\Addressing\Country.php`

## Summary
The `Country` entity represents geographical countries in an addressing/location system, extending Sylius e-commerce framework's base Country model. It exposes country data through a GraphQL API (with query and collection operations) while disabling REST operations, and provides search and ordering capabilities through API filters for querying country information.
