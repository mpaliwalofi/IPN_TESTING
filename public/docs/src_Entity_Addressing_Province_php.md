# Province.php

**Path**: `src\Entity\Addressing\Province.php`

## Summary
This class represents a geographic province entity in a Sylius-based e-commerce application's addressing system. It extends the base Sylius Province model to provide ORM mapping for storing provincial/state data in the `sylius_province` database table. The class serves as a domain entity for managing sub-country administrative divisions used in address management and shipping calculations.
