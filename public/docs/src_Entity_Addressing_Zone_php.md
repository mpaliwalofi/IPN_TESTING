# Zone.php

**Path**: `src\Entity\Addressing\Zone.php`

## Summary
The `Zone` class represents a geographic zone entity for addressing purposes in a Sylius-based e-commerce application. It extends the base Sylius Zone model and configures it as an API Platform resource with GraphQL query support (both single item and paginated collection queries), along with search and ordering filters. This entity is used to define geographical regions for purposes like shipping zones, tax zones, or other location-based business rules.
