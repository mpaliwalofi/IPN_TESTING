# AnimalTypeTranslation.php

**Path**: `src\Entity\Animal\AnimalTypeTranslation.php`

## Summary
This entity class manages translations for animal type names in a multi-language e-commerce system built with Sylius. It stores localized versions of animal type labels (e.g., "Dog" in English, "Chien" in French) with a unique constraint ensuring one translation per locale per animal type, and exposes this data through a GraphQL API for querying translated animal type names.
