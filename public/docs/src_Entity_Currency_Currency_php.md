## Summary
This class represents a Currency entity in an e-commerce application built with Sylius framework. It extends the base Sylius Currency model and exposes currency data through a GraphQL API (with paginated query collection support) while providing search and ordering filter capabilities via ApiPlatform, allowing clients to query and filter available currencies in the system.
