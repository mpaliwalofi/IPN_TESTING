# OrderRuleTranslation.php

**Path**: `src\Entity\Order\OrderRuleTranslation.php`

## Summary
This entity represents a translation record for order rule error messages in a Sylius-based e-commerce system. It extends Sylius's `AbstractTranslation` to store localized error messages that can be displayed when order validation rules fail, with each translation tied to a specific locale and parent order rule entity through a unique constraint.
