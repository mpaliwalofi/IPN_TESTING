# PaymentMethodTranslation.php

**Path**: `src\Entity\Payment\PaymentMethodTranslation.php`

## Summary
This entity represents translations for payment method names and descriptions in different locales, extending Sylius's base payment method translation functionality. It enables multi-language support for payment methods (like "Credit Card", "PayPal", etc.) and exposes these translations through a GraphQL API with query and collection query operations while disabling REST operations.
