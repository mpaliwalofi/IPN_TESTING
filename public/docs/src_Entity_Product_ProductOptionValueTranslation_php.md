# ProductOptionValueTranslation.php

**Path**: `src\Entity\Product\ProductOptionValueTranslation.php`

## Summary
This entity class represents translations for product option values (e.g., "Small", "Medium", "Large" in different languages) in a Sylius e-commerce application. It extends the base Sylius `ProductOptionValueTranslation` model to enable multi-language support for product variant options, storing localized text for each option value in the `sylius_product_option_value_translation` database table.
