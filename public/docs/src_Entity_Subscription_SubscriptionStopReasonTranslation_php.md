## Summary
This entity class represents translations for subscription stop reasons in a multi-language e-commerce system built with Sylius. It stores the localized names of reasons why subscriptions are cancelled or stopped, with a unique constraint ensuring one translation per locale per stop reason. The class extends Sylius's AbstractTranslation and is exposed via GraphQL API for querying translated stop reason names.
