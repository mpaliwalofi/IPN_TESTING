# TaxonImage.php

**Path**: `src\Entity\Taxonomy\TaxonImage.php`

## Summary
This code defines a `TaxonImage` entity that extends Sylius's base taxon (taxonomy/category) image model to handle product classification imagery in an e-commerce system. It configures the entity as an API Platform resource with GraphQL query capabilities (both single item and paginated collection queries) while disabling REST operations, storing image data in the `sylius_taxon_image` database table.
