**Path**: `src\Exception\InvalidArgumentException.php`

## Summary
This class defines a custom exception for handling invalid argument errors in the application. It extends Symfony's `HttpException` to automatically map invalid argument scenarios to HTTP error responses, allowing the framework to handle these exceptions with appropriate HTTP status codes and responses. This provides a domain-specific exception type that integrates with Symfony's HTTP error handling mechanism.
