# ResourceNotFoundException.php

**Path**: `src\Exception\ResourceNotFoundException.php`

## Summary
This is a custom HTTP exception class designed for API Platform that specifically handles 404 "Not Found" errors when API resources cannot be located. It extends Symfony's HttpException and defaults to an HTTP 404 status code, providing a standardized way to throw resource-not-found exceptions throughout the API with consistent error responses.
