# UnitFixedDiscountConfigurationTypeExtension.php

**Path**: `src\Form\Extension\UnitFixedDiscountConfigurationTypeExtension.php`

## Summary
This Symfony form extension enhances Sylius's unit-based fixed discount promotion configuration by adding a "max_units" field. It allows administrators to limit the maximum number of product units that can receive a fixed discount, providing more granular control over promotion rules (e.g., "get $5 off per item, up to 3 items maximum").
