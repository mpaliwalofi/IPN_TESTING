# UnitPercentageDiscountConfigurationTypeExtension.php

**Path**: `src\Form\Extension\UnitPercentageDiscountConfigurationTypeExtension.php`

## Summary
This Symfony form extension enhances Sylius's unit percentage discount promotion configuration by adding an optional `max_units` field. It allows administrators to limit the maximum number of units that can receive a percentage discount in a promotion, adding more granular control over discount applications in the e-commerce platform.
