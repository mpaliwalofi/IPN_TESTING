# ProductVariantAutocompleteChoiceType.php

**Path**: `src\Form\ProductVariantAutocompleteChoiceType.php`

## Summary
This Symfony form type provides an autocomplete choice field specifically for selecting Sylius product variants. It extends the base ResourceAutocompleteChoiceType and configures it to search product variants by name using their code as the value, with autocomplete filtering based on text phrases that contain the search term.
