# PaypalGatewayConfigurationType.php

**Path**: `src\Form\Type\Checkoutcom\PaypalGatewayConfigurationType.php`

## Summary
This Symfony form type configures PayPal payment gateway integration for Checkout.com within a Sylius e-commerce application. It extends a gateway abstract type and adds a required `client_id` field for authenticating PayPal transactions through the Checkout.com payment provider.
