# CustomerAutocompleteChoiceType.php

**Path**: `src\Form\Type\CustomerAutocompleteChoiceType.php`

## Summary
This Symfony form type provides an autocomplete dropdown for selecting Sylius customers by their email addresses. It extends the framework's autocomplete functionality to specifically handle customer resources, configuring the field to search and display customers using their email as the identifier while storing their ID as the value, with support for multiple customer selection.
