# OrderRuleTranslationType.php

**Path**: `src\Form\Type\Order\OrderRuleTranslationType.php`

## Summary
This Symfony form type defines the translation form for order rule entities, specifically handling the localized `errorMessage` field. It extends Sylius's resource bundle to provide a textarea input for translating error messages that will be displayed when order validation rules are violated, allowing multi-language support for order rule feedback.
