# OrderRuleType.php

**Path**: `src\Form\Type\Order\OrderRuleType.php`

## Summary
This Symfony form type defines the structure for creating and editing order rules in an e-commerce system built on Sylius. It provides fields for configuring rule properties (name, description, enabled status), selecting applicable sales channels, defining conditional logic through a collection of conditions, and managing multi-language translations. The form serves as the interface for merchants to set up business rules that can be applied to orders based on specific criteria.
