**Path**: `src\Form\Type\Order\Rule\AmountOnTaxonUntilConfigurationType.php`

## Summary
This form type configures a promotional rule for setting minimum order amounts per taxon (product category) across different sales channels in a Sylius e-commerce application. It extends Symfony's form system to create a collection of channel-specific configurations, where each entry captures the minimum order total threshold for a taxon, automatically populated with the channel's name and base currency code.
