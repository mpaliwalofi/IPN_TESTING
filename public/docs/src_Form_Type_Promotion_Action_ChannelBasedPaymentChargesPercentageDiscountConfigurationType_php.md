# ChannelBasedPaymentChargesPercentageDiscountConfigurationType.php

**Path**: `src\Form\Type\Promotion\Action\ChannelBasedPaymentChargesPercentageDiscountConfigurationType.php`

## Summary
This Symfony form type configures a channel-based percentage discount system for payment charges in a Sylius e-commerce application. It extends the ChannelCollectionType to allow administrators to set different percentage discount configurations per sales channel, with each channel's form entry labeled by the channel name. This enables multi-channel stores to offer varying payment-related discounts based on where the purchase occurs.
