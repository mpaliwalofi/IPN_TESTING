# ChannelBasedItemTotalMaxConfigurationType.php

**Path**: `src\Form\Type\Promotion\Rule\ChannelBasedItemTotalMaxConfigurationType.php`

## Summary
This Symfony form type configures a channel-based promotion rule for setting maximum item total thresholds across multiple sales channels in a Sylius e-commerce application. It extends the ChannelCollectionType to create a collection of ItemTotalMaxConfigurationType forms, where each entry is automatically configured with the appropriate channel name and base currency, allowing merchants to define different maximum order total limits per channel.
