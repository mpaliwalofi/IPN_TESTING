# ContainsVariantConfigurationType.php

**Path**: `src\Form\Type\Promotion\Rule\ContainsVariantConfigurationType.php`

## Summary
This form type configures the rule conditions for a promotion that checks if an order contains specific product variants. It defines the form structure used to select which product variants must be present in a cart/order for a promotional rule to apply, using Symfony's form component to add a product variant collection field to the configuration.
