# EmptyType.php

**Path**: `src\Form\Type\Promotion\Rule\EmptyType.php`

## Summary
This class defines an empty Symfony form type for promotion rules that require no configuration parameters. It serves as a placeholder form type for promotion rule types that don't need any user input or additional fields, allowing the promotion system to handle rules uniformly whether they require configuration or not.
