# SubscribableFilterConfigurationType.php

**Path**: `src\Form\Type\Promotion\SubscribableFilterConfigurationType.php`

## Summary
This Symfony form type configures a promotional filter for product subscription eligibility. It provides a choice field that allows selecting whether a promotion should apply to subscribable products, non-subscribable products, or both (when not required), using constants from the Subscription entity to define these states.
