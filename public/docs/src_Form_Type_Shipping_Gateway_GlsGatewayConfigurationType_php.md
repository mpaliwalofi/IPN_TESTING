# GlsGatewayConfigurationType.php

**Path**: `src\Form\Type\Shipping\Gateway\GlsGatewayConfigurationType.php`

## Summary
This Symfony form type defines the configuration interface for integrating with the GLS (General Logistics Systems) shipping gateway. It collects four required credentials: login, password, API URL, and country code, all validated as mandatory fields within the Sylius e-commerce framework's validation groups.
