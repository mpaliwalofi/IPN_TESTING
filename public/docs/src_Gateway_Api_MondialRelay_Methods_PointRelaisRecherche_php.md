# PointRelaisRecherche.php

**Path**: `src\Gateway\Api\MondialRelay\Methods\PointRelaisRecherche.php`

## Summary
This class implements a search method for Mondial Relay pickup points (relay points) through their WSI4_PointRelais_Recherche API endpoint. It allows searching for pickup locations based on various criteria including geographic parameters (country, city, zip code, coordinates), package characteristics (size, weight), and search constraints (radius, number of results), with country being the only required parameter along with the number of results.
