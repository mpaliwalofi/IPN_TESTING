# CardRepresentation.php

**Path**: `src\Gateway\Model\CardRepresentation.php`

## Summary
The `CardRepresentation` class is a data transfer object (DTO) that encapsulates payment card information including issuer, brand, PAN (Primary Account Number), and expiration date. It serves as a standardized model for representing credit/debit card data within the payment gateway system, providing both constructor-based initialization with nullable properties and array serialization via the `toArray()` method for data exchange or persistence.
