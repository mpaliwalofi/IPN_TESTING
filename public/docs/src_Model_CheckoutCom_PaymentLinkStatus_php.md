# PaymentLinkStatus.php

**Path**: `src\Model\CheckoutCom\PaymentLinkStatus.php`

## Summary
This class defines constant status values for Checkout.com payment links within the application's payment processing system. It serves as a centralized enumeration of possible payment link states: active (awaiting payment), payment received (completed), and expired (no longer valid). This allows consistent status checking and handling throughout the codebase when working with Checkout.com's payment link functionality.
