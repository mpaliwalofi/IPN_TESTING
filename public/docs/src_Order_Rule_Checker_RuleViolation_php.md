**Path**: `src\Order\Rule\Checker\RuleViolation.php`

## Summary
The `RuleViolation` class is a data structure that encapsulates information about a failed cart or order rule validation. It captures the violated rule, its type, configuration, and appropriate error message (prioritizing the order rule's custom message over the default rule message), serving as a standardized way to communicate rule validation failures throughout the order processing system.
