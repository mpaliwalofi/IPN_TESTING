# ProductCriteriaConfigurationType.php

**Path**: `src\Shipping\Form\Type\Rule\ProductCriteriaConfigurationType.php`

## Summary
This Symfony form type defines the configuration interface for product-based shipping rule criteria in a Sylius e-commerce application. It allows administrators to specify shipping rules based on three optional criteria: specific products (via autocomplete), product taxonomies/categories (taxons), and product variants, enabling flexible shipping rule configuration based on different product attributes.
