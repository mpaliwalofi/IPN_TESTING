**Path**: `src\Subscription\Updater\ShippingSubscriptionOrderUpdater.php`

## Summary
The `ShippingSubscriptionOrderUpdater` class manages shipping-related updates for subscription orders, including billing/shipping addresses and shipping method changes. It handles both regular address updates and special pickup point delivery scenarios, ensuring the shipping method is eligible and properly applying order state transitions after updates. This component is essential for maintaining accurate delivery information throughout a subscription's lifecycle.
