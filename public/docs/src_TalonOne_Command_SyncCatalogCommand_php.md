# SyncCatalogCommand.php

**Path**: `src\TalonOne\Command\SyncCatalogCommand.php`

## Summary
This Symfony console command synchronizes product catalog data from a Sylius e-commerce platform to TalonOne's promotion engine. It supports both full and incremental synchronization modes, fetching product variants for a specified sales channel and optionally filtering by date, with configurable batch limits for processing the catalog sync operation.
