# CampaignCollection.php

**Path**: `src\TalonOne\Core\DTO\CampaignCollection.php`

## Summary
This is a Data Transfer Object (DTO) that represents a paginated collection of Campaign entities from the TalonOne integration. It implements both `IteratorAggregate` and `PaginatorInterface` to provide iteration capabilities over campaign data while supporting pagination metadata such as total items, page size, current page, and offset calculations for API responses.
