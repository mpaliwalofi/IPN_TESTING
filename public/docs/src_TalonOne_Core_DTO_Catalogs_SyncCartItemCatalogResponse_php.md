# SyncCartItemCatalogResponse.php

**Path**: `src\TalonOne\Core\DTO\Catalogs\SyncCartItemCatalogResponse.php`

## Summary
This Data Transfer Object (DTO) represents the response structure for syncing cart item catalog data in the TalonOne promotional campaign platform. It encapsulates catalog metadata including identification, timestamps (creation/modification dates), ownership details (account and creator IDs), and configuration properties like name, description, subscribed application IDs, and version tracking. The class serves as a type-safe container for mapping API responses related to cart item catalog synchronization operations.
