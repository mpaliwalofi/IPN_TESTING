# AudienceMembership.php

**Path**: `src\TalonOne\Core\DTO\CustomerProfiles\AudienceMembership.php`

## Summary
This Data Transfer Object (DTO) represents a customer's membership in a marketing audience within the TalonOne promotion engine system. It captures the essential information linking a customer profile to a specific audience segment by storing both the unique identifier and human-readable name of the audience. This structure is used to track which audiences a customer belongs to, enabling targeted campaign delivery and customer segmentation.
