## Summary
This is a Data Transfer Object (DTO) that represents a customer profile in the TalonOne promotion/loyalty platform. It encapsulates customer-related data including identification, activity metrics (closed sessions, total sales), loyalty and audience memberships, and campaign eligibility information used for personalized marketing and promotion management.
