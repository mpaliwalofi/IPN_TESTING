# LoyaltyMembership.php

**Path**: `src\TalonOne\Core\DTO\CustomerProfiles\LoyaltyMembership.php`

## Summary
This Data Transfer Object (DTO) represents a customer's membership in a loyalty program within the TalonOne integration system. It encapsulates essential loyalty membership information including the program identifier and name, member ID, accumulated points balance, and the member's current tier status. This DTO is used to transfer loyalty program data between the TalonOne API and the application's customer profile management system.
