# UpdateCustomerProfileResponse.php

**Path**: `src\TalonOne\Core\DTO\CustomerProfiles\UpdateCustomerProfileResponse.php`

## Summary
This is a Data Transfer Object (DTO) that encapsulates the response data returned after updating a customer profile in the TalonOne promotion engine integration. It contains the updated customer profile along with all the marketing automation results triggered by the update, including triggered campaigns, loyalty changes, promotional effects, awarded giveaways, and any coupons or referrals that were created as a consequence of the profile update.
