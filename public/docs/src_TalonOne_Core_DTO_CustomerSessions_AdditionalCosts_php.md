# AdditionalCosts.php

**Path**: `src\TalonOne\Core\DTO\CustomerSessions\AdditionalCosts.php`

## Summary
This Data Transfer Object (DTO) represents additional costs associated with customer sessions in the Talon.One promotion engine integration. It encapsulates two specific cost types—shipping fees and payment fees—as `AdditionalCostsItem` objects, with serialization support for the 'talon.one.integration' group, enabling these costs to be included when calculating promotions or discounts in customer checkout sessions.
