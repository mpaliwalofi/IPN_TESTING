# CustomerSession.php

**Path**: `src\TalonOne\Core\DTO\CustomerSessions\CustomerSession.php`

## Summary
This class is a Data Transfer Object (DTO) that represents a customer session for integration with the Talon.One promotion engine API. It encapsulates customer shopping session data including profile identification, cart items, promotional elements (coupon codes, referral codes, loyalty cards), store information, and additional attributes needed for evaluating campaign rules and applying promotional offers.
