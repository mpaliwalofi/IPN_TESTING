# UpdateCustomerSessionResponse.php

**Path**: `src\TalonOne\Core\DTO\CustomerSessions\UpdateCustomerSessionResponse.php`

## Summary
This is a Data Transfer Object (DTO) that encapsulates the response data from updating a customer session in the Talon.One promotions engine. It aggregates multiple related entities including the updated customer session, profile, loyalty data, referrals, coupons, triggered campaigns, effects (applied promotions), and any rule failures, serving as a comprehensive container for all outcomes resulting from a customer session update operation.
