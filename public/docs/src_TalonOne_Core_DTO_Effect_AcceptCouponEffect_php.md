# AcceptCouponEffect.php

**Path**: `src\TalonOne\Core\DTO\Effect\AcceptCouponEffect.php`

## Summary
This class represents a specific type of promotional effect in the Talon.One integration that handles coupon acceptance operations. It extends a base `Effect` class and contains properties specific to accepting/validating coupons through the `AcceptCouponEffectProps` object. This DTO (Data Transfer Object) is used to encapsulate the data and configuration needed when a coupon is successfully accepted in the Talon.One promotion engine.
