# AddFreeItemEffect.php

**Path**: `src\TalonOne\Core\DTO\Effect\AddFreeItemEffect.php`

## Summary
This class represents a promotional effect in the TalonOne integration that adds free items to a customer's cart or order. It extends a base `Effect` class and contains properties specific to free item additions (such as SKU, quantity, etc.) encapsulated in the `AddFreeItemEffectProps` object, enabling the system to apply "buy X get Y free" or similar promotional campaigns.
