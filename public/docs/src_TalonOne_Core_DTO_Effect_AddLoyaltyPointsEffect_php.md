# AddLoyaltyPointsEffect.php

**Path**: `src\TalonOne\Core\DTO\Effect\AddLoyaltyPointsEffect.php`

## Summary
This class represents a specific type of effect in the TalonOne promotion engine that adds loyalty points to a customer's account. It extends the base `Effect` class and uses `AddLoyaltyPointsEffectProps` to encapsulate the properties needed to execute the loyalty points addition, such as the amount of points to add and the target loyalty program. This is part of a campaign management system where promotional rules trigger various effects, with this one specifically handling loyalty reward distribution.
