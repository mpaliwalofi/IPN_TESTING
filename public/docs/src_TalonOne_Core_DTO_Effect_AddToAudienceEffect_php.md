# AddToAudienceEffect.php

**Path**: `src\TalonOne\Core\DTO\Effect\AddToAudienceEffect.php`

## Summary
This class represents a campaign effect in the TalonOne promotion engine that adds users/customers to a specific audience segment. It extends the base `Effect` class and contains properties (via `AddToAudienceEffectProps`) that define which audience to add the user to, serving as a data transfer object (DTO) for audience membership operations triggered by campaign rules.
