## Summary
This class represents a Talon.One promotion effect that awards a giveaway to a customer. It extends the base `Effect` class and contains properties specific to giveaway awards (encapsulated in `AwardGiveawayEffectProps`), serving as a Data Transfer Object (DTO) for handling giveaway-related promotional outcomes in the Talon.One integration system.
