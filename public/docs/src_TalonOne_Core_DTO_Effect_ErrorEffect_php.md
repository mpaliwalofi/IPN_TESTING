## Summary
This class represents an error effect in the TalonOne integration system, which is used to capture and handle error conditions that occur during promotion or campaign rule processing. It extends the base `Effect` class and contains error-specific properties through the `ErrorEffectProps` object, allowing the application to track and respond to failures in TalonOne's rule engine execution.
