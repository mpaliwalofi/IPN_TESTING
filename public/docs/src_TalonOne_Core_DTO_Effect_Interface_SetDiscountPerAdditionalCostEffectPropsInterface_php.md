## Summary
This interface defines the contract for discount effects applied to additional costs in the TalonOne promotion engine. It requires implementing classes to provide methods for retrieving an additional cost's identifier, name, and the desired discount value to be applied. This supports TalonOne's campaign management system where discounts can be configured for supplementary charges beyond the base product price.
