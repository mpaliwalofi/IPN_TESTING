# SetDiscountPerItemEffectInterface.php

**Path**: `src\TalonOne\Core\DTO\Effect\Interface\SetDiscountPerItemEffectInterface.php`

## Summary
This interface defines the contract for a "Set Discount Per Item" effect in the TalonOne promotion engine integration. It extends the base discount effect interface and requires implementing classes to provide item-specific discount properties through a `SetDiscountPerItemEffectPropsInterface`, enabling granular discount application at the individual item level rather than order-wide discounts.
