## Summary
This class is a Data Transfer Object (DTO) that holds properties for an "Accept Coupon" effect in the TalonOne promotion engine integration. It specifically stores the coupon value/code when a coupon is accepted or validated during a promotional campaign transaction, serving as a structured container for passing coupon acceptance data between different layers of the application.
