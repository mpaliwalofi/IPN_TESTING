# AddFreeItemEffectProps.php

**Path**: `src\TalonOne\Core\DTO\Effect\Props\AddFreeItemEffectProps.php`

## Summary
This class is a Data Transfer Object (DTO) that represents the properties of a promotional effect in the TalonOne marketing/promotion engine. Specifically, it defines the structure for adding a free item to a cart or order, containing the SKU (stock keeping unit) identifier and the name of the free item being offered as part of a promotion campaign.
