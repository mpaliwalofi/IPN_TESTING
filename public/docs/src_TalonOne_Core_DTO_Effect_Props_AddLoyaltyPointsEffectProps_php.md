## Summary
This is a Data Transfer Object (DTO) that represents the properties/parameters for adding loyalty points in the TalonOne promotion engine system. It encapsulates all the necessary information to execute a loyalty points addition operation, including the loyalty program details (program ID, sub-ledger), point values, recipient identification, optional expiration dates, and transaction tracking information for cart-related contexts.
