# AddToAudienceEffectProps.php

**Path**: `src\TalonOne\Core\DTO\Effect\Props\AddToAudienceEffectProps.php`

## Summary
This class serves as a Data Transfer Object (DTO) for the Talon.One promotion engine's "add to audience" effect, which is used to add customer profiles to marketing audience segments. It holds the necessary identifiers to link a specific customer profile (via integration ID or internal ID) with a target audience (via audience ID or name) when processing campaign rules and their resulting effects.
