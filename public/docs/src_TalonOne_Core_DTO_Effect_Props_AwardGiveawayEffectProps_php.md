# AwardGiveawayEffectProps.php

**Path**: `src\TalonOne\Core\DTO\Effect\Props\AwardGiveawayEffectProps.php`

## Summary
This class serves as a Data Transfer Object (DTO) for storing properties related to awarding giveaway prizes in the TalonOne promotion engine integration. It encapsulates information about a specific giveaway award event, including the giveaway pool details (ID and name), the recipient's integration identifier, the specific giveaway instance ID, and the redemption code generated for the award. This DTO is used to structure effect data when processing promotional campaigns that involve distributing giveaway prizes to customers.
