**Path**: `src\TalonOne\Core\DTO\Effect\Props\ChangeLoyaltyTierLevelEffectProps.php`

## Summary
This Data Transfer Object (DTO) represents the properties of a loyalty tier level change effect in the TalonOne promotion engine integration. It captures details when a customer's loyalty tier is modified, including the rule that triggered the change, the loyalty program identifier, the previous and new tier names, and an optional expiration date for the new tier. This class serves as a structured container for communicating loyalty tier transition data between different parts of the application.
