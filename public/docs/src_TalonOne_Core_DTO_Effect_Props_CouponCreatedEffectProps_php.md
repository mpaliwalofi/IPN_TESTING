**Path**: `src\TalonOne\Core\DTO\Effect\Props\CouponCreatedEffectProps.php`

## Summary
This class is a Data Transfer Object (DTO) that represents the properties of a coupon creation effect in the TalonOne promotional campaign system. It encapsulates the essential data returned when a coupon is successfully created, specifically the coupon's value and the associated customer profile identifier, facilitating communication between different layers of the application when processing coupon generation events.
