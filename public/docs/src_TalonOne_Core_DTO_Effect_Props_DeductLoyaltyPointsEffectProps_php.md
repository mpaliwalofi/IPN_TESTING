# DeductLoyaltyPointsEffectProps.php

**Path**: `src\TalonOne\Core\DTO\Effect\Props\DeductLoyaltyPointsEffectProps.php`

## Summary
This class is a Data Transfer Object (DTO) that represents the properties of a loyalty points deduction effect in the TalonOne promotion engine integration. It encapsulates all necessary information to process a loyalty points deduction transaction, including the loyalty program identifier, the amount to deduct, the associated card/customer identifier, and transaction tracking details (UUID, rule title, sub-ledger ID).
