**Path**: `src\TalonOne\Core\DTO\Effect\Props\RejectReferralEffectProps.php`

## Summary
This Data Transfer Object (DTO) represents the properties of a referral rejection effect in the Talon.One promotion engine integration. It encapsulates the data needed when a referral is rejected, including the referral value, rejection reason, optional campaign exclusion details, and tracking information (condition/effect indices) for debugging or audit purposes. This class serves as a structured container for communicating referral rejection outcomes between the application and the Talon.One API.
