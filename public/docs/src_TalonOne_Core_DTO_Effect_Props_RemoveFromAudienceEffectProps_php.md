# RemoveFromAudienceEffectProps.php

**Path**: `src\TalonOne\Core\DTO\Effect\Props\RemoveFromAudienceEffectProps.php`

## Summary
This class serves as a Data Transfer Object (DTO) for the TalonOne promotion engine's "remove from audience" effect properties. It encapsulates the necessary data to identify both the target audience (via ID or name) and the customer profile (via integration ID or internal ID) that should be removed from a specific audience segment. This DTO is used when processing campaign rules that trigger the removal of customers from marketing audiences or customer segments.
