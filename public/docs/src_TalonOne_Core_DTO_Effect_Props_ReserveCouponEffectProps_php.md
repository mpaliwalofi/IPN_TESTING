# ReserveCouponEffectProps.php

**Path**: `src\TalonOne\Core\DTO\Effect\Props\ReserveCouponEffectProps.php`

## Summary
This class is a Data Transfer Object (DTO) that holds properties for a coupon reservation effect in the TalonOne promotion engine integration. It captures the coupon value being reserved, the customer profile ID it's being reserved for, and whether this is a new reservation or an existing one, enabling the application to track and manage coupon allocations to specific users.
