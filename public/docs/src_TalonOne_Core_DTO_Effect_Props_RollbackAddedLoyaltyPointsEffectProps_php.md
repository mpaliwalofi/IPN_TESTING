## Summary
This class represents a Data Transfer Object (DTO) for loyalty points rollback operations in the TalonOne promotion/loyalty system. It encapsulates the properties needed to reverse a previously added loyalty points transaction, including program identification, ledger details, point values, recipient information, and optional cart item positioning data for transaction tracking and reversal purposes.
