# RollbackDeductedLoyaltyPointsEffectProps.php

**Path**: `src\TalonOne\Core\DTO\Effect\Props\RollbackDeductedLoyaltyPointsEffectProps.php`

## Summary
This class is a Data Transfer Object (DTO) that encapsulates the properties needed to rollback (reverse) previously deducted loyalty points in the Talon.One promotion engine. It stores essential information for the rollback operation including the loyalty program ID, sub-ledger ID, point value to restore, recipient details, transaction UUID for tracking, and optional date constraints and card identifier to ensure the reversal is applied correctly to the right customer account.
