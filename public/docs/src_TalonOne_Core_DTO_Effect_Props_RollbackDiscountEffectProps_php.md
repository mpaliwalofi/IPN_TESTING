# RollbackDiscountEffectProps.php

**Path**: `src\TalonOne\Core\DTO\Effect\Props\RollbackDiscountEffectProps.php`

## Summary
This class serves as a Data Transfer Object (DTO) for rolling back discount effects in the TalonOne promotion management system. It encapsulates the properties needed to reverse or undo a previously applied discount, including the discount name, value, and optional positioning information (cart item position/sub-position) along with additional cost tracking fields. The class enables the system to precisely identify and revert specific discounts that were applied to cart items during promotion processing.
