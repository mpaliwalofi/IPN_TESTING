## Summary
This class is a Data Transfer Object (DTO) that encapsulates properties for a "set discount per item" effect in the TalonOne promotion engine integration. It holds discount-related metadata including the discount name, value, item position/sub-position, scope, bundle information, and both actual and desired discount amounts, enabling item-level discount calculations and tracking within promotional campaigns.
