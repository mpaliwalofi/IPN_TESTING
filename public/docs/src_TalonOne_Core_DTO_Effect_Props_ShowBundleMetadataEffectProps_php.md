# ShowBundleMetadataEffectProps.php

**Path**: `src\TalonOne\Core\DTO\Effect\Props\ShowBundleMetadataEffectProps.php`

## Summary
This class serves as a Data Transfer Object (DTO) for representing bundle metadata display effects in the TalonOne promotion engine integration. It encapsulates properties needed to show bundle information, including a description, bundle attributes (as string array), and item indices (as float array) that identify which items are part of the bundle. The class is used to transfer bundle-related effect data between different layers of the application when processing promotional rules or displaying bundle offers to users.
