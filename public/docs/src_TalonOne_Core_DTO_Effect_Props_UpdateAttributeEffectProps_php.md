# UpdateAttributeEffectProps.php

**Path**: `src\TalonOne\Core\DTO\Effect\Props\UpdateAttributeEffectProps.php`

## Summary
This class is a Data Transfer Object (DTO) that holds the properties for an "UpdateAttribute" effect in the TalonOne promotion engine integration. It encapsulates two pieces of data needed to modify an attribute: a `path` string that identifies which attribute to update, and a `value` object containing the new value to be assigned. This is part of the effect handling system for processing promotional rule outcomes that modify customer or session attributes.
