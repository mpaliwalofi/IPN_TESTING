# ReferralCreatedEffect.php

**Path**: `src\TalonOne\Core\DTO\Effect\ReferralCreatedEffect.php`

## Summary
This class represents a data transfer object (DTO) for a referral creation effect in the TalonOne promotion engine integration. It extends a base `Effect` class and encapsulates the properties specific to when a new referral is created through the TalonOne system, allowing the application to handle and process referral-related campaign effects with strongly-typed property data via the `ReferralCreatedEffectProps` object.
