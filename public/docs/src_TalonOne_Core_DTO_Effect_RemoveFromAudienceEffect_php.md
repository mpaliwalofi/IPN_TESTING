# RemoveFromAudienceEffect.php

**Path**: `src\TalonOne\Core\DTO\Effect\RemoveFromAudienceEffect.php`

## Summary
This class represents a data transfer object (DTO) for a TalonOne promotion engine effect that removes a user from an audience segment. It extends the base `Effect` class and contains properties specific to audience removal operations, encapsulated in the `RemoveFromAudienceEffectProps` object. This effect is used in campaign rules to programmatically exclude customers from targeted audience groups based on specific conditions or actions.
