# ReserveCouponEffect.php

**Path**: `src\TalonOne\Core\DTO\Effect\ReserveCouponEffect.php`

## Summary
This class represents a coupon reservation effect in the Talon.One promotion engine integration. It extends the base `Effect` class and encapsulates the properties specific to reserving a coupon through the `ReserveCouponEffectProps` object, which would contain details like coupon codes and reservation parameters needed when a promotional rule triggers a coupon reservation action.
