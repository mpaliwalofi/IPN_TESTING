## Summary
This class represents a discount effect in the TalonOne promotion engine integration. It serves as a Data Transfer Object (DTO) that encapsulates discount-related promotion effects, extending a base `Effect` class and using `SetDiscountEffectProps` to store the specific discount parameters. The class provides type-safe access to discount effect properties that would be received from or sent to the TalonOne API when processing promotional campaigns.
