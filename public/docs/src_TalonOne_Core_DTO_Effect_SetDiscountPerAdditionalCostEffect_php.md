## Summary
This class represents a TalonOne promotion engine effect that applies a discount to additional costs (e.g., shipping, handling fees) in an e-commerce context. It extends the base `Effect` class and implements `SetDiscountEffectInterface`, encapsulating discount-specific properties through `SetDiscountPerAdditionalCostEffectProps` to enable programmatic application of discounts to non-item costs during promotion rule execution.
