This class represents a specific type of promotional effect in the TalonOne integration that applies discounts to additional costs on individual items (e.g., shipping, handling fees per item). It extends a base `Effect` class and implements the `SetDiscountPerItemEffectInterface`, encapsulating the discount configuration through its `SetDiscountPerAdditionalCostPerItemEffectProps` property, which allows the application to process and apply per-item additional cost discounts as part of a promotional campaign ruleset.
