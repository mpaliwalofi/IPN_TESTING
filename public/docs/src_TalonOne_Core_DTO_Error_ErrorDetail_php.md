# ErrorDetail.php

**Path**: `src\TalonOne\Core\DTO\Error\ErrorDetail.php`

## Summary
This class is a Data Transfer Object (DTO) that structures error information in the TalonOne integration system. It provides a standardized format for capturing error details with three key properties: the source of the error, a title/summary, and detailed information about what went wrong. This DTO enables consistent error handling and reporting across the TalonOne API integration layer.
