# Error.php

**Path**: `src\TalonOne\Core\DTO\Error.php`

## Summary
This is a Data Transfer Object (DTO) that represents error responses from the TalonOne integration (a promotion and loyalty platform). It encapsulates error information with a main error message and an optional array of detailed error objects (`ErrorDetail`), allowing the application to handle and communicate API errors in a structured format.
