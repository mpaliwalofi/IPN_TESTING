# Event.php

**Path**: `src\TalonOne\Core\DTO\Event.php`

## Summary
This is a Data Transfer Object (DTO) for Talon.One's promotional campaign management system, representing an event that occurs within the platform. The Event class stores comprehensive information about customer interactions including the event metadata (id, created, applicationId), customer identifiers (profileId, sessionId, storeIntegrationId), the event type and attributes, as well as the resulting effects, ledger entries, and meta information generated by the rule engine when processing the event.
