# Giveaway.php

**Path**: `src\TalonOne\Core\DTO\Giveaway.php`

## Summary
This is a Data Transfer Object (DTO) that represents a giveaway or promotional code in the TalonOne marketing/promotion system. It encapsulates all properties of a giveaway including its unique code, validity period (start/end dates), association with a pool and customer profile, usage status, and custom attributes. The class serves as a simple data container for transferring giveaway information between different layers of the application without any business logic.
