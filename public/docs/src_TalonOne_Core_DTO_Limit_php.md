# Limit.php

**Path**: `src\TalonOne\Core\DTO\Limit.php`

## Summary
This Data Transfer Object (DTO) represents rate limiting or usage quota constraints in the TalonOne promotional campaign system. It encapsulates the configuration for throttling specific actions by defining what action is being limited, the maximum number of allowed occurrences (limit), the time period for the limit, and optionally which entities (likely customers, campaigns, or coupons) the limit applies to.
