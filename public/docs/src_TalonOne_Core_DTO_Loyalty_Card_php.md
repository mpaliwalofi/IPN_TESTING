# Card.php

**Path**: `src\TalonOne\Core\DTO\Loyalty\Card.php`

## Summary
This is a Data Transfer Object (DTO) representing a loyalty card in the TalonOne integration system. It encapsulates the essential properties of a loyalty card including identification (cardId, programId, userId), lifecycle information (status, issue/expiry dates), and the current points/monetary balance, serving as a structured data container for transferring loyalty card information between application layers.
