# Loyalty.php

**Path**: `src\TalonOne\Core\DTO\Loyalty\Loyalty.php`

## Summary
This Data Transfer Object (DTO) represents a loyalty system structure in a TalonOne integration, encapsulating collections of loyalty cards and loyalty programs. It serves as a container for transferring loyalty-related data between the application and the TalonOne promotional campaign management system, allowing the application to handle multiple loyalty cards and programs simultaneously.
