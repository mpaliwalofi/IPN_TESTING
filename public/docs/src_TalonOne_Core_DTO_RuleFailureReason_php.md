# RuleFailureReason.php

**Path**: `src\TalonOne\Core\DTO\RuleFailureReason.php`

## Summary
This Data Transfer Object (DTO) represents the failure information for a rule in the TalonOne integration system. It encapsulates three key pieces of diagnostic data: the rule's unique identifier, its human-readable name, and the specific reason why the rule failed to execute or validate, enabling detailed error reporting and debugging of TalonOne campaign/promotion rules.
