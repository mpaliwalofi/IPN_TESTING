# Setting.php

**Path**: `src\TalonOne\Core\DTO\Setting.php`

## Summary
This is a Data Transfer Object (DTO) that defines the configuration settings for coupon code generation in the TalonOne integration. It specifies the pattern/format that generated coupon codes should follow and optionally defines which characters are valid for use in those coupons, enabling controlled and consistent coupon code creation across the application.
