## Summary
This class represents an HTTP request to update an existing campaign in the TalonOne promotional campaign management system. It encapsulates the PUT operation for the `/v1/applications/{applicationId}/campaigns/{campaignId}` endpoint, accepting campaign data as payload and returning a Campaign DTO upon successful execution. The class implements the ManagementRequestInterface pattern to standardize API communication for campaign modification operations.
