This class represents an HTTP PUT request to update an existing coupon in the TalonOne Management API. It encapsulates the request details including the application ID, campaign ID, and coupon ID in the URI path, along with configurable payload and query parameters (defaulting to dry-run mode). The request returns a Coupon DTO object and is used to modify coupon properties within a specific campaign context.
