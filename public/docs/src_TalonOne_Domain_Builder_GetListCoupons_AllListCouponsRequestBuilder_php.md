**Path**: `src\TalonOne\Domain\Builder\GetListCoupons\AllListCouponsRequestBuilder.php`

## Summary
This builder class constructs API requests to retrieve lists of coupons from the TalonOne promotional management system for a specific application and campaign. It configures the request to fetch up to 1000 coupons at a time with the `valuesOnly` flag enabled, following the builder pattern to separate request construction logic from the actual API request object.
