**Path**: `src\TalonOne\Domain\Builder\GetListCoupons\ExpiredListCouponsRequestBuilder.php`

## Summary
This builder class constructs API requests to retrieve expired coupons from the TalonOne promotion management system. It implements the Builder pattern to configure a `GetListCouponsRequest` with specific query parameters (page size of 1000, values only, and filtered to expired coupons) for a given application and campaign ID, facilitating the retrieval of expired coupon lists through the TalonOne Management API.
