## Summary
This builder class constructs API requests to close customer sessions in the TalonOne promotional campaign system. It specifically creates a `PutCustomerSessionRequest` that sets a customer session's state to "CLOSED" for a given order, generating the necessary payload with session state and query parameters to finalize the session through TalonOne's integration API.
