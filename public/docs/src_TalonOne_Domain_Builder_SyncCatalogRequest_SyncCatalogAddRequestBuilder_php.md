**Path**: `src\TalonOne\Domain\Builder\SyncCatalogRequest\SyncCatalogAddRequestBuilder.php`

## Summary
This builder class constructs requests to synchronize product catalog data with TalonOne's promotion engine. It processes an array of ProductVariant entities and generates a batch of PATCH actions containing product SKUs, prices, names, and category taxonomies to be added or updated in the TalonOne catalog via their API.
