# UpdateCouponRequestBuilder.php

**Path**: `src\TalonOne\Domain\Builder\UpdateCoupon\UpdateCouponRequestBuilder.php`

## Summary
This class is a builder that constructs API requests for updating coupons in the TalonOne promotion management system. It specifically creates `PutUpdateCouponRequest` objects with the necessary application ID, campaign ID, and coupon ID, then sets the coupon's usage limit as the payload for the update operation. The builder implements the Builder pattern to encapsulate the construction logic for coupon modification requests that will be sent to TalonOne's Management API.
