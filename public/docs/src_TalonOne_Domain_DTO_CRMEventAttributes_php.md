# CRMEventAttributes.php

**Path**: `src\TalonOne\Domain\DTO\CRMEventAttributes.php`

## Summary
This Data Transfer Object (DTO) encapsulates CRM event attributes for Talon.One coupon campaign management, specifically for customer support-related promotional campaigns. It stores campaign configuration details including campaign name, coupon prefix/suffix, validity dates, auto-delete settings, and analytics parameters, then transforms them into a standardized array format via `toArray()` for integration with the Talon.One promotion engine API.
