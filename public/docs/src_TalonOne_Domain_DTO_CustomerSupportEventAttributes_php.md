# CustomerSupportEventAttributes.php

**Path**: `src\TalonOne\Domain\DTO\CustomerSupportEventAttributes.php`

## Summary
This Data Transfer Object (DTO) encapsulates attributes for customer support events in the TalonOne promotional engine integration, specifically for tracking various discount/adjustment types (product discounts, order adjustments, shipping fees, payment fees, and free items). The class stores configuration for different promotional targets with their values, types, validity periods, and analytics metadata, and provides a `toArray()` method to serialize these attributes into a formatted array structure for API communication with TalonOne's customer support event system.
