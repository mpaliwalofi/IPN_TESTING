# NotificationType.php

**Path**: `src\TalonOne\Domain\Enum\NotificationType.php`

## Summary
This file defines an enumeration for categorizing different types of notifications in the TalonOne integration domain. It provides four string-backed notification types: INFO (for configuring coupon descriptions in the frontend), OFFER, ERROR, and MISC, which are used to classify and handle promotional/coupon-related notifications throughout the application.
