# NoCouponCreatedException.php

**Path**: `src\TalonOne\Domain\Exception\NoCouponCreatedException.php`

## Summary
This is a custom domain exception class for the TalonOne integration that indicates a failure in coupon creation operations. It extends PHP's base Exception class and implements a TalonOneException marker interface, providing a standardized way to handle scenarios where the coupon generation process fails to produce a coupon. This exception allows the application to distinguish coupon creation failures from other types of errors when interacting with the TalonOne promotional engine.
