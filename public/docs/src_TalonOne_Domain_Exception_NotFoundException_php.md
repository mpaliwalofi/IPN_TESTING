# NotFoundException.php

**Path**: `src\TalonOne\Domain\Exception\NotFoundException.php`

## Summary
This file defines a custom exception class for handling "not found" scenarios within the TalonOne domain (a promotion/coupon management system). The `NotFoundException` extends PHP's base `Exception` class and implements the `TalonOneException` marker interface, allowing it to be caught specifically when TalonOne resources (such as coupons, campaigns, or promotions) cannot be located in the system.
