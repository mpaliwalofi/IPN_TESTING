# TalonOneException.php

**Path**: `src\TalonOne\Domain\Exception\TalonOneException.php`

## Summary
This file defines a marker interface for Talon.One integration exceptions in the application's domain layer. It extends PHP's `\Throwable` interface to create a specific exception type hierarchy for handling errors related to Talon.One (a promotion and loyalty platform) operations. This allows the application to catch and handle Talon.One-specific exceptions separately from other application exceptions.
