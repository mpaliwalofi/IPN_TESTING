# CustomerBusinessReportUpdaterFilter.php

**Path**: `src\TalonOne\Updater\CustomerBusinessReportUpdaterFilter.php`

## Summary
This class is a filter iterator for customer business reports in the TalonOne integration that selectively processes items based on a whitelist filter. It accepts all items when no filter is provided, or only accepts items whose keys match entries in the provided filter array, enabling selective updates of customer business report data.
