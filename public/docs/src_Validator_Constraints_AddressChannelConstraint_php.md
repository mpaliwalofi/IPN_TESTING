# AddressChannelConstraint.php

**Path**: `src\Validator\Constraints\AddressChannelConstraint.php`

## Summary
This is a Symfony validation constraint class that validates address-channel relationships in the application. It defines a custom validation rule that can be applied to properties or methods to ensure addresses are associated with valid channels. The constraint can be used repeatedly on the same target and includes a translatable error message key for when validation fails.
