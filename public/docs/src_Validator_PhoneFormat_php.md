# PhoneFormat.php

**Path**: `src\Validator\PhoneFormat.php`

## Summary
This is a custom Symfony validation constraint class for validating phone number formats in address-related forms. It defines two error messages: one for incorrectly formatted phone numbers (`messageWrongFormat`) and a general invalid phone message (`message`), which will be used by a corresponding validator to enforce phone number validation rules in the application's address management system.
