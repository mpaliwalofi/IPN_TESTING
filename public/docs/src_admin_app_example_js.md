# app.example.js

**Path**: `src\admin\app.example.js`

## Summary
This is an example configuration file for a Strapi admin panel that demonstrates how to customize admin settings. It serves as a template showing how to configure available locales/languages for the admin interface and implement a bootstrap function that executes during admin panel initialization. The file is meant to be copied and modified by developers to customize their Strapi admin interface, with locale options currently commented out to show available language choices.
