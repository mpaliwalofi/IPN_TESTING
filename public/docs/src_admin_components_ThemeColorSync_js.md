# ThemeColorSync.js

**Path**: `src\admin\components\ThemeColorSync.js`

## Summary
This React component automatically synchronizes color fields with theme selections in Strapi's content management system for specific content types (b2c.edito and b2c.review-large). It monitors theme changes at both the root level and within dynamic zone components, then updates the corresponding color field based on predefined theme-to-color mappings stored in a centralized configuration file, ensuring consistent styling across content entries without manual color selection.
