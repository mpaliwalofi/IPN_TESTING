# lifecycles.js

**Path**: `src\api\b2c-account-page\content-types\b2c-account-page\lifecycles.js`

## Summary
This lifecycle hook module automatically triggers product localization after creating or updating B2C account pages in Strapi. It integrates with the Akeneo import plugin to ensure that content is properly localized across different languages/regions whenever B2C account page content is modified, maintaining data consistency between Strapi CMS and the Akeneo PIM system.
