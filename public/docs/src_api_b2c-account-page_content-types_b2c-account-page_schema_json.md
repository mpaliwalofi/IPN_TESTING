# schema.json

**Path**: `src\api\b2c-account-page\content-types\b2c-account-page\schema.json`

## Summary
This JSON schema defines a Strapi content type for a B2C customer account page ("Mon Compte" - My Account in French). It configures a single-type content model with internationalization support that uses a dynamic zone architecture, allowing content editors to compose the account page using over 40 different component types including account links, order summaries, calendly integrations, carousels, FAQs, and various editorial content blocks. The schema also includes OpenGraph metadata for social sharing and LLM-specific components, with draft/publish workflow capabilities enabled.
