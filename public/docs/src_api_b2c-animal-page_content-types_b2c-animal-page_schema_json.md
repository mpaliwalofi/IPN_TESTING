**Path**: `src\api\b2c-animal-page\content-types\b2c-animal-page\schema.json`

## Summary
This is a Strapi CMS schema definition for a B2C "My Animal" page content type. It defines a localized single-type page structure with OpenGraph metadata, an LLM component, and a dynamic content zone that can include 40+ different component types (carousels, FAQs, product cross-sells, hero sections, etc.) for building flexible pet-related marketing and e-commerce pages. The schema supports draft/publish workflows and internationalization for multi-language content management.
