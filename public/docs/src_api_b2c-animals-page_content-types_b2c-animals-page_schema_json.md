# schema.json

**Path**: `src\api\b2c-animals-page\content-types\b2c-animals-page\schema.json`

## Summary
This is a Strapi CMS schema definition for a B2C "My Animals" page (Mes animaux) that manages pet-related content for customer accounts. The schema defines a single-type content structure with headers for both animal-present and no-animals states, plus two dynamic content zones that can accommodate 40+ different component types including animal lists, subscriptions, orders, carousels, FAQs, and various editorial blocks. This configuration enables content managers to build flexible, personalized pet owner dashboard pages with draft/publish workflow support.
