# algolia.json

**Path**: `src\api\b2c-category\content-types\b2c-category\algolia.json`

## Summary
This configuration file defines Algolia search indexing settings for B2C e-commerce categories in a Strapi application. It specifies that category data should be indexed to "strapi_ecommerce_categories" with a maximum string length of 750 characters, while explicitly excluding certain fields (like products, localizations, timestamps, and internal references) from the search index to optimize performance and relevance.
