# schema.json

**Path**: `src\api\b2c-category\content-types\b2c-category\schema.json`

## Summary
This schema defines a Strapi content type for managing B2C product categories with full internationalization support. It includes e-commerce-specific fields like SKU, Akeneo PIM references, slugs, breadcrumbs, and OpenGraph metadata, along with category guide entrance features (URLs, content for mobile/desktop). The schema enables draft/publish workflows and visibility controls for organizing and displaying product categories in a multilingual B2C storefront.
