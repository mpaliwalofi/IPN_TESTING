# lifecycles.js

**Path**: `src\api\b2c-contact-page\content-types\b2c-contact-page\lifecycles.js`

## Summary
This lifecycle hook module automatically triggers localization after B2C contact page content is created or updated in Strapi. It integrates with the "import-akeneo" plugin's localizer service to ensure content localization is maintained whenever the contact page content changes, likely synchronizing multilingual content from the Akeneo PIM system.
