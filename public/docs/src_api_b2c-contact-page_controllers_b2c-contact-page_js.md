# b2c-contact-page.js

**Path**: `src\api\b2c-contact-page\controllers\b2c-contact-page.js`

## Summary
This Strapi CMS controller manages the B2C (Business-to-Consumer) contact page API endpoint, handling retrieval and transformation of contact page content for customer-facing applications. It fetches the contact page data with deep population of related content, applies component-specific transformations to content blocks (like forms, text, images), adds cross-sell product information, and returns a sanitized, formatted response optimized for frontend consumption.
