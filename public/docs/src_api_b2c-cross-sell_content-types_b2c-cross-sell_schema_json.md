# schema.json

**Path**: `src\api\b2c-cross-sell\content-types\b2c-cross-sell\schema.json`

## Summary
This is a Strapi CMS schema definition for a B2C cross-selling content type that manages product cross-sell recommendations. It defines a localized collection with draft/publish workflow capabilities, containing a unique slug identifier and a required cross-sell component for storing cross-sell product suggestions. The schema enables content editors to create and manage internationalized cross-sell offerings that can be referenced by product pages or shopping cart interfaces.
