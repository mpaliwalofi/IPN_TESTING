# lifecycles.js

**Path**: `src\api\b2c-faq-page\content-types\b2c-faq-page\lifecycles.js`

## Summary
This lifecycle hook module automatically triggers localization whenever a B2C FAQ page is created or updated in Strapi. It integrates with the "import-akeneo" plugin to ensure that FAQ page content is properly localized across different languages/regions immediately after any content changes.
