# b2c-faq-page.js

**Path**: `src\api\b2c-faq-page\controllers\b2c-faq-page.js`

## Summary
This controller manages the B2C FAQ page content in a Strapi CMS, handling retrieval and transformation of FAQ page data. It fetches the FAQ page entity with deep population of related content, sanitizes the output, and applies cross-sell transformations before returning the response to API consumers.
