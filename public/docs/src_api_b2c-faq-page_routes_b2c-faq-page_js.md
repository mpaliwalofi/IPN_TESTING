# b2c-faq-page.js

**Path**: `src\api\b2c-faq-page\routes\b2c-faq-page.js`

## Summary
This file defines a Strapi router for the B2C (Business-to-Consumer) FAQ page content type. It uses Strapi's factory function to automatically generate standard CRUD API endpoints (GET, POST, PUT, DELETE) for managing FAQ page content through the CMS, enabling frontend applications to retrieve and display frequently asked questions for consumer-facing applications.
