**Path**: `src\api\b2c-faq-question\content-types\b2c-faq-question\algolia.json`

## Summary
This JSON configuration file defines Algolia search index settings for B2C FAQ questions in a Strapi CMS. It specifies that FAQ questions should be indexed in "strapi_faq_questions" with a maximum string length of 750 characters, while excluding metadata fields (buffer, llm, createdAt, updatedAt) from the search index to optimize search performance and relevance.
