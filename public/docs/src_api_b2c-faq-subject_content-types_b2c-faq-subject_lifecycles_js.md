# lifecycles.js

**Path**: `src\api\b2c-faq-subject\content-types\b2c-faq-subject\lifecycles.js`

## Summary
This file defines lifecycle hooks for the B2C FAQ subject content type that trigger localization after content creation or updates. It integrates with the Akeneo import plugin's localization service to automatically synchronize or translate FAQ subject data across different locales whenever a FAQ subject record is created or modified in the Strapi CMS.
