# schema.json

**Path**: `src\api\b2c-faq\content-types\b2c-faq\schema.json`

## Summary
This is a Strapi CMS content-type schema definition for a B2C FAQ system. It defines a localized collection with fields for managing FAQ indexes and suggestions, including toggle flags to enable/disable FAQ features and their respective index identifiers. The schema supports draft/publish workflows and internationalization (i18n) for multi-language FAQ content management.
