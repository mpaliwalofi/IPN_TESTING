# b2c-faq.js

**Path**: `src\api\b2c-faq\services\b2c-faq.js`

## Summary
This file defines a Strapi service for managing B2C (Business-to-Consumer) FAQ content. It creates a standard CRUD service using Strapi's factory pattern to handle database operations for FAQ entries that will be displayed to end consumers, providing the backend logic layer between the API routes and the data model.
