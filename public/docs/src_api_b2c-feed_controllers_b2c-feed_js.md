# b2c-feed.js

**Path**: `src\api\b2c-feed\controllers\b2c-feed.js`

## Summary
This file defines a Strapi controller for managing B2C (Business-to-Consumer) feed content. It creates a standard CRUD controller using Strapi's factory pattern to handle API requests for the 'b2c-feed' content type, enabling operations like creating, reading, updating, and deleting B2C feed entries through RESTful endpoints.
