# lifecycles.js

**Path**: `src\api\b2c-footer\content-types\b2c-footer\lifecycles.js`

## Summary
This Strapi lifecycle hook module automatically triggers localization after B2C footer content is created or updated. It integrates with the "import-akeneo" plugin's localizer service to synchronize or distribute footer content across different locales, ensuring consistent footer data management in a multi-language e-commerce context.
