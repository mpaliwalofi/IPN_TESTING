## Summary
This is a Strapi CMS schema definition for a B2C "Forgot Password" page content type. It defines a single-type, localized content structure with OpenGraph metadata, dynamic zone content supporting 40+ reusable components (including password reset forms, carousels, FAQs, CTAs, etc.), and LLM integration. The schema enables content managers to build and customize a multi-lingual forgot password page with flexible component-based layouts while maintaining draft/publish workflow capabilities.
