# b2c-forgot-password-page.js

**Path**: `src\api\b2c-forgot-password-page\routes\b2c-forgot-password-page.js`

## Summary
This file creates a RESTful API router for the B2C (Business-to-Consumer) forgot password page content type in Strapi CMS. It uses Strapi's core router factory to automatically generate standard CRUD endpoints (GET, POST, PUT, DELETE) for managing the forgot password page content, allowing administrators to configure and retrieve page elements like text, images, or form settings for the customer-facing password recovery page.
