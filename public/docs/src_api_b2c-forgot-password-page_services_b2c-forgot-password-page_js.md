# b2c-forgot-password-page.js

**Path**: `src\api\b2c-forgot-password-page\services\b2c-forgot-password-page.js`

## Summary
This code defines a Strapi CMS service for managing the B2C (Business-to-Consumer) forgot password page content. It creates a standard CRUD service using Strapi's factory pattern to handle database operations for the forgot password page's content model, allowing administrators to configure and update the text, images, and other content elements displayed on the customer-facing password recovery page.
