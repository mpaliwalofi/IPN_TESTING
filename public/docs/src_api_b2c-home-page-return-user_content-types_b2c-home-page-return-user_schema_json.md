# schema.json

**Path**: `src\api\b2c-home-page-return-user\content-types\b2c-home-page-return-user\schema.json`

## Summary
This is a Strapi CMS schema definition for a B2C homepage specifically designed for authenticated/returning users. It defines a single-type content model with internationalization support that includes OpenGraph metadata and a flexible dynamic content zone composed of 40+ B2C-specific components (carousels, cross-sells, FAQs, hero sections, newsletters, etc.) that can be arranged to create a personalized homepage experience for logged-in customers.
