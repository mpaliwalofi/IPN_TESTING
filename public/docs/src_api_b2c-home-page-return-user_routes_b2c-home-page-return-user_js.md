# b2c-home-page-return-user.js

**Path**: `src\api\b2c-home-page-return-user\routes\b2c-home-page-return-user.js`

## Summary
This file creates a REST API router for managing the B2C (Business-to-Consumer) home page content specifically tailored for returning users. It uses Strapi's factory pattern to automatically generate CRUD endpoints (create, read, update, delete) for the `b2c-home-page-return-user` content type, enabling the CMS to serve personalized home page data to authenticated or repeat visitors of the B2C platform.
