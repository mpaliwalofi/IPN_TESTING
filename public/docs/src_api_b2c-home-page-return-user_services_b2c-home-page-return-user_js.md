# b2c-home-page-return-user.js

**Path**: `src\api\b2c-home-page-return-user\services\b2c-home-page-return-user.js`

## Summary
This code creates a Strapi service for managing the B2C home page content specifically tailored for returning users. It uses Strapi's factory pattern to generate a core service with standard CRUD operations for the 'b2c-home-page-return-user' content type, enabling the CMS to handle personalized home page data that differs from what new users see.
