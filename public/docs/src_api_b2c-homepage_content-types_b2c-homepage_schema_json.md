# schema.json

**Path**: `src\api\b2c-homepage\content-types\b2c-homepage\schema.json`

## Summary
This JSON schema defines a Strapi CMS content type for a B2C anonymous homepage ("Page Accueil - Anonyme"). It configures a single-type content model with internationalization support, draft/publish workflow, and a dynamic zone containing 40+ reusable component types (carousels, hero sections, FAQs, product cross-sells, etc.) that content editors can arrange to build the homepage layout.
