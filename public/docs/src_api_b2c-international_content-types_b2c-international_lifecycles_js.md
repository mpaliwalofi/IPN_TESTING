# lifecycles.js

**Path**: `src\api\b2c-international\content-types\b2c-international\lifecycles.js`

## Summary
This module defines lifecycle hooks for a B2C international content type in Strapi that automatically triggers localization after content creation or updates. Whenever a B2C international entry is created or modified, it invokes the Akeneo import plugin's localization service to synchronize or translate content across different locales, ensuring consistent internationalized content distribution.
