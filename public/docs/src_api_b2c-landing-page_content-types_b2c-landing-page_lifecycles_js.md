# lifecycles.js

**Path**: `src\api\b2c-landing-page\content-types\b2c-landing-page\lifecycles.js`

## Summary
This lifecycle hook module automatically triggers content localization after B2C landing pages are created or updated in Strapi. It integrates with the Akeneo import plugin's localizer service to ensure landing page content is properly localized across different languages/regions whenever changes are saved.
