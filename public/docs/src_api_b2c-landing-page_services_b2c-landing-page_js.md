# b2c-landing-page.js

**Path**: `src\api\b2c-landing-page\services\b2c-landing-page.js`

## Summary
This file creates a Strapi CMS service for managing B2C (Business-to-Consumer) landing page content. It uses Strapi's factory pattern to generate a standard CRUD service that handles data operations for B2C landing page entities, providing an API layer between the content model and controllers. This service enables content managers to create, retrieve, update, and delete landing page content through Strapi's content management system.
