# b2c-order-page.js

**Path**: `src\api\b2c-order-page\routes\b2c-order-page.js`

## Summary
This file defines a Strapi CMS router for the B2C (Business-to-Consumer) order page content type. It creates a standard REST API router that automatically generates CRUD endpoints (GET, POST, PUT, DELETE) for managing B2C order page content in the Strapi application, enabling frontend applications to retrieve and manipulate order page data through the API.
