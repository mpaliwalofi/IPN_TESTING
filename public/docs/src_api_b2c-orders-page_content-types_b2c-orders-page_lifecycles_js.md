# lifecycles.js

**Path**: `src\api\b2c-orders-page\content-types\b2c-orders-page\lifecycles.js`

## Summary
This lifecycle hook file manages localization synchronization for B2C orders page content in Strapi CMS. It automatically triggers the Akeneo import plugin's localization service whenever a B2C orders page entry is created or updated, ensuring that content translations remain synchronized with the Akeneo PIM system.
