## Summary
This is a Strapi CMS schema definition for a B2C product page default content type. It defines a single-type content model with internationalization support that provides a dynamic zone containing 29 different component types (carousels, cross-sells, FAQs, reviews, CTAs, etc.) for building customizable product detail pages. The schema enables content managers to compose and localize product page layouts using pre-defined modular components while supporting draft and publish workflows.
