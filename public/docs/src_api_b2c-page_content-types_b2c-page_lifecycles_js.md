# lifecycles.js

**Path**: `src\api\b2c-page\content-types\b2c-page\lifecycles.js`

## Summary
This code defines lifecycle hooks for a B2C page content type that automatically triggers localization after page creation or updates. It integrates with an Akeneo import plugin by calling the localizer service to synchronize or localize content whenever a B2C page is created or modified in the Strapi CMS.
