# schema.json

**Path**: `src\api\b2c-page\content-types\b2c-page\schema.json`

## Summary
This is a Strapi CMS schema definition for a B2C static page collection type that supports internationalization and draft/publish workflows. It defines a flexible content structure with localized fields including title, slug, OpenGraph metadata, and a dynamic zone that accepts 38+ different component types (carousels, FAQs, hero sections, newsletters, etc.) for building customizable marketing and content pages. The schema enables content editors to construct complex B2C pages by combining various pre-defined UI components in any order.
