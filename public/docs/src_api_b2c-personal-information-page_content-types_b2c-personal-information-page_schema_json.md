# schema.json

**Path**: `src\api\b2c-personal-information-page\content-types\b2c-personal-information-page\schema.json`

## Summary
This is a Strapi CMS schema definition for a B2C "Personal Information" page (French: "Mes informations personnelles"), configured as a single-type content model with internationalization support. The schema defines a flexible page structure with OpenGraph metadata, an LLM component, and a dynamic zone that can contain 40+ different content components ranging from personal information forms and order history to carousels, FAQs, newsletters, and rich content blocks. It enables content managers to build and localize a customizable user profile/account page by assembling various B2C, B2B, and generic UI components.
