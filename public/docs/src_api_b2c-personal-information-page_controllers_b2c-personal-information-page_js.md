# b2c-personal-information-page.js

**Path**: `src\api\b2c-personal-information-page\controllers\b2c-personal-information-page.js`

## Summary
This Strapi controller manages the B2C (Business-to-Consumer) personal information page content by retrieving and transforming its data structure. It fetches the page entity with deeply populated relations and content fields, applies cross-sell transformations for marketing purposes, and returns a sanitized response suitable for displaying customer personal information management interfaces.
