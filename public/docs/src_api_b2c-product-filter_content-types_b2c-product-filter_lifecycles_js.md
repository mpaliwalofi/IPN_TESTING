# lifecycles.js

**Path**: `src\api\b2c-product-filter\content-types\b2c-product-filter\lifecycles.js`

## Summary
This lifecycle hook configuration for the B2C product filter content type triggers localization after create and update operations. It integrates with the Akeneo import plugin to ensure product filter data is properly localized across different languages/regions whenever filter records are created or modified in the CMS.
