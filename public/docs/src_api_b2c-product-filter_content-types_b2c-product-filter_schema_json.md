# schema.json

**Path**: `src\api\b2c-product-filter\content-types\b2c-product-filter\schema.json`

## Summary
This is a Strapi CMS schema definition for a B2C product filter collection type. It defines a localizable content structure for managing product filtering options with fields for filter labels, unique identifiers (UID), display ordering, and associated product recommendations. The schema enables draft/publish workflow and internationalization (i18n) support for creating multi-language filter interfaces in an e-commerce application.
