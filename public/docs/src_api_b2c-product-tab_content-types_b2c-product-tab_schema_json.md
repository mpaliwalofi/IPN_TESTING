# schema.json

**Path**: `src\api\b2c-product-tab\content-types\b2c-product-tab\schema.json`

## Summary
This is a Strapi CMS schema definition for a B2C product tab collection type that manages tabbed content sections on product pages. It defines a structured data model with fields for tab titles, optional color-coded tags (primary, neutral, sells, subscription), anchor links for navigation, and dynamic content zones that can contain rich text or USP (Unique Selling Proposition) lists. The schema enables content editors to create, organize, and configure multiple product information tabs with flexible content components while maintaining consistent styling and presentation options.
