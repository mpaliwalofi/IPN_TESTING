# schema.json

**Path**: `src\api\b2c-product-tag\content-types\b2c-product-tag\schema.json`

## Summary
This schema defines a Strapi collection type for B2C product tags used to label and categorize products in an e-commerce system. It supports internationalization (i18n) and includes attributes for a customizable label, a theme enumeration (new, promotion, limited edition, etc.), a boolean flag to control display on product listing pages (PLP), and a customizable hex color picker for visual representation.
