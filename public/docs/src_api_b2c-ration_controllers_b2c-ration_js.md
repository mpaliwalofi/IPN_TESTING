# b2c-ration.js

**Path**: `src\api\b2c-ration\controllers\b2c-ration.js`

## Summary
This is a Strapi controller for B2C pet food ration recommendations that calculates feeding portions based on pet age and weight. It provides localized error messages and recommendations across multiple European markets (FR, BE, NL, DE, IT), validating input parameters like age (max 13 months) and weight ranges before returning appropriate feeding guidance. The controller serves as the backend API for an Ultra Premium Direct pet food recommendation system.
