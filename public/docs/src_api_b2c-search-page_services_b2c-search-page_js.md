## Summary
This file defines a Strapi service for the B2C (Business-to-Consumer) search page content type. It creates a standard core service using Strapi's factory pattern, which provides default CRUD operations for managing B2C search page data through the CMS. This service acts as the business logic layer between the API controllers and the database for handling B2C search page content.
