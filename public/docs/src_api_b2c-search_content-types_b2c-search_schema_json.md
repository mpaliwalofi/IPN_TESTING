# schema.json

**Path**: `src\api\b2c-search\content-types\b2c-search\schema.json`

## Summary
This schema defines a Strapi content type for configuring B2C (business-to-consumer) search functionality across multiple content types. It manages search index configurations for articles, categories, products, and product suggestions, with each having an index identifier string and an enabled/disabled toggle, while supporting internationalization and draft/publish workflows.
