# schema.json

**Path**: `src\api\b2c-shop-page\content-types\b2c-shop-page\schema.json`

## Summary
This is a Strapi CMS schema definition for a B2C shop locations page ("Magasins" in French) content type. It defines a single-type content model with internationalization support that includes a title, OpenGraph metadata, Booxi API integration for appointment booking, and a flexible dynamic zone content area supporting 40+ different component types (carousels, product grids, FAQs, hero sections, etc.) for building customizable shop/store locator pages.
