## Summary
This schema defines a Strapi CMS single-type content model for a B2C sponsorships/referral page (French: "Mon Parrainage"). The page uses a dynamic zone architecture that allows content editors to compose the page from 43+ different component types including hero banners, carousels, FAQs, contact forms, subscription management widgets, and a MentionMe referral component. It includes OpenGraph metadata for social sharing and supports draft/publish workflows for content management.
