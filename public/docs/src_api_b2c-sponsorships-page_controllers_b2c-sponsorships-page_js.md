# b2c-sponsorships-page.js

**Path**: `src\api\b2c-sponsorships-page\controllers\b2c-sponsorships-page.js`

## Summary
This is a Strapi CMS controller for managing a B2C sponsorships page as a single-type content entity. It overrides the default `find` method to retrieve the sponsorships page data with deep population of related fields, applies data sanitization and transformation, and enhances the response with cross-sell information before returning it to the client.
