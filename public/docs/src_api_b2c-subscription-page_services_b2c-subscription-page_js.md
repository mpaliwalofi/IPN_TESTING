# b2c-subscription-page.js

**Path**: `src\api\b2c-subscription-page\services\b2c-subscription-page.js`

## Summary
This code defines a Strapi CMS service for managing B2C (Business-to-Consumer) subscription page content. It uses Strapi's factory function to create a standard CRUD service that handles database operations for subscription page data, enabling the CMS to store and retrieve information like pricing plans, subscription tiers, or marketing content related to customer subscriptions.
