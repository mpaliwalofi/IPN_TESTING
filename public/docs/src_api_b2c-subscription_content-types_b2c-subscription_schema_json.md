# schema.json

**Path**: `src\api\b2c-subscription\content-types\b2c-subscription\schema.json`

## Summary
This is a Strapi CMS content-type schema definition for a B2C subscription management page ("Mon abonnement" - "My Subscription" in French). It defines a localized collection with configurable product subscription indices and suggestion indices, each with enable/disable flags, supporting draft/publish workflows and internationalization for managing subscription-related content across different locales.
