# b2c-subscription.js

**Path**: `src\api\b2c-subscription\services\b2c-subscription.js`

## Summary
This code defines a Strapi service for managing B2C (Business-to-Consumer) subscription functionality. It creates a standard CRUD service using Strapi's factory pattern to handle database operations for B2C subscription entities, such as creating, reading, updating, and deleting subscription records for individual consumers.
