# b2c-usp.js

**Path**: `src\api\b2c-usp\controllers\b2c-usp.js`

## Summary
This code defines a Strapi CMS controller for managing B2C USP (Business-to-Consumer Unique Selling Propositions) content. It creates a standard CRUD controller using Strapi's factory pattern, enabling API endpoints to create, read, update, and delete B2C marketing/promotional content that highlights unique selling points to end consumers.
