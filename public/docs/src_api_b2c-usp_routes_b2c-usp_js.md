# b2c-usp.js

**Path**: `src\api\b2c-usp\routes\b2c-usp.js`

## Summary
This file creates a standard REST API router for the B2C USP (Unique Selling Proposition) content type in Strapi CMS. It uses Strapi's factory function to automatically generate CRUD endpoints (create, read, update, delete) for managing B2C marketing USPs through the API, allowing the application to store and retrieve unique selling points targeted at business-to-consumer customers.
