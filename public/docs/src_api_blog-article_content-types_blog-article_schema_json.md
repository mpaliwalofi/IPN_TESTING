# schema.json

**Path**: `src\api\blog-article\content-types\blog-article\schema.json`

## Summary
This is a Strapi CMS schema definition for a blog article collection type. It defines the data structure for blog posts, including required fields like title, slug, and multiple images (main, teaser, hero left/right), along with a flexible content area built from dynamic zone components (rich text, embeds, forms, etc.). The schema supports draft/publish workflow and includes SEO features like OpenGraph metadata, reading time estimation, and recommended product associations.
