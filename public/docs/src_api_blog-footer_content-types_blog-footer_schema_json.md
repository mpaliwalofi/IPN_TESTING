# schema.json

**Path**: `src\api\blog-footer\content-types\blog-footer\schema.json`

## Summary
This is a Strapi CMS schema definition for a blog footer content type. It defines a singleton structure that manages footer content including exactly 6 Instagram post images (via repeatable components) and a list of top articles. The schema enables draft/publish workflow and serves as the data model for displaying social media integration and featured content in a blog's footer section.
