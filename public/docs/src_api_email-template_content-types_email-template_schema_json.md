# schema.json

**Path**: `src\api\email-template\content-types\email-template\schema.json`

## Summary
This is a Strapi CMS schema definition for an email template collection type that stores reusable email templates with internationalization support. The schema defines a structure for managing email templates with attributes including a name, slug, subject line, rich text content (localized), and relationships to email layout and sender configurations. It enables content managers to create and maintain multilingual email templates that can be referenced by their slug and composed with specific layouts and sender profiles.
