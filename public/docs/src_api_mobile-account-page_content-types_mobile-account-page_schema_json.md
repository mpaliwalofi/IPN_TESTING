# schema.json

**Path**: `src\api\mobile-account-page\content-types\mobile-account-page\schema.json`

## Summary
This is a Strapi CMS schema definition for a mobile account page ("Mon Compte") single-type content model. It defines the structure for a user account page with a customizable header (background image and title), reusable link cards component, and contact footer component, enabling content managers to configure the mobile account page layout without code changes.
