# schema.json

**Path**: `src\api\mobile-addresses-page\content-types\mobile-addresses-page\schema.json`

## Summary
This is a Strapi CMS schema definition for a single-type content model that manages the mobile "My Addresses" page (Mes adresses in French). The schema defines a page structure with a single reusable component for a contact footer, supporting draft and publish workflows for content management of the mobile app's address management interface.
