# schema.json

**Path**: `src\api\mobile-animal-page\content-types\mobile-animal-page\schema.json`

## Summary
This is a Strapi CMS schema definition for a mobile application's "My Animal" (Mon Animal) page content type. It defines a single-type content structure with internationalization support and draft/publish workflow, containing a dynamic zone that can include various mobile components such as animal profile headers, information sections, medals, product recommendations, blog content, animal deletion options, and contact footers. The schema enables content managers to build and localize a comprehensive animal profile page by combining these predefined mobile components.
