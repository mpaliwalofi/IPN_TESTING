# schema.json

**Path**: `src\api\mobile-contact-form\content-types\mobile-contact-form\schema.json`

## Summary
This is a Strapi CMS schema definition for a mobile contact form single-type content model with internationalization support. It defines a localized contact page structure with fields for title, background image, form title, and dynamic content zones that can include accordions, upsells, contact footer, and markdown components. The schema enables draft/publish workflow and is specifically designed for mobile application contact form pages with multi-language capabilities.
