# mobile-contact-form.js

**Path**: `src\api\mobile-contact-form\controllers\mobile-contact-form.js`

## Summary
This file creates a Strapi controller for handling mobile contact form submissions. It uses Strapi's factory pattern to generate a standard CRUD controller with default REST API endpoints (create, read, update, delete) for the `mobile-contact-form` content type, enabling mobile applications to submit and manage contact form data through the API.
