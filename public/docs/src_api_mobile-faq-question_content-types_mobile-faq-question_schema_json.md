# schema.json

**Path**: `src\api\mobile-faq-question\content-types\mobile-faq-question\schema.json`

## Summary
This is a Strapi CMS schema definition for a mobile FAQ question collection type. It defines a content structure for storing frequently asked questions with a question field (required string), a rich text response field, and a many-to-one relationship to FAQ subjects/topics (b2c_faq_sujet). The schema enables content management for a mobile application's FAQ system with draft/publish workflow capabilities.
