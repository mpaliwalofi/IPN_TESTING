# mobile-faq-question.js

**Path**: `src\api\mobile-faq-question\routes\mobile-faq-question.js`

## Summary
This file defines the routing configuration for a mobile FAQ question API endpoint in a Strapi CMS application. It uses Strapi's factory pattern to automatically generate standard CRUD (Create, Read, Update, Delete) REST API routes for managing FAQ questions specifically designed for mobile applications. The router handles HTTP requests for operations like listing, creating, updating, and deleting mobile FAQ question entries in the content management system.
