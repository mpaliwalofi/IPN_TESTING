# mobile-faq-question.js

**Path**: `src\api\mobile-faq-question\services\mobile-faq-question.js`

## Summary
This file defines a Strapi service for managing mobile FAQ questions in the application. It uses Strapi's factory pattern to create a standard CRUD service with default operations (create, read, update, delete) for the `mobile-faq-question` content type, enabling backend API functionality for storing and retrieving frequently asked questions specifically designed for mobile applications.
