# schema.json

**Path**: `src\api\mobile-homepage\content-types\mobile-homepage\schema.json`

## Summary
This is a Strapi CMS schema definition for a mobile homepage content type that allows content editors to build a dynamic mobile homepage using a flexible component-based structure. The schema defines a single-type collection with draft/publish capabilities and a dynamic zone field that can contain various mobile-specific components including header, news, order management, subscriptions, animal-related content, shop listings, blog posts, contact footer, and markdown content sections.
