# schema.json

**Path**: `src\api\mobile-order-page\content-types\mobile-order-page\schema.json`

## Summary
This is a Strapi CMS schema definition for a mobile "Order" page ("Ma commande" in French) content type. It configures a single-type content structure with internationalization support and draft/publish workflow, allowing content editors to manage a dynamic mobile order page using reusable components like contact footer, upsells, and markdown content blocks.
