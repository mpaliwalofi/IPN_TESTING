# mobile-order-page.js

**Path**: `src\api\mobile-order-page\routes\mobile-order-page.js`

## Summary
This file defines a RESTful API router for managing mobile order page content in a Strapi CMS application. It uses Strapi's factory function to automatically generate standard CRUD endpoints (create, read, update, delete) for the mobile-order-page content type, enabling API clients to manage the mobile ordering interface configuration and content.
