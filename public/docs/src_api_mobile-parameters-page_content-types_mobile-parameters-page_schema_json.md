# schema.json

**Path**: `src\api\mobile-parameters-page\content-types\mobile-parameters-page\schema.json`

## Summary
This is a Strapi CMS schema definition for a mobile application's "Parameters" (Settings) page content type. It defines a single-type collection that supports internationalization (i18n) and draft/publish workflows, with a dynamic zone content field that can contain link cards and contact footer components. The schema enables content managers to configure and localize the settings page content for a mobile application through the CMS.
