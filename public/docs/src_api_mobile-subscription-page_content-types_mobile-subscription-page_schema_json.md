# schema.json

**Path**: `src\api\mobile-subscription-page\content-types\mobile-subscription-page\schema.json`

## Summary
This is a Strapi CMS schema definition for a single-type content model that manages the "Mon Abonnement" (My Subscription) page in a mobile application. The schema defines a localized content type with draft/publish workflow capabilities, containing a single FAQ component specifically designed for subscription-related questions. It serves as the data structure for managing subscription page content across multiple languages in the mobile app's backend.
