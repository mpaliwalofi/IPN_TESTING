# hero-cover-pro.json

**Path**: `src\components\b2b\hero-cover-pro.json`

## Summary
This is a Strapi CMS component schema definition for a B2B hero section with cover media. It defines a reusable content structure that includes a required cover media element (supporting images, videos, files, or audios), an optional overlay configuration, a cover URL string, and exactly 3 horizontal card components arranged in a repeatable pattern.
