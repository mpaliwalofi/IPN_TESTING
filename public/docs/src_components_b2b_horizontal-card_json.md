# horizontal-card.json

**Path**: `src\components\b2b\horizontal-card.json`

## Summary
This is a Strapi CMS component schema definition for a B2B horizontal card UI element. It defines the data structure for a promotional card containing required fields for media (image/video), title, subtitle, and a call-to-action button, plus an optional overlay configuration. This component is likely used in B2B marketing pages to display featured content or product offerings in a horizontal layout format.
