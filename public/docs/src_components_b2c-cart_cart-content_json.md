# cart-content.json

**Path**: `src\components\b2c-cart\cart-content.json`

## Summary
This is a Strapi CMS component schema definition for a B2C shopping cart's content structure. It defines a reusable "Cart content" component with two configurable attributes: footer text for the order summary and a rich-text header content field, allowing content managers to customize the cart's order summary display without code changes.
