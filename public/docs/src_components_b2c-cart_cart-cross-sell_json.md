# cart-cross-sell.json

**Path**: `src\components\b2c-cart\cart-cross-sell.json`

## Summary
This JSON file defines a Strapi component schema for managing cart cross-sell functionality in a B2C e-commerce system. It specifies a simple data structure with a single boolean attribute "enabled" that controls whether cross-sell recommendations are displayed in the shopping cart, defaulting to true and marked as required.
