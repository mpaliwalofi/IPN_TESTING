# menu-banner-footer.json

**Path**: `src\components\b2c-menu\menu-banner-footer.json`

## Summary
This JSON file defines a Strapi component schema for a menu banner footer in a B2C (business-to-consumer) context. It specifies the data structure for a toggleable banner that can appear at the bottom of menus, containing optional text content and a call-to-action button component. This configuration allows content managers to enable/disable and customize promotional or informational footers within navigation menus.
