# account-links.json

**Path**: `src\components\b2c\account-links.json`

## Summary
This is a Strapi CMS component schema that defines the structure for "Account Links" in a B2C (business-to-consumer) context. It specifies a data model containing a repeatable collection of CTA (Call-to-Action) dashboard components, allowing content editors to create multiple actionable links or buttons that direct users to various account-related dashboards or sections.
