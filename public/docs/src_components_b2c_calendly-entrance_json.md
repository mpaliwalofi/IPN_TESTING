# calendly-entrance.json

**Path**: `src\components\b2c\calendly-entrance.json`

## Summary
This is a Strapi CMS component schema that defines a "Calendly Entrance" content type for B2C (business-to-consumer) use cases. It configures a UI component that integrates Calendly scheduling functionality with customizable presentation elements including title, descriptive text, icons (picto), images with responsive positioning options, and a CTA button to trigger the Calendly booking flow. The schema supports various layout configurations for desktop and mobile views, enabling marketing teams to create branded entry points for appointment scheduling throughout their web properties.
