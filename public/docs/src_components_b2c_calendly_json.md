# calendly.json

**Path**: `src\components\b2c\calendly.json`

## Summary
This is a Strapi CMS component schema definition for integrating Calendly scheduling functionality into a B2C (business-to-consumer) application. It defines the data structure for storing Calendly widget configuration, including the site ID for linking to a specific Calendly account and customization options for appearance (colors) and dimensions (width/height) of the embedded scheduling widget.
