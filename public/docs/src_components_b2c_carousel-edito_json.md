# carousel-edito.json

**Path**: `src\components\b2c\carousel-edito.json`

## Summary
This is a Strapi CMS component schema definition for a B2C carousel editorial content block. It defines the data structure for a promotional carousel that includes a title, rich text content, multiple images, optional CTA button, and layout configuration options (text positioning, background toggle, mobile image ordering, and heading hierarchy). The component is designed to create flexible, editorial-style carousel presentations for business-to-consumer web pages.
