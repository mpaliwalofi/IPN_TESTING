# carousel-generic-item-images.json

**Path**: `src\components\b2c\carousel-generic-item-images.json`

## Summary
This is a Strapi CMS component schema definition for a B2C carousel item that displays media content with responsive variations. It configures a reusable content type that allows editors to specify separate desktop and mobile media assets (images, videos, files, or audio) and control their vertical positioning (top or bottom) within a carousel interface.
