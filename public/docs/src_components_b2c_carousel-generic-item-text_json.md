# carousel-generic-item-text.json

**Path**: `src\components\b2c\carousel-generic-item-text.json`

## Summary
This is a Strapi CMS component schema definition for a B2C carousel item's text content. It defines a reusable content structure that stores rich text content with separate fields for desktop and mobile display variants, allowing content editors to customize carousel text presentation based on device type. This component would typically be used within a larger carousel content type to manage marketing or promotional content on customer-facing pages.
