# carousel-generic-item.json

**Path**: `src\components\b2c\carousel-generic-item.json`

## Summary
This JSON schema defines a Strapi component for a B2C carousel item that displays promotional or informational content. It structures a carousel slide with a title, descriptive text, images, and multiple call-to-action buttons, enabling content editors to create consistent, interactive carousel elements for e-commerce or marketing purposes.
