# carousel-generic.json

**Path**: `src\components\b2c\carousel-generic.json`

## Summary
This JSON file defines a Strapi CMS component schema for a generic B2C carousel widget. It configures a reusable carousel component with customizable properties including auto-rotation interval (1-30 seconds), title, repeatable carousel items, and responsive display settings that allow different numbers of items per slide on desktop (3, 4, or 6 items) versus mobile (1 or 2 items).
