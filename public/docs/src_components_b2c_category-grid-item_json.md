# category-grid-item.json

**Path**: `src\components\b2c\category-grid-item.json`

## Summary
This is a Strapi CMS component schema definition for a B2C (business-to-consumer) category grid item, used to display product or category tiles in an e-commerce interface. The component defines the data structure for items that include a name, image, optional overlay styling, URL link, and price, allowing content managers to create and manage grid-based category or product displays in the storefront.
