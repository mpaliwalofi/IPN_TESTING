# category-grid.json

**Path**: `src\components\b2c\category-grid.json`

## Summary
This is a Strapi CMS component schema that defines a "Category Grid" structure for a B2C (business-to-consumer) application. The component enforces exactly 4 category items (via the `categories` field with min/max constraints of 4) and allows an optional media attachment, enabling content editors to create consistent grid-based category displays for e-commerce or content presentation purposes.
