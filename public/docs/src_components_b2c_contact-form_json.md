# contact-form.json

**Path**: `src\components\b2c\contact-form.json`

## Summary
This is a Strapi CMS component schema definition for a B2C contact form. It defines the data structure for a contact form component that includes a title, subtitle, repeatable form inputs, a submit button, an associated image/media file, an optional overlay configuration, and a collection of USPs (Unique Selling Points). The schema is used by Strapi's content management system to create and validate contact form content entries in the database.
