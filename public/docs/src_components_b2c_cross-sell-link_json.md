# cross-sell-link.json

**Path**: `src\components\b2c\cross-sell-link.json`

## Summary
This is a Strapi CMS component schema definition for a B2C cross-sell link feature. It defines a reusable component that associates a tracking category with a specific cross-sell product or offer through a one-to-one relationship, enabling e-commerce functionality to promote related products to customers while tracking the source of those promotions.
