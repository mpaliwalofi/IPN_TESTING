# cross-sell-main-product.json

**Path**: `src\components\b2c\cross-sell-main-product.json`

## Summary
This JSON schema defines a Strapi component for B2C cross-selling functionality, specifically for displaying a main product alongside related product recommendations. The component configures attributes for a product reference, an associated image, placement positioning (left/center/right), and analytics tracking, enabling merchandisers to strategically position featured products to drive additional sales through cross-sell opportunities.
