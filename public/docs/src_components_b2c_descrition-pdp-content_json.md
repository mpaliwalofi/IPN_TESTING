# descrition-pdp-content.json

**Path**: `src\components\b2c\descrition-pdp-content.json`

## Summary
This is a Strapi CMS component schema definition for a "Description PDP Content" (Product Detail Page content) component used in a B2C e-commerce context. It defines a reusable content structure with a single boolean attribute (`enabled`) that allows content editors to toggle the visibility or activation state of product description content on product detail pages.
