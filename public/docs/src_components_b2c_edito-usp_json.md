# edito-usp.json

**Path**: `src\components\b2c\edito-usp.json`

## Summary
This is a Strapi CMS component schema definition for a B2C "EditoUSP" (Unique Selling Point) content type. It defines the structure for displaying marketing messages with a title, rich text content, and either a selectable icon (from a predefined set of 38 pet-related and e-commerce icons) or an uploaded image. The component is designed for a pet supply e-commerce platform to showcase key business advantages and features to customers.
