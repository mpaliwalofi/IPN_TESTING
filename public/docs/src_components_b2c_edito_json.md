# edito.json

**Path**: `src\components\b2c\edito.json`

## Summary
This is a Strapi CMS schema definition for a B2C "Edito" content component, which represents an editorial content block for displaying rich content with images. The component allows content editors to configure a customizable section with a title, markdown text, multiple images, CTA buttons, layout options (text positioning, mobile image ordering), theming, visibility rules based on user authentication/subscription status, and related overlay sets.
