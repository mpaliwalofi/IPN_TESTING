# expertise.json

**Path**: `src\components\b2c\expertise.json`

## Summary
This is a Strapi CMS component schema definition for a B2C "Expertise" section that structures content for showcasing business expertise or capabilities. It defines a reusable content type with a background image, mobile-optimized image, a title, three descriptive text fields for presenting information in a structured format, and a call-to-action button component. This schema allows content editors to create consistent expertise showcase sections across the website without requiring code changes.
