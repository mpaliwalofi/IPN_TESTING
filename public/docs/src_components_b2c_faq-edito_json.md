# faq-edito.json

**Path**: `src\components\b2c\faq-edito.json`

## Summary
This is a Strapi CMS component schema definition for a B2C FAQ editorial section that combines visual content with question-answer functionality. The component allows configuration of layout options (text alignment, mobile image positioning), supports an optional image with overlay effects, and establishes a one-to-many relationship with FAQ questions. It serves as a structured content type for displaying FAQ content with accompanying imagery in a business-to-consumer context.
