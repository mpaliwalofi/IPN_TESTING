# faqs.json

**Path**: `src\components\b2c\faqs.json`

## Summary
This JSON file defines a Strapi CMS component schema for a B2C FAQ (Frequently Asked Questions) section. It configures a reusable component that contains a title, a one-to-many relationship with FAQ question entities, and customizable HTML heading levels for both the title and individual questions to support proper semantic HTML structure and accessibility.
