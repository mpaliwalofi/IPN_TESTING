# frequently-bought-together.json

**Path**: `src\components\b2c\frequently-bought-together.json`

## Summary
This is a Strapi CMS component schema definition for a "Frequently Bought Together" feature in a B2C e-commerce application. It defines a reusable content structure with a title field and a repeatable list of products, allowing content editors to configure product bundles that are commonly purchased together to encourage cross-selling.
