# funnel-payment-step.json

**Path**: `src\components\b2c\funnel-payment-step.json`

## Summary
This JSON file defines a Strapi component schema for a B2C funnel payment step configuration. It establishes a simple data structure with a single "enable" boolean attribute (defaulting to true) that controls whether a payment step is active in a sales funnel workflow. The component is designed to be reusable across different B2C checkout or sales funnel implementations within the CMS.
