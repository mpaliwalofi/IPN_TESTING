# glossary.json

**Path**: `src\components\b2c\glossary.json`

## Summary
This JSON file defines a Strapi CMS component schema for a B2C Glossary feature. It specifies the data structure for glossary entries with a title, subtitle, configurable heading level (h1-h6, defaulting to h2), and an associated USP (Unique Selling Points) collection list component, allowing content editors to create structured glossary sections with marketing highlights.
