**Path**: `src\components\b2c\hero-cover-category.json`

## Summary
This is a Strapi content type schema that defines a B2C hero cover category component for displaying featured product categories. It structures a promotional banner with a hero product image, title text, and responsive long-form content description. This component is likely used on e-commerce category landing pages to showcase and explain product categories in a visually prominent way.
