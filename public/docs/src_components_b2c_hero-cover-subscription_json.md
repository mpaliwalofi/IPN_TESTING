# hero-cover-subscription.json

**Path**: `src\components\b2c\hero-cover-subscription.json`

## Summary
This is a Strapi CMS component schema definition for a B2C hero/cover section specifically designed for subscription pages. It defines the data structure for a hero banner that includes a customizable title with SEO-optimized heading levels, SEO text fields with URLs, up to two top CTA buttons, a USP (Unique Selling Points) list component, and a bottom CTA button to drive subscription conversions.
