# hero-cover.json

**Path**: `src\components\b2c\hero-cover.json`

## Summary
This is a Strapi CMS component schema definition for a B2C hero cover/banner element. It defines the data structure for a promotional hero section with responsive images (mobile/desktop), title, subtitle, overlay styling, up to 2 CTA buttons, and a full-height display option. This configuration enables content editors to create and manage hero banners for B2C marketing pages without code changes.
