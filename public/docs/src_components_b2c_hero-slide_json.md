# hero-slide.json

**Path**: `src\components\b2c\hero-slide.json`

## Summary
This is a Strapi CMS schema definition for a B2C hero carousel slide component used in e-commerce or marketing websites. It defines the data structure for hero banner slides including separate desktop/mobile images, overlay elements, titles, subtitles, review text, country flags, CTA buttons (1-2 max), and optional disclaimer text with asterisk links. The schema supports responsive design with dedicated mobile-specific fields and includes an active/inactive toggle for controlling slide visibility.
