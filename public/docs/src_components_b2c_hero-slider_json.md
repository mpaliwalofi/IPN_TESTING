# hero-slider.json

**Path**: `src\components\b2c\hero-slider.json`

## Summary
This is a Strapi CMS component schema that defines a B2C hero slider configuration for displaying rotating promotional content on a website. It manages a collection of 1-10 hero slides with a configurable auto-rotation interval between 1-30 seconds (defaulting to 3 seconds per slide). The component serves as a data structure for marketing teams to create and manage homepage carousels or banner rotators in a business-to-consumer context.
