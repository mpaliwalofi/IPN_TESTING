# instant-search-algolia.json

**Path**: `src\components\b2c\instant-search-algolia.json`

## Summary
This is a Strapi component schema definition for an Algolia instant search feature in a B2C (business-to-consumer) context. It defines a simple component with a single boolean attribute "enabled" that controls whether the Algolia instant search functionality is active, allowing content managers to toggle search capabilities on or off through the CMS.
