# item-blog.json

**Path**: `src\components\b2c\item-blog.json`

## Summary
This JSON file defines a Strapi component schema for displaying individual blog items in a B2C (business-to-consumer) context. It establishes a one-to-one relationship with blog articles, allowing this component to reference and display a single blog article entity. This component serves as a reusable content block that can be embedded in various content types to feature or link to specific blog articles throughout the application.
