# jumbotron.json

**Path**: `src\components\b2c\jumbotron.json`

## Summary
This is a Strapi CMS component schema definition for a B2C Jumbotron (hero banner) component. It defines the data structure for storing responsive hero images with separate mobile and desktop versions, plus an optional overlay configuration that links to an overlay-set content type. The schema enables content editors to manage large promotional banners with device-specific imagery and customizable overlay effects.
