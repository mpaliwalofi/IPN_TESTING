# last-order.json

**Path**: `src\components\b2c\last-order.json`

## Summary
This is a Strapi CMS component schema definition for a B2C "last order" UI component. It defines the data structure and configuration options for displaying a customer's most recent order, including boolean flags to control whether the component is enabled, its visual presentation (flipped layout, icon display, inline rendering), and whether to show a header.
