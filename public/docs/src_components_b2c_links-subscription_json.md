# links-subscription.json

**Path**: `src\components\b2c\links-subscription.json`

## Summary
This is a Strapi CMS component schema definition for a B2C "LinksSubscription" content type that manages subscription-related links and calls-to-action. It defines a structure with a title string field and a repeatable collection of CTA buttons, allowing content editors to create grouped subscription action buttons with associated titles for customer-facing interfaces.
