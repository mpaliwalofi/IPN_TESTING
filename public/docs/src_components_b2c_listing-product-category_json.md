# listing-product-category.json

**Path**: `src\components\b2c\listing-product-category.json`

## Summary
This is a Strapi CMS component schema definition for a B2C listing product category page. It defines a structure that includes up to 2 push categories for featured promotions, repeatable category filter groups for product filtering, and a subscription flag, all supporting internationalization (i18n). The component serves as a content model for organizing and displaying product categories with associated filtering and promotional capabilities in an e-commerce context.
