# mention-me.json

**Path**: `src\components\b2c\mention-me.json`

## Summary
This is a Strapi CMS component schema definition for a "MentionMe" B2C (business-to-consumer) referral feature. It defines a reusable content component with a single `model` attribute that can be set to either "landing" (for landing pages) or "dashboard" (for user dashboard views), allowing content editors to configure where and how the MentionMe referral widget appears in the application.
