# order-shipping.json

**Path**: `src\components\b2c\order-shipping.json`

## Summary
This is a Strapi CMS component schema definition for managing order shipping configuration in a B2C (business-to-consumer) context. It defines a reusable component called "Order Shipping" with a single boolean attribute that controls whether shipping is enabled for orders. This component allows content managers to toggle shipping availability as part of order management functionality within the CMS.
