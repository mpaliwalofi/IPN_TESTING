# order-total.json

**Path**: `src\components\b2c\order-total.json`

## Summary
This is a Strapi CMS component schema definition for an "Order Total" component used in B2C (business-to-consumer) e-commerce functionality. It defines a simple reusable component with a single boolean attribute "enabled" that allows toggling the visibility or activation state of order total displays in the system. The component is stored in the "components_b2c_order_totals" collection and can be embedded within other content types to manage order total presentation.
