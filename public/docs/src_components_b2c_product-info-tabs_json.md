# product-info-tabs.json

**Path**: `src\components\b2c\product-info-tabs.json`

## Summary
This is a Strapi CMS component schema definition for a B2C product information tabs feature. It defines a reusable component that contains a repeatable collection of product information items (referencing the "b2c.product-info" component), enabling content editors to create multiple tabbed sections for displaying product details on e-commerce pages.
