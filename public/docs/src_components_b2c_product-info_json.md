# product-info.json

**Path**: `src\components\b2c\product-info.json`

## Summary
This is a Strapi CMS component schema definition for a B2C product information component. It defines a reusable content structure with two required fields: a string-based title and a rich text markdown field, which can be used to display detailed product information across the B2C application. The schema enables content managers to create and manage structured product descriptions that can be embedded in various content types.
