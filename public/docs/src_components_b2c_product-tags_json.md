# product-tags.json

**Path**: `src\components\b2c\product-tags.json`

## Summary
This is a Strapi CMS component schema definition for B2C product tags that allows associating up to two product tags with a product. It defines a reusable component with two one-to-one relationships to the `b2c-product-tag` content type, enabling products to be labeled with a primary and secondary tag for categorization or marketing purposes.
