# products-list.json

**Path**: `src\components\b2c\products-list.json`

## Summary
This is a Strapi CMS component schema definition for a B2C products list feature. It defines a reusable content type that stores product SKU (Stock Keeping Unit) identifiers as required string fields, enabling content editors to reference and display specific products in the system. The component serves as a structured way to link product catalog items to various content pages or sections within the B2C application.
