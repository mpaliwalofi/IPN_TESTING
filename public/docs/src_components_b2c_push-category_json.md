# push-category.json

**Path**: `src\components\b2c\push-category.json`

## Summary
This is a Strapi CMS component schema that defines a "Push Category" content type for a B2C (business-to-consumer) e-commerce system. It structures promotional or featured category data with attributes for a clickable URL, product image, tracking campaign identifier, and an optional overlay set for additional visual content or calls-to-action.
