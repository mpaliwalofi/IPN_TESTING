# review-cms.json

**Path**: `src\components\b2c\review-cms.json`

## Summary
This is a Strapi CMS component schema definition for a B2C review system. It defines the data structure for customer reviews, including fields for author name, numerical rating (note), comment text, date, optional image, and a theme selector with predefined color options (mustard, beige, night blue, water green). The schema is used to create and manage user-generated review content in the CMS with validation rules and default values.
