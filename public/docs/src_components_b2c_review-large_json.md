# review-large.json

**Path**: `src\components\b2c\review-large.json`

## Summary
This is a Strapi CMS schema definition for a B2C "ReviewLarge" component that structures customer review data for display. It defines attributes for review metadata (title, rating up to 5 stars, review counts) and contains a repeatable collection of review items plus an optional call-to-action button. This component enables content editors to create and manage large review sections with aggregated ratings and individual customer reviews for e-commerce or product pages.
