# reviews-product-pdp-content.json

**Path**: `src\components\b2c\reviews-product-pdp-content.json`

## Summary
This is a Strapi CMS component schema definition for managing product review functionality on Product Detail Pages (PDP) in a B2C (business-to-consumer) context. It defines a simple data structure with a single boolean toggle to enable or disable the display of product reviews on PDPs, allowing content managers to control review visibility per product through the CMS interface.
