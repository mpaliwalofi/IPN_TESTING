# sales-force-form.json

**Path**: `src\components\b2c\sales-force-form.json`

## Summary
This is a Strapi CMS schema definition for a SalesForce form component used in B2C (business-to-consumer) interactions. It defines the structure for a customizable lead capture form that includes repeatable form inputs, a configurable image, campaign tracking, success/thank-you messaging, and an optional opt-in component, which can then submit data to Salesforce for marketing and sales purposes.
