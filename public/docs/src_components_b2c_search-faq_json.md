# search-faq.json

**Path**: `src\components\b2c\search-faq.json`

## Summary
This is a Strapi CMS schema definition for a B2C SearchFAQ component that structures FAQ search functionality. It defines a data model with a hero cover media field and establishes one-to-many relationships to FAQ subjects and individual FAQ questions, enabling the organization and display of searchable FAQ content for business-to-consumer interactions.
