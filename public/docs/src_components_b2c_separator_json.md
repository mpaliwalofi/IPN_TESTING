# separator.json

**Path**: `src\components\b2c\separator.json`

## Summary
This is a Strapi CMS component schema definition for a B2C separator UI element. It defines a reusable separator component with three configurable attributes: whether the separator line is dashed (default true) and whether it has top or bottom margins (both default false). The component serves as a visual divider in the B2C frontend, allowing content editors to add customizable horizontal separators between content sections.
