# shop-details.json

**Path**: `src\components\b2c\shop-details.json`

## Summary
This is a Strapi CMS component schema definition for a "ShopDetails" B2C (business-to-consumer) content component. It defines the structure for displaying shop-related information with configurable layout options including text alignment (textRight), background visibility, mobile image ordering (mobileImageFirst), and an embedded call-to-action button component. The schema enables content editors to create flexible, responsive shop detail sections with customizable presentation settings.
