# shop-location-listing.json

**Path**: `src\components\b2c\shop-location-listing.json`

## Summary
This is a Strapi CMS component schema definition for a B2C shop location listing feature. It defines a reusable content type with a single boolean attribute "enabled" (defaulting to true) that controls whether a specific shop location should be displayed in listings. The schema allows content managers to toggle the visibility of individual shop locations in the storefront.
