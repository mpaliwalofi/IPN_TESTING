# similar-products-pdp-content.json

**Path**: `src\components\b2c\similar-products-pdp-content.json`

## Summary
This is a Strapi CMS component schema definition that configures a "Similar Products" section for product detail pages (PDP) in a B2C e-commerce application. It defines three attributes: a customizable title string, a boolean flag to toggle background styling, and an enabled/disabled toggle, both defaulting to true. The schema allows content editors to manage the display and behavior of product recommendation sections on individual product pages.
