# social-link.json

**Path**: `src\components\b2c\social-link.json`

## Summary
This JSON schema defines a Strapi component for managing social media links in a B2C (business-to-consumer) context. It specifies the data structure for social links with required fields including a URL, alternative text for accessibility, and an icon type restricted to seven major social media platforms (Instagram, Facebook, TikTok, Twitter, Pinterest, YouTube, and LinkedIn), plus an optional title field.
