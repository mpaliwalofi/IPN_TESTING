# subscription-abort-reason.json

**Path**: `src\components\b2c\subscription-abort-reason.json`

## Summary
This is a Strapi CMS component schema definition that structures data for tracking subscription cancellation reasons in a B2C (business-to-consumer) context. It defines a simple data model with a single "reason" text field to capture why customers abort or cancel their subscriptions. This component would typically be used to collect feedback during subscription cancellation flows for analytics and customer retention insights.
