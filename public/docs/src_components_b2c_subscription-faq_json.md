# subscription-faq.json

**Path**: `src\components\b2c\subscription-faq.json`

## Summary
This JSON file defines a Strapi content-type schema for a B2C subscription FAQ component. It structures frequently asked questions related to subscriptions, including a title, a one-to-many relationship to FAQ questions, and configurable abort/cancellation reasons with labels and slugs for handling subscription termination scenarios. The schema enables content management for subscription help documentation and tracks why users might cancel their subscriptions.
