# subscription-payment.json

**Path**: `src\components\b2c\subscription-payment.json`

## Summary
This is a Strapi CMS component schema definition that configures a B2C subscription payment feature toggle. It defines a simple data structure with a single boolean attribute "enabled" (defaulting to true) that allows administrators to enable or disable subscription payment functionality. The component serves as a reusable content type for managing subscription payment availability across the application.
