# subscription-upsell.json

**Path**: `src\components\b2c\subscription-upsell.json`

## Summary
This is a Strapi CMS component schema definition for a B2C subscription upsell feature. It defines a reusable content type with a single boolean attribute "enabled" that controls whether subscription upsell prompts are active, allowing content managers to toggle upsell functionality on or off for different contexts in the application.
