# timeline.json

**Path**: `src\components\b2c\timeline.json`

## Summary
This is a Strapi CMS component schema definition for a B2C Timeline feature. It defines a reusable content structure that contains multiple timeline blocks (with a minimum of one required) and a sort type option to display events in either oldest-first or newest-first order. The schema enables content editors to create and manage chronological event displays for business-to-consumer applications.
