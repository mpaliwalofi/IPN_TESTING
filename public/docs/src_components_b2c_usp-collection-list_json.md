# usp-collection-list.json

**Path**: `src\components\b2c\usp-collection-list.json`

## Summary
This JSON file defines a Strapi component schema for managing B2C (Business-to-Consumer) USP (Unique Selling Points) collections. It establishes a one-to-many relationship structure that allows grouping multiple USP items together, enabling the CMS to organize and display marketing value propositions or product benefits as reusable content collections across the B2C platform.
