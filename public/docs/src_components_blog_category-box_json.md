# category-box.json

**Path**: `src\components\blog\category-box.json`

## Summary
This is a Strapi CMS component schema definition for a "CategoryBox" feature in a blog system. It defines a reusable content structure that connects a single blog category with multiple category links through a one-to-one relationship and a repeatable component respectively. This component enables the creation of navigational category boxes that display blog categories along with their associated links in the CMS interface.
