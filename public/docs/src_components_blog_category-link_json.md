# category-link.json

**Path**: `src\components\blog\category-link.json`

## Summary
This is a Strapi CMS component schema definition for a blog category link. It defines a reusable content structure with two fields: a required "label" string for the display text and an optional "url" string for the link destination. This component allows content editors to create and manage clickable category links within the blog system, likely for navigation or categorization purposes.
