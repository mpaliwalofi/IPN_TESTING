# component-blog-push-product.json

**Path**: `src\components\blog\component-blog-push-product.json`

## Summary
This is a Strapi CMS component schema that defines a "PushProduct" content type for promoting products within blog posts. It configures data fields for displaying product information including title, description (rich text), price, URL, image, and an additional promotional link with its own title and URL. The schema enables content creators to embed product promotional cards or widgets directly into blog content.
