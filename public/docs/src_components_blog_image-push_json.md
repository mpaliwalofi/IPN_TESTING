# image-push.json

**Path**: `src\components\blog\image-push.json`

## Summary
This is a Strapi CMS schema definition for a reusable "Category Image Push" component that combines media content with a call-to-action button. It defines a data structure allowing content editors to upload a single media file (image, video, or generic file) alongside a CTA button component, likely used for promotional content blocks or category navigation elements in a blog system.
