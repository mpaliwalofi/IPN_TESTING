# push-products-list.json

**Path**: `src\components\blog\push-products-list.json`

## Summary
This is a Strapi CMS component schema definition for a "PushProductsList" content type that enables content creators to display recommended or upsell products within blog content. The component contains an intertitle field for section headings and a repeatable upSellProducts field that can hold multiple recommended product entries, facilitating product promotion and cross-selling within blog articles.
