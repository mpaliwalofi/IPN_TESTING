# recommended-products.json

**Path**: `src\components\blog\recommended-products.json`

## Summary
This is a Strapi CMS component schema definition that creates a reusable "Recommended Products" component for blog content. It defines a simple data structure with a single `product_reference` string field, allowing content editors to link or reference products within blog posts. This component enables cross-linking between blog content and product catalogs, typically used for product recommendations or affiliate marketing within blog articles.
