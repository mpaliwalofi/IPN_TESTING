# sales-force-form-blog.json

**Path**: `src\components\blog\sales-force-form-blog.json`

## Summary
This is a Strapi CMS component schema definition for a Salesforce-integrated blog form component. It defines the data structure for a lead capture form that includes customizable text fields (title, subtitle, placeholder, call-to-action), thank you messaging, an optional image with layout controls (right/left positioning, mobile display), and Salesforce integration fields (campaign, source, optin) for tracking lead submissions.
