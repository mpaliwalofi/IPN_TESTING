# text-half-image-side.json

**Path**: `src\components\blog\text-half-image-side.json`

## Summary
This is a Strapi CMS component schema that defines a half-text, half-image layout block for blog content. It configures a flexible content section where text and an image can be positioned side-by-side, with the image optionally placed on the left or right side, and includes support for titles with configurable heading levels, rich text content, optional call-to-action buttons, and background styling options.
