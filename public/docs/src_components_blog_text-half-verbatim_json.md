# text-half-verbatim.json

**Path**: `src\components\blog\text-half-verbatim.json`

## Summary
This is a Strapi CMS component schema definition for a blog layout that displays text content alongside a verbatim quote or testimonial in a two-column format. The component allows content editors to configure the layout side (left/right), optional background, heading levels (h1-h6), and includes support for rich text, verbatim text, and an optional call-to-action button.
