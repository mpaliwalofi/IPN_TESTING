# up-sell.json

**Path**: `src\components\blog\up-sell.json`

## Summary
This is a Strapi CMS component schema definition for an "UpSell" feature in a blog context. It defines a data structure that allows content creators to add a repeatable list of recommended products to blog posts for cross-selling purposes. The component acts as a container for multiple `blog.recommended-products` entries, enabling marketers to promote related products alongside blog content.
