# breadcrumb.json

**Path**: `src\components\generic\breadcrumb.json`

## Summary
This is a Strapi CMS component schema that defines a Breadcrumb navigation component structure. It allows content editors to create breadcrumb trails with up to 5 repeatable BreadcrumbItem sub-components, which are used to display hierarchical page navigation paths in the UI. The schema establishes the data model for storing and managing breadcrumb navigation data in the CMS.
