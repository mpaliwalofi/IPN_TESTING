# cta-button.json

**Path**: `src\components\generic\cta-button.json`

## Summary
This is a Strapi CMS component schema definition for a customizable call-to-action (CTA) button. It defines the data structure and configuration options for CTA buttons used throughout the application, including required fields for text and URL, optional settings for email links (`mail_to`), link behavior (`target_blank`, `nofollow`), and visual styling options (color schemes and button styles like solid, ghost, or outline). The schema also supports an optional icon image to enhance the button's visual appeal.
