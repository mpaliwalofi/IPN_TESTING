# cta-dashboard.json

**Path**: `src\components\generic\cta-dashboard.json`

## Summary
This JSON file defines a Strapi CMS content type schema for a call-to-action (CTA) dashboard component. It specifies the structure for dashboard action buttons with three required fields: a label, a link URL, and an icon selector with predefined options related to account management, shopping, FAQ, and certification features. The schema is used to create reusable CTA components that can be embedded in various parts of a dashboard interface.
