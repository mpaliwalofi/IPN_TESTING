# cta-listing.json

**Path**: `src\components\generic\cta-listing.json`

## Summary
This is a Strapi CMS component schema that defines a "CTAListing" (Call-to-Action Listing) content type for managing promotional or action-oriented content blocks. The component allows content editors to create titled sections with an associated icon (from a predefined set of 70+ icons like avatars, arrows, shopping cart, etc.) and multiple CTA buttons, with configurable heading levels for accessibility and SEO purposes.
