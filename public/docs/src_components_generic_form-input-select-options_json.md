# form-input-select-options.json

**Path**: `src\components\generic\form-input-select-options.json`

## Summary
This is a Strapi CMS component schema definition that defines the structure for form select input options. It creates a reusable content type called "FormInputOptions" with a single `label` attribute (string type), which would be used to populate dropdown/select menu choices in forms throughout the CMS.
