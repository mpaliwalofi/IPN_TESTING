# form-input.json

**Path**: `src\components\generic\form-input.json`

## Summary
This is a Strapi CMS component schema definition for a generic form input field. It defines the data structure and configuration options for dynamic form inputs, supporting multiple input types (text, email, checkbox, select, textarea, etc.) with customizable properties like labels, validation requirements, default values, and select options. The schema enables content managers to create and configure reusable form input components within the CMS without writing code.
