# href-lang.json

**Path**: `src\components\generic\href-lang.json`

## Summary
This is a Strapi CMS component schema definition for managing hreflang tags, which are used for international SEO to indicate alternate language versions of web pages. It defines a reusable component with two required fields: a locale enumeration (supporting French, Italian, German, Dutch variants, and a default option) and a URL string, enabling content editors to specify language-specific page URLs for proper search engine indexing across multiple locales.
