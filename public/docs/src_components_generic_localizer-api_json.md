# localizer-api.json

**Path**: `src\components\generic\localizer-api.json`

## Summary
This is a Strapi CMS component schema that defines a localization/internationalization API configuration structure. It manages settings for content translation and localization between different locales, including origin and target locales, domain mappings, phone number localizations, and locale-specific customizations like header exclusions and footer replacements. The component enables multi-language content management with options to control blog visibility and markdown slug translation behavior.
