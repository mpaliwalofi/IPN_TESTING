# opengraph.json

**Path**: `src\components\generic\opengraph.json`

## Summary
This is a Strapi CMS component schema definition for Open Graph metadata, which controls how web pages appear when shared on social media platforms. It defines attributes for SEO and social sharing including title, description, image, robots meta tags (index/follow flags), and alternate language versions (hrefLang). The schema enables content editors to manage social media preview data and search engine indexing directives for web pages.
