# rich-content.json

**Path**: `src\components\generic\rich-content.json`

## Summary
This is a Strapi component schema definition for a "Rich Content" (Contenu Riche) component that stores formatted text. It defines a database collection with a single `markdown` attribute of type `richtext`, allowing content editors to create and store rich text content with formatting capabilities. This component is likely used across multiple content types in the CMS to provide reusable rich text editing functionality.
