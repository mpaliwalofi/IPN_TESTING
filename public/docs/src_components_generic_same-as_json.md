# same-as.json

**Path**: `src\components\generic\same-as.json`

## Summary
This is a Strapi CMS component schema definition for a "sameAs" component, which is used to store URLs that represent the same entity across different platforms or websites. This component implements the schema.org "sameAs" property pattern, commonly used in structured data/SEO to link to an entity's profiles on social media, Wikipedia, or other authoritative sources. It provides a simple reusable content structure with a single URL field that can be embedded in other content types.
