# schema.json

**Path**: `src\components\generic\schema.json`

## Summary
This is a Strapi CMS component schema definition for structured data (likely Schema.org markup) that defines the data model for rich content representation. It specifies attributes for organizational/business information including contact details, addresses, awards, founders, and category relationships. The schema enables content creators to add structured metadata to improve SEO and enable search engines to better understand the website's content through standardized semantic markup.
