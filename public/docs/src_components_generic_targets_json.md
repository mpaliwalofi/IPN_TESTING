# targets.json

**Path**: `src\components\generic\targets.json`

## Summary
This is a Strapi CMS component schema definition for configuring search index targets (likely Elasticsearch or similar). It defines a reusable component that stores target index configuration including the index name, API authentication key, and an optional index suffix for routing or versioning purposes. This component enables multi-target index deployment scenarios where content needs to be indexed to different search destinations.
