# text-seo.json

**Path**: `src\components\generic\text-seo.json`

## Summary
This is a Strapi CMS component schema definition for an SEO-focused text content block. It defines a reusable content structure with a title, rich text content (both visible and collapsible/folded versions), and up to 4 call-to-action button links, allowing content editors to create SEO-optimized text sections with associated actions throughout the application.
