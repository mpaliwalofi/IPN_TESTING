# verbatim-fullwidth.json

**Path**: `src\components\generic\verbatim-fullwidth.json`

## Summary
This is a Strapi CMS component schema definition for a full-width verbatim quote or testimonial block used in blog content. It defines a simple content type with a single required text field, designed to display quoted text that spans the full width of a page layout. The schema enables content editors to add and manage full-width quote blocks within the CMS's component system.
