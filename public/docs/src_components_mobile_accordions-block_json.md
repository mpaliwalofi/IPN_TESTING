# accordions-block.json

**Path**: `src\components\mobile\accordions-block.json`

## Summary
This is a Strapi CMS component schema definition for a mobile accordions block UI component. It defines the data structure for a repeatable accordion section that can be enabled/disabled, optionally display with a beige background, and contains multiple accordion items. This configuration allows content editors to manage collapsible content sections in the mobile interface through the CMS admin panel.
