# animal-page-header.json

**Path**: `src\components\mobile\animal-page-header.json`

## Summary
This is a Strapi CMS component schema that defines a mobile-specific header configuration for animal detail pages. It creates a simple toggle component with a single "enabled" boolean attribute that allows content managers to control whether the animal page header is displayed on mobile devices. The schema establishes the database collection structure and content type fields used by Strapi's admin interface for managing this UI component.
