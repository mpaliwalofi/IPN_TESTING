# animal-page-informations.json

**Path**: `src\components\mobile\animal-page-informations.json`

## Summary
This is a Strapi CMS component schema definition that configures a mobile-specific toggle for animal page information sections. The component provides a simple boolean flag (`enabled`) to control whether animal page information is displayed in the mobile version of the application. It serves as a content management structure for enabling/disabling animal-related content visibility on mobile devices.
