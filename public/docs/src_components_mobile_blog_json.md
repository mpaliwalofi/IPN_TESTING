# blog.json

**Path**: `src\components\mobile\blog.json`

## Summary
This JSON file defines a Strapi component schema for a mobile blog section. It specifies the data structure and fields (title, subtitle, button text/URL, background color toggle, and enabled state) that content editors can configure for displaying blog-related content in a mobile interface. The schema is used by Strapi's content management system to generate the appropriate database tables and admin UI forms for managing mobile blog components.
