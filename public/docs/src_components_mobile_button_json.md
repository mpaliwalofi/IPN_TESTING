# button.json

**Path**: `src\components\mobile\button.json`

## Summary
This is a Strapi CMS component schema definition for a mobile button component. It defines the structure and validation rules for button elements used in mobile interfaces, including required fields for display text and navigation links, an optional webview flag, and an extensive enumeration of 18 visual variants covering primary, secondary, subscription, review, and other button styles. The schema ensures consistent button data storage and rendering across the mobile application by enforcing type safety and required attributes.
