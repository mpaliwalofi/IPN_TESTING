# check-list-item.json

**Path**: `src\components\mobile\check-list-item.json`

## Summary
This is a Strapi CMS component schema that defines the structure for a mobile checklist item UI component. It specifies three attributes: customizable text and icon colors (using a color enumeration component) and rich text content for the checklist item text. The schema enables content editors to create styled checklist items with configurable visual properties for mobile applications.
