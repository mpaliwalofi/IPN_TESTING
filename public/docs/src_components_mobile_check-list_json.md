# check-list.json

**Path**: `src\components\mobile\check-list.json`

## Summary
This is a Strapi CMS component schema definition for a mobile checklist feature. It defines a reusable content structure with a title, repeatable check-list items (referencing another component), and an enabled/disabled toggle. The schema allows content editors to create and manage checklist content in the CMS that can be displayed in mobile applications.
