# contact-faq.json

**Path**: `src\components\mobile\contact-faq.json`

## Summary
This is a Strapi CMS component schema definition for a mobile "Contact FAQ" feature. It defines a reusable content structure that includes an enable/disable toggle, media asset, title/subtitle text, a button with customizable text, and a URL for opening FAQ content in a webview. The component is designed to provide users with a call-to-action that directs them to frequently asked questions through a mobile app webview interface.
