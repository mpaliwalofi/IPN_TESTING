# contact-footer.json

**Path**: `src\components\mobile\contact-footer.json`

## Summary
This is a Strapi CMS component schema that defines a mobile contact footer section for a website. It specifies the data structure for a "Contact Us" component with customizable attributes including a background image, title, subtitle, optional button text, and toggles for beige background styling and enabling/disabling the component. The schema serves as a content model for managing mobile-specific contact call-to-action sections through the CMS.
