# contact-header.json

**Path**: `src\components\mobile\contact-header.json`

## Summary
This JSON file defines a Strapi content component schema for a mobile contact header display element. It specifies a reusable component structure with fields for toggling visibility (enabled), displaying media (image), and presenting textual information (title and subtitle). The component is designed to provide a consistent contact header interface across mobile views in a CMS-managed application.
