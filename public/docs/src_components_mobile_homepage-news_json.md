# homepage-news.json

**Path**: `src\components\mobile\homepage-news.json`

## Summary
This is a Strapi CMS component schema definition for a mobile homepage news section. It defines a reusable content structure with a title, a repeatable collection of mobile sliders, and an enabled/disabled toggle flag. The schema allows content editors to manage news carousel sections on mobile homepages with the ability to show/hide them as needed.
