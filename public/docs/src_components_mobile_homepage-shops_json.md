# homepage-shops.json

**Path**: `src\components\mobile\homepage-shops.json`

## Summary
This is a Strapi CMS component schema definition for a mobile homepage shops section. It defines the data structure for displaying shop-related content on a mobile homepage, including required fields for title, subtitle, an enabled toggle, and associated media (images/videos/audio). The schema serves as a reusable component that can be integrated into Strapi content types to manage shop promotional content in the mobile interface.
