# link-card.json

**Path**: `src\components\mobile\link-card.json`

## Summary
This is a Strapi CMS component schema definition for a "Link Card" UI element designed for mobile applications. It defines a reusable content type with three attributes: display text, a URL link, and a boolean flag (`is_webview`) to determine whether the link should open in an embedded webview or external browser. The schema enables content managers to create consistent, configurable mobile link cards throughout the application.
