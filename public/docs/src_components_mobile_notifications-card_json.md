# notifications-card.json

**Path**: `src\components\mobile\notifications-card.json`

## Summary
This is a Strapi CMS component schema definition for a mobile notifications card UI element. It defines a reusable content structure with a required title field and a repeatable collection of text components, allowing content editors to create standardized notification cards for mobile applications. The schema serves as a data model for managing notification content within the CMS's content type builder.
