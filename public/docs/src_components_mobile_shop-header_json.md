# shop-header.json

**Path**: `src\components\mobile\shop-header.json`

## Summary
This JSON file defines a Strapi component schema for a mobile shop header feature. It specifies a simple data structure with a single boolean attribute "enabled" that controls whether the shop header is displayed, defaulting to true. This configuration allows content managers to toggle the visibility of the shop header in the mobile interface through the Strapi CMS.
