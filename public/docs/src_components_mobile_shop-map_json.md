# shop-map.json

**Path**: `src\components\mobile\shop-map.json`

## Summary
This is a Strapi CMS component schema definition for a "Shop Map" feature in a mobile application. It defines a simple boolean field called "enabled" that controls whether the shop map functionality is active, with a default value of true. This component allows content managers to toggle the visibility or availability of a shop location map feature within the mobile app's interface.
