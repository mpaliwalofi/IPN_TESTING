# static-page-push.json

**Path**: `src\components\mobile\static-page-push.json`

## Summary
This is a Strapi CMS component schema that defines the data structure for mobile push notifications or promotional content cards. It specifies fields for a title, background media (images/videos/audio), call-to-action button (text and URL), a webview toggle flag, and rich text information content. The schema enables content managers to create configurable push/promotional components for mobile applications with customizable media, messaging, and navigation behavior.
