# subscription-faq.json

**Path**: `src\components\mobile\subscription-faq.json`

## Summary
This JSON file defines a Strapi content-type schema for a mobile subscription FAQ component. It structures subscription-related frequently asked questions with a title, one-to-many relationship to FAQ question entries, and configurable abort/cancellation reasons that users can select when ending their subscription. The schema enables content management for mobile app subscription help documentation and the subscription cancellation flow.
