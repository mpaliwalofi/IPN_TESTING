# text-component.json

**Path**: `src\components\mobile\text-component.json`

## Summary
This is a Strapi CMS component schema definition for a mobile text component. It defines a reusable content structure with a single required string field called "value" that can be used within Strapi's content type builder for mobile-related content. The schema enables content creators to add simple text elements to mobile interfaces through Strapi's admin panel.
