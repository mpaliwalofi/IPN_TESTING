# upsells.json

**Path**: `src\components\mobile\upsells.json`

## Summary
This is a Strapi CMS component schema definition for managing mobile upsell functionality. It defines a reusable content type called "Upsells" with a single boolean attribute that controls whether upsells are enabled or disabled in the mobile interface. The schema allows content editors to toggle upsell features on/off across different parts of the mobile application.
