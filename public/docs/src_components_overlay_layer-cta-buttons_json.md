# layer-cta-buttons.json

**Path**: `src\components\overlay\layer-cta-buttons.json`

## Summary
This is a Strapi CMS component schema that defines a reusable "LayerCTAButtons" component for managing call-to-action buttons in overlay layers. It allows content editors to add multiple CTA buttons (via the repeatable "generic.cta-button" component) to overlay interfaces, enabling flexible configuration of user action prompts without hardcoding button layouts.
