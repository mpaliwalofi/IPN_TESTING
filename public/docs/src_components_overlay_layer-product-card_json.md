# layer-product-card.json

**Path**: `src\components\overlay\layer-product-card.json`

## Summary
This is a Strapi CMS component schema that defines a "LayerProductCard" for displaying product information in an overlay context. It configures a card component with product details, optional imagery (including a "Made in France" indicator), customizable heading levels, rich text descriptions, CTA buttons, and a layout option to position products on the left side on desktop views. This component is designed for B2C e-commerce product presentation with flexible content management capabilities.
