# layer-review.json

**Path**: `src\components\overlay\layer-review.json`

## Summary
This is a Strapi CMS component schema definition for a "LayerReview" overlay component that stores and displays customer review information. It defines a data structure with fields for review text, numerical rating (decimal), review count (nb_avis/number of reviews), and configurable font sizing options ranging from 2xs to 31xl, likely used for rendering review widgets or testimonials in a UI overlay system.
