# layer-rich-content.json

**Path**: `src\components\overlay\layer-rich-content.json`

## Summary
This is a Strapi CMS component schema definition for a rich text content layer used in overlay compositions. It defines a reusable content component with three attributes: markdown content (required rich text), configurable font family (Poppins or Cedrat), and font size selection from a comprehensive range of predefined sizes (2xs to 31xl). The schema enables content creators to manage formatted text overlays with consistent typography options across the application.
