# package.json

**Path**: `src\plugins\algolia\package.json`

## Summary
This is a Strapi plugin package configuration that integrates Algolia search functionality into a Strapi CMS application. The plugin provides hooks to automatically index Strapi content to Algolia's search service, enabling powerful search capabilities for content managed within Strapi. It's designed to run on Node.js versions 14.19.1 through 18.x with npm 6.0.0 or higher.
