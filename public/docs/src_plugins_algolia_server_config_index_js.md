# index.js

**Path**: `src\plugins\algolia\server\config\index.js`

## Summary
This module exports the default configuration settings for the Algolia search integration plugin in a Strapi application. It defines essential Algolia credentials (applicationId and apiKey) that will be populated via environment variables, along with a `forceExpurgeObjectBlame` flag for controlling object deletion behavior, and includes an empty validator function for potential configuration validation.
