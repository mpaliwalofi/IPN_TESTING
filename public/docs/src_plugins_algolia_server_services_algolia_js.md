# algolia.js

**Path**: `src\plugins\algolia\server\services\algolia.js`

## Summary
This module provides a Strapi service for integrating with Algolia search, enabling the synchronization of content data to Algolia's search indexes. It manages an Algolia client connection and exposes two core operations: `saveObject` for upserting individual records and `saveCollection` for bulk-replacing entire index contents, allowing Strapi CMS content to be searchable through Algolia.
