# CustomMenu.js

**Path**: `src\plugins\custom-menu\admin\src\components\customMenu\CustomMenu.js`

## Summary
This React component provides a customizable drag-and-drop menu builder interface for Strapi CMS. It allows users to create, reorder, import, and export hierarchical menu structures with support for both standard and "light" menu types, persisting changes via JSON serialization. The component manages menu state, handles drag-and-drop operations for menu reordering, and provides UI controls for adding items, resetting, and importing/exporting menu configurations.
