# Drag.js

**Path**: `src\plugins\custom-menu\admin\src\components\customMenu\drag-and-drop\Drag.js`

## Summary
This component provides a reusable draggable wrapper for menu items in Strapi's custom menu plugin. It uses `react-beautiful-dnd` to enable drag-and-drop functionality, rendering a drag handle icon alongside child content that can be reordered within a menu structure. The component encapsulates the draggable behavior and styling needed for intuitive menu item reordering in the admin interface.
