# Drop.js

**Path**: `src\plugins\custom-menu\admin\src\components\customMenu\drag-and-drop\Drop.js`

## Summary
This component provides a reusable droppable zone wrapper for implementing drag-and-drop functionality in a custom menu interface within Strapi's admin panel. It uses `react-beautiful-dnd`'s `Droppable` component and wraps it with Strapi's `Box` design system component to create consistent drop targets where draggable menu items can be placed during menu customization.
