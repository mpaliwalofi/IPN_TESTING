# index.js

**Path**: `src\plugins\custom-menu\admin\src\components\customMenu\drag-and-drop\index.js`

## Summary
This module serves as a centralized export point for drag-and-drop functionality used in the custom menu plugin's admin interface. It re-exports React Beautiful DnD's `DragDropContext` component (aliased as `DragAndDrop`) along with custom `Drag` and `Drop` wrapper components, providing a simplified API for implementing drag-and-drop menu item reordering throughout the admin UI.
