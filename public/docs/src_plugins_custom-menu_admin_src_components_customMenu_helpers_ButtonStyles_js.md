# ButtonStyles.js

**Path**: `src\plugins\custom-menu\admin\src\components\customMenu\helpers\ButtonStyles.js`

## Summary
This file defines a constant array of button style configuration options for custom menu links in a Strapi admin plugin. It provides three predefined CSS class combinations (Solid, Ghost, and Outline) that can be applied to menu buttons, allowing administrators to select different visual styling options when configuring custom menu items.
