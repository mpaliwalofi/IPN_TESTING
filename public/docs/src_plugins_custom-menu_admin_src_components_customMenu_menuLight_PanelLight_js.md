# PanelLight.js

**Path**: `src\plugins\custom-menu\admin\src\components\customMenu\menuLight\PanelLight.js`

## Summary
This React component manages individual menu panels in a Strapi custom menu plugin, allowing users to configure panel properties like labels, URLs, and an "all products" toggle. It provides drag-and-drop functionality for organizing nested sections within each panel, handles CRUD operations for sections, and triggers auto-save when panel data changes. The component serves as a collapsible accordion item in a larger menu configuration interface, with modal dialogs for deletion confirmation.
