# PanelSectionLight.js

**Path**: `src\plugins\custom-menu\admin\src\components\customMenu\menuLight\PanelSectionLight.js`

## Summary
This React component provides a collapsible accordion panel for configuring individual menu link items in a Strapi custom menu plugin. It allows administrators to edit link properties including text, URL, color scheme, button style, and link behavior options (mailto, target blank, nofollow attributes), with changes automatically saved via a trigger mechanism using useEffect.
