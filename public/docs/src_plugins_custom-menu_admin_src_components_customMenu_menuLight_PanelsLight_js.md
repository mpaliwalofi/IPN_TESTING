# PanelsLight.js

**Path**: `src\plugins\custom-menu\admin\src\components\customMenu\menuLight\PanelsLight.js`

## Summary
This React component renders a drag-and-drop interface for managing "light panels" in a Strapi custom menu plugin. It displays a list of draggable panel items (using the `PanelLight` component), allows users to add new panels via buttons, and handles panel operations like saving, deleting, and reordering through drag-and-drop functionality. The component serves as a container for organizing multiple panel configurations within the custom menu administration interface.

- **Parameters**: `{ data, savePanels, createNewPanel, deletePanel, handleDragEnd }`
