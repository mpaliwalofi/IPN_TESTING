# ImageMobileItem.js

**Path**: `src\plugins\custom-menu\admin\src\components\customMenu\menu\ImageMobileItem.js`

## Summary
This React component provides a form interface for managing mobile image data within a Strapi custom menu plugin. It renders two input fields (URL and alternative text) for configuring a mobile-specific image, using a trigger-based state pattern to propagate changes to the parent component via the `handleChangeImageMobile` callback. The component serves as a reusable form section for mobile image configuration in menu customization workflows.

- **Parameters**: `{ required, dataImageMobile, handleChangeImageMobile }`
