# Menu.js

**Path**: `src\plugins\custom-menu\admin\src\components\customMenu\menu\Menu.js`

## Summary
This React component implements a menu editor for a custom menu plugin in Strapi CMS. It manages the configuration of menu items including their labels, links, visibility settings, button styling (themes/styles/badges), mobile-specific properties, and nested panel structures. The component provides CRUD operations for menu panels, handles state synchronization with parent components through callbacks, and includes features like drag-and-drop reordering and a confirmation dialog for deletions.
