# Tab.js

**Path**: `src\plugins\custom-menu\admin\src\components\customMenu\menu\Tab.js`

## Summary
This React component renders an editable accordion item for managing individual menu tabs in a Strapi custom menu plugin. It allows administrators to configure tab properties including URL, label, associated universe (brand/category), and mobile images, with automatic saving triggered on any field change. The component provides CRUD operations for menu tab entries with features like expandable/collapsible views, delete confirmation dialogs, and form validation.
