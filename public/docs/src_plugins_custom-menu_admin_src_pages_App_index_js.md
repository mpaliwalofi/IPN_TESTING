# index.js

**Path**: `src\plugins\custom-menu\admin\src\pages\App\index.js`

## Summary
This component serves as the main routing wrapper for the Strapi custom-menu plugin's admin interface. It defines two routes: the primary route that displays the HomePage component when accessing the plugin's URL path, and a fallback route that shows an error page for any unmatched paths within the plugin.
