# my-service.js

**Path**: `src\plugins\custom-menu\server\services\my-service.js`

## Summary
This is a Strapi plugin service module that provides a simple example/placeholder service for the custom-menu plugin. It exports a service factory function that returns a single method `getWelcomeMessage()`, which currently only returns a static welcome string, serving as a boilerplate template for developers to implement actual custom menu-related business logic.
