# package.json

**Path**: `src\plugins\firestore\package.json`

## Summary
This is a Strapi plugin package configuration file for integrating Google Cloud Firestore database functionality into a Strapi CMS application. The plugin depends on the official Google Cloud Firestore SDK (v7.0.0) and is designed to work with Node.js versions 18-20, enabling Strapi applications to interact with Firestore as a data storage solution.
