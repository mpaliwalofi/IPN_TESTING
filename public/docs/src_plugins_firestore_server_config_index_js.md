# index.js

**Path**: `src\plugins\firestore\server\config\index.js`

## Summary
This module exports a default configuration object for a Firestore plugin, specifying connection settings (projectId, keyFilename) and localization options (country, defaultLocale set to "fr" for France/French). It provides a placeholder validator function that currently accepts all configurations without performing actual validation. The configuration is disabled by default and requires manual setup before the Firestore integration can be activated.
