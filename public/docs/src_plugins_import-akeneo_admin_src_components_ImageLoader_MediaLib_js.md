# MediaLib.js

**Path**: `src\plugins\import-akeneo\admin\src\components\ImageLoader\MediaLib.js`

## Summary
This React component provides a media library dialog interface for selecting and formatting image assets within the Akeneo import plugin for Strapi. It wraps Strapi's built-in media library component, handling the selection of media files and transforming them into a standardized format with alternative text, prefixed URLs, and MIME types before passing them to the parent component via the onChange callback.
