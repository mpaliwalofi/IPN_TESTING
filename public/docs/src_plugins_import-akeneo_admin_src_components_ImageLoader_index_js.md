# index.js

**Path**: `src\plugins\import-akeneo\admin\src\components\ImageLoader\index.js`

## Summary
This React component provides an image replacement interface for the Strapi Akeneo import plugin. It allows users to select a new image from Strapi's media library to replace an existing Akeneo-imported image, then sends a POST request to the backend to process the image swap while preserving metadata like alternative text. The component manages the media library modal visibility, locks the UI during the update operation, and reloads the page to reflect the changes.
