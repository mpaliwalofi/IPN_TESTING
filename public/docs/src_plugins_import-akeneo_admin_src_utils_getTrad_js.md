# getTrad.js

**Path**: `src\plugins\import-akeneo\admin\src\utils\getTrad.js`

## Summary
This utility function generates namespaced translation keys for the Akeneo import plugin by prefixing a given ID with the plugin identifier. It ensures consistent i18n (internationalization) key formatting across the plugin's admin interface, preventing translation key collisions with other plugins or the core Strapi system.
