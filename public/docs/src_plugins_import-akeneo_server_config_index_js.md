# index.js

**Path**: `src\plugins\import-akeneo\server\config\index.js`

## Summary
This configuration file defines default settings for an Akeneo import plugin, which integrates Strapi with the Akeneo PIM (Product Information Management) system. It exports default configuration including multiple API endpoints (for reviews, bestsellers, prices, Salesforce, blog, guides, and logs), Akeneo authentication credentials (username, password, API keys), localization settings (scope: FR_B2C, locale: fr_FR), and ImageKit integration parameters for handling product catalog images in a staging environment.
