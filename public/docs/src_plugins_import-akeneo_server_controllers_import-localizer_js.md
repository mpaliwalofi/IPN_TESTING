# import-localizer.js

**Path**: `src\plugins\import-akeneo\server\controllers\import-localizer.js`

## Summary
This controller handles HTTP endpoints for the Akeneo import plugin's localization functionality. It provides two actions: a basic health check endpoint (`index`) and a synchronization endpoint (`sync`) that triggers the localization service to synchronize localized content from Akeneo into the Strapi system.
