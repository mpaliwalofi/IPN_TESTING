# index.js

**Path**: `src\plugins\import-akeneo\server\index.js`

## Summary
This is the main entry point module for a Strapi plugin that handles importing data from Akeneo PIM (Product Information Management) system. It aggregates and exports all the plugin's core components including lifecycle hooks (register, bootstrap, destroy), configuration, API routes, controllers, services, and middleware needed to integrate Akeneo's product data into a Strapi application.
