# index.js

**Path**: `src\plugins\import-akeneo\server\services\index.js`

## Summary
This is a service module aggregator for an Akeneo import plugin in Strapi. It exports four core service modules (importer, blog, shop, and localizer) that handle different aspects of importing and processing data from Akeneo PIM (Product Information Management) system into the CMS, including product data, blog content, shop/e-commerce data, and localization/internationalization functionality.
