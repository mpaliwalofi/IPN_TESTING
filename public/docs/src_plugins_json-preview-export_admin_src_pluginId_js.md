# Summary

This module defines and exports a constant string identifier `'json-preview-export'` that serves as the unique plugin ID for the JSON Preview Export plugin within the Strapi admin panel. This identifier is used throughout the plugin's codebase for registration, routing, permissions, and other plugin-specific operations that require a consistent reference to this plugin.
