## Summary
This is a React admin page component for managing marketing/promotional campaigns in a Strapi CMS plugin called "Pixie". It provides a table-based interface for viewing, creating, editing, duplicating, and deleting campaigns with features like search, filtering (by active status), pagination, and campaign configuration including conditions, targets, and actions. The component handles CRUD operations through API requests to manage campaign rules that can be selectively applied based on priorities and conditions.
