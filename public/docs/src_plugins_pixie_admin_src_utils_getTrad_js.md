# getTrad.js

**Path**: `src\plugins\pixie\admin\src\utils\getTrad.js`

## Summary
This utility function generates namespaced translation keys for the Pixie plugin's internationalization (i18n) system. It prefixes any given translation ID with the plugin's identifier to create unique, scoped translation keys (e.g., "pixie.myKey"), preventing key collisions with other plugins or the main application in Strapi's admin panel.
