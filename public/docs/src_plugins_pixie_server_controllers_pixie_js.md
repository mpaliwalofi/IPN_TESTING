# pixie.js

**Path**: `src\plugins\pixie\server\controllers\pixie.js`

## Summary
This is a Strapi plugin controller for a campaign management system called "pixie" that handles HTTP requests for CRUD operations on campaigns. It serves as a routing layer that delegates all business logic to the underlying pixie service, including campaign management (get, create, update, delete) and schema retrieval for actions and conditions. The controller acts as a thin wrapper that connects HTTP endpoints to the service layer for what appears to be a marketing or automation campaign feature.
