# sterilized.html.twig

**Path**: `templates\Animal\Grid\Field\sterilized.html.twig`

## Summary
This Twig template renders a visual indicator for an animal's sterilization status in a data grid. It displays a green checkmark label with "sterilized" text when the animal is sterilized, or a red X label with "not sterilized" text when the animal is not sterilized, using Semantic UI styling components and internationalized translation keys.
