# _header.html.twig

**Path**: `templates\Animal\Show\_header.html.twig`

## Summary
This Twig template renders the header section for an animal detail page in an admin interface. It displays the animal's name with a paw icon on the left side and provides an "Edit" button on the right that links to the animal update form. The template uses Semantic UI styling for a responsive two-column grid layout with a divider at the bottom.
