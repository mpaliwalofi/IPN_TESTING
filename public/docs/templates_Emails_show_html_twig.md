# show.html.twig

**Path**: `templates\Emails\show.html.twig`

## Summary
This Twig template displays a detailed view of an email audit record within a Sylius admin interface. It presents email audit information in a tabular format, showing fields such as the audit ID, template ID, template name, optional message ID, and customer email address. The template is used for tracking and reviewing sent emails in the system for administrative purposes.
