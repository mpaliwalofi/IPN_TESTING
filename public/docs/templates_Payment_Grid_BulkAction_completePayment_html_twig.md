# completePayment.html.twig

**Path**: `templates\Payment\Grid\BulkAction\completePayment.html.twig`

## Summary
This Twig template renders a bulk action form for completing payments in the Sylius payment grid. It generates a submit button with a payment icon that triggers a secure, confirmation-required action to mark selected payments as complete, posting to a configurable route defined in the grid options.
