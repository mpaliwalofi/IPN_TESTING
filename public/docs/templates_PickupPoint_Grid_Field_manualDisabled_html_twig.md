# manualDisabled.html.twig

**Path**: `templates\PickupPoint\Grid\Field\manualDisabled.html.twig`

## Summary
This Twig template renders a status indicator for pickup points in a grid view, displaying either a green "enabled" label or a red "disabled" label based on the `isEnabled` property. It provides visual feedback to administrators about whether a pickup point is currently active or inactive in the system.
