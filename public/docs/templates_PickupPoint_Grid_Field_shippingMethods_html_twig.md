# shippingMethods.html.twig

**Path**: `templates\PickupPoint\Grid\Field\shippingMethods.html.twig`

## Summary
This Twig template renders a list of shipping methods associated with a pickup point in an administrative grid interface. It displays each shipping method's name, enabled/disabled status (with color-coded labels), and external reference identifier, allowing administrators to quickly view which shipping methods are configured for a specific pickup point location.
