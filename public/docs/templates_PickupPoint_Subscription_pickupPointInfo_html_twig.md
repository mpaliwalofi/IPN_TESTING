# pickupPointInfo.html.twig

**Path**: `templates\PickupPoint\Subscription\pickupPointInfo.html.twig`

## Summary
This Twig template displays detailed information about a pickup point for order fulfillment, including its availability status, location address, operating hours, and temporary closing periods. It renders a complete pickup point information card showing the name (with admin link), reference code, associated shipping method, formatted opening hours by day, and any scheduled closure dates. The template is designed for subscription-related order management, allowing users to view comprehensive pickup point details with visual availability indicators (green/red labels) and country flags.
