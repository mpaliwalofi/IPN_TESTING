# _form.html.twig

**Path**: `templates\ShippingZone\_form.html.twig`

## Summary
This Twig template renders a form for configuring shipping zones in an e-commerce system. It displays form fields for zone identification (code, name), zone selection, and member management with custom styling to handle postcode ranges and layout multiple member fields horizontally. The template uses Semantic UI CSS framework for responsive grid layout and styling of the form components.
