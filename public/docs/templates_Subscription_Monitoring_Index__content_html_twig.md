# _content.html.twig

**Path**: `templates\Subscription\Monitoring\Index\_content.html.twig`

## Summary
This Twig template renders a subscription monitoring calendar interface for administrators. It displays a monthly calendar view with navigation controls (previous/next month buttons and a "current month" reset button) and a weekday header row for the calendar grid. The template is designed to allow admins to browse subscription activity or metrics across different months and years using the `app_admin_subscription_monitoring_index` route.
