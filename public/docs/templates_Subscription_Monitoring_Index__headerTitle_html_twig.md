# _headerTitle.html.twig

**Path**: `templates\Subscription\Monitoring\Index\_headerTitle.html.twig`

## Summary
This Twig template renders the header section for the Subscription Monitoring Index page in a Sylius-based application. It imports and uses a reusable header macro from SyliusUI to display a translated title, an optional icon (defaulting to 'cog'), and an optional translated subheader, with all configuration values passed from the controller/context.
