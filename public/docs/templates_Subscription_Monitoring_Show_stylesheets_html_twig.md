# stylesheets.html.twig

**Path**: `templates\Subscription\Monitoring\Show\stylesheets.html.twig`

## Summary
This Twig template file contains CSS styles for a subscription monitoring calendar interface with a daily frequency sidebar. The styles customize FullCalendar events and create a dark-themed sidebar (#303644) that displays frequency groups organized by date, with specific styling for event items, loading states, and text transformations. The styling ensures proper layout using flexbox and manages opacity states for events in different calendar date contexts (current vs. other months).
