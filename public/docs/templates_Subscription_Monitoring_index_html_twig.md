# index.html.twig

**Path**: `templates\Subscription\Monitoring\index.html.twig`

## Summary
This Twig template serves as the main index page for subscription monitoring in a Sylius admin panel. It extends the base admin layout and defines the structure for displaying subscription monitoring data, including a customizable content area triggered by a Sylius template event ('app.admin.subscription_monitoring.index.content') that allows for dynamic content injection. The template also includes dedicated stylesheet and JavaScript files specific to the subscription monitoring index functionality.
