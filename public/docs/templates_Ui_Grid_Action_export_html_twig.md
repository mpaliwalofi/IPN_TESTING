# export.html.twig

**Path**: `templates\Ui\Grid\Action\export.html.twig`

## Summary
This Twig template renders an export button in a Sylius UI grid action context. It imports button macros and creates a default-styled button with a download icon that links to either a provided URL or a generated route path, using the label "Export" (translatable via 'app.ui.export') by default. The template serves as a reusable component for adding export functionality to admin grid interfaces.
