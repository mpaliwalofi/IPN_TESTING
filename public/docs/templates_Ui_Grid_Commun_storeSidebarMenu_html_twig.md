# storeSidebarMenu.html.twig

**Path**: `templates\Ui\Grid\Commun\storeSidebarMenu.html.twig`

## Summary
This Twig template contains JavaScript code that manages the persistence of sidebar visibility state in a grid UI. It uses cookies to store whether the sidebar is visible or hidden when the user clicks the toggle button, allowing the sidebar state to be remembered across page loads. The module initializes on document ready and binds a click event to the sidebar toggle button to save the current visibility state.
