# enabled.html.twig

**Path**: `templates\Ui\Grid\Field\enabled.html.twig`

## Summary
This Twig template renders an enabled/disabled status indicator for grid rows in the Sylius e-commerce framework. It displays a blue label with a checkmark icon and "Yes" text when an entity is enabled, or a gray label with an X icon and "No" text when disabled. This template is used as a reusable field formatter to visually represent boolean enabled states across various admin grid views.
