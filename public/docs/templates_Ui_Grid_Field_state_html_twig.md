# state.html.twig

**Path**: `templates\Ui\Grid\Field\state.html.twig`

## Summary
This Twig template renders state labels in Sylius UI grid fields by constructing a translation key from the state data. It supports custom label templates per state type through an optional `labels` configuration, falling back to a default label template when no custom template is specified. The template is part of Sylius's grid system for displaying entity states (like order status, payment status, etc.) with appropriate visual styling and localization.
