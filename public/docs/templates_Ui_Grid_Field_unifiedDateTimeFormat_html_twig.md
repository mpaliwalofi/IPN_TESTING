# unifiedDateTimeFormat.html.twig

**Path**: `templates\Ui\Grid\Field\unifiedDateTimeFormat.html.twig`

## Summary
This Twig template renders a formatted date/time value within a UI grid field component. It applies a custom datetime formatting filter with a configurable format string (defaulting to "d/m/Y H:i:s" - day/month/year hour:minute:second) to display temporal data consistently across the application's grid interface.
