# _contentWidget.html.twig

**Path**: `templates\UpSelling\Show\_contentWidget.html.twig`

## Summary
This Twig template renders a detailed view widget for displaying up-selling product information in a Sylius e-commerce application. It presents key attributes of an up-sell item in a formatted table, including the name, enabled status (with visual indicators), typology, and classification fields. The template uses Semantic UI styling to create a responsive, user-friendly display with conditional rendering to handle empty values and provide localized labels.
