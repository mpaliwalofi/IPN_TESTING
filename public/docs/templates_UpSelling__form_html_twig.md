# _form.html.twig

**Path**: `templates\UpSelling\_form.html.twig`

## Summary
This Twig template renders a form for managing product upselling configurations in a Sylius e-commerce admin panel. It displays input fields for configuring upsell properties including name, enabled status, typology, classification, and associated product variants, using a custom theme for the variants collection widget. The form is structured with Semantic UI grid layout classes to organize the fields in a responsive eight-column layout.
