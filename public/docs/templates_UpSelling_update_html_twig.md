# update.html.twig

**Path**: `templates\UpSelling\update.html.twig`

## Summary
This Twig template renders the admin interface for updating an existing upsell product configuration in a Sylius e-commerce system. It extends the base admin layout, displays the upsell's name in the page title, and provides template event hooks that allow developers to inject custom content and JavaScript specific to the upsell update functionality.
