# _general.html.twig

**Path**: `templates\bundles\SyliusAdminBundle\Channel\Form\_general.html.twig`

## Summary
This Twig template renders the general settings section of a channel configuration form in Sylius's admin interface. It displays form fields for basic channel properties including code, name, color, and enabled status, organized in a semantic UI layout with a header and segmented container. The template serves as a reusable form component for creating or editing sales channels in the e-commerce platform.
