# _address.html.twig

**Path**: `templates\bundles\SyliusAdminBundle\Common\Form\_address.html.twig`

## Summary
This Twig template renders a standardized address form layout for the Sylius Admin Bundle, displaying fields for collecting complete address information including name, company, street, country, province, postal code, city, and phone number. The template includes AJAX functionality for dynamically loading province/state options based on the selected country code, enabling location-dependent form field rendering. It serves as a reusable form partial for address input across various admin panel contexts like customer management, order processing, and shipping configuration.
