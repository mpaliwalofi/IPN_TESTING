# _template.html.twig

**Path**: `templates\bundles\SyliusAdminBundle\Common\Modal\_template.html.twig`

## Summary
This Twig template defines a reusable modal dialog component for the Sylius admin interface. It provides configurable elements including an optional close icon, header with icon support, content area, and action buttons (cancel/approve) with default translations, allowing consistent modal presentation throughout the admin panel. The template uses Semantic UI styling classes and intelligently merges default actions with custom ones when provided.
