# _address.html.twig

**Path**: `templates\bundles\SyliusAdminBundle\Common\_address.html.twig`

## Summary
This Twig template renders a formatted address display for the Sylius e-commerce admin panel. It shows complete address details including name, company, street address, location (city/province/country with flag), contact numbers, and optionally displays pickup point information for shipping addresses and delivery instructions when available. The template is designed as a reusable component for displaying customer or order addresses throughout the admin interface.
