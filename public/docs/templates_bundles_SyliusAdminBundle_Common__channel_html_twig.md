# _channel.html.twig

**Path**: `templates\bundles\SyliusAdminBundle\Common\_channel.html.twig`

## Summary
This Twig template renders a channel display component for the Sylius e-commerce admin interface. It conditionally displays a channel's country flag (using a macro from SyliusUi) alongside the channel name, allowing administrators to visually identify different sales channels (e.g., regional stores or marketplaces) in the system.
