# _breadcrumbContent.html.twig

**Path**: `templates\bundles\SyliusAdminBundle\Customer\Order\Index\_breadcrumbContent.html.twig`

## Summary
This Twig template defines the breadcrumb navigation for the customer orders index page in Sylius's admin panel. It creates a hierarchical navigation path from Administration → Customers → [Customer Email] → Orders, allowing administrators to track their location while viewing a specific customer's order list and navigate back through parent pages.
