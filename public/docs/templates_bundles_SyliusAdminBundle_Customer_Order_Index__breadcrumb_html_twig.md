# _breadcrumb.html.twig

**Path**: `templates\bundles\SyliusAdminBundle\Customer\Order\Index\_breadcrumb.html.twig`

## Summary
This Twig template renders a breadcrumb navigation component for viewing a customer's orders in the Sylius admin panel. It makes a subrequest to fetch customer information using the customer ID from the request attributes and displays it using a dedicated breadcrumb content template, allowing administrators to maintain navigation context when browsing a specific customer's order history.
