# _email.html.twig

**Path**: `templates\bundles\SyliusAdminBundle\Customer\Show\Details\_email.html.twig`

## Summary
This Twig template displays a customer's newsletter subscription status in the Sylius e-commerce admin panel. It renders a visual indicator (green checkmark or red X icon) based on whether the customer is subscribed to the newsletter, along with a translated label explaining the subscription state.
