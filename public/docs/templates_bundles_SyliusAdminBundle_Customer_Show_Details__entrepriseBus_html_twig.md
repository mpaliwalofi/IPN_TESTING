# _entrepriseBus.html.twig

**Path**: `templates\bundles\SyliusAdminBundle\Customer\Show\Details\_entrepriseBus.html.twig`

## Summary
This Twig template provides an administrative interface button that allows manual/forced synchronization of customer data to WebMethods.io (an enterprise service bus/integration platform). The button triggers a GET request to force send customer information to the enterprise bus system, likely used when automatic synchronization fails or needs to be manually initiated for troubleshooting purposes.
