# _primary.html.twig

**Path**: `templates\bundles\SyliusAdminBundle\Customer\Show\Details\_primary.html.twig`

## Summary
This Twig template displays the primary details section of a customer profile in the Sylius admin panel. It shows the customer's full name as a clickable edit link, their associated channel, registration date, and optional group membership information. The template is part of the customer detail view and provides quick access to key customer identification and categorization data for store administrators.
