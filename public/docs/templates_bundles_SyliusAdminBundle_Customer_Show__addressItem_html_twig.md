# _addressItem.html.twig

**Path**: `templates\bundles\SyliusAdminBundle\Customer\Show\_addressItem.html.twig`

## Summary
This Twig template renders a single table row displaying a customer's address details in the Sylius admin panel. It shows comprehensive address information including ID, alias/hash, personal details (name, company), location data (street, city, postcode, phone), and includes an interactive toggle checkbox that allows administrators to set the address as the customer's default address. The template is part of the customer management interface, specifically designed to be included in a customer detail view showing all associated addresses.
