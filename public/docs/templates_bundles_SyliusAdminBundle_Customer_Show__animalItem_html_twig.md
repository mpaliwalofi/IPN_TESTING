# _animalItem.html.twig

**Path**: `templates\bundles\SyliusAdminBundle\Customer\Show\_animalItem.html.twig`

## Summary
This Twig template renders a single table row displaying detailed information about an animal associated with a customer in the Sylius admin interface. It shows the animal's type, name, gender, sterilization status (with color-coded labels), calculated age from birthday, and breed information (including crossbreed handling). The template also provides action buttons to view or update the animal's details, serving as a reusable component within a customer's animal listing table.
