# _cartItem.html.twig

**Path**: `templates\bundles\SyliusAdminBundle\Customer\Show\_cartItem.html.twig`

## Summary
This Twig template renders a single table row displaying cart/order information for a customer in the Sylius e-commerce admin panel. It shows the cart's ID, creation date, shipping method, total amount with currency formatting, and provides a "Show" button that links to the detailed order view page. The template is used as a reusable component to list individual cart items when viewing a customer's cart history.
