# _carts.html.twig

**Path**: `templates\bundles\SyliusAdminBundle\Customer\Show\_carts.html.twig`

## Summary
This Twig template renders a display section for a customer's shopping carts in the Sylius e-commerce admin interface. It shows a table listing all carts associated with a customer, displaying cart ID, date, shipping method, and total price, with each cart rendered via a separate `_cartItem.html.twig` partial template, or shows an empty state message if no carts exist.
