# _promotionItem.html.twig

**Path**: `templates\bundles\SyliusAdminBundle\Customer\Show\_promotionItem.html.twig`

## Summary
This Twig template renders a single table row displaying promotion details in the Sylius admin customer view. It shows promotion information including ID, name, analytic category, coupon status, usage statistics, and end date, along with action buttons for managing coupons (list, create, generate) and standard edit/delete operations for the promotion itself.
