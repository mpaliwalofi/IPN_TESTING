# show.html.twig

**Path**: `templates\bundles\SyliusAdminBundle\Customer\show.html.twig`

## Summary
This Twig template renders the admin customer detail page in a Sylius e-commerce application, displaying a customer's complete profile including their email, addresses, promotions, animals, and shopping carts. The template includes JavaScript functionality for managing customer addresses with features like creating/updating/deleting addresses, phone number input validation (international telephone input), and postcode validation, with special handling for addresses used in subscriptions.
