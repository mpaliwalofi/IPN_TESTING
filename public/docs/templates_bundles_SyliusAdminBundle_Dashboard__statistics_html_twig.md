# _statistics.html.twig

**Path**: `templates\bundles\SyliusAdminBundle\Dashboard\_statistics.html.twig`

## Summary
This Twig template renders the statistics section of the Sylius admin dashboard. It displays key e-commerce metrics including total sales, number of customers, average order value, and sales trends over time through visual charts and numeric indicators. The template serves as the main analytics overview for store administrators to monitor business performance at a glance.
