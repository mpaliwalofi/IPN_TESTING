# _linkShop.html.twig

**Path**: `templates\bundles\SyliusAdminBundle\Layout\_linkShop.html.twig`

## Summary
This Twig template renders a navigation link in the Sylius admin panel's top bar that allows administrators to quickly open the storefront in a new browser tab. The link uses an icon and translatable text to provide easy access from the admin interface to view the customer-facing shop.
