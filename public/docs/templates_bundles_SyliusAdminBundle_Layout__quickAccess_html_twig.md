# _quickAccess.html.twig

**Path**: `templates\bundles\SyliusAdminBundle\Layout\_quickAccess.html.twig`

## Summary
This Twig template renders a quick access dropdown menu in the Sylius admin panel's top navigation bar. It provides shortcut links to frequently-used administrative actions, specifically creating new orders and promotions, allowing administrators to quickly initiate these common tasks without navigating through the full menu structure.
