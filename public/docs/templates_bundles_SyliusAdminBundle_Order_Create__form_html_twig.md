# _form.html.twig

**Path**: `templates\bundles\SyliusAdminBundle\Order\Create\_form.html.twig`

## Summary
This is a Twig template for creating orders in the Sylius e-commerce admin panel. It renders a form with sections for adding order items, selecting shipping methods (including Mondial Relay pickup points), and applying promotion coupons, along with error message containers for handling various validation failures like missing products or Mondial Relay store selection issues.
