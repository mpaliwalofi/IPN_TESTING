# _cartBreadcrumb.html.twig

**Path**: `templates\bundles\SyliusAdminBundle\Order\Index\_cartBreadcrumb.html.twig`

## Summary
This Twig template generates breadcrumb navigation for the shopping cart administration page in Sylius e-commerce platform. It creates a two-level breadcrumb trail starting from the admin dashboard and ending at the "Carts" section, allowing administrators to navigate back through the application hierarchy while viewing or managing customer shopping carts.
