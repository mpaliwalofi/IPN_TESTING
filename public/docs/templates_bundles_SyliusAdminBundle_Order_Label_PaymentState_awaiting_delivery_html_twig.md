# awaiting_delivery.html.twig

**Path**: `templates\bundles\SyliusAdminBundle\Order\Label\PaymentState\awaiting_delivery.html.twig`

## Summary
This Twig template renders a label component for orders in the "awaiting delivery" payment state within the Sylius e-commerce admin interface. It displays an olive-colored label with clock and truck icons, indicating that payment has been received and the order is waiting to be shipped/delivered. The template supports an optional "attached" parameter to modify the label's CSS styling for different UI layouts.
