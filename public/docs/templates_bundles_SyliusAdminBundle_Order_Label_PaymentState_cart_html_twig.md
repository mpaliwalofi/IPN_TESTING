# cart.html.twig

**Path**: `templates\bundles\SyliusAdminBundle\Order\Label\PaymentState\cart.html.twig`

## Summary
This Twig template renders a Semantic UI label for displaying the payment state of shopping cart orders in the Sylius admin panel. It outputs a translatable payment state value with optional "top attached" styling when the `attached` parameter is set to true, allowing consistent visual representation of cart-stage orders before checkout completion.
