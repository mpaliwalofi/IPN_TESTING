# cart.html.twig

**Path**: `templates\bundles\SyliusAdminBundle\Order\Label\ShippingMethod\cart.html.twig`

## Summary
This Twig template renders a UI label for displaying the shipping method in the cart context within Sylius admin interface. It applies Semantic UI styling with optional "top attached" positioning based on an `attached` parameter, and translates the shipping method value for display.
