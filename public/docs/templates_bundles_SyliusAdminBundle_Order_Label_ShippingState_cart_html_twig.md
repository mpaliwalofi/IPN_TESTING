# cart.html.twig

**Path**: `templates\bundles\SyliusAdminBundle\Order\Label\ShippingState\cart.html.twig`

## Summary
This Twig template renders a shipping state label for orders in the "cart" state within the Sylius e-commerce admin interface. It displays a UI label with the translated shipping state value, with optional styling for attaching the label to the top of another element when the `attached` parameter is true. This component is part of the order management system's visual status indicators for tracking shipment states.
