# _item.html.twig

**Path**: `templates\bundles\SyliusAdminBundle\Order\Preview\_item.html.twig`

## Summary
This Twig template renders a single order item row in the Sylius admin order preview/creation interface. It displays detailed pricing information for each product variant including unit price, discounted price, quantity, subtotal, promotional discounts, taxes, and total, with all monetary values formatted in the order's currency. The template also includes a form row for managing item-level adjustments (discounts, promotions) by aggregating various adjustment types defined in the Sylius e-commerce system.
