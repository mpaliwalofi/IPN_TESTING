# _summary.html.twig

**Path**: `templates\bundles\SyliusAdminBundle\Order\Preview\_summary.html.twig`

## Summary
This Twig template renders an order summary table in the Sylius admin panel, displaying a detailed breakdown of order items with their pricing information. It shows regular order items and free product adjustments, including columns for product name, unit price, discounted price, quantity, and totals, with a footer section for order totals. The template serves as the main structure for previewing order contents before finalization in the e-commerce admin interface.
