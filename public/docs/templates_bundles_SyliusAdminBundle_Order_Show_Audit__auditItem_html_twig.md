# _auditItem.html.twig

**Path**: `templates\bundles\SyliusAdminBundle\Order\Show\Audit\_auditItem.html.twig`

## Summary
This Twig template renders individual audit log entries for order-related entities in the Sylius Admin Bundle. It displays audit trail information in a table row format, showing the timestamp, username, IP address, and a summary of changes for each audited action on an entity, utilizing a helper function to format the change summary.
