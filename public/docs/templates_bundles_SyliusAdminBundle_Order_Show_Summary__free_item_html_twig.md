# _free_item.html.twig

**Path**: `templates\bundles\SyliusAdminBundle\Order\Show\Summary\_free_item.html.twig`

## Summary
This Twig template renders a table row for displaying a free promotional item in a Sylius e-commerce order summary. It shows product variant information, displays "free item" labels for pricing columns (unit price, discounted price, and total), and hardcodes the quantity to 1, representing complimentary products given as part of promotions or special offers.
