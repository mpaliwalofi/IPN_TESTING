# _item.html.twig

**Path**: `templates\bundles\SyliusAdminBundle\Order\Show\Summary\_item.html.twig`

## Summary
This Twig template renders a single order item row in the Sylius admin order summary view. It displays product information, pricing details (unit price, discounted price, quantity, and total), and optionally shows subscription interval information if the item is part of a recurring subscription. The template formats monetary values using the order's currency code and organizes the data in a table row structure for administrative order review.
