# _totals.html.twig

**Path**: `templates\bundles\SyliusAdminBundle\Order\Show\Summary\_totals.html.twig`

## Summary
This Twig template renders the order totals section in the Sylius e-commerce admin panel's order detail view. It displays financial summary information including tax total, items total, and shipping charges with their associated adjustments and promotions, formatting all monetary values in the appropriate currency. The template aggregates and presents various order adjustments (shipping promotions, payment charges, taxes) in a structured table format for order administrators to review.
