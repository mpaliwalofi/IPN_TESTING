# _headerWidget.html.twig

**Path**: `templates\bundles\SyliusAdminBundle\Order\Show\_headerWidget.html.twig`

## Summary
This Twig template renders the header widget for an order detail page in the Sylius e-commerce admin interface. It displays the order number (or "Cart" label for incomplete orders) along with key order metadata including checkout completion date, order state with visual labels, currency code, locale with flag icon, sales channel, and optionally the order token value in a horizontally divided list format.
