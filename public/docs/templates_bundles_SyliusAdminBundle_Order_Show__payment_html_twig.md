# _payment.html.twig

**Path**: `templates\bundles\SyliusAdminBundle\Order\Show\_payment.html.twig`

## Summary
This Twig template renders the payment details section for an order in the Sylius admin panel. It displays payment information including the payment method, amount, state label, creation/update timestamps, and transaction details (like transaction ID and subscription token for completed/refunded payments). For payments with Payzen transaction data, it also provides a modal link to view detailed transaction information.
