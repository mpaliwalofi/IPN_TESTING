# _script.html.twig

**Path**: `templates\bundles\SyliusAdminBundle\Order\Update\_script.html.twig`

## Summary
This Twig template contains JavaScript for managing address selection in the Sylius admin order update form. It toggles between displaying an address form and an existing address selector based on checkbox state, allowing administrators to choose between entering a new address or selecting from existing ones for both shipping and billing addresses.
