# itemCollectionTheme.html.twig

**Path**: `templates\bundles\SyliusAdminBundle\Order\itemCollectionTheme.html.twig`

## Summary
This Twig template file extends the Sylius Admin form theme to customize the rendering of form fields specifically for order item collections. It overrides three form blocks (`choice_row`, `form_row`, and `sylius_resource_autocomplete_choice_row`) to apply Semantic UI styling classes (like "ui dropdown") and handle required/error states, validation messages, and autocomplete functionality for managing order items in the Sylius e-commerce admin panel.
