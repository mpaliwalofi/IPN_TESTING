# preview.html.twig

**Path**: `templates\bundles\SyliusAdminBundle\Order\preview.html.twig`

## Summary
This Twig template renders an order preview page in the Sylius admin panel, specifically for the order creation workflow. It extends the base admin layout and displays a summary of order items using a partial template, allowing administrators to review order details before finalizing the order creation. The template uses Sylius UI components (semantic UI grid and segment) to present the order information in a structured format.
