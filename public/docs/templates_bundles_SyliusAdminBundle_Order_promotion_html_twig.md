# promotion.html.twig

**Path**: `templates\bundles\SyliusAdminBundle\Order\promotion.html.twig`

## Summary
This Twig template provides the admin interface for adding promotions to orders in a Sylius e-commerce system. It renders a form that allows administrators to select and apply a promotion to an order, with the main functionality being promotion selection and submission. The template includes JavaScript that automatically triggers change events on newly added promotion action type selectors when managing complex promotion configurations.
