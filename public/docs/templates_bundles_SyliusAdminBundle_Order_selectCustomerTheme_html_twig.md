# selectCustomerTheme.html.twig

**Path**: `templates\bundles\SyliusAdminBundle\Order\selectCustomerTheme.html.twig`

## Summary
This Twig template extends the Sylius Admin form theme to customize the customer selection widget for order creation. It overrides the customer autocomplete choice field block to provide AJAX endpoints for searching customers and loading customer data by code, enabling administrators to select or search for customers when manually creating orders in the admin panel.
