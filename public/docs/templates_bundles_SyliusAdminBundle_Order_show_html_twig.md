# show.html.twig

**Path**: `templates\bundles\SyliusAdminBundle\Order\show.html.twig`

## Summary
This Twig template renders the order detail view page in the Sylius admin panel. It extends the base admin layout, sets the page title to display the order number, and delegates the main content rendering to a template event system (`sylius.admin.order.show.content`), allowing for flexible customization of order details display through Sylius's event-based templating architecture.
