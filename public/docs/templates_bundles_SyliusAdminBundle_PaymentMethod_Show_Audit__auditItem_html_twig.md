# _auditItem.html.twig

**Path**: `templates\bundles\SyliusAdminBundle\PaymentMethod\Show\Audit\_auditItem.html.twig`

## Summary
This Twig template renders individual audit log entries for payment method changes in the Sylius admin interface. It displays a table row for each audit item showing the timestamp, username, IP address, and a summary of the changes made to the payment method entity, leveraging the DHAuditor bundle's helper functions to format the audit information.
