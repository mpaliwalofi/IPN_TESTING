# show.html.twig

**Path**: `templates\bundles\SyliusAdminBundle\PaymentMethod\Show\Audit\show.html.twig`

## Summary
This Twig template renders an audit log interface for payment method administration in a Sylius e-commerce platform. It displays a scrollable table showing audit trail entries with columns for date, user, IP address, and comments, allowing administrators to track changes and actions performed on payment methods for compliance and monitoring purposes.
