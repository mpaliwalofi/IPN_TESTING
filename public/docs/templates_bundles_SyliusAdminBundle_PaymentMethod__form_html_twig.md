# _form.html.twig

**Path**: `templates\bundles\SyliusAdminBundle\PaymentMethod\_form.html.twig`

## Summary
This Twig template renders the payment method configuration form in the Sylius admin panel. It displays fields for configuring payment method details (code, position, enabled channels, tax category) and availability settings (admin orders, shop orders, subscriptions), along with a section for managing payment method charges with associated error validation display.
