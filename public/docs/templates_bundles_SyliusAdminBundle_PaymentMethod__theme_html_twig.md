# _theme.html.twig

**Path**: `templates\bundles\SyliusAdminBundle\PaymentMethod\_theme.html.twig`

## Summary
This Twig template file customizes the form theme for payment method configuration in the Sylius e-commerce admin panel. It defines custom form widgets for displaying payment method charges with range-based pricing (rangeMin/rangeMax fields) and their associated configuration, as well as a money input field widget and a collection widget for managing multiple payment method charges. The template enables administrators to set up flexible payment pricing structures based on order value ranges.
