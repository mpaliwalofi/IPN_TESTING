# _selectShippingMethod.html.twig

**Path**: `templates\bundles\SyliusAdminBundle\PickupPoint\_selectShippingMethod.html.twig`

## Summary
This Twig template provides a dual-mode shipping method selection interface for Sylius e-commerce orders, allowing users to choose between home delivery and pickup point delivery. It renders form fields for shipping addresses and pickup point options, with a tabbed interface that toggles between displaying standard shipping address forms (with address selection and display) and pickup point selection functionality. The template supports address management features including selecting existing addresses, displaying selected addresses, and adding new shipping addresses.
