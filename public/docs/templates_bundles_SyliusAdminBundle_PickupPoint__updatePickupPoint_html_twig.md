# _updatePickupPoint.html.twig

**Path**: `templates\bundles\SyliusAdminBundle\PickupPoint\_updatePickupPoint.html.twig`

## Summary
This Twig template provides a search and filter interface for managing pickup points in the Sylius e-commerce admin bundle. It includes a search input field for querying pickup points by address, city, or postal code, along with dropdown filters for controlling the number of results displayed (10-50 results). The template uses Semantic UI components and integrates with JavaScript functions to dynamically update pickup point listings without page reload.
