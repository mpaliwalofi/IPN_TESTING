# index.html.twig

**Path**: `templates\bundles\SyliusAdminBundle\Product\index.html.twig`

## Summary
This Twig template extends the base Sylius admin product index template to add custom JavaScript integration points through Sylius's event system. It specifically enables developers to inject custom JavaScript code into the product listing page by firing template events (e.g., `sylius.admin.product.index.javascripts`) that can be hooked into from other parts of the application or bundles.
