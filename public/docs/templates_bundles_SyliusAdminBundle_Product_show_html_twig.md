# show.html.twig

**Path**: `templates\bundles\SyliusAdminBundle\Product\show.html.twig`

## Summary
This Twig template file renders the product detail/show page in the Sylius e-commerce admin interface. It extends the base admin layout and displays comprehensive information about a configurable product by including a dedicated partial template that handles the product's detailed view, with the page title showing "Show Product" followed by the product name.
