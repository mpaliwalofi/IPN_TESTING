# _available_amount.html.twig

**Path**: `templates\bundles\SyliusAdminBundle\PromotionCoupon\Grid\Field\_available_amount.html.twig`

## Summary
This Twig template renders the available amount for a promotion coupon in the Sylius admin grid. It displays a formatted monetary value using the store's currency, or shows a dash ("-") if the coupon is set to use the default amount instead of a custom value.
