# _form.html.twig

**Path**: `templates\bundles\SyliusAdminBundle\Promotion\_form.html.twig`

## Summary
This Twig template renders the admin form for creating/editing promotions in a Syliushe-commerce platform. It provides a structured layout using Semantic UI grid system to organize promotion fields including basic details (name, description), configuration options (usage limits, coupon-based settings, priority, channels), validity dates, and the core promotion logic (rules and actions). The template also conditionally includes a coupon section for coupon-based promotions, enabling administrators to define comprehensive promotional campaigns with specific targeting criteria and discount behaviors.
