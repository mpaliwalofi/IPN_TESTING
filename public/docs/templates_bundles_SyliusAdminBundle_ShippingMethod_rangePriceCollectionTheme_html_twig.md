# rangePriceCollectionTheme.html.twig

**Path**: `templates\bundles\SyliusAdminBundle\ShippingMethod\rangePriceCollectionTheme.html.twig`

## Summary
This Twig template file customizes the form theme for shipping method range-based pricing configuration in the Sylius admin interface. It defines form widgets for entering shipping price ranges (min/max values) with validation error displays, and handles per-zone pricing entries with enable/disable toggles. The template supports multi-zone shipping configurations where prices can vary based on weight/value ranges and geographical zones.
