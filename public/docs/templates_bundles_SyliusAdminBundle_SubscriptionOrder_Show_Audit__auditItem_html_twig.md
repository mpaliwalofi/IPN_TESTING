# _auditItem.html.twig

**Path**: `templates\bundles\SyliusAdminBundle\SubscriptionOrder\Show\Audit\_auditItem.html.twig`

## Summary
This Twig template renders individual audit log entries for subscription orders in the Sylius admin interface. It displays a table row for each audit item showing the timestamp, username, IP address, and a summary of changes made to the audited entity, utilizing the DHAuditor bundle's helper functions to format the audit trail information.
