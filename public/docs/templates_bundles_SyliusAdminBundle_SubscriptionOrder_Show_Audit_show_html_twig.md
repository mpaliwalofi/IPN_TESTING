# show.html.twig

**Path**: `templates\bundles\SyliusAdminBundle\SubscriptionOrder\Show\Audit\show.html.twig`

## Summary
This Twig template displays an audit log table for subscription orders in the Sylius admin panel. It renders a scrollable table with columns for date, user, IP address, and comments, iterating through audit records to show the history of changes or actions performed on a subscription order. The template provides administrative visibility into subscription order modifications for tracking and compliance purposes.
