# _auditItem.html.twig

**Path**: `templates\bundles\SyliusAdminBundle\Subscription\Show\Audit\_auditItem.html.twig`

## Summary
This Twig template renders individual audit log entries for subscription-related entities in the Sylius admin bundle. It displays a table row for each audit item showing the timestamp, username, IP address, and a summarized description of the changes made to the audited entity (such as create, update, or delete actions).
