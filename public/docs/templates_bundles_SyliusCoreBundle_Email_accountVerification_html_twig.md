# accountVerification.html.twig

**Path**: `templates\bundles\SyliusCoreBundle\Email\accountVerification.html.twig`

## Summary
This is a Twig email template for Sylius e-commerce platform that generates an account verification email sent to new users. The template displays a verification token that users need to confirm their email address during the registration process. It extends the base email layout and outputs the token along with localized instructional text in a centered format.
