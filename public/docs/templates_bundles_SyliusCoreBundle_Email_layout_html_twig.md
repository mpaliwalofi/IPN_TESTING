# layout.html.twig

**Path**: `templates\bundles\SyliusCoreBundle\Email\layout.html.twig`

## Summary
This Twig template serves as the base email layout for Sylius e-commerce platform emails. It provides a standardized HTML structure with inline CSS styling that includes the store's logo (derived from the channel hostname), a content block for email-specific content, and a footer link to sylius.com. The template ensures consistent branding and formatting across all transactional emails sent by the Sylius store.
