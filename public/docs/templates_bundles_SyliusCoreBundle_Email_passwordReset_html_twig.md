# passwordReset.html.twig

**Path**: `templates\bundles\SyliusCoreBundle\Email\passwordReset.html.twig`

## Summary
This Twig email template generates a password reset email for Sylius e-commerce platform users. It displays a personalized greeting with the user's username and provides their password reset token, with conditional messaging based on whether the SyliusShopBundle is loaded to determine if users reset via web interface or manual token entry.
