# userRegistration.html.twig

**Path**: `templates\bundles\SyliusCoreBundle\Email\userRegistration.html.twig`

## Summary
This is a Twig email template for Sylius e-commerce platform that generates user registration confirmation emails. It displays a welcome message with the registered username and includes a call-to-action button linking to the store's homepage to encourage the new user to start shopping. The template uses Sylius translation system with locale-specific content and extends a base email layout for consistent styling.
