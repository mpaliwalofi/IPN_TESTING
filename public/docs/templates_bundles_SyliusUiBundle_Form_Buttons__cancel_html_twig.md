# _cancel.html.twig

**Path**: `templates\bundles\SyliusUiBundle\Form\Buttons\_cancel.html.twig`

## Summary
This Twig template renders a "Cancel" button that navigates users back to the previous page in the Sylius e-commerce platform's UI. It creates an anchor link that defaults to the HTTP referer if no explicit path is provided, applies optional CSS classes (including Semantic UI styling and a JavaScript one-click protection class), and displays a translatable "Cancel" label. The template is designed as a reusable component for forms throughout the Sylius admin interface.
