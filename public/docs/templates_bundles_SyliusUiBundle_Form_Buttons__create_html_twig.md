# _create.html.twig

**Path**: `templates\bundles\SyliusUiBundle\Form\Buttons\_create.html.twig`

## Summary
This Twig template renders a form submission button group for creating/adding entities in the Sylius e-commerce platform. It generates a primary action button with customizable CSS classes and translation key (defaulting to 'create' mode), along with an accompanying cancel button, providing a consistent UI pattern for form submission across the application.
