# _update.html.twig

**Path**: `templates\bundles\SyliusUiBundle\Form\Buttons\_update.html.twig`

## Summary
This Twig template renders a form update button section for the Sylius e-commerce platform's UI bundle. It displays a primary "Save Changes" submit button with customizable CSS classes and an accompanying cancel button, using Semantic UI styling with a hidden divider for spacing. The template provides flexibility for parent templates to pass custom CSS classes through the `classes.update` parameter while maintaining consistent update/cancel action patterns across Sylius admin forms.
