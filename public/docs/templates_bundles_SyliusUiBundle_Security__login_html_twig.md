# _login.html.twig

**Path**: `templates\bundles\SyliusUiBundle\Security\_login.html.twig`

## Summary
This Twig template renders the Sylius admin login form with a centered layout using Semantic UI styling. It displays login credentials fields (username, password, remember me), handles error messages from failed login attempts, includes CSRF protection for the admin authentication process, and optionally shows a logo and OAuth provider login buttons. The template serves as the main entry point for administrator authentication into the Sylius e-commerce platform's backend.
