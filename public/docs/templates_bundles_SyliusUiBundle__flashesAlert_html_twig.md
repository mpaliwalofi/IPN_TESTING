# _flashesAlert.html.twig

**Path**: `templates\bundles\SyliusUiBundle\_flashesAlert.html.twig`

## Summary
This Twig template provides a client-side flash message notification system for the Sylius e-commerce platform. It renders a container div and JavaScript module that dynamically generates and displays styled alert messages (success, error, warning, info) with configurable auto-dismiss timers, custom icons, and titles translated via Symfony's translation system. The messages automatically disappear after a default 5 seconds unless marked as permanent, with timer pause functionality on hover for better user experience.
