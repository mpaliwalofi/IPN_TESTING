# _stylesheets.html.twig

**Path**: `templates\bundles\SyliusUiBundle\_stylesheets.html.twig`

## Summary
This Twig template file serves as a stylesheet inclusion point for the Sylius admin interface. It loads the admin-entry CSS bundle via Webpack Encore and defines inline CSS styles for UI components including definition lists, payment items display, and a fixed-position flash message/alert container with responsive positioning and shadowing effects.
