# checkoutcom.en.yaml

**Path**: `translations\checkoutcom.en.yaml`

## Summary
This YAML file contains English translations for Checkout.com payment gateway error codes and messages. It provides human-readable error descriptions for various payment processing scenarios including 3DS authentication failures, card validation issues, authorization problems, and API quota limitations. The file serves as an internationalization resource to display consistent, user-friendly error messages when Checkout.com payment transactions fail or encounter issues.
