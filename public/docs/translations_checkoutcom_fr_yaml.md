# checkoutcom.fr.yaml

**Path**: `translations\checkoutcom.fr.yaml`

## Summary
This file contains French translations for Checkout.com payment gateway error messages. It provides localized error codes and descriptions for various payment processing scenarios including 3DS authentication failures, card validation issues, transaction amount problems, and API quota limits. The translations enable French-speaking users to understand payment errors in their native language when using the Checkout.com payment integration.
