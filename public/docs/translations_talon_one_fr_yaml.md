# talon_one.fr.yaml

**Path**: `translations\talon_one.fr.yaml`

## Summary
This is a French language translation file for the Talon.One promotion and coupon management system integration. It provides localized error messages and status notifications for various coupon validation scenarios, including expired coupons, usage limits, eligibility requirements, and system availability issues. The translations enable French-speaking users to understand why promotional codes cannot be applied or why the integration service is unavailable.
