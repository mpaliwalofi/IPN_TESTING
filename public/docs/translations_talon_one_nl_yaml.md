# talon_one.nl.yaml

**Path**: `translations\talon_one.nl.yaml`

## Summary
This is a Dutch language translation file for Talon.One promotional code and coupon management integration. It provides localized error messages and status notifications for various coupon validation scenarios, including expiration, usage limits, campaign availability, user eligibility, and system availability issues. The translations enable user-facing feedback when promotional codes are applied or rejected in an e-commerce or promotional system.
