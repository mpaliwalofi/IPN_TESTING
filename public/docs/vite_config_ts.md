# vite.config.ts

**Path**: `vite.config.ts`

## Summary
This is a Vite build configuration file for a Vue 3 library or component package. It configures the build to externalize Vue ecosystem dependencies (vue, vue-router, vue-i18n) so they won't be bundled, enables TypeScript declaration file generation via the dts plugin, and sets up auto-importing of Vue composition API functions and router/i18n utilities to eliminate manual import statements during development.
